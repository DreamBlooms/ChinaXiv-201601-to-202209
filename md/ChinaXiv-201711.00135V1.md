ABSTRACT A new metal-organic cadmium coordination polymer $\mathrm { \Delta [ C d ( p z d c ) _ { 0 . 5 } ( L ) ] _ { n } } { \cdot } \mathrm { n H _ { 2 } O }$ (1, H2pzdc $\mathbf { \tau } = \mathbf { \tau }$ pyrazine-2,3-dicarboxylic acid, $\mathrm { { H L } } \ = \ 3$ (2-pyridyl)pyrazole） has been successfully synthesized under hydrothermal conditions. The structure has been determined by single-crystal $\mathrm { \Delta } X$ -ray diffraction analysis, elemental analyses,IR and fluorescence spectrum. Brown crystals crystalize in the orthorhombic system, space group Fdd2 with $a = 1 1 . 2 6 3 6 ( 9 )$ ， $b = 3 8 . 2 9 6 ( 3 )$ ， $c = 1 0 . 5 7 9 5 ( 8 ) .$ A, $V = 4 5 6 3 . 5 ( 6 ) \mathrm { ~ \AA ~ }$ $\mathrm { C _ { 1 1 } H _ { 9 } C d N _ { 4 } O _ { 3 } } , \Lambda$ 公 $= 3 5 7 . 6 2 \$ $D _ { c } = 2 . 0 8 2 \ \mathrm { g / c m ^ { 3 } }$ $\mu ( \mathrm { M o } K \alpha ) = 1 . 9 2 4 ~ \mathrm { m m } ^ { - 1 }$ ， $F ( 0 0 0 ) = 2 8 0 0$ $Z = 1 6$ , the final $R = 0 . 0 1 6 9$ and $w R =$ 0.0443 for 2106 observed reflections $( I > 2 \sigma ( I ) )$ . It shows a two-dimensional network structure and extends into a three-dimensional supramolecular framework through $\pi { - } \pi$ stacking interactions. Moreover, we analyzed Natural Bond Orbital (NBO） by using the PBEO/LANL2DZ method built in Gaussian O3 Program. The calculation results showed obvious covalent interaction between the coordinated atoms and $\operatorname { C d } ( \operatorname { I I } )$ ion.

# 1 INTRODUCTION

The designed synthesis and characterization of metal-organic coordination polymers have gained important progress in supramolecular chemistry and material chemistryl1-8l. The increasing interest in this field is justified not only for their particular beauty and intriguing structural diversities of architecture,but also for their potential applications as catalytic,conductive,luminescent,magnetic,spin-transition,non-linear optical or porous functional materials[9-16].The combination of metal ions with neutral and anionic mixed bridging ligands makes the assembly process more controllable than a single ligand.The prospect of introducing the second or more organic ligands into areaction system provides further impetus for researches on metal-organic supramolecular frameworks.The construction of supramolecular architectures through selectiveand directional non-covalent forces such as hydrogen bonding, $\pi \cdots \pi$ and $\mathrm { C - H } \cdots \pi$ interactions in metal-organic frameworks arouses considerable contemporary interest owing to their potential applications as functional materials.

The hydrothermal technique is well suitable to the preparation of crystals of synthetic minerals,new inorganic materials,and organometalic coordination polymers. Of particular interest to us is the construction of transition metal polymers with new structural features by utilizing hydrothermal synthesis.We report here the preparation and crystal structure of the title complex $\mathrm { \Delta [ C d ( p z d c ) _ { 0 . 5 } ( L ) ] _ { n } { \cdot } n H _ { 2 } O }$ (1),which exhibits a 3D supramolecular network through hydrogen bonds and $\pi { - } \pi$ stacking interactions.

# 3 RESULTSANDDISCUSSION

# 3.1 Structural description

Single-crystal X-ray diffraction analysis displays that compound 1 crystallzes in Fdd2 space group and consists of a two-dimensional network structure. There are one $\operatorname { C d } ( \mathbb { I } )$ ion, one $\mathrm { ~ L ~ }$ ligand,half H2pzdc ligand and one crystal water molecule in the asymmetric unit (Fig. 1). The $\operatorname { C d } ( 1 )$ ion is coordinated by three nitrogen atoms from two diferent L ligands, one nitrogen atom from pzdc ligand in the equatorial plane (Cd(1)-N(1A) $= 2 . 2 5 2 ( 3 )$ ${ \mathfrak { j } } 2 ( 3 ) , { \mathrm { C d } } ( 1 ) - { \mathrm { N } } ( 2 ) = 2 . 2 6 8 ( 3 ) , { \mathrm { C d } } ( 1 ) - { \mathrm { N } } ( 3 ) = 2 . 3 7 1 ( 2 ) , { \mathrm { C d } } ( 1 ) - { \mathrm { N } } ( 4 ) = 2 . 4 1 6 ( 3 ) { \mathrm { ~ \AA ~ } } \delta = 0 . 0 9 8 ( 3 ) { \mathrm { ~ \AA ~ } }$ and one oxygen atom from the pzdc ligand, one oxygen atom from another pzdc ligand at the axial site $( \mathbf { C d } ( 1 ) - \mathbf { O } ( 1 ) = 2 . 5 1 4 ( 2 )$ ， $\mathbf { C } \mathbf { d } ( 1 ) \mathbf { - } \mathbf { O } ( 2 \mathbf { B } ) = 2 . 2 7 3 ( 2 ) \textup { \AA }$ . The coordination angles around $\operatorname { C d } ( \mathrm { I I } )$ ion are in the range $6 7 . 5 9 ( 9 ) \widetilde { \sim } 1 6 7 . 3 3 ( 9 ) ^ { \circ }$

In the crystal structure of complex 1，each molecule of $\mathrm { ~ L ~ }$ binds two cadmium(II) ions through a bidentate-monodentate mode,utilizing two nitrogen atoms to connect one cadmium $( \mathrm { I I } )$ ion,utilizing one nitrogen atom to connect another cadmium $( \mathrm { I I } )$ ion,whereas the H2pzdc ligands adopt a $\mu _ { 6 }$ coordination mode, with two carboxylate groups utilizing four oxygen atoms to connect four different cadmium $( \mathrm { I I } )$ ions, with two nitrogen atoms bridging two different cadmium(II) ions. The coordination mode of the $\mathrm { H } _ { 2 }$ pzdc molecule in1 is different from that in our previously reported compound $\mathrm { \small { [ Z n ( p z d c ) ( m b i x ) ] _ { n } } . n H _ { 2 } O ^ { [ 1 8 ] } }$ . As a result, four $\operatorname { C d } ( \operatorname { I I } )$ ions are linked by two pzdc and two $\mathrm { ~ L ~ }$ ligands,and each pair of the tetranuclear subunits are bridged by pzdc ligands to yield a two-dimensional network structure,as depicted in Fig. 2. The $\operatorname { C d } ( 1 )$ ion exhibits a distorted octahedral coordination architecture.

Furtheranalysisofthecrystalpackingrevealed $\pi { - } \pi$ interactionsincomplex1 between N(3)C(4)C(5)C(6)C(7)C(8) pyridine ring of the $\mathrm { ~ L ~ }$ ligand. The centroid-to-centroid distances are 3.931(3), 3.929(3) $\mathring \mathbf { A }$ and perpendicular distances are 3.411(2), 3.484(2) $\mathring \mathbf { A }$ ，and the dihedral angle is $6 ^ { \circ }$ .Therefore, through $\pi { - } \pi$ interactions，the two-dimensional networks are further extended into a three-dimensional supramolecular framework (Fig. 3).

To confirm the phase purity of complex 1,powder X-ray diffraction (PXRD) patterns were recorded for 1, and it was comparable to the corresponding simulated patterns calculated from the single-crystal diffraction data (Fig. 4), indicating a pure phase of bulky sample.

# 3.4Photoluminescent properties

Luminescence property is very significant in photochemistry and photophysics[20, 21]. So, in this study, we measured the solid-state photoluminescence spectra of 1 (Fig. 5） at room temperature. Excited by $4 4 0 ~ \mathrm { n m }$ complex 1 exhibits blue emission with the maximum peak at $4 9 5 ~ \mathrm { n m }$ . In order to study the nature of these emission bands,we first analyzed the photoluminescence properties of free L, H2pzdc ligands,and confirmed that they do not emit any luminescence in the range of $4 0 0 { \sim } 8 0 0 ~ \mathrm { n m }$ . Therefore,on the basis of the previous literature[22, the emission band could be vested to the emission of ligand-to-metal charge transfer (LMCT). Owing to its strong fluorescent intensity,1 sems to be a good candidate for novel hybrid inorganic-organic photoactive materials.

functional and the LANL2DZ basis set[29].

The major natural atomic charges,natural electron configuration,Wiberg bond indices and NBO bond orders (a.u) for complex 1 are shown in Table 2. It is shown that the electronic configurations of $\operatorname { C d } ( 1 )$ ion, N and O atoms are $5 s ^ { 0 . 2 9 } 4 d ^ { 9 . 9 8 } 5 p ^ { 0 . 3 0 }$ ， $2 s ^ { 1 . 3 4 \sim 1 . 4 1 } 2 p ^ { 3 . 9 5 \sim _ { 4 . 1 6 } }$ and $2 s ^ { 1 . 6 8 ^ { - } 1 . 6 9 } 2 p ^ { 5 . 0 1 ^ { - } 5 . 0 9 }$ , respectively. On account of the above effects,one can infer that the $\operatorname { C d } ( 1 )$ ion coordination with nitrogen and oxygen atoms is mainly on $4 d$ 5s, and $5 p$ orbitals. Nitrogen atoms form coordination bonds with $\operatorname { C d } ( 1 )$ ion utilizing $2 s$ and $2 p$ orbitals. The oxygen atoms provide electrons of $2 s$ and $2 p$ to the $\operatorname { C d } ( 1 )$ ion and form coordination bonds.Thus, the $\operatorname { C d } ( 1 )$ （20 ion gained some electrons from four nitrogen atoms of $\mathrm { ~ L ~ }$ ligand and two oxygen atoms of pzdc ligands[30,31,32]. Therefore,on the basis of valence-bond theory,the atomic net charge distribution and NBO bond orders of complex 1(Table 4) exhibits obvious covalent interaction between the coordinated atoms and $\operatorname { C d } ( 1 )$ ion. The differences of NBO bond orders for Cd-Oand Cd-N make their bond lengths be differentl31],which is in good agreement with the X-ray crystal structural data of compound 1.

As can be seen from Fig.6,the lowest unoccupied molecular orbital (LUMO) mainly consists of L ligand, whereas the highest occupied molecular orbital (HOMO) is mainly composed of pzdc ligand. Thus,the charge transfer from ligand to ligand may be deduced from some contours of molecular orbital of complex 1.

ZHANPei-Ying(战佩英) WANG Shao-Jie(王少杰) LIXiu-Mei(李秀梅) PAN Ya-Ru(潘亚茹)

A new metal-organic cadmium coordination polymer $\mathrm { \Delta [ C d ( p z d c ) _ { 0 . 5 } ( L ) ] _ { n } } { \cdot } \mathrm { n H _ { 2 } O }$ has been successfully synthesized under hydrothermal conditions.The structure has been determined by single-crystal $\mathrm { \Delta } X$ -ray diffraction analysis,elemental analyses, IR and fluorescence spectrum.
