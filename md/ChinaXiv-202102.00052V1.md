摘要：利用大量的地热流体化学成分及环境同位素 $\left( \phantom { - } 8 \mathrm { D } , 8 ^ { 1 8 } \mathrm { O } , ^ { 1 4 } \mathrm { C } \right)$ 测试数据，深入分析了隆起山地构造对流型、沉降盆地传导型地热流体的化学特征及分布规律,进而对地热流体的环境同位素分布特征及形成年龄进行了梳理总结，得出了明确的结论。结果表明：隆起山地构造对流型地热田地热流体主要以断裂上升泉的形式出露,分布于西秦岭-祁连造山带，补给来源为当地及周边大气降水入渗，地热流体形成年龄一般小于 $5 0 0 0 \sim 3 0 0 0 0 \mathrm { a }$ ，水质较好，属“开启型”的地热系统；热储层岩性、断裂规模及水热循环方式和深度等明显控制着地热流体化学类型、环境同位素特征和形成年龄。沉降盆地传导型地热田地热流体主要以管井开采的方式出露；热储埋藏深度小于 $1 6 0 0 \mathrm { m }$ 的地热井，地热流体补给来源为当地及周边大气降水入渗，形成年龄一般小于 $5 0 0 0 \sim 3 0 0 0 0 \mathrm { a }$ ，水质相对较好，属“半开启-半封闭型”的地热系统；热储埋藏深度介于 $1 6 0 0 \sim 2 6 0 0 \mathrm { m }$ 之间的地热开采井,主要为地质历史时期逐步形成的“古水”,水化学类型复杂且水质较差，地热流体形成年龄介于$3 0 0 0 0 \sim 5 0 0 0 0 \mathrm { a } ,$ 属"封闭型”的地热系统;热储层岩性、埋藏深度、地下水在岩层中的滞留时间与循环深度等明显控制着地热流体化学类型、环境同位素富集程度和形成年龄。

关键词：热流体；水化学；环境同位素；形成年龄；地热系统

# 文章编号：

地热系指地球内部所储存的热量，一般构成相对独立的热能储存、运移、转换的系统。按地质成因和能量传递方式可划分为隆起山地构造对流型和沉降盆地传导型二个类型的地热田[1]。甘肃省各时代地层均有分布、地层序列较为完整，新构造运动强烈，岩浆侵入和喷发作用多期次交替发生，火山岩与沉积岩及变质岩相伴出露，各构造单元间或其内褶皱与断裂发育、构造活动复杂，加之山地相对丰富的降水和地质历史时期形成的“古封存水”及沟通山盆之间的导水构造，水热型地热资源禀赋凸显，从而形成了丰富的地热资源。根据60余年地质勘探证实，甘肃境内地热类型齐全，而且具有分布广、埋藏较浅和便于开发的特点，是一个地热资源丰富并具有良好开发利用前景的省份，也是我国的新能源和可再生能源的大省。经初步评价2，省内可供开发利用的水热型地热能资源量达 $1 . 4 1 \times 1 0 ^ { 1 0 }$ +标准煤,全部开发后可实现供暖面积达 $1 . 5 8 \times 1 0 ^ { 1 1 } \mathrm { m } ^ { 2 }$ ，有着巨大的开发利用前景。自2000年以来，有关地学科研人员对甘肃省或某一区域地热资源形成的地质条件、分布特征、成因类型、储存富集规律及开发利用方式和方向等诸多领域进行了富有成效的探索研究，李百祥等[3]、安永康等[4从甘肃省盆山构造格局的角度，定性探讨了对地温场及地热资源类型的控制，概略阐述了地热资源分布特征、开发现状与前景；郑希民等[5]、白福等[6]、张子祥等[7]对兰州-民和断陷盆地地热地质条件进行了深入分析，并提出了热储概念模型;丁宏伟等[8]、王彩霞等[9]、高振荣等"从不同水体环境同位素含量特征，详细分析论证了河西走廊及祁连山区深层地下水的补给来源和深层地温突变的原因。这些研究成果对加快甘肃省地热资源的勘查及开发利用无疑产生了巨大的推动作用。但从水化学及环境同位素特征角度探讨甘肃省地热流体的形成原因、补给来源及形成年龄等方面的研究成果鲜有公开的报导，本文尚属首次。

# 区域地热地质背景条件

甘肃省地处我国五大地质板块交汇地带，具有不同时代的海相火山岩系、多种类型的沉积建造、复杂的沉积型相和种类齐全的变质岩系及强烈的岩浆活动过程,地壳及上地幔主要由变质岩和地壳重熔花岗质岩石组成，发育了不同序列的多个构造单元。多期次挤张收缩、岩层褶皱断裂、陆内地震活跃及沉积作用，控制着形式多样的山盆构造格局和地层分布，区域大地热流值及地温场水平和垂向规律明显，从而也影响着地热资源的形成与分布。

# 1.1区域大地热流值特征

1.1.1 隆起山地造山带通过对全省不同地质构造单元热流值的统计[2-4],大地热流与构造活动性及地质年代之间存在着良好的一致性，构造活动区的基底一般隐伏着来自地壳深部或上地幔的高热流，高热流的存在又增加了地壳浅部构造活动区的活动强度。甘肃省构造作用活跃的祁连山-阿尔金山区大地热流值一般大于 $7 0 \mathrm { m W } \cdot \mathrm { m } ^ { - 2 }$ ，西秦岭、北山地区$6 0 \sim 7 0 ~ \mathrm { m W } \cdot \mathrm { m } ^ { - 2 }$ ,六盘山构造带大地热流值介于$5 0 \sim 6 0 \mathrm { m W } \cdot \mathrm { m } ^ { - 2 }$ 之间(图1)。

# 干旱区地理

以元古-太古界为基底的古老地块，是NEE向左行阿尔金走滑断裂带末段效应产生的NNE向拉分盆地，而裂谷拉张阶段一般具有很高的热流值；兰州-榆中盆地处于NWW向西秦岭造山带北缘断裂和北祁连造山带南缘断裂间，属NNW向的反S型走滑压扭盆地，位于盆地边缘的红古区窑街煤矿高地温场特征明显,5个测温点平均大地热流值为 $8 1 . 4 ~ \mathrm { m W } \cdot \mathrm { m } ^ { - 2 }$ ，属甘肃省内最高，这主要是煤层自燃形成烧变岩引起地下温度升高所导致的；另外，窑街矿区的游离$\mathrm { C O } _ { 2 }$ 含量也较高，这是地下水在深循环过程中引起碳酸盐岩类岩石脱碳作用产生的 $\mathrm { C O } _ { 2 }$ ，在高温条件下“气-水相溶”并存于热水之中的原因所致，与矿区同处断裂带的永登药水沟、龙王沟温泉亦具有游离 $\mathrm { C O } _ { 2 }$ 含量较高的特征。

# 1.2区域地温场特征

甘肃省地温梯度一般介于 $2 . 0 \sim 3 . 5 \mathrm { ^ c \cdot ( 1 0 0 m ) ^ { - 1 } }$ 之间，最高达 $4 . 0 ^ { \circ } \mathrm { C } \cdot ( 1 0 0 \mathrm { m } ) ^ { - 1 }$ ,存在自西向东、由北向南地温梯度逐渐升高的变化规律[2-4]。其中北山地区地温梯度最低，一般小于 $2 . 0 \operatorname { \mathrm { ~ ‰ ~ } }$ ；祁连山-阿尔金山及西秦岭地温梯度介于 $2 . 5 \sim 3 . 5$ $\mathrm { { } ^ { q } C \cdot ( 1 0 0 \ m ) ^ { - 1 } }$ 之间，西秦岭北麓地温梯度平均3.0$\mathrm { { } ^ { 9 } C \cdot ( 1 0 0 m ) ^ { - 1 } }$ 以上，局部达 $4 . 0 ^ { \circ } \mathrm { C } \cdot ( 1 0 0 \mathrm { m } ) ^ { - 1 }$ 。河西走廊盆地群地温梯度一般介于 $2 . 0 \sim 3 . 0 \mathrm { ‰ }$ 之间，中西部的敦煌-瓜州盆地、酒泉东盆地、张掖-民乐盆地局部可达 $3 . 5 ~ \mathrm { { ^ { \circ } C } } \cdot ( 1 0 0 ~ \mathrm { { m } } ) ^ { - 1 }$ ;陇西黄土高原地温梯度一般介于 $2 . 0 \sim 2 . 5 ~ \mathrm { ^ c \cdot ( 1 0 0 ~ m ) ^ { - 1 } }$ 之间,其中兰州-榆中盆地、靖远-会宁盆地较高，兰州-榆中盆地局地和边缘可达 $4 . 0 \ \mathrm { { ^ { \circ } C } \cdot ( 1 0 0 \ m ) ^ { - 1 } }$ ，,靖远-会宁盆地局地达 $3 . 0 ^ { \circ } \mathrm { C } { \cdot } ( 1 0 0 \mathrm { m } ) ^ { - 1 }$ ;陇东黄土高原地温梯度一般介于 $2 . 0 \sim 3 . 5 ^ { \circ } \mathrm { C } \cdot ( 1 0 0 \mathrm { m } ) ^ { - 1 }$ 之间，存在自西向东逐渐增大的变化规律，磴口-平凉断裂以西的西缘逆冲褶皱带小于 $2 . 0 \mathrm { { ^ { \circ } C } \cdot ( 1 0 0 \mathrm { { m } ) ^ { - 1 } } }$ ，以东的白垩系盆地为 $2 . 5 \sim$ $3 . 5 ^ { \circ } \mathrm { C } \cdot ( 1 0 0 \mathrm { m } ) ^ { - 1 }$ ,子午岭以东大于 $3 . 5 ^ { \circ } \mathrm { C } { \cdot } ( 1 0 0 \mathrm { m } ) ^ { - 1 }$ 。

由此不难看出，甘肃省区域地温场与大地热流值具有基本相同的变化规律，即大地热流值高的区域地温梯度亦高，大地热流值低的区域地温梯度亦较低。区域大地热流值、地温梯度隆起山地造山带一般高于沉降盆地区 $5 \sim 1 0 \mathrm { \ m W } \cdot \mathrm { m } ^ { - 2 }$ 和 $0 . 5 \sim 1 . 0$ $\mathrm { { } ^ { 9 } C \cdot ( 1 0 0 m ) ^ { - 1 } }$ 。

# 1.3地热异常点及分布特征

据最新资料统计[2],截止2018年12月，全省共有地热异常点78处（含水温 $\geqslant 1 5 \ \mathrm { { ^ \circ C } }$ 的温泉、地热井）。其中温泉24处，占地热点总数的 $3 0 . 7 \%$ ，均分布于祁连山-阿尔金山、北山及西秦岭，水温均小于$6 0 ^ { \circ } \mathrm { C }$ ,其中 $4 0 \sim 6 0  { \mathrm { ~ \circ ~ } }  { \mathrm { C } }$ 的中低温泉水3处， $2 5 \sim 4 0 \mathrm { ^ { \circ } C }$ 的低温泉水11处， $1 5 \sim 2 5 ^ { \circ } \mathrm { C }$ 的热水异常泉水10处；地热井54眼，占地热点总数的 $6 9 . 3 \%$ ,分布于祁连山-阿尔金山、北山、西秦岭及河西走廊盆地群、陇西-陇东黄土高原,井口水温一般介于 $1 7 . 5 \sim 7 7 . 0 \mathrm { ~ \textdegree C }$ 之间。其中大于 $6 0 ~ \mathrm { { ^ { \circ } C } }$ 的中低温热水井7眼， $4 0 \sim 6 0 ~ \mathrm { ^ { \circ } C }$ 的中低温热水井17眼， $2 5 \sim 4 0 ~ \mathrm { ^ { \circ } C }$ 的低温热水井16眼， $1 5 \sim 2 5 ^ { \circ } \mathrm { C }$ 的热水异常井14眼。现状热流体井口水温大于 $7 0 \ \mathrm { { ^ { \circ } C } }$ 的地热井共计有4眼，主要分布在兰州-榆中盆地和张掖-民乐盆地，分别为兰州市西固区深沟桥天庆国际新城 $7 1 \ \mathrm { { ^ { \circ } C } }$ 、兰州市七里河区西部欢乐园 $7 3 \mathrm { { ^ \circ C } }$ 、兰州市安宁区仁寿山庄 $7 2 \%$ 和张掖市民乐县民乐六坝工业生态园 $7 7 \mathrm { { ^ \circ C } }$ 。

进一步研究发现，位于全球大气降水线和西北大气降水线附近的样品，主要分布于西部敦煌-瓜州盆地、酒泉东盆地和东部陇东白垩系盆地，地热井深度介于 $6 8 0 \sim 1 ~ 6 0 0 ~ \mathrm { m }$ 之间，热储埋藏深度相对较浅，盖层(不含第四系松散层)为厚度 $1 5 0 \sim 3 5 0 \mathrm { m }$ 的新近系泥质砂岩和下更新统砂岩、砂砾岩，盖层较薄且岩性致密性相对不足，与上覆中上更新统松散岩类孔隙水（主要接受当地大气降水及其地表水径流人渗补给)及其他水体存在不同程度的水力联系，因此热水来源可以认为是当地大气降水，流体受到一定程度的蒸发影响， $8 ^ { 1 8 } 0 . 8 \mathrm { D }$ 与热水中溶解的氢氧组分交换几率较小， $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 漂移程度较小、较容易富集，总体概化为“半开启-半封闭型"的地热系统；而距离降水线较远的地热流体，主要分布于西部酒泉东盆地、张掖-民乐盆地、中部兰州-榆中盆地及东部陇东白垩系盆地，地热井深度介于 $1 6 0 0 \sim$ $2 6 0 0 \mathrm { ~ m ~ }$ 之间，热储埋藏深度大，盖层为厚度 $8 0 0 \sim$ $1 0 0 0 \mathrm { m }$ 的新近系-古近系大厚度的泥岩和砂质泥岩，盖层巨厚且岩性致密，与上覆新统松散岩类孔隙水及其他水体基本不存在水力联系，热流体主要为地质历史时期逐步形成的“古水”，流体受蒸发的程度弱， $8 ^ { 1 8 } \mathrm { O } , 8 \mathrm { D }$ 与热水中溶解的氢氧组分交换几率大， $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 漂移程度大、容易富集，总体概化为“封闭型"的地热系统。评价结果与陕西关中盆地地下热水赋存特征有一定的相似性[12-13]。

4.3.3影响地热流体形成年龄的因素分析对隆起山地构造对流型地热田而言，地热流体主要以断裂上升泉的形式出露，热储层岩性、断裂规模及水热循环方式和深度等明显控制着地热流体化学类型和形成年龄。地热流体补给来源为当地及周边大气降水入渗，主要沿断裂带循环加热，循环深度浅、交替迅速，地下水在岩层中的滞留时间较短、“水-岩互助"作用弱， $\mathbf { 8 } ^ { 1 8 } \mathbf { 0 } \mathbf { . 8 } \mathbf { D }$ 含量与热水中溶解的氢氧离子交换几率小， $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 漂移程度小、不宜富集，热流体TDS含量相对较低，水质相对较好，属地热流体形成年龄一般小于 $5 0 0 0 \sim 3 0 0 0 0 \mathrm { a }$ 、“开启型”的地热系统。总体来说，隆起山地构造对流型地热流体形成年龄相对较新。

对沉降盆地传导型地热田而言，地热流体赋存于盆地深部并以管井开采的方式出露(部分自流），热储层岩性、埋藏深度、地下水补给来源与循环深度等明显控制着地热流体化学类型和形成年龄。热储埋藏深度小于 $1 6 0 0 \mathrm { m }$ 的地热井，盖层厚度薄且岩性致密性相对不足，热流体与上覆松散岩类孔隙水(来源于当地及周边大气降水入渗补给)及其他水体之间存在不同程度的水力联系，但流体补给强度较弱，主要靠来自地壳深部的热传导增温，流体循环深度浅，地下水在岩层中的滞留时间相对较短、“水-岩互助"作用相对较弱， $8 ^ { 1 8 } \mathrm { O } , 8 \mathrm { D }$ 与热水中溶解的氢氧组分交换几率相对较小， $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 漂移程度弱且不容易富集，热流体TDS相对较低，水质相对较好，属地热流体形成年龄一般小于30000a、“半开启-半封闭型"的地热系统;热储埋藏深度介于$1 6 0 0 \sim 2 6 0 0 \mathrm { m }$ 之间的地热井，盖层厚度巨大且岩性致密，热流体与上覆松散岩类孔隙水及其他水体之间基本不存在水力联系，主要为地质历史时期逐步形成的“古水”，流体补给强度极弱或基本没有补给，主要靠来自地壳深部和上地幔的热传导增温，热储埋深大、水循环交替缓慢，地下水在岩层中的滞留时间较长、“水-岩互助"作用强， $8 ^ { 1 8 } 0 . 8 \mathrm { D }$ 与热水中溶解的氢氧组分交换几率相对较大， $\mathbf { 8 ^ { 1 8 } O }$ 漂移程度强且容易富集，地热流体TDS较高，水化学类型复杂且水质较差，属地热流体形成年龄介于$3 0 \ 0 0 0 \sim 5 0 \ 0 0 0$ a、“封闭型”的地热系统。总体来说，沉降盆地传导型地热流体形成年龄相对较老，存在热储埋藏深度越深，地质条件越封闭，8D、 $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 越富集，流体形成年龄越老的地热赋存规律。

# 5结语

（1）已有的研究结果表明[12-14],热流体环境同位素可以用于研究地下水的补给循环及其赋存环境。地热流体的8D值的变化主要取决于热水接受补给时的环境温度及其补给高程，此外混合作用亦有一定的影响； $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 值的变化除与8D有相同的影响因素外，还主要取决于热水与围岩交换的程度； $^ { 1 4 } \mathrm { C }$ 属放射性同位素，其值提供了地下热水的形成及滞留时间等宝贵信息。因此，环境同位素技术是一种不可替代的查找地球深部地热流体起源、成因及其运移循环的有效手段[13]

（2）本次研究发现，甘肃省隆起山地构造对流型地热田水质一般相对较好，形成年龄较新，部分样品中化学成分以重碳酸盐或重碳酸盐 $+$ 硫酸盐占绝对优势，局地TDS小于 $0 . 5 \ \mathrm { g \cdot L ^ { - 1 } }$ 、锶含量为 $0 . 4 1 \sim$ $1 . 2 9 \mathrm { m g } \cdot \mathrm { L } ^ { - 1 }$ 的地热流体，达到了锶型天然饮用矿泉水的标准[15],可以进一步勘查及开发利用;而沉降盆地传导型地热田水质一般较差，形成年龄相对较老，大部分样品中化学成分以氯化物或氯化物 $^ +$ 硫酸盐占绝对优势，TDS大于 $3 . 0 \sim 1 0 . 0 \ \mathrm { g \cdot L } ^ { - 1 }$ ,井口水温 $5 5 \sim 7 7 ~ \mathrm { ^ { \circ } C }$ 。这种高盐、高温且伴随大量 $\mathrm { H } _ { 2 } \mathrm { S }$ 气体溢出的地热流体对人体健康十分有害，对上下水管道、水泵、供暖设施等均具有高度腐蚀性，开发时应予以重视。

（3）由于本次未收集到研究区不同海拔高程大气降水、冰川及永久积雪融水的 $\delta ^ { \mathrm { 1 8 } } \mathrm { O } , \delta \mathrm { D }$ ） $^ { 1 4 } \mathrm { C }$ 等环境同位素测试数据，故未能依据氘盈余量[16-17]就地热流体特别是沉降盆地传导型地热流体与水资源形成
