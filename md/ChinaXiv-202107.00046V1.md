# The hypoglycemic activities of different solvent extracts from Pyrrosia lingua (Thunb.) Farwell in vitro

Abstract: In order to provide scientific basis for the further development and utilization of Pyrrosia lingua, the $\mathfrak { a }$ -glucosidase and $\mathfrak { a }$ -amylase inhibitory activities of different polar extracts of $9 5 \%$ ethanolextractsofPyrrosia lingua were investigatedby using p-nitrophenyl- $\mathbf { \nabla } \cdot \mathbf { a }$ -D-glucopyranoside (pNPG) and DNS (3,5-dinitro salicylic acid (DNS) methods with acarbose as positive control. Enzyme kinetics method and Lineweaver-Burk curve were used to analyze the types of inhibition of $\mathfrak { a }$ -glucosidase for the most active extract. The results showed that the $\mathrm { I C } _ { 5 0 }$ values of the water extract were $( 4 . 7 1 \pm 0 . 7 2 ) \mu \mathrm { g } \cdot \mathrm { m L } ^ { - 1 }$ for $\mathfrak { a }$ -glucosidase and （48.40 $\pm \ : 0 . 3 2 ) \ : \ : \mu \mathrm { g } { \cdot } \mathrm { m L } ^ { - 1 }$ for $\mathfrak { a }$ -amylase respectively,which were significantly stronger than those of the other extracts $( P < 0 . 0 5 )$ . The enzymatic kinetics showed that the inhibitory type on $\mathfrak { a }$ -glucosidase of water extract was reversible inhibition, while the Lineweaver-Burk curve showed a competitive inhibition. In conclusion, the water extract from Pyrrosia lingua displayed potential inhibitory activity on $\mathfrak { a }$ -glucosidase and $\mathfrak { a }$ -amylase,which could be the natural sources of $\mathfrak { a }$ -glucosidase inhibitorwith less gastrointestinal side effects.

2019 年全球 20\~79 岁糖尿病患者约 4.63 亿，中国约1.164 亿（International DiabetesFederation,2019），糖尿病已成为的严重威胁人类健康的第三大疾病（杨文英，2018）。餐后血糖持续升高是引起糖尿病患者血管内皮损伤、视网膜病变等并发症的主要原因，有效控制餐后血糖上升有利于防治糖尿病及其并发症的发生（中华医学会糖尿病学分会，2018）。以 $\mathfrak { a }$ -葡萄糖苷酶和 $\mathfrak { a }$ -淀粉酶为降低餐后血糖有效分子靶标开发的阿卡波糖、伏格列波糖等药物已成为临床降低餐后血糖的首选药物。有研究表明，由于阿卡波糖对 $\mathfrak { a }$ -淀粉酶具有强抑制作用，长期服用会引起引发肠胃胀气、腹部不适等胃肠道副作用（苏青，2018；袁向华等，2016），从中药及天然药物中寻找新型高效、副作用小的 $\mathfrak { a }$ -淀粉酶和 $\mathfrak { a }$ -葡萄糖苷酶抑制剂已引起医药工作的关注，近年来也取得了一系列的研究成果（范莉等，2016）。

石韦（Pyrrosialingua）为水龙骨科石韦属植物，别名石剑、石苇，主产区为广东、福建、台湾等地，具有通淋利水、泄热清肺等功效，临床用于治疗淋痛、尿血、尿路结石、肾炎等（国家药典委员会，2015）。石韦为民间防治糖尿病的常用中草药（周悦，2016），现代药理研究也证实石韦可影响小鼠血糖、糖耐量（王兵和黄传贵，2008）、改善糖尿病肾病（刘绪林等，2018）。此外，本课题组前期对梅州客家地区特色和优势药用植物进行了体外降血糖活性初步筛选，发现石韦具有较好的体外降血糖活性。为进一步明确石韦对 $\mathfrak { a }$ 葡萄糖苷酶和 $\mathfrak { a }$ -淀粉酶抑制作用和作用机制，本研究以阿卡波糖为阳性对照，4-硝基苯 $\mathbf { \nabla } \cdot \mathbf { a } .$ -D-吡喃葡萄糖苷和淀粉为底物，评价石韦 $9 5 \%$ 乙醇提取物的不同极性萃取物对 $\mathfrak { a }$ -葡萄糖苷酶和 $\mathfrak { a }$ 号淀粉酶的抑制活性，并对活性最强的萃取物通过酶促动力学方法与Lineweaver-Burk曲线明确其 $\alpha \cdot$ -葡萄糖苷酶的抑制作用类型，为进一步开展石韦降血糖方面的化学成分和药效学研究提供科学依据。

# 3讨论与结论

石韦为民间防治糖尿病的常用中草药（周悦，2016），现代药理研究显示石韦可影响小鼠血糖、糖耐量（王兵和黄传贵，2008）、改善糖尿病肾病（刘绪林等，2018），但其作用靶点是否与抑制 $\mathfrak { a }$ -葡萄糖苷酶和 $\mathfrak { a }$ -淀粉酶的研究未见报道。实验首次发现石韦 $9 5 \%$ 乙醇提取物的不同溶剂萃取物对 $\mathfrak { a }$ -葡萄糖苷酶和 $\mathfrak { a }$ -淀粉酶具有良好的抑制活性，丰富了石韦发挥降血糖作用的靶点研究。

目前临床常用的 $\mathfrak { a }$ -葡萄糖苷酶抑制剂由于对 $\alpha$ -淀粉酶的过度抑制，常导致胃肠道副作用（李波等，2015），研究表明其抑制作用强度与假双糖结构连接的葡萄糖分子数目有关（顾觉奋和陈紫娟，2009）。本实验对石韦 $9 5 \%$ 乙醇提取物的不同溶剂萃取物开展较为系统的体外 $\mathfrak { a }$ -葡萄糖苷酶和 $\mathfrak { a }$ -淀粉酶抑制活性评价，结果显示，石韦各萃取物对 $\mathfrak { a }$ -葡萄糖苷酶和 $\mathfrak { a }$ 淀粉酶抑制活性大小均为水萃取物 $>$ 正丁醇萃取物 $\vartriangleright$ 乙酸乙酯萃取物 $>$ 石油醚萃取物（ $P <$ 0.05），且活性最强的水萃取物对 $\mathbf { \boldsymbol { a } } .$ -葡萄糖苷酶的抑制类型为竞争性抑制，与阳性对照阿卡波糖相同，可能与石韦中具有与阿卡波糖抑制 $\scriptstyle { \mathfrak { a } } .$ -葡萄糖苷酶和 $\mathfrak { a }$ -淀粉酶所必须的假双糖结构有关，该结构由环已糖醇及氨基糖构成，为极性大的寡糖类似物（顾觉奋和陈紫娟，2009），但石韦在体内对 $\mathfrak { a }$ -葡萄糖苷酶的抑制作用是否与阿卡波糖对 $\mathfrak { a }$ -葡萄糖苷酶的抑制作用相同，还需要进一步深入研究。此外，结果表明石韦萃取物对 $\mathbf {  { a } }$ -葡萄糖苷酶的抑制活性较强，而 $\alpha \cdot$ 淀粉酶抑制活性较弱，可能与石韦中发挥 $\mathfrak { a }$ -淀粉酶抑制活性物质结构连接的葡萄糖分子数目与阿卡波糖连接葡萄糖分子数目不同有关。

本研究揭示了石韦 $9 5 \%$ 乙醇提取物的不同溶剂萃取物中水萃取物具有最强的 $\mathbf {  { a } }$ -葡萄糖苷酶和 $\scriptstyle { \mathfrak { a } } .$ -淀粉酶的抑制活性，且对 $\scriptstyle { \mathfrak { a } } .$ -葡萄糖苷酶为可逆竞争性抑制，表明石韦大极性化学部位中富含抑制 $\mathfrak { a }$ -葡萄糖苷酶和 $\mathfrak { a }$ -淀粉酶的活性成分，是天然的、胃肠道副作用小的 $\scriptstyle { \mathfrak { a } } .$ 葡萄糖苷酶抑制剂来源。但石韦中抑制 $\mathfrak { a }$ -葡萄糖苷酶和 $\mathbf {  { a } }$ -淀粉酶的具体活性成分、体内外作用机制以及构效关系等还不明确，后续研究应重点对大极性化学部位开展系统的化学成分、作用机制、构效关系等研究，有望阐明石韦降血糖作用的科学内涵。
