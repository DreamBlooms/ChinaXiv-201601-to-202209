随着人们生活水平的提高，羊肉在餐桌上越来越受到欢迎，这主要是由于羊肉具有营养  
丰富、口感好以及食疗药补等优点，被认为是一种保健型的肉类1。但与养羊业发达的国家  
相比，我国在羊肉的品质方面还存在着很大的提升空间。羊肉中脂肪酸的组成是肉品质的主  
要影响因素，因此研究羔羊脂肪酸的沉积对提高肉品质具有重要的意义[2]。通过控制断奶时间来改善羊肉肉品质已成为反刍动物营养领域的一个热点。通过控制断  
奶时间可以改变反刍动物的饲粮摄入，反刍动物饲粮粗精比[3-4]、能量[5-6]和蛋白质水平[7]直  
接影响动物肌肉中脂肪酸的组成，从而起到改变肌肉中脂肪酸的组成达到改善肉品质的目的。

# 1.3.2 肌肉脂肪酸测定

# 1.3.2.1 样品预处理

样品总脂肪的提取及脂肪酸甲酯化参考O'Fallon 等[13]的方法进行。 $\mathrm { ~ 1 ~ g ~ }$ 的半冷冻样品均匀地置于室温条件下的研磨机中研磨10\~15s,然后放置于1个 $1 6 \mathrm { { m m } } \times 1 2 5 \mathrm { { m m } }$ 带螺帽耐热玻璃装有 $0 . 7 ~ \mathrm { m L } ~ 1 0 ~ \mathrm { m o l / L }$ 的KOH溶液和 $6 . 3 ~ \mathrm { m L }$ 的甲醇的培养管。将管子置于 $5 5 \mathrm { ~ } ^ { \circ } \mathrm { C }$ 水浴$1 . 5 \mathrm { h }$ ，每 $2 0 \mathrm { m i n }$ 用力用手晃动管子5s使样品正常渗透、溶解和水解。通过自来水水浴至室温后，加入 $0 . 5 8 ~ \mathrm { m L } 1 2 ~ \mathrm { m o l / L }$ 的 $\mathrm { H } _ { 2 } \mathrm { S O } _ { 4 }$ 。管子通过倒置混合，会产生 $\mathrm { K } _ { 2 } \mathrm { S O } _ { 4 }$ 沉淀，然后继续置于 55℃水浴 $1 . 5 \mathrm { h }$ ，每 $2 0 ~ \mathrm { m i n }$ 用力晃动管子 $5 \mathrm { ~ s ~ }$ 。脂肪酸甲酯（FAME）形成后，管子置于自来水中冰浴。加入 $3 \mathrm { m L }$ 正己烷，管子涡旋 $5 \mathrm { m i n }$ ，然后将管子离心 $5 \mathrm { m i n }$ 。将含有FAME的正己烷层抽取到气相色谱分析（GC）小瓶中。GC小瓶加盖置于 $- 2 0 \mathrm { ~ } ^ { \circ } \mathrm { C }$ 下保存至分析。

# 1.3.2.2 气相色谱分析方法

制备好的脂肪酸甲酯进行GC（Trace1300，Thermo Fisher)，毛细色谱柱为 $1 0 0 \mathrm { m } { \times } 0 . 2 5$ $\mathrm { m m } { \times } 0 . 2 \mu \mathrm { m }$ （HP-88，Agilent)。程序升温条件为:初始温度166 $\mathrm { { } ^ { \circ } C }$ 维持 $3 9 \mathrm { m i n }$ ，以 $1 0 \ \mathrm { ^ { \circ } C / m i n }$ 升至 $2 3 0 \mathrm { ~ \textdegree ~ }$ ,维持 $1 0 \mathrm { m i n }$ ，以 $3 ~ \mathrm { { ^ { \circ } C / m i n } }$ 升至 $2 3 5 ^ { \circ } \mathrm { ~ C ~ }$ ，维持 $8 \mathrm { m i n }$ ，进样口温度为 $2 5 0 \mathrm { ~ \textdegree C }$ ，火焰离子检测器(FID)温度为 $2 5 0 \mathrm { ~ \textdegree C }$ ，载气氮气 $( \mathbf { N } _ { 2 } )$ 流速为 $1 . 2 ~ \mathrm { m L / m i n }$ ,分流比25:1。脂肪酸甲基酯标样均购于 Sigma 公司。

# 3.1 营养物质各养分采食量

在本课题组的另一项研究中，对同等条件下羔羊营养物质采食量进行了分析，由于羔羊随母羊群饲，养分采食量数据未进行统计分析。在1\~28日龄时，28日龄断奶组和56日龄断奶组羔羊干物质采食量不高且接近（36.88g/dvs.40.01g/d），在29\~70日龄时28日龄断奶组采食量（205.05\~643.62g/d）高于56日龄断奶组（152.59\~569.03g/d），在71\~84日龄时28日龄断奶组采食量低于28日龄断奶组（1008.33g/d vs.1156.41g/d）[18]。
