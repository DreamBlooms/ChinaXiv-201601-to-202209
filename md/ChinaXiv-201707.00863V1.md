水资源主要是指与人类社会和生态环境保护息息相关而又能不断更新的地表水和地下水[1],随着近年来全球水质型缺水日益严重[2],人们对清洁水源的需求愈发迫切。森林生态系统具有调节、净化和稳定水质的作用[3],被认为是清洁水源的发祥地[4],发挥森林的水源涵养功能是解决水资源问题的重要途径之一[5]目前,森林与水质的研究成为了森林生态水文学研究的重点和热点,分析森林水环境的化学变化对了解森林净化水源的作用机制具有重要意义[6-7]。

森林生态系统丰富的结构层次能够对大气降水进行重新分配和有效调节,发挥森林生态系统特有的生态水文功能。大气降水进入森林后,林冠层对降水有着再分配作用,一部分形成穿透雨,直接降落到林下植被和枯落层中,还有一部分沿树干流下,成为树干径流,随后进入到土壤层,这些水分变为地表径流、壤中流和地下水,水中的化学成分和pH值随着介质的变化会再度发生改变。林冠层对降水 $\mathrm { \ p H }$ 值的改变不尽相同,某些树种的林冠层可以使降水 $\mathrm { \ p H }$ 值下降[8-12],某些树种林冠层有提高降水 $\mathrm { \ p H }$ 值的作用[13-15]。而对贵州典型森林植被研究表明,林冠层对降雨pH值具有缓冲作用,可以使 $\mathrm { \ p H }$ 值稳定在一定区间内[16],同时林冠层对降水中离子的改变也因树种而异[17-9]。树干径流pH值和离子质量浓度也因树种而异[2-23],有研究表明同一树种在不同区域的树干径流中的元素含量也有显著差异[24]。枯枝落叶层是森林水文功能的第二作用层,其具有截留降水,增加入渗,增加地表粗糙度,拦截地表径流,抑制土壤蒸发,防止土壤溅蚀,增加土壤有机质,改变土壤质地,改善土壤结构,进而增强土壤抗蚀能力和蓄水减沙能力等多种功能[25],目前我国对枯枝落叶层的水分涵养机理研究主要是集中在枯落物对水分下渗、地表蒸发、径流含沙量等方面的影响和持水过程分析,对枯落物水质的研究仅见少数报道[26-29]。

黑河是我国西北地区第二大内陆河,流经青海省、甘肃省和内蒙古自治区的11个县市,其水质的好坏对流域内居民的生产生活有着十分重要的影响。祁连山区是黑河流域的发源地,其内的水源涵养林在净化水质方面发挥着巨大的生态作用[30]，山区内主要树种为青海云杉（Piceacrasifola）和祁连圆柏（Sabinaprzewalskii）,因此研究这两种林木的生态水文化学特征不仅有具有重要的环境意义,还具有深远的社会意义。但是，目前对其的研究多集中在生态水文过程的物理过程和生态效应方面,对生态水文化学过程研究较少，仅有金博文等[31]和王顺利等[32]对祁连山区青海云杉穿透雨元素含量进行过研究,而对其树干径流和枯透水元素质量浓度变化鲜有报道，对祁连圆柏林的水质研究更是未见报道,因此，本文将测定青海云杉和祁连圆柏林穿透雨，树十径流，枯透水的 $\mathrm { \ p H }$ 值和离子质量浓度，为祁连山区水源涵养林水质净化机制研究提供数据，同时为祁连山森林生态系统生态服务功能研究奠定基础。

# 2.4水样化学分析

# 2.4.1 pH值测定

# 4讨论

本流域内大气降水呈碱性,其可归因于两个方面,一方面,流域内水体和土壤呈碱性,形成的碱性凝结核使降水 $\mathrm { \ p H }$ 值升高,另一方面,研究区外来沙源主要为巴丹吉林沙漠、腾格里沙漠和塔克拉玛干沙漠，这些地区土壤 $\mathrm { \ p H }$ 值较高[34-35]，输送而来的碱性沙尘溶于降水,使其偏碱;林冠对降水 $\mathrm { \ p H }$ 的改变因树种而异[8-16],有研究认为某些林冠之所以可以使降水 $\mathrm { \ p H }$ 值变小,可能是由于有机酸和无机酸溶解于降水之中[36],而部分林冠能够使 $\mathrm { \ p H }$ 值升高的原因可能是林木组织内盐基阳离子与降水中 $\mathrm { H ^ { + } }$ 发生置换,盐基离子取代 $\mathrm { H } ^ { + }$ ,从而使得林内雨的 $\mathrm { \ p H }$ 值上升[9],圆柏与云杉的穿透雨 $\mathrm { \ p H }$ 均小于林外雨,这与部分研究结果吻合[8-12],这可能是由

于两种林分冠层分泌的酸性物质溶于降水所致,其中和了部分降水和林冠干沉降中的碱性物质,虽然 $\mathrm { \ p H }$ 值较林外雨有所降低,但两种林分穿透雨仍为碱性;两种林分树干径流 $\mathrm { \ p H }$ 值较穿透雨升高，这可能是由于流域内干沉降多为碱性物质,云杉和圆柏表皮粗糙,裂隙多,容易承接干沉降,树干产生径流时,水分与树干上的碱性沉降物反应,使降水 $\mathrm { \ p H }$ 值升高。云杉和圆柏枯透水 $\mathrm { \ p H }$ 值与穿透雨相比升高，这与杜敏等[29]的研究结果一致,其成因与树干径流类似,枯落物分解时产生碱性物质[37融入其中,导致其比穿透雨pH值升高。

不同地区的大气环境差异显著,降水中各养分浓度具有很强的空间异质性[38],本研究区林外雨阳离子中$\mathrm { C a } ^ { 2 + }$ 质量浓度最多,这与很多研究结果一致[3,8,18,23,27],但是,宋文龙等[16]在研究贵州森林群落时发现大气降水中 $\mathrm { N H } _ { 4 } ^ { + }$ 质量浓度最高,田大伦等[39]在研究第杉木幼林时,得出大气降水中K元素质量浓度最高,而巩合德等[40]发现S元素质量浓度最高。降水中的 $\mathrm { C a } ^ { 2 + }$ 主要来源于空气中和陆地上的尘埃和有机物[41],因此质量浓度高低与大气飘尘和污染物有关。 $\mathrm { { N a } ^ { + } }$ 在林外雨中质量浓度最低,这与卢晓强等[42 的研究结果一致。有研究认为大气中 $\mathbf { M } \mathbf { g } ^ { 2 + }$ 大多来自于海洋[43],本研究区距海洋较远,故其质量浓度较低， $\bar { \mathsf { K } ^ { \dagger } }$ 虽仅次于 $\mathrm { C a } ^ { 2 + }$ ,但其绝对质量浓度较低,这与苏日娜[44]的研究一致。

与林外雨相比，除 $\mathrm { C u } ^ { 2 + }$ 和圆柏穿透雨的 $\mathrm { N H } _ { 4 } ^ { + }$ 外,穿透雨中各阳离子质量浓度均有升高。相关研究表明，$\mathrm { C a } ^ { 2 + }$ 和 $\mathbf { M } \mathbf { g } ^ { 2 + }$ 较难从植物组织中淋溶出来[39.45],而 $\mathbf { K } ^ { + }$ 和 $\mathrm { { N a } ^ { + } }$ 易发生溶解迁移[8.41,44],所以 $\mathrm { { \dot { \mathrm { { G a } } } } } ^ { 2 + }$ ， $\mathbf { M } \mathbf { g } ^ { 2 + }$ 的增加主要是由于林冠的干沉降溶于降水所致，而 $\mathbf { K } ^ { + }$ 和 $\mathrm { { N a } ^ { + } }$ 的增加除了干沉降溶解外，还有淋溶作用。两种林分穿透雨中 $\mathrm { { N a } ^ { + } }$ 质量浓度均有显著增加,因林外雨中 $\mathrm { { N a } ^ { + } }$ 含量较低,故其绝对质量浓度并不太高,与云杉相比,圆柏穿透雨中 $\mathrm { \Delta N a } ^ { + }$ 质量浓度增幅较高，这表明圆柏冠层会分泌较多的 $\mathrm { { N a } ^ { + } }$ $\mathrm { \Delta N a } ^ { + }$ 易发生溶解迁移,降雨通过冠层后其 $\mathrm { { N a } ^ { + } }$ 质量浓度增加。 $\mathrm { N H } _ { 4 } ^ { + }$ 是是植物生长所需的重要营养成分,研究表明无机态氮可被叶片吸收或通过离子交换而固定在植物体内[38,46],因此圆柏穿透雨中 $\mathrm { N H } _ { 4 } ^ { + }$ 降低,而云杉穿透雨中 $\mathrm { N H } _ { 4 } ^ { + }$ 小幅增加,这可能是因为某些树种的冠层对 $\mathrm { N H } _ { 4 } ^ { + }$ 有调节作用[14],当林外雨中 $\mathrm { N H } _ { 4 } ^ { + }$ 较少时,会使穿透雨中 $\mathrm { N H } _ { 4 } ^ { + }$ 增加,反之降低。 $\mathrm { C u } ^ { 2 + }$ 难以淋失但易被吸附固定[29],在两种林分穿透雨含量减少,云杉降低的幅度比圆柏稍大,说明云杉冠层对 $\mathrm { C u } ^ { 2 + }$ 的吸附要略好于圆柏。

与穿透雨相比，除云杉树干径流的 $\mathrm { G } \bar { \mathrm { u } } ^ { 2 + }$ 和 $\mathrm { N H } _ { 4 } ^ { + }$ 表现为升高外,两种林分树干径流中阳离子质量浓度均有所下降。 $\mathrm { C a } ^ { 2 + }$ ） $\mathbf { M } \mathbf { g } ^ { 2 + }$ ） $\mathrm { K } ^ { + }$ 和 $\mathrm { \Delta N a } ^ { + }$ 下降可能是因为两种林分树干粗糙,对离子有较强的吸附作用[3]。云杉树干径流中 $\mathrm { N H } _ { 4 } ^ { + }$ 质量浓度升高,这可能是因为云杉主要分布在研究区阴坡和半阴坡,郁闭度大,林内湿度大,树干上多附生生物，其分解有机物后形成 $\mathrm { N H } _ { 4 } ^ { + }$ ,导致其质量浓度升高。圆柏树干径流中 $\mathrm { N H } _ { 4 } ^ { + }$ 下降可能是因为圆柏分布于阳坡,林内空气湿度小,树干上附生生物少,分解产生的 $\mathrm { N H } _ { 4 } ^ { + }$ 较少,树干的吸附量高于产生的 $\mathrm { N H } _ { 4 } ^ { + }$ 量，因此降低。 $\mathrm { C u } ^ { 2 + }$ 易被吸附固定[29],所以圆柏树干径流中 $\mathrm { C u } ^ { 2 + }$ 降低,云杉树干径流中虽然 $\mathrm { C u } ^ { 2 + }$ 含量升高,但幅度较小，可以认为是云杉树干对 $\mathrm { C u } ^ { 2 + }$ 的吸附性不佳，同时有部分干沉降溶于树干径流中所致。 $Z \mathrm { n } ^ { 2 + }$ 难以淋失和易被固定[21],在所有样本中,只有在云杉树干径流检测出 $Z \mathrm { n } ^ { 2 + }$ ,这可能是云杉树干活体组织会析出微量的$Z \mathrm { n } ^ { 2 + }$ 所致。

与穿透雨相比,云杉枯透水各阳离子质量浓度均呈下降趋势,而圆柏枯透水中 $\mathbf { K } ^ { + }$ 、 $\mathrm { C a } ^ { 2 + }$ 和 $\mathrm { M g } ^ { 2 + }$ 质量浓度升高， $\mathrm { N H _ { 4 } ^ { + } } \mathrm { \Omega } \mathrm { \backslash N a } ^ { + }$ 和 $\mathrm { C u } ^ { 2 + }$ 质量浓度降低。云杉枯透水中 ${ { \mathrm { C a } } ^ { 2 + } } , { { \mathrm { M g } } ^ { 2 + } }$ ） $\mathrm { K } ^ { + }$ 和 $\mathrm { { N a } ^ { + } }$ 下降可能是由于枯落物对离子的吸附作用,两种林分中 $\mathrm { N H } _ { 4 } ^ { + }$ 的降低可能是因为枯落物中附生生物能把 $\mathrm { N H } _ { 4 } ^ { + }$ 吸收利用,并转换为有机态[42],而$\mathrm { C u } ^ { 2 + }$ 易被吸附固定,所以在云杉和圆柏中质量浓度均下降,圆柏枯透水中 $\mathrm { K } ^ { + }$ ） $\mathrm { C a } ^ { 2 + }$ 和 $\mathrm { M g } ^ { 2 + }$ 质量浓度升高,可能是因为圆柏枯落物在分解过程中释放出少量的 $\mathrm { K } ^ { + }$ ） $\mathrm { C a } ^ { 2 + }$ 和 $\mathbf { M } \mathbf { g } ^ { 2 + }$ 。

林外雨中 $\mathrm { N O } _ { 3 } ^ { - }$ 和 $\mathrm { S O } _ { 4 } ^ { 2 - }$ 质量浓度基本相等， $\mathrm { C l } ^ { - }$ 质量浓度稍低,因大气中离子含量异质性较高[38],故其含量由本研究区的地理状况决定。 $\mathrm { N O } _ { 3 } ^ { - }$ 在两种林分穿透雨中质量浓度降低，主要是无机氮可以被植物吸收利用，同时部分 $\mathrm { N O } _ { 3 } ^ { - }$ 以挥发性酸 ${ \mathrm { H N O } } _ { 3 }$ 的形式存在于水中，当雨滴击溅林木叶面时，加速了HNO3的挥发，从而使 $\mathrm { N O } _ { 3 } ^ { - }$ 质量浓度降低[2]， $\mathrm { S O } _ { 4 } ^ { 2 - }$ 和 $\mathrm { C l } ^ { - }$ 在两种林分穿透雨中质量浓度升高,可能是干沉降溶于水中,也有可能是

大气降水可以洗脱叶内 $\mathrm { S O } _ { 4 } ^ { 2 - }$ 和 $\mathrm { C l } ^ { - }$ 所致。

与穿透雨相比,两种林分树干径流和枯透水中的阴离子质量浓度均有升高, $\mathrm { N O } _ { 3 } ^ { - }$ 质量浓度升高可能是因为两种林分树干附生生物分解产生无机氮,因云杉树干附生生物较多,所以产生的 $\mathrm { N O } _ { 3 } ^ { - }$ 较多,使其基本恢复到林外雨的水平,而圆柏树干附生生物较少,因此 $\mathrm { N O } _ { 3 } ^ { - }$ 比穿透雨仅有小幅度上升，未达到林外雨水平。至于为何圆柏枯透水中的 $\mathrm { C l } ^ { - }$ 和 $\mathrm { S O } _ { 4 } ^ { 2 - }$ 大幅显著增加,而在其树干径流中只有小幅度提升,云杉枯透水中CI和 $\mathrm { S O } _ { 4 } ^ { 2 - }$ （2质量浓度基本与穿透雨持平,而在其树干径流中会产生大量 $\mathrm { C l } ^ { - }$ 和 $\mathrm { S O } _ { 4 } ^ { 2 - }$ ,这一问题需要进一步做深入的研究。

# 5 结论

(1)天涝池流域大气降水 $\mathrm { \ p H }$ 均值为7.74,呈碱性,降水中离子绝对质量浓度较低。(2)青海云杉和祁连圆柏冠层有降低降雨 $\mathrm { \ p H }$ 值的作用,云杉林冠层对 $\mathrm { N H } _ { 4 } ^ { + }$ 有升高作用,圆柏林冠层对  
$\mathrm { N H } _ { 4 } ^ { + }$ 有降低作用,两种林冠层对 $\mathrm { N O } _ { 3 } ^ { - }$ 和 $\mathrm { C u } ^ { 2 + }$ 有降低作用,对其它离子质量浓度均表现为升高作用。(3)典型林分树干径流有提高穿透雨 $\mathrm { \ p H }$ 值的作用,与穿透雨相比,两种林分树干径流中阴离子均有升  
高,其中云杉树干径流中的 $\mathrm { C l } ^ { - }$ 和 $\mathrm { S O } _ { 4 } ^ { 2 - }$ 增高较多，圆柏树干径流中所有阳离子质量浓度均有下降,云杉树干径  
流中 $\mathrm { C a } ^ { 2 + }$ 、 $\mathrm { K } ^ { + }$ ） $\mathbf { M } \mathbf { g } ^ { 2 + }$ 和 $\mathrm { { N a } ^ { + } }$ 减少， $\mathrm { N H } _ { 4 } ^ { + }$ 和 $\mathrm { C u } ^ { 2 + }$ 增加。(4)典型林分枯透水有提升穿透雨 $\mathrm { \ p H }$ 值的作用,与穿透雨相比,两种林分枯透水中阴离子质量浓度均有  
升高,其中圆柏枯透水中的 $\mathrm { C l } ^ { - }$ 和 $\mathrm { S O } _ { 4 } ^ { 2 - }$ 显著升高,云杉枯透水各阳离子均有降低,圆柏枯透水中 $\mathrm { C a } ^ { 2 + }$ ） $\mathrm { K } ^ { + }$ 和  
$\mathbf { M } \mathbf { g } ^ { 2 + }$ 质量浓度升高， $\mathrm { N H } _ { 4 } ^ { + }$ ） $\mathrm { { N a } ^ { + } }$ 和 $\mathrm { C u } ^ { 2 + }$ 质量浓度下降。(5)在采集的所有样本中， $\mathrm { P b } ^ { 2 + }$ 和 $\mathrm { C d } ^ { 2 + }$ 均未检出,而 $\mathrm { Z n } ^ { 2 + }$ 仅在云杉树干径流中检出。
