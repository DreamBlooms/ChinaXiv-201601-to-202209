# 黄土区不同土地利用类型下砂质壤土地表 $\mathbf { C O } _ { 2 }$ 通量变化特征——以清凉寺沟流域为例

许志平¹，邵天杰¹，张连凯²，邵明玉²，牛俊杰³（1.陕西师范大学地理科学与旅游学院,陕西 西安710119；2.中国地质科学院岩溶地质研究所，广西 桂林541004；3.太原师范学院汾河流域科学发展研究中心,山西 太原030012)

摘要：利用LI-8100A土壤碳通量自动测量系统对晋西北黄土丘陵沟壑区清凉寺沟流域地表土壤 $\mathrm { C O } _ { 2 }$ 通量进行测量，探讨了 $\mathrm { C O } _ { 2 }$ 通量时空变化特征、不同土地利用类型下的 $\mathrm { C O } _ { 2 }$ 通量特征以及环境因子对 $\mathrm { C O } _ { 2 }$ 通量的影响，对于晋西北黄土丘陵沟壑区小流域地区地表土壤 $\mathrm { C O } _ { 2 }$ 通量的研究进行补充。结果表明：(1) $\mathrm { C O } _ { 2 }$ 通量在生长季呈现单峰变化,于日出前后达单日最低值,在16:00—22:00达到单日最高值;在非生长季则呈现不规律变化。（2）农耕地的 $\mathrm { C O } _ { 2 }$ 通量相较于荒草地以及枣林地的 $\mathrm { C O } _ { 2 }$ 通量要低;不同季节的 $\mathrm { C O } _ { 2 }$ 通量关系为8月 $\mathrm { C O } _ { 2 }$ 通量最大，5月 $\mathrm { C O } _ { 2 }$ 通量次之，11月 $\mathrm { C O } _ { 2 }$ 通量最低。（3）地表土壤 $\mathrm { C O } _ { 2 }$ 通量与光合有效辐射、大气温度、土壤温度、土壤含水率、土壤有机质、土壤质地等因素密切相关，通过科学的土地利用方式可有效提高土壤的固碳能力。

关键词：土壤 $\mathrm { C O } _ { 2 }$ 通量；时空变化特征；影响因素；土地利用类型；黄土丘陵沟壑区；晋西北

地表 $\mathrm { C O } _ { 2 }$ 通量作为碳循环的关键一环，成为目前全球变暖研究必不可少的关键内容之一。作为土壤碳库以及大气碳库的关键枢纽，其微小变化都显著影响着全球碳循环和碳平衡[1-2]。黄土高原地处干旱、半干旱气候区，黄土深厚，土壤资源丰富，碳库储量大，处于我国生态极敏感区且地表 $\mathrm { C O } _ { 2 }$ 通量不稳定[3-6]。同时，黄土高原作为半干旱向半湿润的过渡地区，土地利用方式多样，不合理的土地利用方式会影响土壤和植物的固碳能力，使地表碳排放不稳定的现象加剧[7-9]。目前,在黄土高原地区开展的大规模退耕还林，是以生态建设为目的调整不合理土地利用方式的具体体现,科学的土地利用方式可以增加土壤的固碳能力,从而减缓全球变暖[10-12]

土壤呼吸也可定义为土壤表面 $\mathrm { C O } _ { 2 }$ 通量，土壤呼吸不仅包含自养呼吸还包括异养呼吸，并且在整个过程中受到其他环境因子的影响：温度、水分、光照和有机质等，同时土壤呼吸复合控制机制在多种时空尺度上的交互作用，使得土壤呼吸机理的过程变得更加复杂[13-14]。关于土壤呼吸主要影响因子的分析表明，土壤呼吸速率是由多种影响因子相互影响共同决定的，土壤温度和土壤含水率在土壤呼吸中起到主导作用[15]。由于温度是调节和控制许多生态学过程的关键因素，所以土壤碳循环中多个环节会直接或间接受到温度的影响，土壤呼吸是随着温度的上升而增加[16-18]。土壤含水率也对土壤呼吸有着极大的影响，影响方式也更为复杂，含水率过高或过低都会抑制土壤呼吸[19]。土壤有机质是土壤呼吸的重要基质，且与土壤呼吸速率呈正相关[20]。土地利用方式会影响大气当中的 $\mathrm { C O } _ { 2 }$ 的碳含量，并且不同的土地利用方式对于土壤呼吸所产生的影响也略有不同[21]。农田生态系统作为人类活动较为密集的区域，其抑制土壤释放 $\mathrm { C O } _ { 2 }$ 到大气环境，对减缓全球变暖有着积极作用[22]

在黄土高原土壤侵蚀与生态恢复综合治理过程中，小流域是基本结构单元[23]。过往的研究集中在单一植被类型或单一生态系统，而本文以黄土丘陵沟壑区清凉寺沟流域为研究对象，通过利用LI-8100A土壤碳通量自动测量系统，对晋西北黄土区不同土地利用类型土壤 $\mathrm { C O } _ { 2 }$ 通量进行系统分析。在研究区域结合时间和空间双重尺度的情况下，同时综合环境因子研究土壤表面 $\mathrm { C O } _ { 2 }$ 通量的变化特征，以此分析土壤呼吸对碳汇的影响。黄土丘陵沟壑区土壤 $\mathrm { C O } _ { 2 }$ 通量对土地利用和气候因素的响应、以及其变异规律是该地区生态建设、土地利用结构优化的重要基础，这对于未来执行合理的农田管理措施和土地利用方式并进一步减缓甚至改善气候变暖有着重要意义。

# 1.2 研究方法

以山西临县清凉寺沟流域不同土地利用类型的地表 $\mathrm { C O } _ { 2 }$ 通量作为研究对象，利用LI-8100A土壤碳通量自动测量系统对研究区域的农耕地、枣林地、荒草地进行生长季与非生长季的地表土壤 $\mathrm { C O } _ { 2 }$ 通量的观测。根据研究区域1989—2019年气象数据，绘制年均降水量距平及年均温度距平图（图2)。2019年的年均降水量为 $4 4 2 . 5 \ \mathrm { m m }$ ，年均气温为$9 . 3 9 ~ \mathrm { ^ { \circ } C }$ ,其与1989—2019年的平均值近似。由图2可以看出，2019年是具有代表性的1a,未出现极端气候条件，故观测于2019年进行。所有观测区每季度观测1次（每个观测区连续观测 $2 4 \mathrm { h }$ )，连续观测1个水文年。考虑到研究区春季干旱多风少雨，夏季炎热雨量集中，秋冬季大气温度较低，观测时间根据观测的环境因素选择近期无降水影响、气候条件正常的晴朗天气。时间的选择是因为要排除降水对于土壤含水率的影响；当天的气候条件要与往年同期的气象数据进行对比，避免大气温度过高或过低对于土壤 $\mathrm { C O } _ { 2 }$ 通量的异常影响，晴朗天气是观测光合有效辐射的基本条件。观测分别于5月、8月以及11月进行。11月因夜间大气温度低于测量仪器的最低工作温度，故仅监测白天时段。

野外原位观测实施步骤与参数设置：先将10cm短期调查气室的土壤环埋至样点，并露出地面2cm，后将气室水平轻放于土壤环之上。其中样点选择在观测区内地表均匀平坦之处，避开虫洞、裂隙、细沟等地表破坏位置，刘割和清除土壤环内的植被和枯落物，并尽可能排除人为因素对于观测结果的影响。在临近 $\mathrm { C O } _ { 2 }$ 通量测量点将8100-203土壤温度探针和Delta-TTheta土壤湿度探针插入土壤 $5 \mathrm { c m }$ 深处测量土壤温度及土壤湿度。5月及8月对样点进行 $2 4 \mathrm { ~ h ~ }$ 观测，11月对样点进行白天时段观测，每$0 . 5 \mathrm { { h } }$ 观测1次，并设置重复测量3次。观测时长设置为 $1 2 0 ~ \mathrm { s }$ ，气室闭合与测量开始的时间设置为30s,气室闭合前的混合时间设置为 $3 0 ~ \mathrm { s }$ ，气室打开后的混合时间设置为 $4 5 \mathrm { ~ s ~ }$ 。光合有效辐射采用LI-8100A土壤碳通量自动测量系统配置的LI-190R光合有效辐射传感器测定，土壤温度采用LI-8100A土壤碳通量自动测量系统配置的8100-203土壤温度传感器测定，大气温度采用液体金属汞温度计测定。上述数据记录间隔均为 $0 . 5 \mathrm { h }$ O

# 3.1 $\mathbf { C O } _ { 2 }$ 通量在不同土地利用类型下的时空变化特征

不同的土地利用方式会影响人类活动和自然过程对于碳循环的影响[24-25]。由研究区观测数据可知，不同土地利用方式 $\mathrm { C O } _ { 2 }$ 通量浓度季节变化规律一致，均表现为8月的 $\mathrm { C O } _ { 2 }$ 通量浓度最大，5月的 $\mathrm { C O } _ { 2 }$ 通量浓度次之，11月的 $\mathrm { C O } _ { 2 }$ 通量浓度最小（图3）。这可能与温度以及光合有效辐射对植物呼吸速率的影响有关。

5月 $\mathrm { C O } _ { 2 }$ 通量由高到低为杨家坡枣林地、张国村荒草地、张国村农耕地以及清凉寺农耕地。农田土地利用措施的 $\mathrm { C O } _ { 2 }$ 通量小于荒草地生态系统与林地生态系统的 $\mathrm { C O } _ { 2 }$ 通量，这说明合理的土地利用措施对于土壤碳有一定的固定作用。与王剑等[]、毛琴琴[1等的研究结果相似,即科学的土地利用方式可增加固碳能力。4个观测区在日尺度上的 $\mathrm { C O } _ { 2 }$ 通量最低值均出现在8：00，其中清凉寺农耕地和张国村农耕地的变化规律较为一致，随着日出后气温升高， $\mathrm { C O } _ { 2 }$ 通量也随之升高并在16：00达到最大值，后逐渐下降。杨家坡枣林地和张国村荒草地的变化规律较为一致，但两地的最大值分别出现在22：00和20:00。

光合有效辐射与 $\mathrm { C O } _ { 2 }$ 通量关系密切。如范连连等[27]不仅发现二者的密切关系，还指出光合有效辐射对于 $\mathrm { C O } _ { 2 }$ 通量的影响主要是通过影响表层土壤的温度以及改变植被的光合作用的速率，即对光合作用的直接作用和气孔导度的间接作用进而影响群落碳截获。植被生理特性研究表明，光合作用的大小受光合有效辐射影响较大[28-29]

11月所代表的秋/冬季时期，光合有效辐射较春、夏季有着明显的下降， $\mathrm { C O } _ { 2 }$ 通量也有着明显的下降(图4c)。其中，张国村荒草地和张国村农耕地的光合有效辐射较为强烈，与 $\mathrm { C O } _ { 2 }$ 通量均呈微弱负相关。而杨家坡枣林地以及清凉寺农耕地的 $\mathrm { C O } _ { 2 }$ 通量与光合有效辐射呈微弱正相关。在非生长季，植物的枯死使得光合有效辐射与 $\mathrm { C O } _ { 2 }$ 通量的相关性变小。3.2.2温度对土壤表层 $\mathrm { C O } _ { 2 }$ 通量的影响由图5可知， $5 \mathrm { c m }$ 深度的土壤温度的变化是随着大气温度的变化而变化，但变化反应总是滞后于大气温度。当大气温度达到日尺度的峰值时(5月为16：00、8月为14：00)，土壤温度会在这之后达到日尺度的峰值。在生长季 $\mathrm { C O } _ { 2 }$ 通量是随着温度的变化而变化，但总滞后于温度的变化。

3.2.3 土壤粒径分布特征及其对 $\mathrm { C O } _ { 2 }$ 通量的影响晋西北黄土丘陵沟壑区的气候相对于关中平原、渭北旱塬等典型黄土区而言更为干旱少雨，其土壤质地更粗，除少数裸露的基岩、沙砾河漫滩外，普西北地区主要为砂质土壤类型，因此砂质类型的土壤质地在晋西北黄土区占主导地位，具有较好的典型性和代表性。研究区内主要分布黑泸土，黑泸土在我国西北黄土侵蚀地区广泛分布，是我国西北黄土地区的主要土类[33]。黑泸土分布区界限西至临夏、兰州一线,南至天水、宝鸡、韩城一线,东至太原、大同一线，北至兰州、靖边、大同一线。Zhu等34将黑泸土分为典型黑鲈土、黏黑鲈土、轻黑鲈土和草甸黑鲈土，加上黑麻土共5个亚类。研究区的黑泸土属黏黑坡土亚类。在土壤 $\mathrm { C O } _ { 2 }$ 通量研究中，土壤粒径分布特征对于土壤 $\mathrm { C O } _ { 2 }$ 通量影响的相关研究较少，也是被忽略的一环。不同质地的土壤，其土壤 $\mathrm { C O } _ { 2 }$ 通量有较为明显的差异，准确掌握不同质地下王壤$\mathrm { C O } _ { 2 }$ 通量变化特征对于高效利用土地以及控制碳排放具有重要指导意义，因此深入研究土壤质地对于碳通量的影响十分必要。本文将观测区YZ、QY、ZC、ZY采集的表层土壤样品使用激光粒度分析仪进行分析，将各观测区的样品粒径数据绘制成Shepard三角图(图6)。根据研究区4个样点耕作层土壤的粒度含量分布特征可知，各监测点的粒径水平相当，各剖面的粒度含量分布特征较为接近。

3.2.4土壤含水率与土壤有机质对于 $\mathrm { C O } _ { 2 }$ 通量的影响土壤含水率是影响表层土壤 $\mathrm { C O } _ { 2 }$ 通量的环境要素之一，对植被的生长、根系分布和微生物活性等与土壤 $\mathrm { C O } _ { 2 }$ 通量密切相关的生物因子起重要控制作用[38]。陈全胜等[19]发现对于 $\mathrm { C O } _ { 2 }$ 通量来说接近田间持水力是最佳水分状态，当土壤处于过干或过湿状态时，土壤呼吸就会受到抑制。

# 4结论

（1）研究区地表土壤 $\mathrm { C O } _ { 2 }$ 通量的昼夜变化规律特征为：在生长季呈现单峰变化，于16：00—22：00达到单日最高值，在日出左右达到单日最低值；在非生长季则呈现不规律变化。

(2) $\mathrm { C O } _ { 2 }$ 通量的时空特征为：农耕地的 $\mathrm { C O } _ { 2 }$ 通量与荒草地、枣林地相比要低,表明合理的土地利用措施对碳排放有着积极作用;同时，不同季节的 $\mathrm { C O } _ { 2 }$ 通量关系为：8月 $\mathrm { C O } _ { 2 }$ 通量 $^ { > 5 }$ 月 $\mathrm { C O } _ { 2 }$ 通量 $^ { \cdot > 1 1 }$ 月 $\mathrm { C O } _ { 2 }$ 通量，这与5月及8月研究区降水充足、气候温暖适宜、植被处于生长期，土壤根系和土壤微生物呼吸速率较快相关。

(3）光合有效辐射和大气温度影响着土壤温度，并且三者同 $\mathrm { C O } _ { 2 }$ 通量有显著正相关关系。土地利用类型以及含水率对于土壤质地也有不同影响，科学的土地利用类型会减小土壤孔隙度，进而提高土壤固碳能力。土壤高含水率对地表土壤 $\mathrm { C O } _ { 2 }$ 通量起抑制作用；有机质在生长季并非主导因素，在非生长季则占据主导位置，同 $\mathrm { C O } _ { 2 }$ 通量呈负相关关系。

# Study on the change of surface $\mathbf { C O } _ { 2 }$ flux in sandy loamy soil under different land use types: An example from the Qingliangsi ditch watershed

XU Zhiping'， SHAO Tianjie'， ZHANG Liankai²， SHAO Mingyu²， NIU Junjie³ (1.School of Geographical Science and Tourism,Shaanxi Normal University,Xi'an71O119,Shaanxi, China; 2.Institute of Karst Geology, Chinese Academy of Geological Sciences,Guilin 541004,Guangxi, China; 3.Research Center for Scientific Development ofFenhe River Vally,Taiyuan Normal University,Taiyuan 030012, Shanxi, China)

Abstract: Global climate change is closely related to the carbon cycle,and exploring changes in surface carbon emissions is an essential aspect of climate change research. In this study, the $\mathrm { C O } _ { 2 }$ fluxes of surface soils in Qingliangsigou watershed of Loess Hills and Gulles in northwest Shanxi were measured using an LI-8100A soil carbon flux automatic measurement system,the soil particle size was measured using a laser particle size analyzer (mastersizer 2Ooo),the soil organic mater was measured using a chemical titration method,and the soil water content was measured using the drying method. Additionally, we explored the diurnal and spatial variation of surface soil $\mathrm { C O } _ { 2 }$ fluxes, temporal and spatial changes, $\mathrm { C O } _ { 2 }$ fluxes under different land use types,and the influence of environmental factors on surface soil $\mathrm { C O } _ { 2 }$ fluxes to complement the study of surface soil $\mathrm { C O } _ { 2 }$ fluxes in small watershed areas in the Loess Hills and Gullies of northwest Shanxi.We found that $\mathrm { C O } _ { 2 }$ fluxes showed a single peak in variation during the growing season,reaching a single day minimum around sunrise and a single day maximum between l6:OO and 22:00,and in the non-growing season, there was iregular variation. The $\mathrm { C O } _ { 2 }$ fluxes of farmland were lower than the $\mathrm { C O } _ { 2 }$ fluxes of waste grassland and jujube forest land. The highest $\mathrm { C O } _ { 2 }$ fluxes were observed in August, followed by May,and the lowest $\mathrm { C O } _ { 2 }$ flux was in November. The $\mathrm { C O } _ { 2 }$ flux of surface soil was closely related to photosynthetic effective radiation,atmospheric temperature,soil temperature, soil water content,soil organic mater,and soil texture.Therefore,the carbon fixation of soil can be effectively improved through scientific land use.In conclusion,the results of this study can be applied to most of the Loess Plateau and used to improve the ecological environment.
