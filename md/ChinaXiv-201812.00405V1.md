山羊绒是一种细而柔软的动物纤维，其纺织品通常被美誉为"软黄金”。陕北白绒山羊是以辽宁白绒山羊为父本，陕北黑山羊为母本，采用杂交方式培育而成的以产绒为主、绒肉兼用型山羊新品种[1]，是三大国审品种之一，其产绒量、绒长度及细度等方面在我国均处于领先水平。绒山羊毛被生长是一个复杂的生理过程，受营养、环境及遗传因素的影响。在营养方面，微量元素锌是产绒必需的营养成分。因此，明确生绒期绒山羊饲粮适宜的锌水平对提高其产绒性能至关重要。锌对绒山羊产绒性能具有一定的影响，当饲粮严重缺乏锌时会影响硫和氮的吸收，从而严重影响羊毛的正常生长，主要表现角质化不全、羊毛生长速度减慢、掉毛及脱毛，甚至皮肤损伤等[2]。索宝[3研究认为，生绒期绒山羊每增重 $1 \mathrm { k g }$ 机体沉积98.04mg 锌，每生长 $1 \mathrm { k g }$ 被毛机体沉积 $1 1 0 . 8 7 \mathrm { m g }$ 锌。研究认为生绒期辽宁绒山羊饲粮锌供给量为 $8 6 ~ \mathrm { m g / k g ~ D M ^ { \left[ 4 - 5 \right] } }$ 。由此可见，在绒山羊饲粮中补锌非常重要。然而，目前舍饲条件下生绒期陕北白绒山羊对锌的需要量尚未确定。因此，本试验通过研究饲粮锌水平对生绒期绒山羊生长性能、产绒性能及绒生长相关激素含量的影响，旨在为陕北白绒山羊锌需求标准的制订及完善绒毛生长的营养调控理论提供科学依据。

# 1.2 饲养管理

试验羊分别在每天08:00及16:00定量饲喂2次，先喂粗料，后喂精料，自由饮水（饮水中未检测出锌）。

# 1.3样品采集与预处理

正试期的前1天，在试验羊的肩胛骨后缘 $1 0 \mathrm { c m } { \times } 1 0 \mathrm { c m }$ 面积上用染发剂染色。在试验结束时，剪下染色部位的羊毛及羊绒，测定产绒性能指标。分别于试验初期、试验中期及试验末期（其中每个试验期间隔为 $1 5 { \mathrm { d } } { \mathrm { } }$ ）对试验羊颈静脉采血 $2 0 ~ \mathrm { m L }$ ， $3 ~ 5 0 0 ~ \mathrm { r / m i n }$ 离心 $1 0 \mathrm { m i n }$ 取血清，置于 ${ } _ { - 2 0 } { } ^ { \circ } \mathrm { C }$ 保存。

1.4测定指标及方法

1.4.1 生长性能

生长性能包括干物质采食量（DMI）、平均日增重（ADG）及料重比（F/G）。试验羊  
分别于试验开始及结束时称体重，采用羊笼及磅秤称重，计算各组试验羊的ADG。正试期  
内，每次饲喂前准确记录试验羊的饲喂量及剩余量，计算试验羊DMI。计算公式如下：$\mathbf { A D G } =$ （试验末期体重一试验初期体重）/试验天数；$\mathrm { F / G } =$ 饲养期内所耗标准饲粮量/同期增重。

# 1.4.2 产绒性能

产绒性能指标包括羊绒细度、羊绒长度、羊绒生长量、羊绒生长率及产绒量。羊绒长度采用钢尺测量，测定100根。羊绒细度采用纤维细度测定仪测定。产绒量于5月初抓绒测定。

# 1.4.3 绒生长相关激素含量

血清胰岛素样生长因子-1（IGF-1）、褪黑激素（MT）、催乳素（PRL）、生长激素（GH)含量均利用放射免疫法测定，测定方法参照试剂盒说明书，试剂盒购于北京英华生物技术研究所。

# 1.5数据处理与分析

试验数据用 Excel2010进行初步统计，并采用 SPSS17.0 统计软件的ANOVA模型进行单因素方差分析，采用Duncan氏法进行多重比较检验。试验初期各组羊绒长度差异较大，因此试验末期各组羊绒长度采用协方差方法进行分析。极显著水平设为 $P { < } 0 . 0 1$ ，显著水平设为 $P { < } 0 . 0 5$ ，结果用平均值 $\pm$ 标准差表示。

3讨论

3.1饲粮锌水平对生绒期陕北白绒山羊生长性能的影响

锌作为营养性饲料添加剂，被誉为"生命元素”，是目前所发现的具有多种生理功能的微量元素之一[]。锌能够使口腔味蕾细胞迅速再生，增进动物食欲；同时，锌能提高消化道上皮细胞内各种消化酶的活性，增加食物在肠道中的存留时间，从而提高消化率及增加采食量[7]。NRC（2001）提出，动物缺锌会导致采食量及生长速率降低。动物机体本身没有贮存锌的能力，因此需要从饲粮中采食锌来维持机体的健康与营养物质平衡[6]。研究表明饲粮锌水平对生绒期内蒙古白绒山羊[3]及辽宁绒山羊[8]的日增重及饲料转化率有显著影响。然而，Aditia 等[9]研究认为饲粮锌水平对山羊ADG、DMI及饲料转化率等的影响均不显著。本试验中，饲粮锌水平对陕北白绒山羊DMI及F/G 的影响均不显著，但是对陕北白绒山羊 ADG有显著影响，IⅢ组ADG 最高且F/G 最低，表明组饲粮锌水平可促进陕北白绒山羊饲粮的消化利用效率，从而提高其生长性能。因此，从生长性能角度考虑，建议生绒期陕北白绒山羊饲粮适宜的锌水平为 $7 0 \mathrm { m g / k g  D M }$ 0

3.2饲粮锌水平对生绒期陕北白绒山羊产绒性能的影响

目前关于饲粮锌水平对山羊产绒性能影响的研究较少。研究发现，基础饲粮（锌水平为22.30 mg/kgDM）添加 $4 5 \mathrm { m g / k g  D M }$ 锌对辽宁绒山羊羊绒长度、羊绒生长率及羊绒细度没有显著影响[4.8]。Liu 等[5]在基础饲粮（ $4 5 . 9 \mathrm { m g / k g  D M }$ ）中分别添加0、20、40及 $8 0 \mathrm { m g / k g }$ DM锌后发现对羊绒生长率及羊绒细度没有产生显著影响。而韦月平等[10]研究发现，饲粮添加蛋氨酸锌可以提高绒毛纤维的密度及和直径，从而提高产绒量。目前研究者关于锌对羊毛生长影响的研究结果比较相似，认为锌可促进山羊羊毛的生长及产毛量[1I-12]。本试验中，饲粮锌水平虽然对羊绒细度及产绒量没有显著影响，但是Ⅲ组产绒量最高。此外，饲粮锌水平对试验末期羊绒长度具有显著影响，羊绒生长量及生长率Ⅲ组显著高于I组和IV组，可见饲粮锌水平对陕北白绒山羊羊绒生长具有营养调控作用。由上述结果可知，生绒期陕北白绒山羊饲粮锌水平在 $5 0 { \sim } 7 0 \mathrm { m g / k g D M }$ 较适宜，同时考虑到陕北白绒山羊生绒期即是繁殖期，为了满足其生绒及繁殖的营养需要，建议饲粮锌水平为 $7 0 \mathrm { m g / k g  D M }$ 。

3.3饲粮锌水平对生绒期陕北白绒山羊绒生长相关激素含量的影响   
锌与激素之间存在相互作用，锌不但对激素受体的效能和靶器官的反应性产生影响，而 且还在激素的生成、储存、分泌和代谢中起作用。反之，激素也影响锌的吸收和代谢[13]。 绒山羊的绒毛生长也随着光照周期性变化而变化，其毛囊也出现生长、退行、休止等周期性 现象。研究表明 MT、PRL、GH及IGF-I在调节绒山羊毛囊生长及毛发生长周期中均具有 重要作用[14-16]。目前已经明确与锌有关的激素有GH、PRL、胰岛素及IGF-1等[17-18]。研究 证实缺锌动物血清胰岛素含量下降，同时血清 IGF-1 含量也相应减少，补锌能够提高血清 GH及胰岛素含量[17.19]。本试验中，随着陕北白绒山羊生绒期的推进，其血清 MT、IGF-1 及 GH含量呈现增加趋势，而血清 PRL 含量则呈降低趋势。在整个试验期内陕北白绒山羊 血清 MT及PRL 含量各组之间差异均不显著，表明饲粮锌水平对血清MT及PRL 含量的影 响不显著。此外，饲粮锌水平对陕北白绒山羊血清 IGF-1及GH含量具有显著影响，表现为 试验中期ⅢI组血清IGF-1含量显著高于IV组，试验末期III组血清 IGF-1含量显著高于I组及 IV组，试验末期Ⅲ组血清GH含量显著高于I组；同时，羊绒生长量及羊绒生长率Ⅲ组显著 高于I组和IV组，表明饲粮锌可能通过促进IGF-1及GH分泌进而调控陕北白绒山羊羊绒生 长。   
4结论   
生绒期陕北白绒山羊饲粮适宜的锌水平为 $7 0 \mathrm { m g / k g }$ DM。
