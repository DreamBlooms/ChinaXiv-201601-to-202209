3讨论

3.1东北玉米特性分析

试验1对20种东北玉米进行了常规养分分析，并分析得出东北玉米的特性。试验结果表明，不同东北玉米之间，EE、NDF及ADF 的含量差异相对较大，其他养分含量的差异相对较小，与李全丰[1对我国玉米养分含量变异看法类似。本试验测定的 20 个东北玉米的CF 含量平均值为2.78%，高于赵养涛[2测定的17个玉米样品的平均值 $1 . 7 0 \%$ ，但 STC 含量在本试验中测得的平均值为 $6 8 . 6 4 \%$ ，低于赵养涛[2]的 $7 3 . 2 1 \%$ 。这说明采集的样品养分含量与品种、年份等种植条件有关，采用人工玉米来进行模拟研究时，可在更大范围内准确评价其生物学有效性。

# 3.2日龄对养分表观消化率和代谢能的影响

本研究表明， $2 5 { \sim } 2 7$ 日龄肉鸡对同种人工玉米代谢饲粮的CP、EE、STC 表观消化率及AME、AMEn均高于 $1 1 \sim 1 3$ 日龄的相应指标。Kato 等[1]采用替代法测定了 $1 { \sim } 7$ 、 $8 \sim 1 4$ 、 $1 5 { \sim } 2 1$ 、 $2 2 \sim$

$2 8 . 2 9 \sim 3 5$ 和 $3 6 { \sim } 4 2$ 日龄肉鸡玉米代谢能，发现 $1 { \sim } 7$ 日龄肉鸡的代谢能显著低于其他日龄阶段，$2 2 { \sim } 2 8$ 日龄肉鸡玉米 AME（16.29 MJ/kg DM)显著高于 $1 4 { \sim } 2 1$ 日龄肉鸡玉米 AME（16.10 MJ/kgDM)。Huang 等[12]认为 28 日龄肉鸡对玉米氨基酸的消化率显著高于14 日龄，28日龄与42 日龄相似。De Lima 等[13]认为肉鸡对玉米蛋白粉的代谢能在 $1 0 { \sim } 4 0$ 日龄间按照 $3 . 1 1 \mathrm { k J / ( k g } \cdot \mathrm { d } )$ 逐渐升高，说明肉鸡不同生长阶段饲粮代谢能应不同。
