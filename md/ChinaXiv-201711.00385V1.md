补喂支链氨基酸对速步马 $1 \mathrm { k m }$ 速步赛成绩及赛前、赛后血浆生化指标的影响李晓斌 马军聂彪彪 杨景焘 乔春江 杨开伦\*（新疆农业大学动物科学学院，新疆肉乳用草食动物营养重点实验室，乌鲁木齐830052）

摘要：本试验研究补喂支链氨基酸对速步马 $1 \mathrm { k m }$ 速步赛成绩及赛前、赛后血浆抗氧化指标以及激素、肌酸、葡萄糖、乳酸、肌酐含量的影响，为支链氨基酸在速步马训练、比赛等的应用提供参考数据。试验选取年龄相近（4 岁左右)、体重相近[（457±50）kg]并经过严格训练的伊犁马公马8匹（速步赛用马)，随机分为2组，分别为对照组、试验组，每组4匹。每天每匹马分别饲喂 $3 ~ \mathrm { k g }$ 颗粒精料，苜蓿干草自由采食，在此基础上试验组每天每匹马补喂 $\boldsymbol { 7 2 } \mathrm { g }$ 支链氨基酸（由 $3 5 \mathrm { g }$ 亮氨酸、 $1 6 . 6 \ \mathrm { g }$ 异亮氨基酸、 $2 0 . 4 ~ \mathrm { g }$ 缬氨酸组成)，进行为期38d（预试期7d，正试期31d）的补饲试验及训练试验。结果表明：补喂支链氨基酸可提高速步马的比赛成绩，同时显著提高速步马赛后 $3 0 \mathrm { m i n }$ 血浆总抗氧化能力及赛后 $2 4 \mathrm { h }$ 血浆超氧化物歧化酶活力（ $P { < } 0 . 0 5 )$ ，但对血浆中激素、肌酸、葡萄糖、乳酸、肌酐含量无显著影响 ( $. P { > } 0 . 0 5$ ）。由此得出，补喂支链氨基酸可缩短速步马比赛用时，提高速步马机体的抗氧化能力，但对血浆中激素、葡萄糖、肌酸、乳酸、肌酐含量无显著影响。

5 关键词：支链氨基酸；伊犁马；运动性能；血浆生化指标

支链氨基酸（branched-chain amino acid,BCAA）由亮氨酸、异亮氨酸、氨酸组成，不仅可作为机体的能源物质，进入机体后参与肌肉细胞的合成代谢，为肌肉的收缩储备能量，还可参与机体代谢，抑制机体氧化应激、调节激素水平[I]。Chang等[2]研究证实BCAA可提高

人的肌肉力量，从而提高运动性能。王良峡等[3认为BCAA可参与供能与糖代谢，具有调节体内蛋白质代谢、抗运动性疲劳、清除自由基、保护线粒体等作用。伊犁马主要产于新疆伊犁哈萨克自治州，具有体型外貌基本一致、遗传性能稳定、力速兼备、繁殖性能好、耐粗饲、抗病能力强等特点，是良好的速步马品种[4]，经测定，伊犁马速力测试成绩 $5 \mathrm { k m }$ 用时6min 35s，但与国外一流速步马相比，伊犁马仍存在速步性能差、速步训练中易疲乏、恢复慢等问题，严重制约了伊犁马良好运动性能的发挥。因此，本试验基于BCAA对动物机体供能、抗氧化及有益于运动后机体恢复的研究，以伊犁马速步马为试验动物，通过饲喂试验探究BCAA对速步马运动性能、机体抗氧化能力及血浆激素含量的影响，为改善速步马运动性能提供参考依据。

在正试期第31天进行速步赛，分别于在赛前 $^ { \mathrm { ~ 1 ~ h ~ } }$ 、赛后 $3 0 ~ \mathrm { m i n }$ 、赛后 $2 4 \mathrm { ~ h ~ }$ 在速步马颈静脉处采集血样 $5 \mathrm { m L }$ 至肝素钠采血管中， $1 5 0 0 \times g$ 离心 $1 5 \mathrm { m i n }$ ，制取血浆并-20℃冷冻保存备用。

# 1.4测定指标及方法

血浆中过氧化氢酶(catalase,CAT）、谷胱甘肽过氧化物酶(glutathion peroxidase,GSH-Px）、超氧化物歧化酶（superoxidedismutase,SOD）活力，总抗氧化能力（total antioxidantcapacity,T-AOC）以及丙二醛（malondialdehyde,MDA）、胰岛素（insulin,INS）、胰高血糖素（glucagon,Gc）、皮质醇（cortisol,COR）、葡萄糖（glucose,Glu）、乳酸（lactic acid,LA）、肌酐（creatinine,Cre）含量使用南京建成生物工程研究所生产的试剂盒测定；血浆中肌酸（creatine）含量使用美国 Sigma 公司生产的试剂盒测定。

# 1.5 数据处理

试验结果均以平均值 $\pm$ 标准差表示。试验数据采用SPSS16.0软件中的ANOVA程序进行单因素方差分析，差异显著者采用Duncan 氏法进行多重比较检验。 $P { > } 0 . 0 5$ 为差异不显著， $P { < } 0 . 0 5$ 为差异显著， $P { < } 0 . 0 1$ 为差异极显著。

3讨论

3.1补喂BCAA对速步马运动性能的影响

随着对BCAA的深入研究发现，BCAA不仅影响机体内蛋白质的合成、分解，还可作为  
能量底物、糖异生底物，调节肌肉中蛋白质的代谢。Kim等[5研究表明在进行耐力运动前给  
运动员补充BCAA可减少肌肉损伤、提高耐力运动的持续时间、降低疲劳反应。Watson等[6  
研究表明给运动员补充12gBCAA可提高部分运动员的比赛成绩。在本试验中，补喂BCAA可提高速步马 $1 ~ \mathrm { k m }$ 速步赛速度，这是由于BCAA是一类重  
要的，也是特殊的分解供能氨基酸。在肌肉中分解代谢旺盛，1分子的亮氨酸、异亮氨酸、  
缬氨酸完全氧化后，分别产生42、43、32分子的ATP[7]。试验组速步马连续补喂了30d的  
BCAA，BCAA可在比赛中发挥其快速供能的作用，以及速步马在补喂BCAA 的基础上进  
行有效的速步训练，从而提高了比赛成绩。成年马正常的心跳范围为 $3 0 { \sim } 4 2$ bpm[8]。Gehlen 等[9]报道，纯血马在训练前静息状态  
下平均心率为 $3 1 . 7 \mathrm { b p m }$ 。Colahan等[1报道，马匹备鞍后，骑手骑在马上测定的马的平均心

率(38±5) bpm，当以 $( 5 4 8 { \pm } 9 0 ) \mathrm { m / m i n }$ 的平均速度奔跑时心率达到了 $1 8 4 . 2 3 \mathrm { b p m }$ 。在本试验中，各组马匹赛前心率略高于前人报道，这可能与速步马备鞍有关。赛后即刻和赛后 $3 0 \mathrm { m i n }$ ，试验组心率均低于对照组，表明补喂 BCAA不仅能够提高速步马的运动速度，还可降低比赛时心率。因此，补喂 BCAA 能够提高速步马比赛速度、降低赛时心率，同时有利于速步马赛后心率的恢复。

3.2补喂BCAA对速步马 $1 \mathrm { k m }$ 速步赛赛前、赛后血浆中抗氧化指标的影响  
速步马在高强度运动过程中机体内会发生氧化反应，对机体组织造成一些不可逆的损伤，导致运动机能减弱，严重影响速步马的运动性能[1]。研究表明，BCAA 在机体抗氧化过程中发挥明显的作用。BCAA可通过机体代谢经血液循环进入到组织细胞，阻断自由基在机体细胞膜上的链式反应，并清除机体运动或损伤后产生的过多自由基，减少脂质过氧化反应终产物MDA的产生[12]，同时提高机体内 SOD 和GSH-Px 等抗氧化酶的活力。  
本试验中，补喂BCAA可提高速步马赛后 $3 0 \mathrm { m i n }$ 血浆中SOD活力及T-AOC，提高赛后24h血浆中CAT、GSH-PX、SOD活力及T-AOC，表明BCAA在速步马体内发挥了一定的抗氧化作用。SOD、GSH-Px属于蛋白质酶，其合成代谢过程中的翻译或转录均会受到氨基酸浓度的影响[13]。部分BCAA还可通过血液循环进入组织细胞，有利于清除自由基，阻断脂质被进一步氧化，保护机体免受损伤[14]。在本试验中，连续30d补喂BCAA可提高速步马血浆中SOD、GSH-Px活力。

3.3补喂BCAA对速步马 $1 \mathrm { k m }$ 速步赛赛前、赛后血浆中激素含量的影响动物体内激素调控受多种因素影响，如运动会导致血浆中 INS、Gc 和COR 含量的变化，且与糖代谢供能有直接的关系。Xiao 等[15]报道，训练可提高运动员训练后血浆中COR含量，降低 INS 含量。Davies 等[16研究不同运动强度下血浆COR 含量的变化，当运动强度低于 $5 0 \% \mathrm { V _ { O 2 } m a x }$ 运动时，会引起血浆COR含量的下降，而当以运动强度大于60%Vozmax运动时，血浆COR的含量增加。

在本试验中，赛后 $3 0 \mathrm { m i n }$ 及赛后 $2 4 \mathrm { h }$ 对照组与试验组血浆中INS含量与赛前1h相比均有所升高，是由于 INS 是加强糖异生作用的主要激素，速步马在比赛中需要大量糖原，肝糖元异生作用也随之加强，则使INS分泌增强。在血浆Gc含量方面，各组赛后 $2 4 \mathrm { h }$ 血浆中 Gc 含量均高于赛前1h及赛后 $3 0 \mathrm { m i n }$ ，这可能是由于赛后 $2 4 \mathrm { h }$ 为速步马赛后恢复期，Gc含量的增加有利于机体内Glu含量的恢复。血浆中COR含量受运动强度的影响，且高强度运动后COR含量会持续降低，直到肾上腺皮质分泌大量的COR后才得以恢复[17]。本试验中各组速步马赛后 $3 0 ~ \mathrm { m i n }$ 及赛后 $2 4 \mathrm { h }$ 血浆中COR含量持续降低，与Davies 等[16]的研究结果一致。

3.4补喂BCAA对速步马 $1 \mathrm { k m }$ 速步赛赛前、赛后血浆中肌酸、Glu含量的影响  
Glu 和肌酸是动物体内直接和间接的能源物质，在运动中对能量的供应至关重要。动物体内肌酸主要在肝脏和肾脏中合成，广泛分布于骨骼肌、心肌、大脑等组织中。肌酸主要以甘氨酸为骨架，精氨酸提供胱基，在胱基转移酶和甲基转移酶的催化下合成。血浆中肌酸含量受到动物品种、饲粮中氨基酸组成及生理状态有关。血浆中的Glu可通过饲粮中脂肪、蛋白质转化而来。Sewell等[18]报道，在静息状态下纯血马血浆中肌酸含量为 $8 { \sim } 1 0 3 ~ \mathrm { \textmu m o l / L }$ 0Falavigna等[9]以成年大鼠为研究对象，探究补喂BCAA对成年大鼠的影响，结果表明补喂BCAA后，大鼠运动前血浆中Glu含量显著高于运动后。  
在本试验中，对照组赛前1h、赛后 $3 0 \mathrm { m i n }$ 、赛后24h血浆中肌酸含量无显著变化，而试验组赛后24h血浆中肌酸含量较赛前 $1 \mathrm { h }$ 、赛后 $3 0 \mathrm { m i n }$ 有所增加，表明补喂BCAA可提高血浆中肌酸含量，其原因可能是BCAA可调节动物体内蛋白质合成过程[3]，对肌酸的合成有积极作用。试验组赛前1h、赛后 $3 0 \mathrm { m i n }$ 、赛后24h血浆中Glu含量均高于对照组，这是由于补喂的BCAA具有氧化功能，特别是长时间的耐力运动会使BCAA在肌肉中的氧化增加而参与供能[20]，从而使血浆中Glu含量增加。

3.5补喂BCAA对速步马 $1 \mathrm { k m }$ 速步赛赛前、赛后血浆中LA、Cre含量的影响

在剧烈运动时，速步马体内会进行无氧代谢，产生大量LA，使呈弱碱性的体液变为  
酸性，影响细胞顺利吸收营养和氧气，削弱细胞的正常功能，严重影响速步马的运动性能[21]。  
Falavigna 等[19]给成年大鼠长期补喂不同剂量的 BCAA，测定训练后1h 和力竭时血浆中LA  
的含量，结果表明补喂 BCAA可显著降低血浆中LA 含量。刘建红等[22]证实给划船运动员  
补充 BCAA可显著降低血浆中LA含量。在本试验中，试验组马匹赛后 $3 0 \mathrm { m i n }$ 及赛后24h血浆中LA含量均低于对照组，表明补喂  
BCAA可减少速步马运动后血液中LA的堆积，降低机体的疲劳，可能与补充BCAA可使骨骼  
肌乳酸脱氢酶活力接近正常组水平，从而减少骨骼肌中LA的积累和向血清的转移，减少血  
浆中LA的含量有关[14]。Cre是肌酸和磷酸肌酸代谢的终产物，它主要由肌肉中磷酸肌酸的非  
酶促反应生成，Cre的生成影响机体的运动能力。在本试验中，补喂BCAA对速步马赛前1h、  
赛后 $3 0 \mathrm { m i n }$ 、赛后24h血浆中Cre含量无显著影响，这可能与速步马的生理状态有关，有待于  
进一步研究。  
4结论在试验条件下，补喂BCAA可缩短速步马比赛用时，提高速步马机体的抗氧化能力，  
但对血浆中激素、葡萄糖、肌酸以及乳酸、肌酐含量无显著影响。
