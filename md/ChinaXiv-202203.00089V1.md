# 准噶尔东部地区3种土壤类型 $^ { 1 3 7 } \mathbf { C } \mathbf { s }$ 与土壤颗粒的关系

杨建军'，王和平¹，王明力²，关添泽¹(1.新疆大学资源与环境科学学院,新疆 乌鲁木齐 830046；2.核工业二一六大队,新疆 乌鲁木齐 830011)

摘要：为更加深入地了解土壤颗粒对 $^ { 1 3 7 } \mathrm { C s }$ 分布的影响,选取新疆准噶尔东部地区沙土、灰棕漠土和灰漠土3种主要的土壤类型,探讨不同土壤类型及自然地理环境条件下 $^ { 1 3 7 } \mathrm { C s }$ 与土壤颗粒的关系。结果表明：沙土、灰棕漠土和灰漠土均以 $> 5 0 ~ \mu \mathrm { m }$ 的颗粒占据主导地位,其占比介于 $5 1 . 3 5 \% { \sim } 9 0 . 2 2 \%$ ,中值粒径(D50)表现出风沙土 $>$ 灰棕漠土 $>$ 灰漠土的趋势;灰棕漠土 $^ { 1 3 7 } \mathrm { C s }$ 质量活度变化范围为 $0 . 3 2 { \sim } 1 5 . 7 0 \ \mathrm { B q \cdot k g ^ { - 1 } }$ ,其土壤剖面分布与 $< 2 ~ { \mu \mathrm { m } }$ 和 $2 { \sim } 5 ~ \mu \mathrm { m }$ 的土壤颗粒具有较强的一致性;风沙土 $^ { 1 3 7 } \mathrm { C s }$ 比活度的分布分别与 $> 5 0 ~ \mu \mathrm { m }$ 土壤颗粒( $( 0 { \sim } 1 4 ~ \mathrm { c m } )$ 含量及D50剖面(0\~12cm)分布趋于一致，而灰漠土 $^ { 1 3 7 } \mathrm { C s }$ 比活度与各粒级土壤颗粒剖面并未表现出相同的变化趋势;风沙土和灰漠土中，对 $^ { 1 3 7 } \mathrm { C s }$ 质量活度影响最大的均为 $2 { \sim } 5 ~ \mu \mathrm { m }$ 粒级的土壤颗粒，而灰棕漠土中 $^ { 1 3 7 } \mathrm { C s }$ 受 $1 0 { \sim } 5 0 ~ \mu \mathrm { m }$ 粒级土壤颗粒影响最大。准噶尔东部地区3种土壤类型 $^ { 1 3 7 } \mathrm { C s }$ 均与土壤颗粒存在密切关系，通过土壤剖面不同粒级土壤颗粒含量预测不同剖面 $^ { 1 3 7 } \mathrm { C s }$ 质量活度具有一定参考价值。

关键词： $^ { 1 3 7 } \mathrm { C s }$ ；土壤颗粒；土壤类型；准噶尔盆地东部地区

过去几十年来, $^ { 1 3 7 } \mathrm { C s }$ 在全世界成功地用作土壤侵蚀评估的示踪剂[1-3]。在应用 $^ { 1 3 7 } \mathrm { C s }$ 作为估算土壤侵蚀或沉积速率的示踪剂时，充分了解 $^ { 1 3 7 } \mathrm { C s }$ 在不同土壤粒度组分之间的分布非常重要[4-5]。土壤颗粒作为 $^ { 1 3 7 } \mathrm { C s }$ 的吸附体和再分配的载体，对 $^ { 1 3 7 } \mathrm { C s }$ 含量及其再分配过程起到决定作用，同时 $^ { 1 3 7 } \mathrm { C s }$ 随土壤颗粒在土壤侵蚀过程中存在的分选作用也对研究结果存在潜在影响[7-8]。但迄今为止，有关 $^ { 1 3 7 } \mathrm { C s }$ 含量与土壤颗粒之间关系的研究还较为薄弱，一些研究结果也存在着不同的差异[9],需要更多的学者投入相关研究以推动 $^ { 1 3 7 } \mathrm { C s }$ 示踪技术的进一步向前发展。

有关学者发现，棕壤和黄棕壤类型土壤 $^ { 1 3 7 } \mathrm { C s }$ 浓度随粒径减小而增加，并与中值粒径(D50)之间存在极好的负指数关系[10]。其中 $< 2 ~ \mu \mathrm { m }$ 和 $2 { \sim } 5 ~ \mu \mathrm { m }$ 粒级土壤颗粒对 $^ { 1 3 7 } \mathrm { C s }$ 有着较强的吸附能力，而5\~10$\mu \mathrm { m }$ 和 $1 0 { \sim } 5 0 ~ \mu \mathrm { m }$ 粒级的土壤颗粒含量与 $^ { 1 3 7 } \mathrm { C s }$ 质量活度之间存在着明显的负相关[]。此外,许多研究表明 $^ { 1 3 7 } \mathrm { C s }$ 活度与黏粒和胶体的含量成正相关关系，黏粒对于 $^ { 1 3 7 } \mathrm { C s }$ 在土壤中的吸附和固定有着十分重要的作用[12-15]。但是,一些研究也得出了不同的结论，比如砂质土中 $0 . 0 1 { \sim } 0 . 1 \ \mathrm { m m }$ 粒级对 $^ { 1 3 7 } \mathrm { C s }$ 含量的影响最大，而壤质土中 $^ { 1 3 7 } \mathrm { C s }$ 活度主要与粗粉砂1 $0 . 0 1 { \sim } 0 . 0 5 ~ \mathrm { m m }$ )含量有关，与 $< 0 . 0 1 \ \mathrm { m m }$ 的物理性黏粒含量无关[，且黄土高原区的土壤颗粒与 $^ { 1 3 7 } \mathrm { C s }$ 含量在土壤剖面分布趋势上并不具有一致性[]。此外,对红壤丘陵区的研究表明， $^ { 1 3 7 } \mathrm { C s }$ 活度在不同土层均与砂粒含量显著正相关，而与其他粒级土壤颗粒含量的关系并不显著[17]。以上结果表明,不同区域$^ { 1 3 7 } \mathrm { C s }$ 与土壤颗粒的关系存在一定差异,进一步探讨土壤颗粒与 $^ { 1 3 7 } \mathrm { C s }$ 的关系有助于放射性核素在水土流失示踪研究中的应用。

水土流失指在水力、重力、风力等外营力的作用下水土资源和土地生产力的破坏和损失，准噶尔盆地东部地区降雨量低，风速大，植被覆盖度低，风力侵蚀严峻,土壤肥力差[18]。本研究选取准噶尔东部地区主要土壤类型，探究不同粒级土壤颗粒对$^ { 1 3 7 } \mathrm { C s }$ 的吸附作用,以期更加深入地了解不同土壤类型及自然地理环境条件下 $^ { 1 3 7 } \mathrm { C s }$ 吸附的粒度效应，并探讨土壤颗粒对 $^ { 1 3 7 } \mathrm { C s }$ 分布的影响，从而为准噶尔东部地区土壤侵蚀定量模型提供必要的参数依据。

# 1材料与方法

# 1.1 研究区概况

准噶尔东部地区是新疆准噶尔盆地东部的彩南至北塔山的狭长地带，海拔高度 $8 0 0 { \sim } 1 4 7 3 \ \mathrm { m }$ ，地理位置为 $8 8 ^ { \circ } 1 0 ^ { \prime } { \sim } 9 1 ^ { \circ } 1 0 ^ { \prime } \mathrm { E } , 4 3 ^ { \circ } 3 0 ^ { \prime } { \sim } 4 5 ^ { \circ } 0 0 ^ { \prime } \mathrm { N }$ ，总面积约 $2 . 2 3 \times 1 0 ^ { 4 } ~ \mathrm { k m } ^ { 2 }$ ,属典型的极端干旱大陆气候,多年平均降水量为 $1 8 3 . 5 ~ \mathrm { m m }$ ,多年平均蒸发量为2042.3$\mathbf { m } \mathbf { m }$ ，四季多风，最多大风日数45d，风力一般在3\~5级,多年平均风速在 $2 . 7 { \sim } 3 . 7 \mathrm { m } { \cdot } \mathrm { s } ^ { - 1 }$ ,最大风速 $1 6 \mathrm { { m } \cdot \mathrm { { s } ^ { - 1 } } }$ ，属强度风力侵蚀，是研究干旱区土壤侵蚀的代表性地区。土地利用类型以沙地和裸地为主，也有少量的耕地和草地，植被覆盖度较低 $( 3 \% { \sim } 5 \% )$ )，有机质积累量少，是典型的荒漠地带，是新疆水土流失重点治理的区域。

# 1.2样品采集与测定

样品采集时间为2017年5—8月，实测点位分布较为均匀，基本能够覆盖研究区。根据研究区景观和土壤类型的分布特点，共选择了27块具有代表性的样地;同时，为了减少空间异质性，保证代表性样地的坡向、坡度等小地形条件类似，从中选取风沙土、灰棕漠土和灰漠土的代表样地分别为6块、5块和3块(表1)。每块样地面积 $2 0 \mathrm { m } { \times } 3 0 \mathrm { m }$ ,在各样地内按照"品"字形设置样点3个，首先用铁铲去除地表砾石、植物残体等杂物，再以 $2 \mathrm { c m }$ 为间隔在 $0 \sim$ $2 0 ~ \mathrm { c m }$ 垂直分层采样，有研究表明 $^ { 1 3 7 } \mathrm { C s }$ 基本分布于

1.2.1 土壤样品 $^ { 1 3 7 } \mathrm { C s }$ 的测定采集的土壤样品在实验室内自然风干后，经进一步去杂质、研磨过2$\mathbf { m } \mathbf { m }$ 筛后，取 $1 0 0 0 \ \mathrm { g }$ 封装保存待测。 $^ { 1 3 7 } \mathrm { C s }$ 活度在核工业北京地质研究院利用GMX-20A-Plus高纯锗y能谱仪测定，采用配备N型高纯锗探头的低能量、低本底y能谱仪，根据 $6 6 1 . 6 \mathrm { K e V }$ 谱峰面积测算样品 $^ { 1 3 7 } \mathrm { C s }$ 含量。样品测重 ${ \tt > } 4 0 0 \mathrm { \ g }$ ，测定时间 ${ \geqslant } 5 0 0 0 0 { \mathrm { ~ s ~ } }$ ，测试误差为 $\pm 5 \% ( 9 5 \%$ 信度）。

# 3讨论

土壤颗粒组成是土壤最基本的要素，土壤粒度代表了土壤中矿物颗粒直径组合状况，可被用来表示土壤质地状况、理化与结构等性质，其组成会影响土壤抗侵蚀与水土保持能力，美制标准将土壤颗粒划分为砂粒！ $( 0 . 0 5 { \sim } 2 . 0 \ \mathrm { m m }$ ）、粉粒（0.002\~0.05$\mathbf { m } \mathbf { m }$ )和黏粒 $( < 0 . 0 0 2 \mathrm { m m } ) ^ { [ }$ 24]

$^ { 1 3 7 } \mathrm { C s }$ 作为以其独特的性质,作为研究土壤侵蚀良好的示踪剂，与传统方法比具有成本低、见效快及精度高已成为共识[25]。陈思宇等[26]研究发现 $^ { 1 3 7 } \mathrm { C s }$ 质量活度与土壤黏粒含量正相关性，黏粒土壤颗粒越细、表面积大，其所含负电荷能吸持阳离子而使其不被淋失，因此 $^ { 1 3 7 } \mathrm { C s }$ 更易吸附在粒级较小的黏粒上,且使得 $^ { 1 3 7 } \mathrm { C s }$ 在土壤中很难迁移和被置换。一般认为 $^ { 1 3 7 } \mathrm { C _ { s } }$ 随土壤加深剧烈减少，张婷婷等[27]认为土壤中 $5 0 \% { \sim } 8 0 \%$ 的 $^ { 1 3 7 } \mathrm { C s }$ 含量集中在表层土壤 $1 0 \ \mathrm { c m }$ 以内，本文中灰棕漠土与上述研究结果一致，风沙土 $< 1 0 ~ \mu \mathrm { m }$ 的土壤颗粒在 $0 { \sim } 1 0 ~ \mathrm { c m }$ 的含量基本为零，而 $> 5 0 ~ \mu \mathrm { m }$ 和中值粒径D50与 $^ { 1 3 7 } \mathrm { C s }$ 变化趋势较一致，可能由于灰棕漠土代表样地表面粗糙，未受侵蚀扰动，而风沙土受风力侵蚀影响较大，表现为侵蚀剖面[28]。灰漠土 $^ { 1 3 7 } \mathrm { C s }$ 随剖面加深呈递减趋势，但与各土壤颗粒无显著相似变化趋势， $^ { 1 3 7 } \mathrm { C s }$ 与 $< 2$ $\mu \mathrm { m } \ 、 2 \sim 5 \ \mu \mathrm { m }$ 和 $5 { \sim } 1 0 ~ \mu \mathrm { m }$ 土壤颗粒含量负相关， $^ { 1 3 7 } \mathrm { C s }$ 质量活度在土壤剖面 $0 { \sim } 6 ~ \mathrm { c m }$ 急剧下降为零，且表层土壤 $< 5 0 ~ \mu \mathrm { m }$ 土壤颗粒含量非常少，可能由于所在样地受到强力的风蚀作用，具体原因尚不明确。有学者[29]认为放射性核元素在土壤中的迁移、转化和固定不仅与土壤黏土矿物的种类和含量有关，且在很大程度上也跟土壤腐植酸含量有关。另外，李勇等[30]认为 $^ { 1 3 7 } \mathrm { C s }$ 比活度测量值与土壤样品中O的化学组成呈正相关，与Na的含量呈负相关。

本文研究了新疆准噶尔东部地区3种土壤类型各粒级土壤和 $^ { 1 3 7 } \mathrm { C s }$ 的关系，明确了与 $^ { 1 3 7 } \mathrm { C s }$ 存在紧密关系的土壤颗粒，并获得了具有一定精度的回归模型。需要指出的是，本研究虽明确了不同颗粒与$^ { 1 3 7 } \mathrm { C s }$ 的关系,而没有获取各粒级土壤颗粒的 $^ { 1 3 7 } \mathrm { C s }$ 含量、黏土矿物组成、有机质等信息，这是后继研究中需要解决的问题，以进一步阐明不同粒级颗粒与$^ { 1 3 7 } \mathrm { C s }$ 之间关系的内在机制。

# 4结论

通过对新疆准噶尔东部地区3种土壤类型各粒级的土壤颗粒含量和 $^ { 1 3 7 } \mathrm { C s }$ 的关系探究表明，灰棕漠土 $^ { 1 3 7 } \mathrm { C s }$ 质量活度的深度分布趋势与 $< 2 ~ \mu \mathrm { m }$ 和2\~5$\mu \mathrm { m }$ 的土壤颗粒含量具有较强的一致性，风沙土 $^ { 1 3 7 } \mathrm { C s }$ 质量活度则与 $> 5 0 ~ \mu \mathrm { m }$ 的土壤颗粒含量和D50分别在 $0 { \sim } 1 4 ~ \mathrm { c m }$ 和 $0 { \sim } 1 2 ~ \mathrm { c m }$ 范围内具有一致性。灰漠土$^ { 1 3 7 } \mathrm { C s }$ 质量活度与各粒级土壤颗粒含量并未表现出相同的变化趋势。风沙土和灰漠土的 $^ { 1 3 7 } \mathrm { C s }$ 质量活度均与 $< 5 0 ~ \mu \mathrm { m }$ 的各粒级土壤颗粒含量呈负相关，与$> 5 0 ~ \mu \mathrm { m }$ 的土壤颗粒含量呈正相关，而灰棕漠土与之相反。对 $^ { 1 3 7 } \mathrm { C s }$ 质量活度影响最大的颗粒粒级风沙土和灰漠土均为 $2 { \sim } 5 ~ \mu \mathrm { m }$ ，灰棕漠土为10\~50$\mu \mathrm { m }$ 。在一定程度上，可以通过土壤颗粒含量来预测$^ { 1 3 7 } \mathrm { C s }$ 质量活度。

# Relationship between $^ { 1 3 7 } \mathbf { C 5 }$ and soil particles in the main soil types in the eastern Jungar Basin

YANG Jianjun'， WANG Heping'， WANG Mingli²， GUAN Tianze' (1. College of Resources and Environment Science,Xinjiang University,Urumqi 83o046,Xinjiang, China; 2. Geology Party No.216, CNNC, Urumqi 830011, Xinjiang, China)

Abstract:We selected three main soil types in the the eastern Jungar Basin,and explored the relationship between $^ { 1 3 7 } \mathrm { C s }$ and soil particles and natural geographic environment conditions to further understand the influence of soil particles on the distribution of $^ { 1 3 7 } \mathrm { C s }$ . The three soil types were dominated by soil particles $> 5 0 ~ \mu \mathrm { m }$ accounting for $5 1 . 3 5 \% - 9 0 . 2 2 \%$ of the three soil types. The median particle size (D50) of aeolian sandy soil was greater than that of gray-brown desert soil, which was greater than that of gray desert soil. The range of $^ { 1 3 7 } \mathrm { C s }$ mass activity in gray-brown desert soil was $0 . 3 2 - 1 5 . 7 0 \mathrm { \ B q \cdot k g ^ { - 1 } }$ ，The distribution of $^ { 1 3 7 } \mathrm { C s }$ specific activity in aeolian sandy soil is consistent with the particle content of soil ${ > } 5 0 ~ \mu \mathrm { m }$ and the distribution of D50 profile,but the specific activity of $^ { 1 3 7 } \mathrm { C s }$ in gray desert soil followed a different trend from that of soil particles of various sizes.Among aeolian sand and gray desert soils,those with the greatest impact on the $^ { 1 3 7 } \mathrm { C s }$ mass activity were $2 -$ $5 ~ { \mu \mathrm { m } }$ soil particles,whereas $^ { 1 3 7 } \mathrm { C s }$ in the gray-brown desert soil were most affected by $1 0 - 5 0 ~ { \mu \mathrm { m } }$ soil particles. The content of soil particles in diffrent grain fractions of the soil profile has certain reference value for predicting the mass activity of $^ { 1 3 7 } \mathrm { C s }$ in different profiles.

Keywords: $^ { 1 3 7 } \mathrm { C s }$ ；soil particle; soil type； the eastern Jungar Basin
