甜菜碱，又称为三甲基甘氨酸，是动物体内天然存在的无毒物质，最早从甜菜糖蜜中分离出来。甜菜碱是高效的甲基供体，参与机体重要的生理调节过程[1]。大量研究表明，甜菜碱具有参与蛋白质和脂肪代谢[2-3]，调节动物体正常渗透压等功能[4-5]。在单胃动物生产中，甜菜碱可以作为饲料添加剂用于动物生长发育、提高动物采食量、降低料肉比[6-9]。但是,在反刍动物生产中直接饲喂甜菜碱会被瘤胃微生物降解成三甲胺[10]，导致其添加效果不理想，因此反刍动物生产中多采用过瘤胃甜菜碱(rumen-protected betaine,RPB)的形式进行添加。樊晓京等[1]和连红等[12]研究发现，饲粮中添加过瘤胃甜菜碱能够有效提高奶牛的泌乳性能。Wang 等[13]研究表明，甜菜碱能够提高泌乳牛对饲粮中干物质（DM)、粗蛋白质（CP）、中性洗涤纤维（NDF）和酸性洗涤纤维（ADF）的消化率。此外Loest 等[14]在肉牛饲粮中添加甜菜碱，可以提高肉牛的采食量,提高日增重,降低料肉比。目前围绕过瘤胃甜菜碱的研究逐渐增多，相应的产品开发成为厂家和科研单位关注的焦点，但是受产品保护工艺、包被材料以及保护程度的影响，需要对相关产品进行应用效果评定。随着肉羊育肥产业的兴起，调控肉羊生长速度及肉品质的添加剂逐步得到应用,利用甜菜碱应用于肉羊生产刚刚开始。迄今，国内过瘤胃甜菜碱产品已经开发上市，在肉羊育肥中的应用及效果有待评定。本试验旨在研究添加甜菜碱和过瘤胃甜菜碱对湖羊生长性能和消化代谢的调控作用，为甜菜碱在肉羊生产中的高效利用提供科学依据。

3讨论

3.1饲粮中添加甜菜碱和过瘤胃甜菜碱对生长性能的影响

甜菜碱可以作为甲基供体替代蛋氨酸，减少了甲基供体蛋氨酸和胆碱的需要量，并参与动物机体其他重要生理过程，对动物生长具有促进作用[2]。本试验表明，饲粮中添加甜菜碱和过瘤胃甜菜碱均对肉羊的 ADFI均无显著性影响，但过瘤胃甜菜碱可以提高肉羊的 ADG,并随着过瘤胃甜菜碱添加量的增加呈现增加的趋势，以 $4 \ : \mathrm { g / d }$ 添加量效果最好，同时过瘤胃甜菜碱组降低了F/G。Huang 等[17]研究表明，饲粮中补充甜菜碱能够提高生长激素和胰岛素样生长因子-I的含量，通过不同途径促进蛋白质合成，并保证机体的正氮平衡，进而提高动物的生长性能。研究发现在热应激状态下，饲粮中添加包被甜菜碱也可以提高热应激肉牛的日增重及转化率[18]。由于反刍动物瘤胃内环境的特殊性使得饲粮中甜菜碱易被易被瘤胃微生物降解成三甲胺，造成应用效果不显著。本试验中，B-2.2 组对肉羊的生长性能与Con组相比差异不显著，这主要可能因为甜菜碱易被瘤胃微生物大量降解有关。除甜菜碱的形式影响应用效果之外，饲粮蛋白质水平也会影响甜菜碱应用效果。Fermandez 等[19]研究表明，饲粮中添加甜菜碱 $( 2 \ \mathrm { g / k g } )$ ）和过瘤胃甜菜碱（ $( 2 \ \mathrm { g / k g } )$ ）对肉羊的生长性能无显著性影响，推测是其饲粮蛋白质水平较高所致。

# 3.2饲粮中添加甜菜碱和过瘤胃甜菜碱对营养物质表观消化率的影响

添加甜菜碱在一定程度上可以提高动物的消化率，这是因为甜菜碱具有高渗透压特性，其在细胞内积聚可以增强肠道保水力，能够促进肠道上皮细胞增殖，并且可以增强肠道的张力，增加肠道长度，表面积增大，进而提高动物体对营养物质的消化率[20-21]。Wang 等[13]研究表明，在荷斯坦奶牛饲粮中添加不同水平甜菜碱可以显著提高饲粮中 DM、CP、NDF和ADF 的消化率，李华伟等[22]研究表明，饲粮中添加过瘤胃胆碱可以提高杜湖杂交1代公羊的 DM、有机物和 ADF 的表观消化率。但是也有关于添加过瘤胃甜菜碱对动物的表观消化率没有显著影响的报道，田兴舟等[23]研究表明对黔北麻羊补饲过瘤胃胆碱对 DM、CP、EE、NDF 的表观消化率无显著影响，但可显著提高ADF 和粗灰分的表观消化率。本试验中饲粮中添加甜菜碱和过瘤胃甜菜碱均对肉羊DM、CP、NDF 和 ADF 的消化率无显著影响，但均随过瘤胃甜菜碱添加量增加呈现升高的趋势。试验结果的不一致，可能与动物品种，饲粮组成及饲养条件等多方面原因相关。

# 3.3饲粮中添加甜菜碱和过瘤胃甜菜碱对血液生化指标的影响

大量研究表明甜菜碱可以参与动物机体的脂肪代谢和蛋白质代谢。其中TG 和TC 的含量是反映脂肪代谢的重要指标，添加甜菜碱有助于卵磷脂合成，进而促进肝脏中脂蛋白的合成和分泌，TG与脂蛋白结合后运输到肝外，被肝外组织利用，从而降低动物体产生脂肪肝的风险[24]。本试验表明，饲粮中添加过过瘤胃甜菜碱可以降低血清中 TC、TG 的含量，且当过瘤胃甜菜碱添加量为 $2 \ \mathrm { g / d }$ 时，血清中TC 和 TG 的含量最低，这与张冬梅等[25]、朱定贵等[7]的研究一致。甜菜碱可以转化为肉碱，促进脂肪酸氧化，从而降低血清中TG的含量。此外HDL可以与胆固醇结合，将血液中胆固醇运到肝脏中进行代谢，使血液中胆固醇含量降低。本试验结果表明，饲粮中添加甜菜碱，能有效降低育肥羊胆固醇和TG 的沉积，促进了脂肪分解，减少体脂沉积，降低脂肪肝风险。

本试验结果表明，饲粮中添加甜菜碱对血清中TP的含量无显著性影响，然而添加过瘤胃甜菜碱能够显著提高血清中TP 的含量。血清中的TP含量与肌肉率呈现极显著的正相关性，当血液中TP含量增加时，蛋白质的合成代谢也随之增强。此外，血清中的UN是蛋白质氨基酸代谢的终产物，当UN含量降低时，说明动物体内的氨基酸分解作用减弱，本试验结果说明，在饲粮中添加一定量的甜菜碱有利于蛋白质合成代谢加强和蛋白质沉淀增加。这与甜菜碱可以提供蛋白质合成所需的甲基供体有关，进一步满足肉羊生长需要。这与董冠[8]研究结果一致，即饲粮中添加一定水平的过瘤胃甜菜碱可以提高蛋白质的消化吸收程度。

4结论

饲粮中添加过瘤胃甜菜碱，能够提高肉羊生长性能，改善蛋白质和脂肪代谢，适宜添加量为 $4 \mathrm { g / d }$ 。
