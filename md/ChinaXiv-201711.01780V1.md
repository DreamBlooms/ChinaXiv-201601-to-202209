3 （湖南农业大学动物科学技术学院，饲料安全与高效利用教育部工程研究中心，湖南畜禽安

摘要：本试验旨在评定黄羽肉鸡不同来源高梁的表观代谢能（AME），并比较其养分代谢率的差异。试验采用单因素随机设计，选择56日龄健康黄羽肉鸡公鸡24羽，随机分为4个组，每组6个重复，每个重复1羽。用美国高梁、湖南高梁和内蒙古高梁分别替代基础饲粮中 $20 \%$ 玉米作为试验饲粮，采用套算法计算美国高梁、湖南高梁和内蒙古高梁的AME，并比较其养分代谢率的差异。试验期7d，分为预试期（第 $1 { \sim } 4$ 天）和正试期（第 ${ \bar { 5 } } { \sim } 7$ 天）2 个阶段。结果表明：1）内蒙古高梁的AME 最高，比美国高梁和湖南高梁分别高 $1 2 . 1 7 \%$ 和 $1 5 . 8 1 \%$ （ $P { < } 0 . 0 1$ ）。2）内蒙古高梁的粗纤维（CF）表观代谢率最高，分别比美国高梁和湖南高粱高 $23 . 8 1 \%$ 和 $2 7 . 8 7 \%$ ( $P { < } 0 . 0 1$ ）；美国高梁次之，比湖南高梁高 $3 . 2 8 \%$ （ $P { < } 0 . 0 1$ ）。内蒙古高梁的粗脂肪（EE）表观代谢率极显著低于湖南高梁 $0 . 5 0 \%$ ( $P { < } 0 . 0 1$ ）。不同来源高梁的粗蛋白质（CF）、干物质（DM）、粗灰分（Ash）和无氮侵出物（NFE）表观代谢率均无显著差异（ $P { > } 0 . 0 5$ ）。由此得出，通过套算法测定了美国高梁、湖南高梁和内蒙古高梁的 AME分别为13.39、12.97和 $1 5 . 0 2 \mathrm { M J / k g }$ 。不同来源高梁除 AME、EE 和CF表观代谢率差异显著外，其余各养分表观代谢率均无显著差异。

关键词：黄羽肉鸡；高梁；单宁；代谢率；套算法高梁为一年生禾本科高梁属草本植物，属于古老的谷类作物之一[]，具有抗旱、耐涝的特点。我国高梁种植广泛、产量高、养分含量丰富、价格低廉，是作为饲用作物开发的资源之一。目前，畜牧业的快速发展使我国大宗饲料原料相对缺乏，国内玉米价格节节攀升，进口量同比增长，依靠进口玉米解决能量饲料资源问题已不能满足养殖业发展的需要，而合理使用高梁则能在较大程度上缓减饲用玉米的不足。Nyannor 等[2用 $9 3 5 \ \mathrm { \ g / k g }$ 高消化高梁（PHD1和PHD2）、普通高梁和玉米饲喂肉鸡研究高梁的营养价值，结果显示，玉米的表观代谢能（AME）显著高于PHD1 和普通高梁，与 PHD2无显著差异。Truong 等[3研究#3和#5 高梁对肉鸡养分利用率的影响，结果表明，饲喂#3 高梁肉鸡的AME、代谢能（ME）：总能（GE）和真表观代谢能（AMEn）较#5 高梁均有所提高。不同品种高梁的营养成分含量不同，所产生的 AME 和养分利用率也会存在差异。因此，本试验以玉米-豆粕型饲粮为基础饲粮，采用套算法评估美国高梁、湖南高粱和内蒙古高粱对黄羽肉鸡AME 的影响并比较其养分代谢率的差异，为不同来源高梁代谢能值的评定以及高梁饲粮配方的设计提供数据参考。

# 1.4饲养管理

试验鸡采用单笼独立饲养，鸡舍自然通风和光照，定期打扫舍内卫生，相对湿度保持在$5 5 \% { \sim } 6 5 \%$ ，自由饮水、采食。每天早晨回收残料并称重记录。预试期间，观察每只鸡的采食量。绝食 $1 2 \mathrm { h }$ 后进入正试期，少量多次饲喂，采用全收粪法进行代谢试验。

# 3讨论

# 3.1黄羽肉鸡不同来源高粱AME

本试验研究发现，美国高梁、湖南高粱和内蒙古高梁的AME分别为13.39、12.97和15.02MJ/kg，这可能是由于不同来源高梁中抗营养因子含量的不同所导致的。李筱倩等[4研究显示，高单宁含量褐高梁与低单宁含量高梁的 AME 差异极显著，与本试验结果相一致。Sannamani等[5用红、白、黄3种高梁替代 $50 \%$ 、 $7 5 \%$ 、 $100 \%$ 的玉米，结果显示， $50 \%$ 黄高梁组的能量代谢率极显著高于对照组。不同单宁含量高梁的能量代谢率不同，这是由于单宁含量过高影响饲粮的适口性，直接影响家禽的采食量，进而影响能量的代谢率。不同来源高梁的品质受品种、收获期、贮存方法、环境、气候条件和土壤条件等因素的影响。胡贵丽等[研究显示，美国高梁的单宁含量最低，为 $0 . 3 1 \%$ ；内蒙古高梁次之，为 $1 . 1 \%$ ；湖南高梁最高，为 $1 . 3 8 \%$ 。用不同来源高梁等比例替代玉米会使饲粮的单宁含量同高梁原料单宁含量产生一致性变化。本试验中，不同高梁饲粮的单宁含量以湖南高梁饲粮为最高，内蒙古高梁饲粮次之，美国高粱饲粮最低。观察高粱籽实的颜色可知，湖南高粱颜色偏深，内蒙古高梁次之，美国高梁颜色偏浅。由此推测，高梁籽实的颜色是预测单宁含量的一个重要指标[7]。

# 3.2黄羽肉鸡不同来源高梁营养物质代谢率

本试验结果显示，不同来源高梁的CF表观代谢率差异极显著，内蒙古高梁最高，美国高梁次之，湖南高梁最低；内蒙古高梁的EE 表观代谢率却极显著低于湖南高梁；不同来源高梁的 DM、Ash、NFE 表观代谢率均无显著差异。Tandiang 等8用低单宁含量的高梁替代1/3、2/3、3/3 的玉米饲喂肉鸡，结果显示，各组间 EE 代谢率差异极显著，DM、Ash 代谢率均无显著差异，与本试验结果相一致。本试验结果也得出，不同来源高梁的CP表观代谢率均在 $40 \%$ 以上，其中美国高梁的CP表观代谢率最高，湖南高梁次之，内蒙古高梁最低，但3者无显著差异，这可能是由于受高梁所含的抗营养因子单宁的影响。Duodu 等[9研究发现，高梁含有的 $\gamma$ 和 $\beta$ 醇溶蛋白位于蛋白质体的外周，阻碍了位于中心的主要贮藏蛋白质的消化。Sultan 等[]发现高梁的 CP 代谢率低于玉米。3 种高梁的 NFE 表观代谢率比较一致（85.80%～85.83%），与Garcia 等[1]研究不一致，原因可能与被测高梁的替代比例、肉鸡品种和饲粮组合效应等因素有关。

# 3.3套算法评定黄羽肉鸡不同来源高梁营养物质代谢率

套算法是评定单个饲料原料营养价值的经典方法。根据被测饲料原料的性质而确定其所占的比例，传统的套算法试验饲粮由 $50 \% \sim 8 0 \%$ 基础饲粮和 $2 0 \% \sim 5 0 \%$ 待测饲料原料组成[12]。张子仪等[13]以6 周龄来航鸡为试验动物，证明能量饲料的表观代谢能随待测饲料原料在试验饲粮中比例的增加而呈递减规律。这表明基础饲粮和被测饲料原料营养水平的差异也会影响被测饲料原料生物学效价的评定。本试验中不同来源高粱替代玉米的比例为 $20 \%$ 在今后的类似试验中，一方面可以适当增加被测高梁的替代比例，另一方面也可在基础饲粮中使用一定比例的被测高梁，以减小试验误差。

4结论

通过套算法测定了美国高梁、湖南高梁和内蒙古高梁的AME分别为13.39、12.97和15.02MJ/kg。不同来源高梁除 AME、EE 和CF表观代谢率差异显著外，其余各养分表观代谢率均无显著差异。
