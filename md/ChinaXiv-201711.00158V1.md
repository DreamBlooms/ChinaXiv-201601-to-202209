# 1 INTRODUCTION

The design and synthesis of coordination complexes have been one of the most atracting points in coordination polymers[1-5] as a result of their structural diversity and versatile applications in various areas including photochemistryl6],catalysis[7],gas-absorption and separation[8,9],energy storageand conversion[0.], etc.Employing functional ligands has become a widespread method to purposively construct coordination polymers with certain structures and properties[12-15]. Fluorescein,a classic dye which was synthesized in the 19th century firstlyl16], has particularly arisen our interest recently since it is widely used in human's daily life as colorant and biochemical probes due to its broad emiting spectrum, high fluorescence quantum yield and non-toxic propertyl17,18]. Among all the reported metal coordination complexes with fluorescein, fewof them are companied with crystal structures[19,20].

Herein we report the synthesis and structure of a six-coordinated cadmium complex based on fluorescein, $\{ \mathrm { [ C d _ { 2 } ( C _ { 2 0 } H _ { 1 0 } O _ { 5 } ) _ { 2 } ( H _ { 2 } O ) _ { 4 } ] \cdot 1 0 H _ { 2 } O \} _ { \mathrm { n } } ( { \bf 1 } ) }$ ，which possesses a unique $\mathrm { C d } _ { 2 } \mathrm { O } _ { 1 0 }$ cluster.

# 2.2Synthesis procedures

Fluorescein ( $1 6 . 7 \ \mathrm { m g } , \ 0 . 0 5 \ \mathrm { m m o l } )$ was dissolved in $1 . 8 ~ \mathrm { m L }$ DMF, then $2 . 2 ~ \mathrm { m L }$ aqueous solution of $\mathrm { C d } ( \mathsf { N O } _ { 3 } ) _ { 2 }$ (14.9 mg, O.O5 mmol) was added. After ten minutes supersonic processing in open air, the mixture was sealed in a $1 0 ~ \mathrm { m L }$ glass vessel, which was heated to $3 8 3 ~ \mathrm { K }$ and kept for two days,and then slowly cooled to room temperature for another two days. Red block single crystals were filtrated and washed with pure water, and dried in air. Yield $27 \%$ (based on fluorescein). Calculated $( \% )$ : C, 49.71; H, 2.90; O,23.18. Found $( \% )$ : C, 49.16; H, 2.72; O, 23.85.

# 3RESULTS AND DISCUSSION

# 3.1 Structural description

Single-crystal X-ray diffraction study reveals that 1 crystalizes in the monoclinic system with space group $P 2 / n$ ,showing a two-dimensional layer which packs up to afford the 3D structure.In one asymmetric unit, there exist two cadmium atoms,two fluorescein ligands,and four coordinating water molecules. There are four Cd atoms bonded with a single fluorescein ligand (Fig. la),while each $\operatorname { C d } ( \operatorname { I I } )$ ion is hexa-coordinated by four O from four diferent fluorescein ligands and two water molecules with octahedral coordination geometry (Fig.1b).The disordered O atoms are simplified for clarification.In particular,the two adjacent Cd atoms are bonded up by two $\mu$ -O from two independent fluorescein ligands (Fig.lc),and each is connected with another two fluorescein ligands and two water molecules, forming a unique ${ \mathrm { C d } } _ { 2 } { \mathrm { O } } _ { 1 0 }$ cluster. The Cd-O bond lengths vary from 2.218(4) to 2.361(5) $\mathring \mathrm { A }$ while the Cd-O-Cd bond angles are $1 0 4 . 0 7 6 ( 1 ) ^ { \circ }$ and

$1 0 4 . 3 8 8 ( 1 ) ^ { \circ }$ ，and the O-Cd-O bond angles change from from $7 5 . 7 1 1 ( 1 ) ^ { \circ }$ to $1 7 2 . 2 3 4 ( 1 ) ^ { \circ }$ . Two of the six coordinating fluorescein ligands related to this cluster contribute the two $\mu$ -O atoms, O(5) and O(10); two give the other coordinating O atoms, O(3)and O(8); the rest two ligands provide the two coordinating O atoms on their carboxyl, O(1) and O(6); the other four coordinating sites are occupied by water molecules.Two kinds of weak $\pi \cdot \pi$ interactions are found between two sets of parallel fluorescein ligands with the centroid-to-centroid distances as 4.312/4.317 $\mathring \mathrm { A }$ and the perpendicular distances to be 3.326/3.387 A,respectively. The $\mathrm { C d } _ { 2 } \mathrm { O } _ { 1 0 }$ clusters are linked up by fluorescein ligands via the weak π-π interactions and the first seven kinds of hydrogen bond interactions (Table 2),affording the wonderful 2D layer structure (Fig.2） which further packs up by weak $\pi { - } \pi$ interactions and the C(23)-H(23)\*·O(14) hydrogen bond to form the whole 3D structure with two types of nano-channels (Fig. 3).

# 3.2 Characterizations

# 3.2.1 Powder X-raydiffraction

The crystalline phase purity of 1 was confirmed by powder X-ray difraction.The experimental pattern agreed basically with that of simulation (Fig. 4a). The sample had been kept in open air for two days with no significant loss of crystallinity, indicating that 1 is quite stable towards water and oxygen.

# 3.2.2IR spectroscopy

The infrared spectroscopy was performed to give the characteristic absorption of 1. IR data (KBr, $\mathrm { c m ^ { - 1 } }$ 》 $\nu _ { \mathrm { s } } ( \mathrm { A r - O ^ { - } } )$ : 1290 (s), 1328(s); $\nu _ { \mathrm { s } } ( \mathrm { C O O - } )$ : 1513 (m),1207 (m); $\nu _ { \mathrm { a s } } ( \mathrm { C O O - } )$ : 1577 (s), 1452 (s).

# 3.2.3Thermo gravimetric analysis

Thermo gravimetric analysis was conducted to determine the contents of solvents and thermal stability of 1. For 1, the TGA curve (Fig. 4b) shows several processes of weight loss in the full range ( $( 3 0 { \sim } 9 0 0 ~ \mathrm { ^ { \circ } C }$ of measurement temperature. The first stage is $3 0 { \sim } 1 2 0 \ \mathrm { ^ { \circ } C }$ ,corresponding to the loss of 10 water molecules of crystallization. The second one is $1 2 0 { \sim } 1 5 8 \ \mathrm { ^ \circ C }$ due to the release of four constitution water molecules. Then a platform from 158 to $4 7 5 \ \mathrm { ~ \textdegree C }$ follows, indicating the framework possesses quite excellent thermal stability in this range of temperature. The sudden weight drop after $4 7 5 \ \mathrm { ~ \textdegree C }$ can be attributed to the collapse of the framework.

# 4CONCLUSION

In summary, a novel cadmium-fluorescein coordination complex has been synthesized via hydrothermal reaction and systematically characterized. Single-crystal X-ray difraction analysis reveals that 1 possesses a unique ${ \mathrm { C d } } _ { 2 } { \mathrm { O } } _ { 1 0 }$ core,which is linked up by fluorescein to afford a 2D layer that packs up to afford a 3D supramolecule,and two different types of 1D channels are found.
