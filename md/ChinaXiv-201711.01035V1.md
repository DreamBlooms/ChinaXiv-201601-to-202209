铜是毛皮动物必需的微量元素，在机体能量代谢、造血和被毛着色等生理过程中发挥重要作用[I。铜缺乏会导致动物贫血、生长迟滞、骨骼异常以及被毛脱色等症状，而适宜铜水平则能够促进动物生长，提高饲粮的利用率2]。冬毛期是蓝狐夏毛换冬毛、脂肪囤积及皮张成熟的关键时期[3]，微量元素铜在此时期发挥着重要作用。Aulerich 等[4-5]对冬毛期水貂的研究发现，饲粮中添加铜（100 或 $2 0 0 \mathrm { m g / k g }$ 硫酸铜）能够促进水貂生长并使毛色加深。Wu等研究表明，饲粮中添加铜能够提高冬毛期母貂采食量及脂肪的消化率，降低料重比。Zhong等[7]研究表明，饲粮中添加 $3 0 \mathrm { m g / k g }$ 铜（铜源为柠檬酸铜）能够提高冬毛期银狐的毛皮品质。白玉妍等8研究表明，在乌苏里貉饲粮中添加蛋氨酸铜能提高毛皮的质量。此外，本课题组前期研究表明，饲粮铜水平为 $4 0 \mathrm { m g / k g }$ 时能促进育成期蓝狐的生长，提高饲粮蛋白质及脂肪的消化率9]。目前，有关蓝狐冬毛期饲粮铜的适宜水平的研究尚未见报道，铜对冬毛期蓝狐生长及毛皮品质的影响还不可知。本试验以此为切入点，研究饲粮铜水平对冬毛期雌性蓝狐生长性能、营养物质消化率、血清生化指标及毛皮品质的影响，为生产实践中合理配制蓝狐饲粮提供理论指导。

# 1.2 试验设计与饲养管理

试验采用单因素完全随机试验设计，将50只150日龄左右的健康雌性蓝狐随机分成 5组（Ⅰ～V组)，每组10个重复，每个重复1只，分别饲喂对应试验饲粮。试验蓝狐均为单笼(105cmx80 cmx70cm)饲养，每日08:00与15:00 分别饲喂1次，自由采食，自由饮水，每日记录实际采食量。预试期7d，正试期 $6 0 \mathrm { d }$ ，试验地点为农业部长白山野生生物资源重点野外科学观测试验站。

# 1.3消化代谢试验

正式试验开始后30d，每组随机挑选8只健康蓝狐进行为期5d的消化代谢试验。试验采用全收粪法，采用粪尿可分离式接粪盘收集蓝狐每日的粪便及尿液。消化代谢试验期间饲养管理与日常饲养管理完全相同。每天收集的粪便称重后按鲜重的 $5 \%$ 加入 $10 \%$ 的硫酸溶液，并加入少量甲苯防腐，保存于-20℃备用。每日收集的尿液按每 $1 0 0 ~ \mathrm { m L }$ 加入 $1 0 \mathrm { m L } 1 0 \%$ 硫酸溶液，滴加4滴甲苯防腐，保存于-20℃备用。

# 1.4 测定指标及方法

正试期第1天早晨对蓝狐进行空腹称重，为试验动物的初始体重，以后每隔15d称重1次，记录体重，并以试验结束时的体重作为终末体重；记录每只蓝狐每天的给料量和剩料量，计算母只监狐的米食量以及每组的米食量，开计算出平均日米食量、平均日增里和科里比。饲粮及排泄物中的干物质含量采用65℃烘干法测定，参照GB/T6435-2006；采用凯氏定氮法测定粗蛋白质的含量，参照GB/T6432-1994；采用索氏抽提法测定粗脂肪的含量，参照GB/T6433-2006。采用火焰原子吸收法测定饲粮铜水平及肝脏及排泄物中铜的含量，参照 GB/T 13885-2003。

3讨论

3.1饲粮铜水平对冬毛期雌性蓝狐生长性能的影响

NRC（1982）[10]指出，水貂和蓝狐饲粮中铜的需要量为 $4 . 5 { \sim } 6 . 0 \ \mathrm { m g / k g }$ ，经测定，本试验中基础饲粮中铜水平为 $7 . 7 8 ~ \mathrm { m g / k g }$ ，可以满足蓝狐对铜的需要量。大量研究表明饲粮中添加高剂量的铜（ $1 2 5 { \sim } 2 5 0 \mathrm { m g / k g }$ ）能够促进猪[14]、肉鸡[15]及獭兔[16]的生长，提高饲料利用率。本研究结果表明，饲粮中添加适量的铜同样能够提高雌性蓝狐的平均日增重，降低料重比。铜促生长作用的机理主要有增加采食量、提高消化酶活性和营养物质消化率、抗微生物作用以及促进生长相关激素的分泌等[17]。本试验中，饲粮中添加铜并未提高蓝狐的平均日

采食量，这与Zhou等[18]在猪上的研究结果不一致，可能是不同物种的生理特性不同导致的。Dove[19]研究发现，饲粮中添加 $2 5 0 \mathrm { m g / k g }$ 铜能够提高猪对饲粮中营养物质的消化率。本试验中铜的添加提高了蓝狐对饲粮中脂肪的消化率，这与Dove[19]在猪上的研究结果一致。由于蓝狐为肉食动物，对饲粮中脂肪水平要求较高[20]，脂肪消化率的提高在很大程度上改善了饲粮的利用率，并促进了冬毛期蓝狐脂肪的囤积和体重的增加。LaBella等[21研究证明铜可以刺激牛垂体分泌生长激素（GH)。Yang 等[22研究表明，饲粮中添加 $1 2 5 \mathrm { m g / k g }$ 硫酸铜或蛋氨酸铜能够促进猪生长激素释放激素（GHRH）的分泌，抑制生长抑素（SS）的分泌。铜对蓝狐生长相关激素的影响仍不清楚，需要进一步的研究。  
3.2饲粮铜水平对冬毛期雌性蓝狐营养物质消化率的影响  
本研究表明饲粮中适宜水平的铜能够促进饲粮中脂肪的消化率，这与在猪[19]及水貂[23]上的研究结果一致。Luo 等[24]研究发现，饲粮中添加铜能够提高断奶仔猪小肠脂肪酶和磷脂酶A的活性。胆汁是脂肪消化过程中的重要物质，能够通过乳化脂肪形成脂肪微粒，提高脂肪与脂肪酶的接触面积，促进脂肪的消化[25]。由于胆汁是机体铜排泄的主要途径，摄入高水平的铜会促进胆汁的分泌[26]，进而促进脂肪的消化。饲粮中干物质消化率、蛋白质消化率及氮沉积并未受到饲粮铜水平的影响,这与在Wu等在水貂上的研究结果一致。Roberts等[27]研究表明绝大多数（ $8 5 \%$ ）的饲粮铜由胆汁分泌并经由粪便排出体外。因此，随着饲粮铜水平的升高，蓝狐粪便中铜的含量即铜排出量急剧升高。  
3.3饲粮铜水平对冬毛期雌性蓝狐血清生化指标的影响

蛋白质是血清的主要成分，对于维持机体渗透压及保持水分平衡极为重要。血清中的总蛋白主要包括白蛋白和球蛋白。白蛋白在机体铜的转运过程中发挥着重要的作用。本试验结果表明饲粮中添加铜提高了血清白蛋白和总蛋白的含量，这与在水貂[23]上的研究结果一致。尿素是体内氨基酸分解代谢后的最终产物之一，血液中的尿素来源于肝脏，并通过是肾脏排出体外。李道林[28研究发现獭兔血清尿素氮含量会随着饲粮中铜水平的升高而降低，这与

本试验的结果不一致，结果差异可能是蓝狐与獭兔不同的生理特性造成的。血清 ALT和 AST活性是检测肝功能的重要指标，研究表明高铜造成动物肝脏损伤，致使血清 AST 和 ALT 活性升高[29-30]。本试验中，饲喂最高铜水平饲粮组的蓝狐血清 AST与ALT 活性较高，但与其他组相比差异并不显著。由于本试验所设饲粮最大铜水平为 $1 6 0 \mathrm { m g / k g }$ ，远远低于上述研究的添加量（600 mgCu/kgDM)，对肝脏的伤害也较小，因此结果差异不显著。铜蓝蛋白是由肝脏合成的一种含铜氧化还原酶，能够参与体内铁的代谢。Zhong 等[7研究表明饲粮中添加铜并未显著提高银狐血清铜蓝蛋白的活性，与本研究结果相一致。Cu-Zn SOD 是动物体内一种重要的抗氧化物质，能够清除体内的自由基，保护细胞免受伤害。冯杰等[31研究表明，饲粮中添加高铜能够提高仔猪血清Cu-Zn SOD 的活性。本研究中，Cu-Zn SOD 的活性随饲粮铜水平的升高而提升，这与冯杰等[31]在仔猪上的研究结果相一致。  
3.4饲粮铜水平对冬毛期雌性蓝狐血清铜、锌、铁及肝脏铜含量的影响  
本研究表明，血清铜、锌、铁含量并未受到饲粮铜水平的显著影响，而肝脏铜含量随着饲粮铜水平的升高而增加。Danzeisen 等[32]报道,动物血浆中铜的含量受到的机体严格调控，并不能反映当前动物体内铜的状态。在可调节的范围内，机体可以通过调节十二指肠铜的吸收和胆汁铜的排放来维持血清中铜含量的稳定[33]。目前，唯一能准确反映机体铜过量的指标是肝脏铜含量[34]。在本研究中，肝脏铜含量随饲粮铜水平的升高而增加，这反映了蓝狐正处于铜过量的状态，而由于机体对血清铜的调节作用，因此，各组蓝狐血清铜含量差异不显著。本研究结果与 Zhong 等[7]在银狐上的报道一致。尽管铜与锌之间[35]和铜与铁之间[36]存在着互作效应，但在本试验中，血清中锌、铁的含量并未受到饲粮铜水平的显著影响，原因可能是饲粮铜水平较低，不足以影响另外2种元素的吸收和代谢。  
3.5饲粮铜水平对冬毛期雌性蓝狐毛皮品质的影响  
Lorek 等[37研究表明，蓝狐体长与皮长之间（ $R ^ { 2 } { = } 0 . 6 1 6$ ）以及体重与皮长（ $R ^ { 2 } { = } 0 . 7 5 4 \rangle$

之间存在显著的正相关。本研究中II、IⅣ组蓝狐有较大的体重和体长，因此皮长指标优于其他组蓝狐。饲粮的营养水平是影响毛皮品质的重要因素[38-39]，但饲粮铜水平对蓝狐毛皮品质的影响尚未有报道。Zhong 等7研究发现，饲粮中添加 $3 0 \mathrm { m g / k g }$ 柠檬酸铜（总铜水平为35mg/kg）能够提高雌性银狐针毛长度和皮板厚度。Wu等[40]研究表明，铜能够增加水貂皮长，提高毛皮品质，加深毛色。本研究中，饲粮铜水平为 $4 0 \mathrm { m g / k g }$ 时，蓝狐的针绒毛长度较好，这与 Zhong[7等在银狐上的研究结果一致。铜是酪氨酸酶的重要组成部分，而酪氨酸酶是黑色素合成过程中的关键限速酶。李继兴[4I研究表明，卡拉库尔羊体内铜含量与酪氨酸酶基因的表达呈正相关，饲粮中补饲铜能够提高酪氨酸酶活性。本试验中饲喂含 $1 6 0 \mathrm { m g / k g }$ 铜的饲粮的蓝狐皮张颜色较对照组深，原因可能是高剂量铜促进了酪氨酸酶基因的表达，提高了酪氨酸酶的活性，进而促进了黑色素的合成，使得毛色加深。这与Aulerich等[5在水貂上的研究结果一致。

# 4结论

$\textcircled{1}$ 饲粮铜水平为 $4 0 . 0 0 { \sim } 8 0 . 0 0 \mathrm { m g / k g }$ 时，冬毛期雌性蓝狐可获得较快的生长，饲粮脂肪的消化率较高；饲粮铜水平为 $4 0 . 0 0 \mathrm { m g / k g }$ 时雌性蓝狐的皮长及毛皮品质较好;饲粮铜水平为 $1 6 0 . 0 0 \mathrm { m g / k g }$ 时能加深毛皮的颜色，但会造成肝脏铜的沉积并增加铜的排放。  
$\textcircled{2}$ 综合得出，饲粮铜水平为 $4 0 . 0 0 \mathrm { m g / k g }$ （铜添加水平为 $3 2 . 2 2 \mathrm { m g / k g }$ ）时，冬毛期雌性蓝狐可获得较好的生长性能、营养物质消化率及毛皮品质，同时在饲养中向环境排放的铜较低。
