锌是动物体内不可缺少的微量元素之一，是体内300多种酶的必需组分和激活因子，参与维持细胞膜的完整性和体内系列化学反应，维持正常的代谢和内环境平衡[1]。因多数饲料原料中锌含量偏少，且可利用率较低，故饲粮中一般需额外添加锌以满足蛋禽的营养需要。本文简要综述了锌在蛋禽中的生物学功能、在蛋禽生产中的应用研究、存在的问题及展望，以期为锌在蛋禽生产中的合理应用提供理论参考。1锌在蛋禽中的生物学功能  
锌作为动物体内重要的微量元素之一，广泛参与体内代谢，具有多种生物学功能。锌在机体内不仅发挥酶的催化活性，在结构组成上也发挥重要作用[2]。含锌的金属酶存在于六大酶类中，包括：氧化还原酶类（超氧化酶歧化酶等）、转移酶类（RNA聚合酶等）、水解酶类（碱性磷酸酶等）、裂解酶类（碳酸酐酶）、异构酶类（磷酸甘露糖酶）、合成酶类（tRNA 合成酶）[3]。因此，锌参与体内核酸和蛋白质的合成、能量代谢、氧化还原等生化代谢过程，影响机体的物质代谢和动物的生长发育。  
机体的免疫和抗氧化状态是维持蛋禽体内稳态的重要部分。锌可降低氧化应激，提高机体的免疫功能[2]。锌可维持生物膜正常的结构和功能，不仅通过铜锌超氧化物歧化酶（Cu,Zn-SOD)清除体内多余的自由基，还可通过金属硫蛋白（MT）保护膜蛋白的构象和抑制过氧化物的产生，从而维持氧化还原状态的平衡。此外，锌可促进体内免疫器官的生长发育，提高体液和细胞免疫，维持淋巴细胞的正常功能，从而改善机体的免疫能力[3]。因此，锌对蛋禽的健康状态发挥重要作用。

锌是碳酸酐酶(CA)的重要辅助因子，在蛋壳形成中具有重要作用。CA 可促进二氧化碳( $\mathrm { C O } _ { 2 }$ ）

水合形成碳酸氢根离子，促进蛋壳腺内碳酸钙的沉积。体外试验研究表明，锌可促进方解石形成过程中晶体的生长[4]。同时，锌是碱性磷酸酶（ALP）的组成部分，研究表明，子宫和峡部内 ALP活性与钙离子的浓度之间存在显著的负相关关系[5]，由此推测蛋壳腺内 ALP 活性与碳酸钙的沉积存在一定关系。此外，ALP还具有去磷酸化作用，对蛋壳基质中某些磷酸化蛋白[骨桥蛋白、Ovocleidin-116（OC-116）等]可能起到一定的调控作用，从而直接或间接地影响碳酸钙晶体的形成[。可见，锌可调控蛋禽的蛋壳品质。

# 2.1锌对蛋禽生产性能的影响

研究表明，饲粮添加维生素E（40、80IU）或锌（100、 $2 0 0 \mathrm { m g / k g }$ ，有机锌）对蛋鸭（22\~30周龄）的采食量和蛋重无显著影响，但显著提高了产蛋率和饲料利用率，且锌与维生素E的效果类似，但剂量之间无显著差异[9]。蛋鸭（27\~39 周龄）饲粮（锌含量为 $3 7 \mathrm { m g / k g }$ ）添加锌（ $1 5 { \sim } 9 0$ mg/kg，硫酸锌）显著影响其产蛋率和饲料利用率，并随剂量增加呈二次曲线变化，其中30 和 45mg/kg 添加组生产性能较佳[10]。然而,蛋鸡饲粮(锌含量约为 $2 7 \mathrm { m g / k g } \mathrm { \cdot }$ 添加硫酸锌 $( 3 5 { \sim } 1 4 0 ~ \mathrm { m g / k g } )$ （20对蛋鸡（54\~63 周龄）的生产性能无显著影响[11-12]，但添加有机锌线性降低了蛋鸡（ $6 3 { \sim } 6 8$ 周龄)的平均蛋重[12-13]。与75 和 $1 0 0 ~ \mathrm { { m g / k g } }$ 添加组相比，饲粮（锌含量为 $3 3 . 4 0 ~ \mathrm { m g / k g }$ ）添加 $5 0 \mathrm { m g / k g }$ 锌（硫酸锌、氧化锌、甘氨酸锌）即可使蛋鸡（36\~45周龄）的生产性能达到最佳，较高剂量反而降低平均蛋重和饲料利用率[14]。此外，高剂量螯合锌（ $1 3 7 { \sim } 6 5 5 ~ \mathrm { m g / k g }$ ）具有降低蛋鸡蛋重的趋势（57\~60 周龄)，显著降低产蛋率（48\~56周龄）[15]。饲粮（锌含量为 $6 0 \mathrm { m g / k g } ^ { \cdot }$ ）添加300和

$6 0 0 \mathrm { m g / k g }$ 硫酸锌和氨基酸锌对产蛋高峰后蛋鸡（42\~54 周龄）的生产性能亦无显著改善作用[16]。可见，锌对蛋鸡和蛋鸭生产性能影响的研究结果并不一致，造成上述差异的原因可能与动物的品种、基础饲粮锌含量、锌添加剂量、饲喂周期及饲养环境等的不同有关。综上，饲粮添加高剂量（ $\mathrm { > 5 0 ~ m g / k g }$ ）锌对产蛋鸡的生产性能总体上并无显著改善作用，甚至会降低平均蛋重，尤其是有机锌。以生产性能为评判指标，基础饲粮中含有的锌（约为 $3 0 \mathrm { m g / k g }$ 即可满足蛋鸡生产需要，不需额外添加，这与 NRC（1994）推荐的剂量（约为 $3 5 \mathrm { m g / k g }$ ）不谋而合。锌可提高蛋鸭的产蛋率和饲料利用率，高峰期蛋鸭锌的需要量为基础饲粮（锌含量约为30mg/kg）额外添加 $3 0 { \sim } 4 5 ~ \mathrm { m g / k g }$ 。锌对蛋鸭生产性能的改善作用可能与其提高抗氧化状态有关，额外添加30 和 $4 5 \mathrm { m g / k g }$ 锌提高了蛋鸭的产蛋率和饲料利用率，而抗氧化能力在这2 组较高[10]。此外，锌也可能通过影响营养物质的消化率而调控生产性能，锌（30、 $6 0 \mathrm { m g / k g }$ ）可提高动物对干物质、有机物、粗蛋白质和粗脂肪的消化率[17]；锌不足降低动物的食欲和饲料利用率，导致动物生长受阻[7]。但这些作用机理均有待验证。

# 2.2锌对蛋禽产品品质的影响

近年来，蛋壳破损已成为制约蛋禽业健康发展的重要瓶颈之一。蛋壳品质的降低不仅影响蛋禽产蛋量和经济效益，降低蛋品质，而且还降低种蛋的孵化率和胚胎的成活率[18]。因此，改善蛋壳品质是蛋禽生产中亟待解决的重要问题。CA是一种含锌酶，可促进蛋壳腺内碳酸钙的沉积[19]。抑制蛋壳腺内CA活性和基因表达，蛋壳品质降低[20-21],提高蛋鸡体内CA活性可改善蛋壳品质[22]。研究表明，在产蛋后期蛋鸡（59\~63周龄）的基础饲粮（锌含量为 $2 4 . 1 0 \mathrm { m g / k g }$ ）添加锌（ $7 0 { \sim } 1 0 4$ mg/kg，硫酸锌）可通过提高血浆和蛋壳腺内CA活性，改善蛋壳超微结构，提高蛋壳厚度[I];随添加剂量的增加（ $3 5 { \sim } 1 4 0 ~ \mathrm { m g / k g }$ ，硫酸锌和氨基酸锌）蛋鸡（ $6 3 { \sim } 6 8$ 周龄）蛋壳腺内CA活性和蛋壳厚度呈线性和二次变化，且氨基酸锌的效果较硫酸锌更加显著[13]，但对蛋壳强度无显著影响[11-13]。热应激下，添加 $\mathrm { 1 ~ 0 0 0 ~ m g / k g }$ 锌[硫酸锌、蛋氨酸锌、乙二胺四乙酸(EDTA)-螯合锌）]可提高蛋鸡（60\~66 周龄）的蛋壳重量，减少蛋壳破损[23-24]，这主要是通过提高钙结合蛋白含量和CA 活性实现的[24]。额外添加锰/锌 $( 3 0 / 5 0 \mathrm { m g / k g }$ ,62\~70 周龄)或锰/锌/铜（30/30/5、60/60/10 mg/kg,69\~78 周龄)可提高蛋鸡的蛋壳品质，对青年蛋鸡无显著影响[25-26],锌、锰、铜还可改善蛋鸡（47\~62周龄）的蛋壳超微结构[27]。可见，锌可通过提高CA活性和改善蛋壳超微结构而提高蛋壳厚度，且在老龄蛋鸡上的效果更佳，但对蛋壳强度无显著影响。以蛋壳品质为评价指标，蛋鸡锌的需要量为基础饲粮（锌含量约为 $3 0 \mathrm { m g / k g } ^ { \cdot }$ ）额外添加 $7 0 { \sim } 1 0 0 \ \mathrm { m g / k g }$ 。相较于蛋鸡，锌调控蛋鸭蛋壳品质的研究较少。基础饲粮（锌含量为 $3 7 \mathrm { m g / k g } \mathrm { \cdot }$ ）添加硫酸锌（ $1 5 { \sim } 9 0 \ \mathrm { m g / k g } \$ ）对蛋鸭（27\~39周龄）的蛋壳品质无显著影响[10]。而蛋氨酸锌（ $2 0 { \sim } 4 0 ~ \mathrm { m g / k g }$ ）显著提高了青年蛋鸭的蛋壳强度，但对蛋壳厚度无显著影响[28]。此外，蛋壳超微结构对蛋壳品质具有决定性作用，且逐渐成为评价蛋壳品质的重要指标，但关于超微结构在蛋鸭生产上的研究报道甚少，饲粮锌对蛋鸭蛋壳品质的影响缺乏系统性研究，锌源和添加剂量对蛋鸭蛋壳品质的影响有待进一步研究。禽蛋是营养素强化的理想载体。由于锌的多种生物学功能，通过饲粮调控可将锌富集于禽蛋，为人类锌的补充提供膳食来源。每克鸡蛋中锌的含量一般为 $1 0 { \sim } 1 1 \ \mu \mathrm { g } ^ { [ 2 9 - # 2.3锌对蛋离机体健康的影响

机体内各种细胞代谢产生大量的活性氧等物质，尤其蛋禽体内旺盛的脂质代谢，更需良好的抗氧化系统来维护自身的平衡。机体内存在大量的抗氧化物质，如多种抗氧化酶，包括谷胱甘肽过氧化物酶（GSH-Px）、超氧化物歧化酶（SOD）、过氧化氢酶（CAT）等，还存在许多非酶抗氧化剂，如谷胱甘肽（GSH)、维生素、MT等；机体通过上述酶或非酶抗氧物质等来抑制氧化反应和清除产生的氧化产物，从而维持机体的氧化还原平衡[36]。锌可维持MT含量、作为Cu,Zn-SOD的必要成分、作为硫醇和其他化学成分的保护剂，保护生物结构免受自由基的损伤[3]。研究表明，饲粮添加锌（ $1 0 0 \mathrm { m g / k g }$ ，氨基酸锌）可提高产蛋鸡（45\~65 周龄）肝脏和脾脏中 SOD 活性，总抗氧化能力（T-AOC）和GSH含量，降低丙二醛（MDA）的产生[37]。基础饲粮（锌含量为27.95mg/kg）添加锌（ $3 0 { \sim } 1 2 0 \mathrm { m g / k g }$ ，硫酸锌）线性提高了褐壳蛋鸡（20\~40 周龄）肝脏中Cu,Zn-SOD的活性，但添加剂量超过 $6 0 \mathrm { m g / k g }$ 后，其活性不再显著升高[38]。基础饲粮（锌含量为 $2 9 . 0 4 \mathrm { m g / k g } )$ （20添加锌（35、70、 $1 4 0 \mathrm { m g / k g }$ ，硫酸锌和蛋氨酸锌）可显著提高蛋鸡（ $5 8 { \sim } 6 4$ 周龄）血浆和肝脏内Cu,Zn-SOD 活性，提高 T-AOC 和抗超氧阴离子能力，降低 MDA含量，且 $7 0 \mathrm { m g / k g }$ 硫酸锌组的抗氧化性能最佳[12]。此外，基础饲粮（锌含量为 $3 7 \mathrm { m g / k g }$ ）添加锌（ $1 5 { \sim } 9 0 ~ \mathrm { m g / k g }$ ，硫酸锌）可显著影响蛋鸭（27\~39周龄）的抗氧化状态，提高血浆GSH-Px和总超氧化物歧化酶（T-SOD）的活性，且随添加剂量的增加呈二次变化，30 和 $4 5 \mathrm { m g / k g }$ 添加组的效果较佳[10]。育成蛋鸭饲粮（锌含量为 $3 2 . 9 \mathrm { m g / k g }$ ; $4 { \sim } 1 2$ 周龄）添加30 和 $6 0 \mathrm { m g / k g }$ 锌（硫酸锌）可提高血清T-AOC含量，但差异不显著；再进一步增加添加剂量（200、 $\mathrm { 1 ~ 0 0 0 ~ m g / k g }$ )，T-AOC反而下降[39]。可见，适宜添加剂量的锌可改善蛋禽体内的抗氧化状态，在基础饲粮锌含量约为 $3 0 \mathrm { m g / k g }$ 时，建议产蛋鸡饲粮锌的额外添加量为 $6 0 { \sim } 7 0 ~ \mathrm { m g / k g }$ ，产蛋鸭饲粮锌的额外添加量为 $3 0 { \sim } 4 5 ~ \mathrm { m g / k g }$ 锌调控蛋种禽抗氧化机能的研究较少，但大量研究表明锌与机体的免疫系统密切相关[40-42]。锌可维持免疫器官的正常功能，促进免疫细胞增殖和分化，调节多种细胞因子的分泌和活性，还可调节免疫细胞的凋亡[43]。产蛋前，蛋禽已基本完成免疫系统的生长发育，因此，锌对育雏育成期蛋禽免疫能力的调控作用更明显。缺锌（ $2 2 . 9 \mathrm { m g / k g }$ VS. $1 0 0 . 0 \mathrm { m g / k g }$ ）严重抑制雏鸭（1\~49日龄）免疫器官的生长发育和外周血T淋巴细胞的活化，显著降低血清免疫球蛋白（IgG）的含量和红细胞的免疫功能，并对免疫器官造成明显的病理损伤，导致免疫功能下降[44]。基础饲粮（锌含量为 $3 3 . 3 ~ \mathrm { m g / k g }$ ）添加硫酸锌可提高蛋雏鸭（0\~4周龄）免疫器官的发育，提高血清免疫球蛋白和白细胞介素-2（IL-2）浓度，额外添加 $4 8 . 2 { \sim } 5 7 . 9 \mathrm { m g / k g }$ 锌效果较好[45]。基础饲粮（锌含量为$3 2 . 9 \mathrm { m g / k g }$ ）添加锌（0、30、60、200、 $\mathrm { 1 ~ 0 0 0 ~ m g / k g }$ ，硫酸锌）对育成鸭免疫性能影响的结果表明，低锌（ $0 \mathrm { m g / k g }$ ）和高锌（ $\mathrm { 1 ~ 0 0 0 ~ m g / k g }$ ）明显抑制了蛋鸭的免疫性能， $3 0 \mathrm { m g / k g }$ 的添加剂量即可满足免疫器官的正常发育[39]。此外，锌可促进蛋鸡免疫器官的发育，增强细胞免疫功能[37],基础饲粮（锌含量为 $4 5 \mathrm { m g / k g }$ ）添加40和 $7 0 \mathrm { m g / k g }$ 锌可显著提高蛋鸡（57\~72 周龄）T淋巴细胞转化指数和血清IgG 含量，且蛋氨酸锌的效果较硫酸锌好[46]。可见，锌对蛋禽的免疫功能具有积极作用，但锌对产蛋期蛋鸭和蛋种禽免疫方面的研究未见报道。以免疫性能为评价指标，在基础饲粮锌含量约为 $3 0 \mathrm { m g / k g }$ 时，建议蛋鸭产蛋前饲粮锌的额外添加量为 $3 0 { \sim } 6 0 ~ \mathrm { m g / k g }$ ，产蛋期蛋鸡饲粮锌的额外添加量为 $4 0 { \sim } 7 0 \ \mathrm { m g / k g }$

# 2.4蛋禽对锌的营养需要量研究

NRC（1994）对家禽饲粮锌的一般推荐量为 $3 0 { \sim } 4 0 ~ \mathrm { m g / k g } .$ 。锌不足导致细胞氧化损伤增加[2.47],影响蛋禽的食欲和饲料利用率[7]，降低产蛋率和孵化率[48]，降低蛋壳品质[19.22]，增加经济损失。种蛋缺锌导致胚胎骨骼畸形，孵化的小鸡不能站立、进食和饮水[49]。锌在蛋禽体内主要在十二指肠被动吸收。当体内锌含量较高时，小肠细胞分泌大量的MT，与锌紧密结合，防止锌与富含半胱氨酸的蛋白结合而转运至血液，引起中毒[8]。因此，成熟蛋禽可耐受一定量的锌，但添加剂量为 $2 ~ \mathrm { g / k g }$ （丙酸锌）时会抑制蛋鸡产蛋，引起脱毛[50]。

商品蛋鸡饲粮锌的一般推荐量为 $7 0 { \sim } 1 4 0 \ \mathrm { m g / k g ^ { [ 1 1 \mathrm { - } 1 2 , 5 1 ] } }$ ，再高（150或 $2 0 0 \mathrm { m g / k g }$ ，硫酸锌）对蛋壳品质无显著改善作用[52]，过高( $1 3 7 { \sim } 6 5 5 \mathrm { m g / k g }$ ，螯合锌）则影响产蛋性能，降低蛋壳重量[15]。此外，基础饲粮（锌含量为 $2 7 . 9 5 \mathrm { m g / k g }$ ）添加锌 $3 0 { \sim } 1 2 0 \mathrm { m g / k g }$ (硫酸锌)，以骨锌含量为评价指标，褐壳产蛋鸡（ $2 0 { \sim } 4 0$ 周龄）锌的最适添加剂量为 $4 4 ~ \mathrm { m g / k g ^ { [ 3 8 ] } }$ 。虽然，稍过量的锌对蛋禽生产和蛋品质无显著负面作用，但会增加锌排放[51]，影响环境健康。综合考虑，建议产蛋期蛋鸡饲粮锌含量为：基础饲粮（锌含量 $2 5 { \sim } 3 0 \mathrm { m g / k g }$ ）额外添加 $6 0 { \sim } 8 0 \mathrm { m g / k g }$ 锌。饲粮添加适宜剂量（ $\cdot 3 0 { \sim } 4 5 ~ \mathrm { m g / k g }$ ）的锌（硫酸锌）可提高蛋鸭的产蛋率和饲料转化率，提高抗氧化机能，但添加剂量再增加（ $4 5 { \sim } 9 0 \ \mathrm { m g / k g } )$ ，无进一步改善作用[10]。基础饲粮（锌含量为32.9 mg/kg）外源添加 $3 0 { \sim } 6 0 ~ \mathrm { m g / k g }$ 锌（硫酸锌）即可满足育成期蛋鸭（4\~10 周龄）的生长需要[39]。综合生长性能、机体抗氧化功能、激素代谢和免疫机能结果，育雏期（0\~4周龄）蛋鸭饲粮的锌含量为：基础饲粮（锌含量为 $3 3 . 3 ~ \mathrm { m g / k g }$ ）额外添加 $5 1 . 8 { \sim } 5 3 . 0 \ \mathrm { m g / k g }$ 锌（硫酸锌）[45]。综合推荐，产蛋鸭饲粮锌含量为：基础饲粮（锌含量约为 $3 0 \mathrm { m g / k g }$ ）额外添加 $3 0 { \sim } 5 0 \mathrm { m g / k g }$ 锌。近期关于蛋种禽对锌需要量的研究较少。早期研究表明，基础饲粮中含有 $2 8 \mathrm { m g / k g }$ 的锌即可满足蛋种鸡正常的生产需要和后代发育，额外添加10、20、 $4 0 \mathrm { m g / k g }$ 碳酸锌[53]，甚至更高剂量（20、200、2000 mg/kg）的硫酸锌无显著的积极作用[54]。但也有研究表明，缺锌降低种蛋孵化率和胚胎的生长发育[48]。可见，锌在蛋种鸡生产上的应用结果不同。蛋种鸭对锌需要量的研究甚少，国内外均缺乏相应的标准。仅有研究表明，不同锌源（氧化锌、纳米氧化锌、碱式氯化锌、氨基酸螯合锌）和添加剂量（20、 $8 0 \mathrm { m g / k g }$ ）对山麻鸭的平均蛋重、产蛋率、料蛋比、受精率以及孵化率无显著影响[5]，但研究并不系统。随着育种技术发展及饲养模式和环境的变化，更需探究锌在蛋种禽的需要量。

# 3小结

综上，适宜剂量的锌可提高蛋禽的生产性能、产品品质和机体免疫和抗氧化功能。研究虽取得较大进展，但仍存在一些问题：1）多数研究围绕产蛋蛋禽展开，缺乏锌在蛋种禽生产应用的系统性研究。一方面，锌在蛋种禽的需要量有待确定；另一方面，锌在蛋黄的沉积或胚胎给养锌对种蛋孵化率、胚胎的存活率及后代生长发育的影响有待研究。2）与蛋鸡相比，锌对蛋鸭影响的研究相对较少。蛋鸭养殖是我国畜牧业的重要部分，并育有多种品系，深入营养素调控蛋鸭生产和品质的研究，可更好地促进蛋鸭养殖业的健康发展，如：富锌鸭蛋的研究；锌对蛋鸭蛋壳品质的影响及调控机理。3）锌在蛋禽中的应用研究缺少机理性的探索。文献阅读时，通常会发现相似地试验设计，不同的试验结果，根本原因是缺乏机理性的探讨，如：锌调控蛋禽生产性能及免疫机能等的机理。从机理入手，可更好地解释试验结果及不同结果之间的差异，从中获得更有价值的研究结果，更好地指导和应用于生产。

Abstract: The regulation of body health,egg and eggshell quality of laying bird have atracted much attention to promote the healthy and sustainable development of poultry industry. Zinc $( Z \boldsymbol { \mathrm { n } } )$ is one of the essential trace elements in laying birds,which participates the body's metabolism as the component or activator of enzymes and active proteins,and exerts important biological functions.It can modulate the antioxidant and immune status,and affect the quality of eggs and eggshells of laying birds.Therefore, this paper summarizes the biological functions of $Z \mathrm { n }$ in laying birds and the research progress of $Z \mathrm { n }$ application in laying bird production, and puts forward the problems existed and prospects for further study, and then provides practical techniques for Zn application in laying bird production. Key words: zinc; performance; egg quality; antioxidant and immune status; laying birds
