# 克服窗口玻璃影响的辐射测量方法

赵越」徐东耀」俞伦鹏²（」中国矿业大学（北京）化学与环境工程学院，北京100083

摘要：对于有窗口玻璃的高温炉的温度测量，如何克服窗口玻璃的影响一直是个难题。本文介绍一种方法，采用增加1块相同窗口玻璃2次（简称为 $T _ { \mathrm { l } } + \Delta T$ ）测量方法，以达到消除窗口玻璃的影响；通过理论分析，通过数值模拟计算、实验验证证明该方法是可行的。通过高温黑体校准和高温辐射温度计校准应用事例，证明 $T _ { \mathrm { l } } + \Delta T$ 克服窗口玻璃的辐射测温方法，具有科学性，合理性，方法简单，方便实用。对于有窗口玻璃的高温炉辐射温度测量、高温辐射温度计的量值传递等应用，具有指导意义。

# 0前言

对大于 $1 2 0 0 ^ { \circ } \mathrm { C }$ 以上温度测量，通常采用辐射测量方法。在冶金、材料、以及材料热物性、温度计量测试等行业中，所涉及的高温炉温度有 $1 4 0 0 ^ { \circ } \mathrm { C }$ 、$2 0 0 0 ^ { \circ } \mathrm { C }$ 、 $2 5 0 0 ^ { \circ } \mathrm { C }$ 、 $3 0 0 0 ^ { \circ } \mathrm { C }$ ，最高温度可以达到$3 2 4 0 ^ { \circ } \mathrm { C }$ 。

这些高温炉，大部分是封闭的，内部抽真空或充有氮气、氩气等惰性气体。高温炉通常留有能透光的窗口，使用辐射温度计和辐射温度传感器通过窗口对高温炉内的温度进行测量和控制。窗口材料通常为石英玻璃、红外玻璃等，但也有采用氟化钙晶体、锗晶体等其他材料。为了方便起见，以下统称“窗口玻璃”。

对于有窗口玻璃的情况下，为了克服窗口玻璃的影响，找出一种既方便又适用的辐射测温方法，很有实际指导意义和实用价值，这就是本文的目的。

# 1理论分析

# 1.1全辐射测量原理

对于实际物体辐射能量与温度的关系用斯忒藩一波尔兹曼定律表达为

$$
\begin{array} { r l r } { { \cal E } } & { { } = } & { \varepsilon \sigma _ { \mathrm { ~ 0 ~ } } { \cal T } ^ { \mathrm { ~ 4 ~ } } } \end{array}
$$

$E$ 一辐射能量（也称为辐射力）， ${ \mathbb W } / { \mathfrak { m } } 2$ ； $T$ 一物体的绝对温度 K； $\sigma _ { \mathrm { ~ 0 ~ } }$ 一斯忒藩一波尔兹曼常数，5.67$\times 1 0 { - } 8 \mathbb { W } / \left( \mathrm { m 2 } \cdot \mathrm { K 4 } \right)$ ； $\varepsilon$ 一发射率。当有窗口玻璃存在时，式(1）表达为

$$
{ \cal E } _ { 1 } \ = \ \tau { \cal E } \ = \ \tau \varepsilon \sigma _ { 0 } T ^ { 4 } \ = \ \varepsilon \sigma _ { 0 } T _ { 1 } ^ { 4 }
$$

$E _ { 1 }$ 一有1块窗口玻璃的辐射能量（也称为辐射力)， $\mathbb { W } / \mathrm { m } 2$ ； $\tau$ 一为窗口玻璃的透射率 $\tau { \bf \Psi } = { \bf \Psi } { \cal E } _ { 1 } / E$ 。$T _ { 1 }$ 一为包含1块窗口玻璃表观温度。如果我们在辐射窗口再加一块相同的窗口玻璃，式（2）表达为

$$
{ \cal E } _ { 2 } = \tau { \cal E } _ { 1 } = \tau \tau \varepsilon \sigma _ { 0 } T ^ { 4 } = \varepsilon \sigma _ { 0 } T _ { 2 } ^ { 4 }
$$

$E _ { \mathrm { 2 } }$ 一有2块窗口玻璃的辐射能量（也称为辐射力）， ${ \mathbb W } / { \mathfrak { m } } 2$ ； $T _ { \mathrm { 2 } }$ 一为包含2块窗口玻璃表观温度。

从式（1）～式（3）化简得出

$$
\begin{array} { r l r } { T } & { { } = } & { { { \cal T } _ { 1 } } ^ { 2 } \Big / { \cal T } _ { 2 } } \end{array}
$$

设 $T _ { 1 } \ - \ T _ { 2 } \ = \ \Delta { T }$ ，式（4）化简为

$$
 { T } =  { T } _ { 1 } + \Delta  { T } + \Delta  { T } ^ { 2 } \big / T _ { 2 }
$$

前面已经叙述，对于有窗口的高温炉，工作温度都在 $1 0 0 0 ^ { \circ } \mathrm { C }$ 以上,对于式(5),让 $\Delta { T } ^ { 2 } / T _ { 2 } \ \leq \ 0 . \ 5$ （注意计算中的温度单位为K)，得出

$$
\begin{array} { r l r } & { 1 0 0 0 ^ { \circ } \mathrm { C } \quad } & { \Delta T \ \leq \ 2 5 \ . 2 3 \ ^ { \circ } \mathrm { C } } \\ & { 2 0 0 0 ^ { \circ } \mathrm { C } \quad } & { \Delta T \ \leq \ 3 3 \ . 7 1 \ ^ { \circ } \mathrm { C } } \\ & { 3 0 0 0 ^ { \circ } \mathrm { C } \quad } & { \Delta T \ \leq \ 4 0 \ . 4 5 \ ^ { \circ } \mathrm { C } } \end{array}
$$

满足上述条件，式（5）直接化简为

$$
\begin{array} { r l r } { T } & { { } = } & { T _ { 1 } + \Delta \ T } \end{array}
$$

式（7）就是一个将舍弃误差控制在 $0 . 5 ^ { \circ } \mathrm { C }$ 简化的计算公式，只要满足式（6）的要求，对于有玻璃窗口的高温炉，克服玻璃窗口的辐射温度测量变得非常简单。也就是首先测量高温炉温度，增加一块相同的玻璃窗口再测量1次，炉内温度为第1次测量温度值，加第1次与第2次温度测量的差值，以下简称 $\begin{array} { r } { T \ = \ T _ { 1 } + \Delta T } \end{array}$ 方法。

# 1.2单波长辐射测量原理

上面分析是基于全辐射的原理，导出的表达式简单直观。但实际测量使用的辐射温度计并不是全波长，而是单波长或波段，因此本节使用单波长辐射原理进行分析。

对于实际物体单色辐射能量与温度的关系用普朗克定律表达为

$$
E _ { \lambda } ~ = ~ \varepsilon _ { \lambda } ~ \frac { C _ { 1 } \lambda ^ { - 5 } } { e ^ { C _ { 2 } / \lambda T } ~ - ~ 1 }

$$
\dot { E _ { \lambda } } = \tau _ { \lambda } E _ { \lambda } = \tau _ { \lambda } \varepsilon _ { \lambda } \frac { C _ { 1 } \lambda ^ { - 5 } } { e ^ { C _ { 2 } / \lambda T } - 1 } = \varepsilon _ { \lambda } \frac { C _ { 1 } \lambda ^ { - 5 } } { e ^ { C _ { 2 } / \lambda T _ { 1 } } - 1 }

$\boldsymbol { \tau } _ { \lambda }$ 一为窗口玻璃的光谱透射率。 $\tau _ { \lambda } ~ = ~ E _ { \lambda } ^ { ' } / E _ { \lambda }$ ：$T _ { \mathrm { ~ 1 ~ } }$ 一为包含1块窗口玻璃表观温度。如果我们在辐射窗口再加一块相同的窗口玻璃，式（9）表达为

# 1.3带宽辐射测量原理

在温度一定时，黑体光谱辐射特性遵循普朗克定律。如果有窗口玻璃，尽管窗口玻璃的透过率一致，也就是说对辐射能量衰减的比例相同，但是显现出温度变化是不一样的。从表9中可以看出，对于波长 $0 . 6 6 1 1 6 \mu \textrm { m }$ 和 $1 . 6 \mu \mathrm { ~ m ~ }$ ， $\Delta T$ 分别为 $1 1 . 4 ^ { \circ } \mathrm { C }$ 和 $2 7 \mathrm { { ^ \circ C } }$ ，尽管波长相差约 $0 . 9 4 \mu \mathrm { ~ m ~ }$ ，而 $\Delta T$ 相差约$1 5 . 6 ^ { \circ } \mathrm { C }$ 。

这说明，在校准辐射温度计时，不同的工作波长， $\Delta T$ 会有很大的变化， $\Delta T$ 在不同波长之间不能互用。就是相同波长辐射温度计产品，在生产时其工作波长总是存在一定差异，用本文介绍的 $T _ { 1 } + \Delta T$ 方法，现场实验实时克服窗口玻璃的影响，更具有准确性和实用性。

# 5结论

对于有窗口玻璃的高温炉的温度测量，如何克服窗口玻璃的影响一直是个难题。本文通过理论分析、数值计算、实验验证和应用事例，使用克服窗□玻璃的辐射测温方法，具有科学性，合理性，方法简单，方便实用。对于有窗口玻璃的高温炉辐射温度测量、高温辐射温度计的量值传递等应用，具有指导意义。
