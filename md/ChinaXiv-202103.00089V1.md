# 肥料酸化对灌漠土pH的影响

陈震¹²，车宗贤¹²，张久东'，崔云玲¹，张立勤¹（1.甘肃省农业科学院土壤肥料与节水农业研究所,甘肃 兰州730070;2.甘肃农业大学资源与环境学院，甘肃 兰州730070)

摘要：明确水肥一体化条件下不同酸度水肥溶液对灌漠土pH的改良效果,为碱性土壤合理施肥提供参考。本试验设置5个灌溉溶液pH:A1(1.5)、A2(3.5)、A3(5.5)、A4(7.5)(对照)和A5(9.5),两个灌水量 $\mathrm { W } 1 ( 1 5 0 0 \mathrm { ~ m } ^ { 3 } \cdot \mathrm { h m } ^ { - 2 } )$ 和W2$( 3 0 0 0 \mathrm { m } ^ { 3 } { \cdot } \mathrm { h m } ^ { - 2 } )$ ,两个施氮量 $\mathrm { N 1 } ( 4 5 0 \mathrm { k g } \cdot \mathrm { h m } ^ { - 2 } )$ 和 $\mathrm { N } 2 ( 9 0 0 \mathrm { k g } \cdot \mathrm { h m } ^ { - 2 } )$ ,共20个处理,通过盆栽试验进行研究。结果表明：（1）土壤pH与灌溉溶液pH显著正相关，A1处理下 $0 { \sim } 5 ~ \mathrm { c m }$ 土层土壤pH可降至7.16\~7.4，相比A2、A3、A4、A5四个处理降低了 $0 . 4 5 \mathrm { \sim } 0 . 8 1$ ,与各处理间差异显著。(2)增加灌水量或施氮量对土壤 $\mathrm { \ p H }$ 无显著影响。(3）土壤pH在灌溉溶液的当天最低，然后慢慢恢复，且随着灌溉次数的增加，土壤 $\mathrm { \ p H }$ 越来越低。（4）各处理下土壤pH随土层深度的增加而增加，其中A1处理下 $0 { \sim } 5 \mathrm { \ c m } \ . 5 { \sim } 1 0 \ \mathrm { c m } \ . 1 0 { \sim } 2 0 \ \mathrm { c m } \ . 2 0 { \sim } 3 0 \ \mathrm { c m } \ . 3 0 { \sim } 4 0 \ \mathrm { c m }$ 土层土壤 $\mathrm { \ p H }$ 分别为7.27、7.67、7.84、7.98、8.08，土层间差异最大。酸性溶液能显著降低灌漠土 $\mathrm { \ p H }$ ,不论是灌溉溶液 $\mathrm { \ p H }$ ,还是灌水量，或是施氮量，主要是改变了耕层土壤的pH,对底层土壤pH影响不大。所以,在农业生产中,可通过调节溶液酸度,降低碱性土壤pH,为作物的健康生长提供保障。

关键词：酸性肥料；土壤pH；灌漠土；黑麦草；水肥一体化

作为一个巨大的缓冲体系，土壤对酸、碱具有较强的缓冲性能。在我国，土壤呈现出南酸北碱的趋势，虽然近年来土壤酸化问题比较严重，但在北方，土壤pH仍处于一个比较高的水平，有些地区能达到9以上。作物的生长需要一个适宜的pH范围，土壤pH过高或过低都会对作物造成一定影响。大多数作物最适宜的 $\mathrm { \ p H }$ 范围为6.5～7.5,碱性土壤严重降低土壤养分的有效性，制约着农业的绿色发展。

土壤中 $\mathrm { \ p H }$ 深度对于植物的重要性不言而喻，对大多数作物来说，降低碱土 $\mathrm { \ p H }$ ,提供一个适宜的$\mathrm { \ p H }$ 环境可以促进其健康生长。李燕婷等研究了酸性根际肥与石灰性土壤 $\mathrm { \ p H }$ 之间的关系，结果表明，这种新型肥料可显著降低肥料颗粒 $2 \mathrm { c m }$ 范围内的土壤pH,提高了该区域土壤中有效铁的含量。张莉等研究发现，磷酸脲能有效减轻菠菜的碱胁迫程度，并作为碱土的改良剂促进植物健康生长。田吉林等3发现，通过灌溉酸化水，土壤pH会显著降低，从而改善烤烟品质。一些有机物料对土壤pH有很好的改良效果，郭军玲等4在大同盆地苏打盐化土上的研究表明,风化煤和牛粪处理下的土壤pH降幅最大，改良效果较好。唐继伟等[5研究发现，化肥配施有机肥，可维持盐化潮土 $\mathrm { \ p H }$ 的稳定性。此外，秸秆及其制成的生物炭也能有效地改良土壤$\mathrm { p H } ^ { [ 6 - 9 ] }$ 。大量研究表明,酸性物质可降低碱土的pH,在此背景下，本试验通过对肥料进行不同程度地酸化，配合水肥一体化技术,研究肥料pH及用量与土壤pH之间的关系，以期为碱土改良提供参考，并为后期新型肥料的研发提供理论支持。

# 2结果与分析

# 2.1模拟试验期间土壤pH动态变化

随着灌溉溶液的加入，各处理下的土壤pH处于不断地变化状态(图2)，灌溉溶液加入当天的土壤 $\mathrm { \ p H }$ 小于前一天。随着灌溉次数的增加，S1\~S6的土壤pH呈现出一个逐渐降低的趋势，但S2和S5处理中的土壤pH在试验后期有一个先上升后下降的过程，具体原因有待考证。S1和S6的土壤pH在试验前期波动较大，而中后期的变化比较平稳，与S1、S6不同,S2、S3、S4、S5的土壤pH一直处于相对波动较大的变化趋势。主要由于碱溶液或过酸溶液带入的氢氧根离子或氢离子较多，一直与土壤中的相关离子发生中和反应，从而使得土壤pH处于一个相对稳定的状态。而近中性溶液或酸性稍弱的溶液在灌溉当天就中和完毕，然后土壤pH又开始慢慢恢复。

# 3讨论

土壤 $\mathrm { \ p H }$ 的改变是一个十分缓慢的过程，近年来，很多学者围绕如何降低碱性土壤pH开展了大量的工作,也促进了一些新技术、新型肥料[2.11-12]的出现,比如,王晓刚等[13]发现液体肥料对土壤 $\mathrm { \ p H }$ 有明显的改良作用。本试验是以水肥一体化技术为支撑进行研究的，在模拟试验中，土壤pH随着灌溉溶液的加入上下波动，相比对照，酸性溶液对土壤$\mathrm { \ p H }$ 的降低程度显著， $\mathrm { \ p H }$ 为1的灌溉溶液处理下的土壤 $\mathrm { \ p H }$ 相比对照降低了0.35,显著低于其他各处理，可见，酸性物质对灌漠土pH的影响是非常大的，这与前人的研究一致[1.3.14]。张皓禹等[15]在滴灌条件下研究了几种酸化剂或酸性肥料对石灰性土壤的改良效果，发现这几种物质都能降低土壤pH,并提高养分的有效性，且土层越深，各处理下的土壤 $\mathrm { p H }$ 越高。本试验也有类似结果，但对于灌溉溶液 $\mathrm { \ p H }$ 为9.5的处理， $0 { \sim } 5 ~ \mathrm { c m }$ 土层土壤 $\mathrm { \ p H }$ 要略高于 $5 { \sim } 1 0 ~ \mathrm { c m }$ 土层，主要由于碱性溶液对表层土壤的作用比较强，导致土壤 $\mathrm { \ p H }$ 的降幅较小。同时， $\mathrm { \ p H }$ 为1.5的灌溉溶液对表层土壤pH降低程度非常明显，与原土相比，降幅超过1,相比当地灌溉水 $\mathrm { \ p H }$ 的平均水平,其他两个酸性灌溉溶液也能显著降低土壤pH。

一般来说,土壤中酸性物质的施入量越大，土壤pH降低幅度越大[16]。刘佳等[12]研究发现,化肥的长期施用会严重加剧旱地红壤的酸化程度。吴曦等"通过盆栽试验发现碱土pH随硫磺使用量的增加而呈降低趋势，但硫水平过高会对油菜的生长不利。在盆栽试验中，高灌溉量处理下， $\mathrm { \ p H }$ 为1.5的灌溉溶液处理下 $0 { \sim } 5 ~ \mathrm { c m }$ 土层土壤 $\mathrm { \ p H }$ 为7.16,而低灌溉量处理下为7.35\~7.4,两者差异显著，可见，土壤pH与灌水量，也就是进人土壤中的 $\mathrm { H ^ { + } }$ 的量呈显著正相关。常青等[18]通过研究不同稀释倍数的木醋液也得到了类似结果。对比模拟试验与盆栽试验的土壤pH,可以发现，尽管模拟试验和盆栽试验所设的灌溉溶液 $\mathrm { \ p H }$ 差异不大,但在盆栽试验耕层，尤其是 $0 { \sim } 5 ~ \mathrm { c m }$ 土层土壤 $\mathrm { \ p H }$ 比模拟试验的土壤pH要低很多，这主要是由于在盆栽试验中，大量的灌溉溶液首先与表层土壤反应，导致表层土壤pH产生了非常大的变化。

近年来，由于大量地使用氮肥，加快了土壤酸化的进程[19],尿素进入土壤后，虽短时间内能提高土壤 $\mathrm { p H } ^ { [ 2 0 - 2 2 ] }$ ,但最终还是会使土壤慢慢酸化。不过在本试验中，增加灌水量或施氮量仅能使土壤的平均 $\mathrm { \ p H }$ 降低0.03,差异并不显著，这主要是由于尿素对土壤 $\mathrm { \ p H }$ 的降低是一个长期累积的过程,而本试验持续时间短，施氮量并不足以引起土壤 $\mathrm { \ p H }$ 的显著变化。

# 4结论

酸性溶液能显著降低灌漠土土壤pH,灌溉溶液进入王壤后，土壤pH先是下降，然后慢慢恢复，且随着灌溉次数的增加，土壤 $\mathrm { \ p H }$ 越来越低。pH为1.5的灌溉液可使 $0 { \sim } 5 ~ \mathrm { c m }$ 土层土壤pH降至7.16\~7.4,与各处理间差异显著。各处理下土壤pH随土层深度的增加而增加，当灌溉溶液 $\mathrm { \ p H }$ 为1.5时，各土层间土壤pH差异最大。增加灌水量或施氮量对土壤pH影响不显著。不论灌溉溶液pH,还是灌水量或施氮量，主要是改变了耕层土壤的 $\mathrm { \ p H }$ ,对底层土壤 $\mathrm { \ p H }$ 影响不大。
