我国是芦笋生产大国，2015年芦笋产量约为696万t，芦笋下脚料产量约为200 万t。

芦笋下脚料是芦笋加工副产物，它含有丰富的维生素、多糖、必需氨基酸和生物活性物质（皂甙类和黄酮类化合物）[1-2]。研究发现，给泌乳牛羊饲喂芦笋下脚料能显著提高其产奶量和乳汁品质[3]，给育肥猪补饲芦笋下脚料能提高其日增重和饲料转化率[4]。新鲜芦笋下脚料存在难保存、粗纤维含量高和适口性差等问题，限制了其在动物生产中的广泛应用；而经过微贮处理则能有效延长新鲜芦笋下脚料的保存时间，并降低其粗纤维含量[5]。本课题组前期研究发现，芦笋下脚料经固态发酵后能提高其营养价值，改善其适口性，给妊娠后期和哺乳期母猪补饲后能增强其肠道蠕动，减少便秘并改善乳汁质量[]。但有关饲喂发酵芦笋下脚料对母猪及其仔猪血清指标影响的研究尚未见报道。本试验通过给妊娠后期及哺乳期母猪补饲发酵芦笋下脚料，研究其对母猪和哺乳仔猪血清抗氧化指标及激素、免疫球蛋白和炎性因子含量的影响，为发酵芦笋下脚料在母猪健康养殖生产中的合理使用提供技术支持。

1材料与方法

# 1.1 试验材料

发酵芦笋下脚料：将新鲜芦笋下脚料洗净后，经粉碎机粉碎至 $3 \ \mathrm { c m }$ 左右，自然晾干至手抓后指缝间无汁液流出（水分含量约为 $6 5 \%$ )。将秸秆纤维分解剂（主要成分为酶制剂、乳化剂、益生菌等)、玉米粉和豆粕分别按 $0 . 1 \%$ 、 $5 . 0 \%$ 和 $5 . 0 \%$ 的添加量搅拌均匀，然后与晾干后的芦笋下脚碎料充分混匀，最后喷洒 $1 . 0 \%$ 氯化钠溶液，再混匀。把混匀后的芦笋下脚料装于 $2 5 \mathrm { k g }$ 的发酵袋内，装至袋子的2/3处，尽量排除袋子里的空气，用扎带扎紧袋口，在室温条件下发酵 $\boldsymbol { 1 4 \mathrm { d } }$ ，制成发酵芦笋下脚料，其主要营养成分及含量见表1。

# 1.2 试验设计

选择15头中等膘情、胎龄为2\~3胎且预产期接近（1周之内）的“长白 $\times$ 大约克夏”杂种母猪，随机分成3组，每组5个重复，每个重复1头猪。A组（对照组）母猪饲喂基础饲粮,B和C组母猪在饲喂基础饲粮的同时每头每天分别补饲0.25和 $0 . 5 0 \mathrm { k g }$ 发酵芦笋下脚料。各组母猪每头每天的基础饲粮饲喂量相同，基础饲粮参照NRC(1998)标准配制，基础饲粮组成及营养水平见表2。

# 1.3 饲养管理

试验于江西新余市姚于镇万佳猪场进行。正式试验从母猪妊娠期的第85天开始至产后第 21天结束，试验期间记录各组仔猪的死亡头数，在仔猪初生和21日龄早晨分别对哺乳仔猪进行空腹称重。发酵芦笋下脚料每天分2次饲喂，饲喂2h后再饲喂全价饲粮。正式试验开始前先进行7d的预备试验，其余饲养管理和防疫等按猪场的正常程序进行。

# 1.4样品采集

母猪血样：分别于母猪分娩后 $2 4 \mathrm { h }$ 内和断奶当天08：00空腹采血。采用耳缘静脉采血${ \mathfrak { s } } _ { \mathrm { m L } }$ ， $4 \mathrm { { ^ \circ C } }$ 静置 $3 0 \mathrm { m i n }$ 、 $3 \ : 0 0 0 \ : \mathrm { r / m i n }$ 离心 $1 5 \mathrm { m i n }$ 后吸取血清，分装于 $0 . 5 ~ \mathrm { m L }$ 离心管后，于$- 2 0 \ \mathrm { ^ { \circ } C }$ 冷冻保存待测。

仔猪血样：分别于仔猪10和21日龄08：00空腹采集健康、体重相近仔猪的前腔静脉血液，方法与母猪血样同。

# 1.5 测定指标与方法

血清免疫球蛋白[免疫球蛋白A(immunoglobulinA,IgA)、免疫球蛋白G(immunoglobulinG,IgG）、免疫球蛋白M(immunoglobulinM,IgM)]、炎性因子[白细胞介素-1β(interleukin- $1 \beta$ IL-1β)、白细胞介素-6(interleukin-6,IL-6）、肿瘤坏死因子 $- \alpha$ ( tumor necrosis factor-a, TNF- $\mathbf { \nabla } \cdot \mathbf { a }$ 0和激素[生长激素(growth hormone，GH)、胰岛素（insulin，INS)]含量均采用酶联免疫法测定，血清总超氧化物歧化酶(total superoxide dismutase，T-SOD)活性采用羟胺法测定，血清丙二醛(malondialdehyde，MDA)含量采用硫代巴比妥酸（TBA）法测定。试剂盒均购自南京建成生物工程研究所。

# 1.6 数据处理与统计分析

原始数据用 Excel 2003 进行整理，采用 SPSS 17.0 软件进行单因素方差分析（one-wayANOVA)，组间平均值采用Duncan 氏多重比较法进行差异显著性检测，结果以“平均值±标准误”表示。

# 3讨论

# 3.1母猪补饲发酵芦笋下脚料对断奶仔猪数的影响

母猪年产断奶仔猪数是衡量现代猪场生产成绩最常用和最重要的指标之一，也是衡量猪场经济效益的主要指标，断奶仔猪数越多，猪场所获得的经济效益越高。提高哺乳仔猪成活率是提高断奶仔猪数的重要措施之一，而哺乳仔猪抗病力的高低直接影响到哺乳仔猪的成活率。研究表明，提高哺乳母猪乳汁中免疫球蛋白含量对增强哺乳仔猪的抗病力有重要作用，而给哺乳母猪补饲发酵芦笋下脚料能有效提高母猪初乳和常乳中免疫球蛋白含量并降低乳汁中促炎因子的含量。当哺乳仔猪吮吸免疫球蛋白含量高的乳汁后，有助于提高仔猪对疾病的抵抗力，从而提高其成活率，这是本试验中母猪补饲发酵芦笋下脚料能有效提高断奶仔猪数的重要原因之一。

3.2母猪补饲发酵芦笋下脚料对母猪和哺乳仔猪血清抗氧化指标的影响

T-SOD和MDA是反映机体抗氧化能力的生化标志物。T-SOD是机体内重要的抗氧化酶，可以清除机体产生的内源性自由基，减少机体组织细胞由于自由基的作用所产生的伤害。MDA会攻击细胞生物膜上的不饱和脂肪酸,导致脂质过氧化作用的产生，对细胞具有毒性。

研究表明，芦笋多糖对自由基有较强的清除作用，在大鼠血清、细胞和细胞器等水平上都有较好的抗氧化活性[7]。芦笋黄酮能够保护 T-SOD，防止其失活[8]，降低 MDA 含量[9]，对损伤的组织和细胞起到修复作用[10]。芦笋笋基乙醇提取物能够显著提高小鼠血清 T-SOD 活性，高剂量可降低血清 MDA含量[1]。与对照组相比，补饲发酵芦笋下脚料的母猪乳汁中 T-SOD活性显著升高，MDA 含量有下降趋势，而乳汁中的抗氧化物质主要来自于血液。本试验结果显示，每头母猪每天补饲 $0 . 5 0 ~ \mathrm { { k g } }$ 的发酵芦笋下脚料，能极显著提高断奶时母猪血清T-SOD 活性，极显著降低分娩和断奶时母猪血清 MDA含量，说明补饲发酵芦笋下脚料能够提高母猪机体的抗氧化能力。母猪血清中的抗氧化物质能够转运到乳汁，而乳汁中抗氧化物质的含量可能会影响仔猪的抗氧化能力。毛春瑕等[研究发现，补饲发酵芦笋下脚料能够显著提高母猪乳汁 T-SOD 活性，乳汁 MDA 含量有下降趋势。本试验结果显示，每头母猪每天补饲 $0 . 2 5 ~ \mathrm { k g }$ 发酵芦笋下脚料，10 日龄哺乳仔猪血清T-SOD 活性显著高于对照组，血清MDA 含量显著低于对照组，说明给母猪补饲发酵芦笋下脚料能同时提高母猪和哺乳仔猪的抗氧化能力。

3.3母猪补饲发酵芦笋下脚料对母猪和哺乳仔猪血清激素含量的影响

GH 能促进三大营养物质的代谢，提高机体对饲粮养分的吸收能力[12]。提高猪体内GH含量可显著提高猪的增重和母猪的泌乳量[13]。芦丁具有弱雌激素作用，雌激素可以通过调节 GH 等激素的分泌来发挥间接作用[14]。乳源性 INS 能够刺激新生仔猪小肠生长并影响到小肠的组织形态[15]。研究发现，给母猪补饲发酵芦笋下脚料对乳汁中GH和 INS 含量无显著影响[。本试验结果表明，C组母猪分娩时血清GH含量显著高于对照组，说明每头母猪每天补饲 $0 . 5 0 ~ \mathrm { k g }$ 发酵芦笋下脚料促进了母猪体内GH的产生。这可能与母猪分娩时芦笋黄酮发挥了弱雌激素功能，从而提高了母猪体内GH含量有关。但各组哺乳仔猪10 和21日龄时血清GH 和 INS 含量均无显著差异，这可能是由于母猪补饲发酵芦笋下脚料后并没有改变母猪乳汁GH和 INS 含量。

3.4母猪补饲发酵芦笋下脚料对母猪和哺乳仔猪血清免疫球蛋白含量的影响

免疫球蛋白有抗体活性，广泛存在于细胞内液和外分泌液中。免疫球蛋白可以与体内病原微生物结合，从而清除体内病原体，减少病原体对机体的伤害。IgG 是一种重要的免疫球蛋白，约占机体免疫球蛋白总量的 $7 5 \% ^ { [ 1 6 ] }$ ，具有中和细菌、抗病毒和增强机体免疫细胞对病原体的吞噬等作用，对维护动物体健康起着重要的作用。已有研究表明，芦笋植株中含有皂苷和多糖等多种能增强动物机体免疫功能的物质[8.17-18]。本试验结果表明，C 组母猪分娩时血清IgA和IgG含量均显著高于对照组，说明每头母猪每天补饲 $0 . 5 0 \mathrm { k g }$ 发酵芦笋下脚料能提高母猪血清免疫球蛋白含量，加强母猪对病原微生物的抵抗作用。

3.5母猪补饲发酵芦笋下脚料对母猪和哺乳仔猪血清炎性因子含量的影响

IL-1β、IL-6 和 TNF- $\mathbf { \sigma } \cdot \mathbf { a }$ 是3种重要的白细胞介素。 $\mathrm { I L } { - } 1 \beta$ 主要由巨噬细胞产生，B淋巴细胞和自然杀伤（NK）细胞等也能产生，促进抗体形成抗原递呈。IL-6能刺激B细胞产生抗体，诱导T细胞增殖和分化并参与机体的免疫应答，是炎性反应的促发剂。TNF- $\mathbf { \nabla } \cdot \mathbf { a }$ 是炎性反应过程中重要的炎性介质，由单核细胞和巨噬细胞产生。蔡春水等[19研究发现，机体补充黄酮类物质能改善机体的炎性反应，降低血清中促炎性因子的含量，对机体的健康有积极的影响。芦笋提取物可提高小鼠的细胞免疫能力，提高 NK 细胞活性[20]。芦笋黄酮可以缓解炎性反应[21]。本试验结果表明，B和C 组母猪血清 IL-6 含量在分娩和断奶时均极显著低于对照组，C组母猪断奶时血清TNF- $\mathbf { \nabla } \cdot \mathbf { a }$ 含量显著低于对照组，表明补饲发酵芦笋下脚料能降低母猪血清炎性因子含量。

# 4结论

$\textcircled{1}$ 每头母猪每天补饲 $0 . 5 0 \mathrm { k g }$ 发酵芦笋下脚料，能显著提高断奶时母猪血清T-SOD活性和分娩时母猪血清GH、IgA 和IgG 含量，显著降低分娩和断奶时母猪血清 MDA、IL-6含量和断奶时母猪血清TNF- $\mathbf { \sigma } \cdot \mathbf { a }$ 含量。

$\textcircled{2}$ 每头母猪每天补饲 $0 . 2 5 \mathrm { ~ k g }$ 发酵芦笋下脚料，能显著提高10日龄哺乳仔猪血清T-SOD 活性，显著降低哺乳仔猪血清 MDA含量；每头母猪每天补饲 $0 . 5 0 ~ \mathrm { k g }$ 发酵芦笋下脚料，能显著降低10日龄哺乳仔猪血清TNF- $\mathbf { \sigma } \cdot \mathbf { a }$ 含量。

Effects of Supplementary Feeding of Sows with Fermented Asparagus By-Product on Antioxidant Capacity, Immune Function and Contents of Inflammatory Factors in Serum of Sows and Suckling Piglets   
LAI Tao’WENHong2\*SHI Xianliang'LI Lingyu1HE Yuyong'LU Weil\*\* (1. Jiangxi Province Key Lab ofAnimal Nutrition,Jiangxi Agricultural University, Nanchang 330045, China; 2. Jiangxi Province Feed Industry Office, Nanchang 330046, China) Abstract: This study was conducted to investigate the effects of supplementary feeding of sows with fermented asparagus by-product on antioxidant capacity, immune function and the contents of inflammatory factors in serum of sows and suckling piglets from late-pregnancy to the end of lactation. Fifteen pregnant sows with similar body condition, parity and parturition time were randomly assigned to three groups with five replicates per group and one sow per replicate. Sows were fed a basal diet supplemented with O, 0.25 and $0 . 5 0 \mathrm { k g }$ fermented asparagus by-product each day in groups A, B and C,respectively. The experiment lasted from day 85 after pregnancy to day 21 after delivery. Blood samples of sows were collected from the auricular vein at the time of sows delivery and piglets weaning, and blood samples of suckling piglets aged at 10 and 21 days were collected from anterior vena cava, and serum indexes were measured, respectively. The results showed as follows: 1) the activity of superoxide dismutase (T-SOD) in serum of sows in group C was significantly higher than that in control group at piglets weaning $( P { < } 0 . 0 1 )$ , and the

content of malondialdehyde (MDA） in serum in group C was significantly lower than that in control group at sows delivery and piglets weaning $( P { < } 0 . 0 1 )$ . The activity of T-SOD in serum of suckling piglets aged at 1O days in group $\mathbf { B }$ was significantly higher than that in control group （204号 $( P { < } 0 . 0 5 )$ ,and the content of MDA in serum in group B was significantly lower than that in control group $( P { < } 0 . 0 5 )$ . 2） The content of growth hormone (GH) in serum of sows in group C was significantly higher than that in control group at sows delivery $( P { < } 0 . 0 5 )$ .3）The contents of immunoglobulin A (IgA） and immunoglobulin G $\mathrm { ( I g G ) }$ in serum of sows in group C were significantly higher than those in control group at sows delivery $( P { < } 0 . 0 5 )$ .4） The content of interleukin-6 (IL-6) in serum of sows in groups $\mathbf { B }$ and C was significantly lower than that in control group at sows delivery and piglets weaning $( P { < } 0 . 0 1 )$ .The content of tumor necrosis factor- $\mathbf { \alpha } _ { \mathrm { ~ \normalfont ~ a ~ } }$ (TNF- $\mathbf { \nabla } \cdot \mathbf { a }$ ) in serum of sows in group C was significantly lower than that in control group at piglets weaning $( P { < } 0 . 0 5 )$ . The content of TNF- $\mathbf { \nabla } \cdot \mathbf { a }$ in serum of suckling piglets aged at 10 days in group C was significantly lower than that in control group $( P { < } 0 . 0 5 )$ ．It is concluded that supplementary feeding sows with $0 . 5 0 ~ \mathrm { k g }$ fermented asparagus by-product each day can improve antioxidant capacity and immune function, increase GH content and decrease the inflammatory response of sows, meanwhile, increase antioxidant capacity and decrease inflammatory response of piglets.

Key words: fermented asparagus by-product; sows; suckling piglets; serum indexes
