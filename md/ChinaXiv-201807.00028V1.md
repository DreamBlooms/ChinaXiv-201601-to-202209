# 1引言

智库，又称之为智囊团，最早诞生于二战时期的美国。在全球化、信息化飞速发展背景下，现代智库逐渐成为政府治理与决策体系中的外脑，成为国家“软实力”和“话语权”的重要体现。在当前中国学界话语体系中，智库普遍被认为是以公共政策为研究对象、以影响政府决策为研究目标、以公共利益为研究导向、以社会责任为研究准则的专业研究机构，学界一般将其划分为官方智库、高校智库和民间智库3种类型。从我国智库作用发挥总体情况看，官方智库在3类智库中居于主要地位。党的十八大以来，习近平总书记对建设中国特色新型智库专门做出批示，一系列推动智库发展的文件出台，中国特色新型智库建设上升为党中央战略部署和国家重要发展战略。根据美国宾夕法尼亚大学“智库研究项目组”（TTCSP）最新编写的《全球智库报告2017》显示，中国智库数量达512 家，继续位于全球第2位。这也证明了中国智库建设进入需要更迫切、要求更高、发展更迅速的“黄金时代”。

2015年1月中共中央办公厅、国务院办公厅印发的《关于加强中国特色新型智库建设的意见》明确，到2020年我国拟建成具有中国特色的新型智库体系，这个体系集党政部门、社科院、党校行政学院、高校、军队、科研院所和企业、社会智库为一体，协调发展，布局合理，定位清晰[2]。新时代党校智库建设要赶上发展黄金期，必须围绕这个方向用力。上海社会科学院智库研究中心出版的《2017中国智库报告》中数据显示，中共中央党校智库综合影响力2016 年排名第3，2017年排名第 $4 ^ { [ _ { 3 } ] }$ 。因此，党校作为党委的重要部门，作为党委、政府决策咨询的主阵地，开展新型智库建设具有重大意义。党校图书馆一直是党校的信息资源中心，在保障党校的教师和学员开展教学和科研活动中起着重要的作用。建设党校智库也离不开图书馆知识资源的保障和支持作用，同时新型智库建设对党校图书馆实现服务转型提出了要求。党校图书馆要充分把握智库建设面临的宝贵机遇，充分发挥自身信息资源优势，完善特色资源库，开展更适合党校智库建设的高质量服务，同时让新型智库建设为图书馆实现服务转型提供长足的支持和保障。

从党校图书馆服务智库建设研究相关文献看，学者对党校图书馆服务智库建设研究从2013年起步、到2016年共有7篇文章，但2017年全年已经有7篇文献发表。这从一定程度上反映出党校图书机构对服务新型智库建设长期以来缺乏足够的意识和重视，也缺乏信心与合理规划，更没有清晰认识到党校图书机构在智库建设过程中的作用功效与身份定位。令人振奋的是，从2017年文献发表数量增加的趋势来看，一些学者已经开始意识到党校图书馆服务智库建设的重要作用。图书馆是党校新型智库建设的重要组成部分，为智库建设提供不可或缺的知识和服务保障工作。2013年，施静等率先提出党校图书馆智库建设的可能性，并从培育政策分析市场、加强资政信息资源建设、构建支持智库建设的组织管理机制等方面提出党校图书馆智库建设的路径4。随后，一些学者围绕如何将智库理念融入党校图书馆决策咨询服务进行相关研究和探讨。杨蔚琪指出，随着“智库”理念的不断深化，对图书馆来说既是全新的挑战，又是难得的发展机遇，党校图书馆要明确角色定位和大力进行服务创新[5]。储流杰提出服务党校智库建设是图书馆转型发展的客观必然，服务党校智库建设图书馆大有可为，党校图书馆拥有资源、技术、服务、人才优势，可为党校新型智库建设提供强大的基础支撑和服务保障°。张鹏等总结了党校图书馆开展智库型服务需要具备资源优势、技术支持、专业馆员支撑、良好服务意识等4大要素。赵秋华分析了我国党校图书馆存在着使用率低、投入不足、服务管理质量

高等问题，并从树立服务意识、构建特色资源智库、策划专题活动加强政府部门合作等方面提出解决的方法和策略[a]。

这些研究为党校图书馆实现转型升级、更好服务智库建设进行了的理论探索。大部分学者现有研究的逻辑起点是在建设新型智库背景下，党校图书馆如何为建设党校智库发挥好服务保障作用；只有少数学者认为党校图书馆自身建设智库具有一定的条件和可能性。笔者认为，目前各层级的党校发展之间具有较大差距，党校图书馆应根据自身情况，首先要做好智库信息资源保障工作。各方面资源条件相对较好、投入力度较大的中央党校图书馆及一些省级、副省级党校图书馆可以在做好智库服务工作的同时，逐步探索自身智库建设的转型发展之路。

可以看出，相对于其他直辖市和省会城市党校图书馆，中央党校图书馆实力相对雄厚，其开展智库服务实践较有特点。一是资源更加丰富、服务形式更加多样化。不仅编印了内部刊物《网上动态》和《网文参阅》，还编辑了公开刊物《党政干部参考》。《党政干部参考》为半月刊，该刊物主要内容是宣传党的路线、方针、政策和反映现实中的重点、热点、难点问题，基本宗旨是增强全国各级党政干部的理论水平、政策水平和自身素质。二是自身特色资源丰富、推动党校系统建设特色资源成效明显。中央党校图书馆不仅加大自身的特色资源库建设，如中央党校文库、中央党校古籍典藏库、中央党校自建书籍库等，还在“十二五”期间，牵头在全国党校系统推进“三大文库”建设，即中国共产党历史文库、马克思主义理论文库和中国国情与地方志文库，“十三五”期间力推“四大专题数据库”建设，即马克思主义理论教育专题数据库、习近平总书记系列重要讲话精神专题数据库、领导干部党性修养专题数据库、新型高端智库建设专题数据库。这些特色数据库整合了不同层级的党校资源，在全国党校系统实现了共建共享，为新时期党校图书馆服务智库建设提供了重要的信息资源保障。三是服务机构较为健全。中央党校图书馆设有参考咨询部，主要开展日常参考咨询工作，围绕学校教学科研等中心工作提供学科化服务，还对图书馆馆藏资源进行二次开发与研究，这些都服务于中央党校新型智库建设。

从地方图书馆服务智库建设来看，部分直辖市和省级党校进行了取得一定成效的探索。北京市委党校图书馆同时挂有北京市委党校市情研究中心的牌子。每年编写的《北京市情数据手册》，主要反映北京市年度各方面基本数据和比较数据。该中心还初步建成了北京市情研究与地方志文库。此外，图书馆还编辑了内刊《观园》杂志，该杂志着力反映领导干部执政过程中所面临的重大理论与现实问题，着力解决北京在中国特色世界城市建设中出现的问题，对于领导干部开阔视野、拓展领导思维和提高工作能力具有积极作用。上海市委党校图书馆自2007年底开始探索开展学科服务工作，经过多年的实践，上海市委党校图书馆学科服务取得了一定的成效。馆内还设有信息咨询部，主要负责信息咨询、专题资料编写及学科服务，还编印了内刊《决策·理论·参考》。馆藏有十九大专题数据库、中国共产党思想理论资源库、“一带一路”战略支撑平台数据、人民舆情信息数据库、全球案例发现系统等专题数据库，为服务智库建设，发挥决策咨询作用提供了丰厚的电子资源保障。广东省委党校图书馆也内设有参考咨询部，并且编印了《教研信息》《广东省情快讯》和《台港澳文摘》等刊物供相关领导、学校老师和学员参阅。图书馆还自建了中共广东党史与人物专题研究数据库、广东省情研究专题数据库、广东省情快讯等特色资源库等。浙江省委党校图书馆除了设置传统的业务部门外，还附设有省情研究所和社会科学信息中心专门提供决策咨询服务。该馆编有电子版刊物《信息简讯》《当代社科视野》和《读吧》，还自建了习近平同志在浙江、浙江省情研究和地方志全文数据库、浙江党史研究与人物全文数据库等。从2007年开始，全省市级党校可以通过党校虚拟专用网络，共享省委党校的数字图书馆资源，从而使浙江省委党校图书馆成为全省党校系统的信息咨询和信息服务中心。除此之外，还有许多党校图书馆也在为服务智库建设积极进行转型升级，开展不同类型的智库服务工作。

虽然近年来党校图书馆在开展智库服务工作中取得了不少成绩，也摸索出了一些创新之路，但从整个党校系统来考量，党校图书馆智库服务工作还处于一个逐步成长阶段，特别是县（市）一级的基层党校图书馆在转型发展中还面临着一些困境和问题，需要我们在今后的工作实践中不断克服和解决。

# 3党校图书馆服务智库建设存在的主要问题

# 3.1资金投入不均衡，地区之间差异大

网络调查发现，东部发达地区和西部落后地区的党校图书馆提供智库服务的差距较大。因为党校图书馆的资金来源主要是靠国家和地方财政的统一拨付，经济发达地区对于党校和党校图书馆的投入力度较大，而经济欠发达地区的投入力度较小，特别是有些党校认为图书馆是边缘化部门，投入的办公经费更是少之又少，这些地区的党校图书馆仅能维持传统的借还书服务，并没有足够的人力和资金来开展多样化的智库服务。直辖市和发达地区省会城市的党校图书馆网站一般是外网，广大用户可以直接进行信息获取。并且网站内容丰富，基本上都有馆办刊物和丰富的自建特色数据库资源。而一些西部地区的党校图书馆网站却无法访问，馆办刊物的能力和数据库建设的水平都有待提高。

资金投入的不均衡，影响了基层党校图书馆从传统服务模式中的转型升级，制约了其融入智库建设的进程。

# 3.2方式方法落后，服务层次不够高

党校新型智库建设要求图书馆能以主动积极的态度融入教学、科研和决策咨询一线工作，及时为领导干部、老师和学员提供有效的信息资源。但目前全国党校系统图书馆的服务很多还停留在传统的文献借还阶段，服务方式落后、单一，信息的获取和敏感性都较为薄弱，与智库用户的联系不够紧密，诸此种种，造成了图书馆的功能逐渐被弱化，甚至在有些党校图书馆变成了可有可无的部门。从当前实践来看，即使是目前开展智库服务比较好的党校图书馆，提供智库服务的层级还处于基础服务阶段，即根据用户需要，通过网络、报刊搜集相关的信息资料传递给用户，或者在搜集原文的基础上进行一些简单的信息加工，整理成总结性的专题文摘。这些是属于信息传递和参考咨询阶段，远未达到对信息的深加工，提出自己原创性的观点和建议的决策咨询阶段，影响了党校图书馆服务智库建设的发展力度。

# 3.3专业人才紧缺，专业服务功能有欠缺

智库建设和图书馆建设关键在于人才。党校图书馆服务智库建设需要有一批高端人才提供的专业性、高效的信息咨询服务。图书馆要向这种专业化服务转型，必须重视工作人员的综合素质、专业技能、知识水平等方面的提升。近些年来，中央党校图书馆、经济条件较好的直辖市和省级、副省级党校图书馆不断引进了一些博士、硕士学历的专业人才，优化了图书馆员的学科结构，提高了馆员的综合素质，为图书馆探索建立学科馆员制度，开展不同类型的智库服务工作发挥了重要作用。但是从整体来看，一些市县级党校图书馆由于投入力度不够，收入相对较低，又无地缘优势，发展前景也不明朗，吸引不了优秀的毕业生去安心工作，出现了人员结构失调，老龄化严重，整体素质不高，科研能力不强等问题。有些图书馆本科学历的人才都不多，甚至出现无一人是图书馆专业背景的情况，

这在很大程度上制约了党校图书馆开展智库服务工作以及专业化服务功能的发挥。

# 4党校图书馆服务智库建设的转型发展策略

# 4.1更新服务观念，全面增强咨政服务意识

党校是党委的重要部门，是培训干部教育的主渠道，相对于其他机构，与党委、政府联系更加紧密，为决策者提供决策咨询参考，建设党校新型智库是大势所趋。新时期党校图书馆要向服务党校智库建设工作模式转型，首先要从思想观念上高度重视，明确自身的定位，加强为党校智库建设提供建言和咨询服务的意识。长期以来，有些图书馆创新服务很少，仅仅固守传统的借阅服务工作，服务的对象也比较狭窄，与用户关系疏远，其主要原因在于主动服务意识不够，对图书馆参与党校智库服务建设的认识不足。只有图书馆员充分意识到发挥咨政服务的重要性，内设相关的咨政机构，主动组织相关调研，策划课题，编写咨政专刊等，才能真正实现图书馆的转型发展，提高智库服务的能力和效果。

# 4.2拓宽经费渠道，为智库建设服务提供经费保障

充足的资金来源是图书馆全面开展智库服务工作的前提和保障。党校图书馆要按照国家相关规定和参照同级别党校图书馆的办公经费标准，通过单位财务部门积极向地方财政申请业务专项经费，要求适当增加投入比例，争取经费足额拨付并最大程度地给予政策倾斜支持。还要向国外优秀智库和社会研究机构学习，吸引更多其他部门资金的纳入。党校图书馆可以整合内部资源，承接一些党校以外的课题和科研项目的研究工作，比如可以加强与宣传部、社科联等社科机构的相关合作，从而获得一定的经费支持，对其扩大智库服务范围，提高参考咨询能力，进一步优化转型提供了经费保障。

除此之外，党校图书馆还要对每年的经费进行合理的规划和分配，严格监督经费的使用情况，力求公开、透明、不浪费，以保证图书馆的智库服务目标可以顺利实现。

# 4.3注重资源整合，强化文献资源开发和专题数据库建设

4.5江里贝你歪口，独化人酚贝砾儿及怕令题双增干廷议丰富的文献信息资源是进行智库研究活动的基础，在转型发展中，党校图书馆要进一步加强文献信息的采集、加工和整理工作，注重文献资料收集的完整性和系统性，专题资料的选取要紧密围绕党和国家的中心工作和方针政策，要密切关注国内外形势发展变化，要聚焦社会的热点、难点问题，地方党校专题资料的选择还要多与当地的经济社会发展相结合，贴近党校发展和学员培训需求。同时，党校图书馆还要注重优秀学员论文、学员调研报告、政府机关内部交换的信息资料以及党校老师的教学科研成果等这些未公开出版的灰色文献的收集和整理。这些文献资料为党校图书馆特色的馆藏资源，所涉及的信息面广、内容新颖、观点独到，对于及时了解党委和政府的最新政策走向，决策者的各种观点具有重要的作用。此外，党校图书馆还要继续加强特色专题数据库建设，馆藏数据库既要体现世情、国情、党情，又要凸显地方特色。这些特色数据库将无序分散的文献资源进行系统化、规范化、有序化整理，并经过深层次加工和挖掘，方便了用户的学习和查阅，有利于提高馆藏资源的利用率，为图书馆开展智库服务提供了扎实的基础。

# 4.4完善数字网络，推动各层级党校共建共享信息资源

从各个地区党校实践情况来看，各个层级的财政拨款存在很大差异，有些落后地区的基层党校图书馆资金紧缺，根本无力开展文献资源的建设工作。在这样的情况下，可以在完善全省党校网络系统的前提下，整合各个市县的党校图书馆资源，实现一个区域的党校资源的共建共享，这样能弥补和解决基层党校图书馆资金不足和技术欠缺的问题，还可以提高对数字资源的利用效率。浙江省委党校从2003年开始，就组织全省党校系统对数字资源实行联合采购，2005年正式启动了全省党校系统数字资源的共建共享机制，目前已实现了省、市、县3级党校虚拟专网的互联互通，使得文献信息资源的共建共享成为可能[9]。党校系统网络资源的共享机制有利于提升党校图书馆决策咨询的整体水平，有利于图书馆进一步发挥好服务智库建设的功效。

# 4.5健全学科馆员制度，大力优化服务方式

党校图书馆要以服务智库建设为契机，转变服务理念，创新服务方式，变被动服务为主动的嵌入式服务，大力推行学科馆员制度，开展学科化的高端信息服务。学科馆员不仅需要具备图书馆基本业务工作能力，还要具备一定的学科专业背景。主要职责是积极联系服务对象，及时了解用户的需求，向服务对象提供所需的信息资料或供参考咨询的专题资料。同时在日常的工作中要密切关注相应学科的最新动态和前沿信息，搜集整理一些焦点、热点问题，定期或不定期向用户推介更新的信息咨询，搭建好与用户交流互动的桥梁。另外，还要不断完善学科导航系统，整合各个学科的网络信息资源，给用户提供一站式信息资源服务。上海市委党校从2007年底开始就探索开展学科服务工作，组建了一支兼职学科馆员队伍，分别对应当时的7个教研部，初步建立了学科馆员联系教研部的基本通道和机制[0]。学科馆员在服务教研部老师的基础上，还应该以一些智库服务项目为抓手，将服务对象拓展到相关的党政机关工作人员，这样可以提高党校图书馆的影响力，打造特色的智库服务品牌。

# 4.6培养人才队伍，全面提高馆员综合素质

作者简介：姜晓婷（ORCID：0000-0002-9678-2611），中共广州市委党校图书馆馆员，法学硕士，E-mail： jiangxiaoting34@163.com。

智库服务人才应该是具有多学科知识背景，具备较强的调查研究和对策分析能力的复合型人才。建立一支知国情、懂政策、理论功底扎实的高素质人才队伍是党校图书馆服务智库建设的关键。图书馆要优化现有人员的组织架构，引导图书馆员逐渐融入到科研的状态和氛围。通过积极为馆员争取各种培训机会、专家论坛、专题调研的等方式，提升馆员的研究能力和工作水平，推动人才队伍的转型升级。完善人才的评聘标准和激励制度，充分调动馆员的工作热情和积极性。做好学科规划建设，有计划有目标的引进一些不同专业背景的高学历人才，解决目前人才瓶颈问题。鼓励馆员“走出去”，与公共图书馆、高校图书馆及兄弟党校图书馆多学习、多交流，并积极参与到各教研部的研究项目和课题中去。只有这样，才能真正提高图书馆员的综合素质，推动图书馆员的转型升级，才能为党校智库建设提供高水平的信息服务。

# 5小结

党校新型智库建设是中国特色新型智库建设的重要组成部分，智库建设为党校图书馆转变传统服务模式，实现转型升级提供了机遇和平台。党校图书馆要提高政治敏锐性，增强智库服务的意识，发挥好党校智库信息源和信息服务窗口的前端支持作用。通过不断挖掘自身潜力，创新服务手段，与其他部门通力合作，最大程度地为党校建设新型智库提供服务和保障。
