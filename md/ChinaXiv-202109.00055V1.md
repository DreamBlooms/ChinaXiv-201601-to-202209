# 基于地理探测器分析青藏高原降水 ${ \bf \delta } ^ { 8 } { \bf O }$ 空间分异特征

席文涛¹²，高晶13

(1.中国科学院青藏高原研究所 中国科学院青藏高原环境变化与地表过程重点实验室,北京100101;2.中国科学院大学,北京100049；3.中国科学院青藏高原地球科学卓越创新中心,北京 100101)

摘要：地理探测器是研究地理现象的空间分异性,并且定量分析其驱动因子的一种统计方法。利用地理探测器方法对青藏高原24个站点的降水 $\mathbf { \delta } \mathbf { \delta } ^ { 1 8 } \mathbf { O }$ 年均值进行了分析，该方法可在一定程度上反映青藏高原降水 $\mathbf { \delta } \mathbf { \delta } ^ { 1 8 } \mathbf { O }$ 年均值的空间分异性，得出纬度、海拔、经度和降水量对青藏高原降水 $\mathbf { \delta } ^ { 8 } \mathbf { ^ { 1 8 } } \mathbf { O }$ 年均值空间分异的解释力分别为0.82、0.71、0.57和0.49，温度对青藏高原降水 $\mathbf { \delta } \mathbf { \delta } ^ { 1 8 } \mathbf { O }$ 年均值空间分异的解释力不显著;因子之间的共同作用增强了降水 $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ^ { 1 8 } O } }$ 年均值在空间上的分异性。讨论了青藏高原站点降水 $\mathbf { \delta } ^ { 8 } \mathbf { ^ { 1 8 } } \mathbf { O }$ 年均值与纬度、经度、海拔、年降水量和年均温之间的关系，并对降水$\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 主控因子的季节变化进行分析,得出纬度对青藏高原降水 $8 ^ { 1 8 } 0$ 年均值、夏季均值和冬季均值空间分异的解释力均为最强。

关键词：青藏高原；降水 $\mathbf { \delta } ^ { 8 ^ { 1 8 } } \mathbf { O }$ ；空间分异性；地理探测器

水体稳定同位素 $( \mathfrak { d } ^ { 1 8 } \mathrm { O } , \mathfrak { d } \mathrm { D } ) ,$ 是有效的环境示踪剂。1964年Dansgaard利用全球降水同位素监测网（Global Network of Isotopes in Precipitation,GNIP）早期收集的降水 $\mathbb { \cdot } ^ { 8 ^ { 1 8 } } \mathrm { O }$ 数据发现降水同位素的变化与气温、降水量和海拔等因子有关。此后大量研究关注于降水 $\mathbf { \delta } ^ { 8 } \mathbf { O }$ 的气候和环境意义[2-5],并且已有大量研究表明降水 $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 的变化表现出温度效应、降水量效应、纬度效应、大陆效应和海拔效应，以及受水汽来源地和区域综合性对流活动的影响[6-10]。随着20世纪80年代青藏高原冰芯研究工作的开展，中国学者启动了青藏高原降水同位素监测计划(TibetanPlateau Network of Isotopes in Precipitation, TNIP）,对20多个站点的降水同位素变化进行了多年连续监测，对该地区冰芯记录解释和水文循环等研究提供理论基础。基于这项计划，学者们对青藏高原降水$\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 的时空变化规律及主控因素已经获得了较为深入的认识[1I-16],发现青藏高原南北部分别受控于印度季风环流和西风环流，降水 $8 ^ { 1 8 } 0$ 在夏季表现为北部高于南部，冬季表现为南部高于北部。目前，鲜有专门针对青藏高原降水 $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 空间分异性的量化研究。

地理探测器统计方法的基本原理为方差分析。该方法通过比较类别型数据或数值型数据层内与层间方差大小，定量研究地理现象的空间分异性及其控制因子之间的交互作用。与其他空间分异性研究方法相比，地理探测器方法具有更高解释效力的优势。该方法已经成功的应用于疾病传播规律、土地利用变化和地理分界线的讨论等研究之中[17]

本文在前人研究工作的基础上，利用青藏高原地区GNIP和TNIP共计24个观测站点多年的降水$\mathbf { \delta } ^ { 8 } \mathbf { O }$ 数据，结合地理探测器方法对青藏高原降水$\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 的空间分异性进行了分析，旨在定量探析不同影响因子对青藏高原降水 $\mathfrak { \delta } ^ { 1 8 } \mathrm { O }$ 空间分异的解释程度。

# 1.2地理探测器方法

地理探测器方法本质上为方差分析，由王劲峰等[17提出,用来探测地理现象的空间分异性。该方法通过比较层内方差与层间方差，可以定量表达研究对象的空间分异特征，将不同影响因子进行交互分析，还可以分析不同影响因子之间的交互作用。本文将影响降水同位素的每一个地理因子（如纬度、经度、海拔、温度和降水)分为不同的层，在不同的层内这些地理因子具有不同的属性值，如果某个地理因子主控降水 ${ \mathfrak { F } } ^ { 1 8 } \mathrm { O }$ 的空间分异，那么降水 ${ \mathfrak { F } } ^ { 1 8 } \mathrm { O }$ 将会和这个因子表现出相似的分布规律。地理探测器中的因子探测器可以探测各个解释指标在多大程度上解释了降水 $\mathbf { 8 ^ { 1 8 } O }$ 的空间分异，以 $\boldsymbol { q }$ 统计量来度量，计算公式如下：

$$
q = 1 - \frac { \displaystyle \sum _ { h = 1 } ^ { L } N _ { h } \sigma _ { h } ^ { 2 } } { N \sigma ^ { 2 } }
$$

式中： $q$ 为因子对降水 $8 ^ { 1 8 } 0$ 空间分异的解释力， $q$ 的值注：图d是由夏季 $\vdots 8 ^ { 1 8 } 0$ 均值减去冬季 $\vdots 8 ^ { 1 8 } 0$ 均值得到的 $\mathbf { \delta } ^ { \mathrm { \tiny { \cdot } } } 8 ^ { 1 8 } 0$ 季节变幅。

# 1.3计算步骤

将多年期间各个站点降水 ${ \mathfrak { F } } ^ { 1 8 } \mathrm { O }$ 数据，按照降水量加权平均计算得出各个站点降水 ${ \mathfrak { F } } ^ { 1 8 } \mathrm { O }$ 的年均值,这代表了多年期间各个站点降水 $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 的平均状态;将多年期间各个站点夏、冬季降水 $\mathbf { 8 ^ { 1 8 } O }$ 数据，分别按照其降水量进行加权平均，计算各个站点降水 $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 的夏、冬季均值，本文在计算加权平均值时所采用的方法均为算术加权平均。

地理探测器方法要求解释变量为类别型变量，首先将降水 $\mathbf { 8 ^ { 1 8 } O }$ 的各个解释变量按照属性值大小根据等间隔分类转化为类别型变量；然后利用地理探测器程序包求出各个指标的 $q$ 统计量及其显著性水平，其余统计分析工作均由R语言完成。

# 3青藏高原降水 $\mathbf { \delta } ^ { \mathbf { 8 } } \mathbf { O }$ 空间分异的主控因素

# 3.1 $\mathbf { \delta } ^ { \mathbf { \delta } } \mathbf { \delta } ^ { \mathbf { \delta } } \mathbf { \delta } ^ { \mathbf { \delta } } \mathbf { \delta } ^ { \mathbf { \delta } }$ 年均值与纬度和降水量的关系

通过地理探测器方法得出纬度对高原降水 $\mathbf { \delta } ^ { 8 } \mathbf { ^ { 1 8 } } \mathbf { O }$ 年均值空间分异的解释力超过0.8，纬度和降水共同作用的解释力达到0.93。这实际上反映了青藏高原从南至北不同的气候模态和水汽传输特征。已有大量研究表明，青藏高原气候表现为西风模态、季风模态和过渡模态，决定了降水稳定同位素的时空分布特征[18-23]。高原南部降水同位素的变化主要受控于印度季风带来的暖湿性水汽[24-25],北部地区降水同位素的变化主要受控于西风带来的高纬度海洋或大陆性水汽[26-27]。因此,本研究中的纬度和降水的共同作用主要反映的是西风和印度季风不同传输水汽的控制作用。

对所有站点的降水 $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 与对应的年降水量进行相关分析，发现站点降水 $\mathbf { 8 ^ { 1 8 } O }$ 年均值与降水量之间无显著相关性 $( 8 ^ { 1 8 } 0 \mathrm { = } \mathrm { - } 0 . 0 1 p \mathrm { - } 9 . 5 4 , R ^ { 2 } \mathrm { = } 0 . 0 8 , P \mathrm { = } 0 . 2 1 , p$ 为降水量，单位为 $\operatorname* { m m } \cdot \mathrm { a } ^ { - 1 }$ )。然后分区进行讨论，得出高原南部站点降水 $\mathbf { \delta } ^ { 8 } \mathbf { ^ { 1 8 } } \mathbf { O }$ 与降水量之间存在显著正相关关系 $\langle \hat { \bf 8 } ^ { 1 8 } 0 = 0 . 0 1 3 p - 2 0 . 5 5 , R ^ { 2 } = 0 . 4 6 , P < 0 . 0 5 )$ ，北部则不显著。这与前人针对南部站点降水 $\mathbf { 8 ^ { 1 8 } O }$ 季节变化与降水量之间的负相关关系不同[18]。这是由于本文在空间上探讨降水 $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 与降水量的关系，这种正相关关系实质上反映了各个站点距离主要水汽源地的距离，即反映大陆效应。高原南部夏季主要受海洋性水汽控制，随着气团向内陆移动，湿绝热降温过程进行，气团水汽质量减小，饱和温度降低，随后分馏系数增大，降水 $8 ^ { 1 8 } 0$ 值也偏低。

# 3.2 $\mathbf { \delta } ^ { \mathbf { 8 } } \mathbf { 0 }$ 年均值与海拔之间的关系

地理探测器结果显示，海拔对降水 $8 ^ { 1 8 } 0$ 空间分异的解释力达到了0.71，仅次于纬度，这表明海拔在驱动青藏高原降水 $\mathbf { 8 ^ { 1 8 } O }$ 空间分异方面发挥着重要作用。已有学者对青藏高原不同区域的海拔效应进行过研究，例如，孙从建等[20在高原西北部利用5个站点的数据，得出降水 $\delta ^ { 1 8 } \mathrm { O }$ 显著的高程效应( $8 ^ { 1 8 } \mathrm { { O } = }$ $- 0 . 0 0 1 2 6 \mathrm { h } - 9 . 0 1 , R ^ { 2 } = 0 . 8 2 , h$ 为海拔，单位 $\mathbf { m }$ ，即降水 $8 ^ { 1 8 } 0$ 随着海拔的升高表现为降低，垂直递减率为$- 0 . 1 2 6 \% o \cdot ( 1 0 0 \mathrm { m } ) ^ { - 1 }$ 。文蓉等[22在喜马拉雅山脉南坡利用3个站点的数据，得出降水 ${ \mathfrak { F } } ^ { 1 8 } \mathrm { O }$ 的垂直递减率为 $- 0 . 1 7 \% o \cdot ( 1 0 0 \mathrm { m } ) ^ { - 1 }$ ,二者在青藏高原不同地区的研究均显示降水 $8 ^ { 1 8 } 0$ 的垂直递减率明显低于全球平均值 $0 . 2 8 \% o \cdot ( 1 0 0 \mathrm { m } ) ^ { - 1 [ 2 3 ] }$ 。本研究依据青藏高原24个站点的数据，得出青藏高原降水 $\mathbf { 8 ^ { 1 8 } O }$ 年均值与海拔同样呈显著负相关关系 $( 8 ^ { 1 8 } 0 \mathrm { = } \mathrm { - } 0 . 0 0 2 7 h \mathrm { - } 3 . 4 1 , R ^ { 2 } \mathrm { = } 0 . 6 5$ $\scriptstyle P < 0 . 0 0 1 .$ ),其垂直递减率为 $- 0 . 2 7 \% o \cdot ( 1 0 0 \mathrm { m } ) ^ { - 1 }$ ,与全球平均值接近，这可能与不同研究选用站点的空间代表性有关，本文使用24个站点分析降水 $\mathbf { \delta } ^ { 8 } \mathbf { ^ { 1 8 } } \mathbf { O }$ 与海拔之间的关系，相对而言代表整个青藏高原地区的海拔效应。

图3显示，高原南部的堆村、定日和翁果站海拔较高，温度偏低，降水量较少，无论是年均值还是夏季均值，降水 $8 ^ { 1 8 } 0$ 均表现为严重贫化，甚至堆村达到研究区内的最低值，这是由于夏季海洋性水汽在翻越喜马拉雅山过程中，受控于瑞利分馏过程，降水$\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 值降低[26]。但随着纬度的增加，降水 $\mathbf { 8 ^ { 1 8 } O }$ 值又表现出升高的趋势，这可能与气团混合了局地蒸散发水汽有关[27-29],并且受不饱和空气中云下蒸发的作用，降落过程中的雨滴与环境水汽进行同位素交换时，重同位素更倾向于进入雨滴，从而使得降水$\mathbf { \delta } ^ { 8 } \mathrm { { ^ { 1 8 } O } }$ 值偏高[30]

# 3.3 $\mathbf { \delta } ^ { \mathbf { 8 } } \mathbf { 0 }$ 年均值与温度之间的关系

根据瑞利分馏模式，当水汽源地和混合过程的影响较小，或者不考虑这种影响时，那么从水汽输送路径的角度出发，空间上可以推导沿途降水 $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 与饱和温度之间的关系为正相关。已有学者通过对塔什库尔干大气水汽 $\{ \delta ^ { 1 8 } 0$ 的研究，认为该区域全年受瑞利分馏控制[29]。但是也有研究表明，高原北部春秋两季的水汽除了西风输送外，还混合了局地的蒸散发水汽，夏季的水汽来源复杂，同时混合了西风水汽、再循环水汽以及少量的东南季风水汽，仅在冬季西风水汽占有绝对优势[31]。除了水汽源地的影响以外，研究表明该区域夏季温度较高，蒸散发强烈，雨滴下落过程中受云下不饱和空气中动力蒸发的影响,降水 $\mathbb { \cdot } ^ { 8 ^ { 1 8 } 0 }$ 也会偏正[32-34]。学者们对高原北部降水 $8 ^ { 1 8 } 0$ 季节变化的研究，表明降水 $\mathbf { \delta } ^ { \mathrm { 8 0 } }$ 受控于温度效应[18.3.35],而南部受控于降水量效应[18.25-26]。

本文利用24个站点的资料，结合地理探测器方法，发现温度对降水 $\delta ^ { \mathrm { 1 8 } } \mathrm { O }$ 年均值的空间分异特征解释不显著。然后分区进行讨论，对高原北部站点降水 $\mathbf { \delta } ^ { 8 } \mathbf { { ^ { 1 8 } } }$ 的年均值与温度进行相关分析，得出二者显著正相关 $( \ S ^ { 1 8 } 0 \mathrm { = } 0 . 5 1 T \mathrm { - } 1 1 . 9 1 , R ^ { 2 } \mathrm { = } 0 . 5 2 , P { < } 0 . 0 5 , T$ 为年均温度，单位 $\mathrm { { ^ \circ C } }$ ),而高原南部不显著。这表明高原北部降水 $\mathbf { \delta } ^ { 8 } \mathbf { \mathrm { ~ ‰ ~ } }$ 在空间上也存在显著的温度效应，即站点温度越高，降水 $8 ^ { 1 8 } 0$ 值也偏高。从图2和图3可以看出，高原北部的站点降水 $8 ^ { 1 8 } 0$ 值自北向南降低，一定程度上叠加了海拔的影响，但也不符合沿水汽输送路径降低的规律。因此，高原北部这种空间上的温度效应可能与局地蒸发有关[31-34]

# 4结论

基于青藏高原24个观测站点多年降水 $\mathbf { 8 ^ { 1 8 } O }$ 雨量加权平均数据，利用地理探测器方法，得出纬度、海拔、经度和降水量对青藏高原降水 $\mathbf { 8 ^ { 1 8 } O }$ 年均值空间分异的解释力分别为 $0 . 8 2 , 0 . 7 1 , 0 . 5 7$ 和 $0 . 4 9 ( P <$ 0.05)，温度的解释力不显著。因子之间的共同作用对青藏高原降水 $\mathbf { \delta } ^ { 8 } \mathbf { 0 }$ 年均值空间分异的解释力均表现为增强，纬度和降水量共同作用的解释力最强为0.93，因子之间的共同作用增强了降水 $8 ^ { 1 8 } 0$ 的空间分异性。同时纬度对青藏高原降水 $\mathbf { \delta } ^ { 8 } \mathbf { 0 }$ 年均值、夏季均值和冬季均值空间分异的解释力均表现为最强。

# Spatial heterogeneity of annual precipitation $\mathbf { \delta } ^ { \mathbf { 8 } } \mathbf { O }$ over the Tibetan Plateau based on the use of a geographical detector

XI Wentao1²，GAO Jing1,3

(1.KeylaboratoryofTibetan Environment Changesand Land SurfaceProcess,Institute of Tibetan Plateau Research, Chinese Academy of Sciences,Beijing 100101, China; 2. University of Chinese Academy of Sciences,   
Beijing10oo49,China; 3.CenterforExcellnce in Tibetan Plateau Earth Sciences,Chinese Academyof Sciences, Beijing 100101, China)

Abstract: The annual precipitation $\delta ^ { 1 8 } \mathrm { O }$ from 24 stations over the Tibetan Plateau was analyzed based on the use of a geographical detector， which was applied to probe the spatial heterogeneity of precipitation $\delta ^ { \mathrm { 1 8 } } \mathrm { O }$ and to quantitatively understand the drivers.This method was used to successfully reflect the spatial heterogeneity of the annual precipitation $\delta ^ { 1 8 } \mathrm { O }$ over the Tibetan Plateau. The latitude,altitude,longitude,and precipitation amounts respectively explained 0.82, 0.71,0.57,and 0.49 of the spatial variation in annual precipitation $\delta ^ { \mathrm { 1 8 } } \mathrm { O }$ Additionally, the combined effects of different factors enhanced the explanation of spatial variation.The possible influences of latitude,precipitation amount, altitude,and temperature on annual precipitation $\delta ^ { 1 8 } \mathrm { O }$ and seasonal variations in influential factors that cause the spatial heterogeneity of precipitation $\delta ^ { \mathrm { 1 8 } } \mathrm { O }$ were discussed. We conclude that latitude has the strongest explanatory power in relation to the spatial heterogeneity of annual,summer,and winter mean precipitation $\delta ^ { 1 8 } \mathrm { O }$ over the Tibetan Plateau.

Keywords: Tibetan Plateau; precipitation $\delta ^ { 1 8 } \mathrm { O }$ ； spatial heterogeneity; geographical detector
