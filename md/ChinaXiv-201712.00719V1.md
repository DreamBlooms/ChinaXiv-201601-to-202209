# 3讨论

我们前期研究[8-10发现，结直肠癌血行转移等恶性演进过程与肿瘤转移抑制基因Kiss-1基因表达下调密切相关，而其表达水平上调可抑制结直肠癌细胞的增殖、侵袭和迁移能力。

本研究通过构建pGC-LV-Kiss-1-EGFP慢病毒载体并转染结直肠癌HCT116细胞,使其Kiss-1基因过表达后发现NF- $\mathbf { \sigma } _ { \kappa \mathbf { B } }$ 信号传导通路中抑制性蛋白I-KB的表达明显升高，同时其下游效应蛋白MMP-9表达明显降低。MMP-9是基质金属蛋白酶家族的重要成员，具有降解细胞外基质和降低细胞粘附性的功能，能够促进肿瘤细胞的侵袭和转移[]。与此同时，CCK-8和Transwell小室实验结果表明Kiss-1基因转染后HCT116细胞的增殖能力、侵袭和迁移能力均受到明显的抑制。因此推测结直肠癌细胞中Kiss-1基因有可能也是通过上调 $\scriptstyle \mathbf { I - \kappa B }$ 蛋白的表达抑制NF-kB信号通路的激活，抑制p50和p65亚基进入胞核，减少与MMP-9启动子的结合，下调MMP-9的表达，进而减少ECM的降解和增加细胞之间的粘附效应，从而抑制结直肠癌细胞的侵袭和迁移能力。恶性肿瘤的转移需要多种信号通路参与和调控， ${ \mathrm { N F } } { \mathrm { - } } { \mathrm { \kappa B } }$ 通路即是其中之一[5-6],在结直肠癌中参与调控血管内皮生长因子和MMP-9的表达,促进肿瘤转移[7.12]。在乳腺癌[3中，通过干扰NF- $\mathbf { \sigma } _ { \kappa \mathbf { B } }$ 与MMP-9的启动子区的结合，MMP-9在乳腺癌细胞中表达的下调，降低了细胞的侵袭、迁移能力。Huang等[4研究显示，NF- $\mathbf { \sigma } _ { \kappa \mathbf { B } }$ 途径受到抑制后，高转移性前列腺癌PC23细胞的侵袭、转移能力亦受到明显抑制。张辉等[15通过合成Kiss-1基因特异性siRNA转染肾癌Caki-1细胞的研究显示，Kiss-1基因能够通过NF- $\boldsymbol { \kappa } \mathbf { B }$ 途径抑制MMP-9的表达，增加细胞的粘附性进而抑制肾癌细胞的侵袭和迁移能力。这些研究表明NF $\mathbf { \sigma } _ { \cdot \mathbf { \kappa } \mathbf { B } }$ 信号传导通路参与了Kiss-1基因抑制肿瘤转移的过程，与本研究结果相似。

综上所述，本研究初步证实肿瘤转移抑制基因Kiss-1可以通过NF $\mathbf { \nabla \cdot } \kappa \mathbf { B }$ 信号传导通路参与抑制结直肠癌血行转移等恶性演进过程。然而，结直肠肿瘤转移涉及十分复杂的分子生物学机制，Kiss-1基因抑制结直肠癌转移很可能与其他信号传导有关，还需要进一步的深入研究，这对揭示结直肠癌转移机制、寻找阻断转移的发生是十分有益的。
