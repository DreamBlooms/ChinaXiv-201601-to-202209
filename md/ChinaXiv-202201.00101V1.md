# 盐碱地食叶草细根对干旱复水后的响应

郑旭，杨志鑫，郝东梅，王润润，李鲁华，张凤华，王家平(石河子大学农学院,新疆 石河子832000)

摘要：盐生植物灌溉周期合理制定是对提高干旱区水资源利用效率的关键因素。以食叶草(Rumex hanus by.）细根( $\scriptstyle \cdot d \leqslant 1 { \mathrm { ~ m m } } ,$ 为研究对象,采用干旱-复水方法,测定复水后不同时间食叶草细根总根长、根尖数、平均根直径,研究根系增长率、死亡量、根寿命和周转率的变化特征，为制定最佳灌溉周期提供参考依据。结果表明：根长和根尖数在复水后7d增长幅度最大,10\~15d增长幅度减弱。细根增长率(RER)在复水后第4d达到最大,第10d出现显著下降( $\scriptstyle \left. P < 0 . 0 5 \right)$ ，在不同土层( $( 0 { \sim } 2 0 \mathrm { c m }$ 和 $2 0 { \sim } 4 0 ~ \mathrm { c m }$ 和根系直径（ $( 0 { \sim } 0 . 5 ~ \mathrm { m m }$ 和 $0 . 5 { \sim } 1 . 0 \ \mathrm { m m }$ )之间均有极显著差异( $P <$ 0.001)。细根净生产量(NRP)与RER二者均呈现先增长后降低的规律。复水后 $1 5 \mathrm { d } , 0 { \sim } 2 0 \mathrm { c m }$ 土层和 $2 0 { \sim } 4 0 ~ \mathrm { c m }$ 土层的细根存活率分别为 $3 . 6 \%$ 和 $1 6 . 9 \% , 0 . 5 { \sim } 1 . 0 \ \mathrm { m m }$ 直径的细根存活率高于 $0 { \sim } 0 . 5 ~ \mathrm { m m }$ 细根。研究表明复水后细根中位寿命在 $8 . 0 9 { \sim } 1 3 . 8 3 \mathrm { d } , 2 0 { \sim } 4 0 \mathrm { c m }$ 土层细根寿命显著高于 $0 { \sim } 2 0 \ \mathrm { c m }$ 土层 $( P { < } 0 . 0 5 )$ 。食叶草细根通过增加 $2 0 { \sim } 4 0 ~ \mathrm { c m }$ 土层中 $0 . 5 { \sim } 1 . 0 ~ \mathrm { m m }$ 细根应对和适应干旱及复水，从而达到生存目的。综合考虑食叶草生长和农业水资源供给情况，在夏季盐碱地食叶草最佳灌溉周期为 $1 0 \mathrm { d }$

关键词：盐碱土；细根；日增长率；死亡量；周转率；中位寿命

水分是影响作物生长发育的重要因素[1],也是干旱区盐碱地形成的主要原因[2-3]。作物生长过程中水分供应不足导致植株受到干旱胁迫，产生作物生长受抑、早衰、萎蔫等一系列不可逆的负面效应[4-5]干旱区经常出现季节性干旱和间歇性干旱,近年来发生频率逐年增加，根系是植物吸收土壤水分、养分的重要器官，对干旱最先响应胁迫[]。根系研究是揭示植物对干旱环境适应性的关键。但由于植物根系调查困难，干旱胁迫对植物根系生长认识目前还不很明确，干旱胁迫及复水后根系的恢复调控机制研究较少。

植物生长发育高度依赖根系对水分和养分的吸收功能[8]。细根是植物根系中重要的组成部分，作为植物进行物质吸收、能量交换的器官[9]。目前国内外学者认为，细根主要是直径小于 $2 ~ \mathrm { m m }$ 的根系[8,10-11]。由于细根具有重要的功能属性,依据研究目的分为细根分级理论和根序级分级理论，细根分级理论，主要是按照直径大小分级，其优点是操作性强，缺点是方法标准不统一。根序级分级理论，以末端最短最细的根尖作为一级根，生长一级根的母根称为二级根，生长二级根的母根为三级根，其他根级依此类推，其优势是考虑了不同根序级的功能差异，缺点是分级工作费时费力。根序模块理论综合上述两种方法的优点，主要基于功能将细根分成吸收根和运输根[12-13]。Liu等[14]研究分析上述两种方法，研究发现根系吸收功能集中在直径 $< 0 . 5$ $\mathbf { m } \mathbf { m }$ 的根系，运输功能主要是直径 $\mathrm { > } 1 \ \mathrm { m m }$ 的根系，而直径 $0 . 5 { \sim } 1 \ \mathrm { m m }$ 的根系既有吸收功能也有运输功能。因此，细根生长和发育对植物生长发育和灌溉周期制定具有重要意义。

干旱和复水在自然界中是循序出现的，植物根系的生长发育在干旱-复水过程中存在着明显差异[15]。研究表明,伊犁绢蒿（Seriphidium transil-iense)在干旱复水后提高了根系活力和渗透调节物质，降低了抗氧化酶的活性，具有更强的抵御干旱胁迫的能力[i6]；干旱后复水对燕麦(Avenasativa)根系总长、根体积、根系平均根直径等根系指标具有促进作用；研究表明，干旱复水后树木幼苗的细根长、根直径、根干重增加，分支强度等根系性状均有增强,从而提高了树木幼苗的存活率[18-19]。目前干旱-复水下植株根系的研究主要聚焦在总根系，对直径小于 $1 \mathrm { m m }$ 的细根研究关注不足，因此，探究干旱胁迫复水对植物细根形态变化、周转和寿命的研究可为盐生植物干旱适应及灌溉周期的制定提供实践依据。

食叶草(Rumexhanusby.)是一种高产优质的粮饲作物，具有耐盐碱、耐干旱和耐寒等特点，是盐碱地改良适宜植物[20]。关于食叶草的研究主要集中在食叶草安全性评价和食叶草在食品、饲草和荒漠化土地改良利用等方面[20-21],食叶草根系在干旱区生长发育动态变化及合理灌溉方面研究尚未开展。本研究以3a生食叶草为研究对象，阐明干旱区盐碱地食叶草细根生长、死亡和周转对干旱后复水的响应及制定合理灌溉周期，研究结果为干旱区盐碱地耐盐植物根系周转和盐碱地修复治理提供实践依据。

# 1材料与方法

# 1.1试验地概况

试验于2020年4月10日—6月25日在石河子总场五分场 $( 4 4 ^ { \circ } 3 6 ^ { \prime } 1 . 7 5 ^ { \prime \prime } \mathrm { N } , 8 5 ^ { \circ } 5 7 ^ { \prime } 3 5 . 7 2 ^ { \prime \prime } \mathrm { E } )$ 进行，试验期间白天平均气温为 $3 2 . 5 ~ \mathrm { ^ { \circ } C }$ ,夜晚平均气温为$1 7 . 1 \ \mathrm { { ^ { \circ } C } }$ 。小区试验土壤为灰漠土，盐含量 $0 . 4 \%$ 2$0 . 6 \%$ ， $\mathrm { \ p H }$ 值8.3\~8.7,为盐碱土。 $0 { \sim } 2 0 ~ \mathrm { c m }$ 土层土壤有机质为 $5 . 6 \mathrm { g } \bullet \mathrm { k g } ^ { - 1 }$ 、全氮 $0 . 2 8 \mathrm { g } { \cdot } \mathrm { k g } ^ { - 1 }$ 、全磷 $0 . 7 5 \mathrm { g } { \cdot } \mathrm { k g } ^ { - 1 }$ 、全钾 $2 1 . 2 \ \mathrm { g \cdot k g ^ { - 1 } }$ 。食叶草于2018年8月5日种植,株距 $4 0 \ \mathrm { c m }$ ，行距 $6 0 \ \mathrm { c m }$ 。

# 1.2 试验设计

2020年4月10日至6月10日，进行干旱处理，试验设置3个重复，每个小区 $3 . 6 ~ \mathrm { m } { \times } 1 0 ~ \mathrm { m } { = } 3 6 ~ \mathrm { m } ^ { 2 }$ ，小区间设有 $1 \mathrm { m }$ 保护行。复水前用CI-600根系监测系统扫描试验植株根系，校验检测区域是否存在直径 $< 1 \ \mathrm { m m }$ 根系。于6月10日进行复水，复水一次，灌水量为 $1 5 0 \mathrm { m } ^ { 3 } { \cdot } \mathrm { h m } ^ { - 2 }$ 。

# 1.3微根管埋设

试验采用微根管技术对食叶草根系生长状况进行研究。于2018年8月在食叶草幼苗期安装CI-600根系监测系统(CIDBio-science，Camas，WA,USA)，使用底部密封的透明塑料根管（与CI-600配套，内径 $6 4 ~ \mathrm { m m }$ ,长度 $6 0 0 \mathrm { m m } .$ 监测食叶草根系生长状态。微根管埋设采用双管法，在植株两侧，与地面呈 $9 0 ^ { \circ }$ 夹角。根管密封的一端插入到土壤中，管子需露出地面 $1 0 \ \mathrm { c m }$ 左右，根管距植物 $5 \mathrm { c m }$ 。不监测时盖上黑色塑料盖子以防止其他杂物落入管中，并在外管壁缠绕黑色胶带，以阻止光线透过根管传播而影响食叶草根系生长。

# 1.4测定方法

在复水后第 $2 \mathrm { d } , 4 \mathrm { d } , 7 \mathrm { d } , 1 0 \mathrm { d } , 1 5 \mathrm { d }$ 用根系监测系统对根系进行监测。使用拉杆送下扫描头，在与扫描头连接的拉杆上，每隔 $2 0 \mathrm { c m }$ 安装拉杆，使用拉杆将扫描头送至待测量的位置，根据埋设角度计算，得到 $( 0 { \sim } 2 0 \ \mathrm { c m } \ . 2 0 { \sim } 4 0 \ \mathrm { c m } )$ 不同深度根系分布图。每次获取的图片长 $\times$ 宽为 $2 1 . 6 0 ~ \mathrm { c m } { \times } 1 5 . 6 0 ~ \mathrm { c m }$ 像素为 $1 2 0 0 \mathrm { d p i }$ ,以TIFF格式保存在电脑里,利用系统配备的WINRHIZOTRON2015a进行分析。

将最新的图像与前期图像叠加，描述被追踪根的演变并报告新特征和状态：如果已经加长或保持乳白色，则为活根，如果已死亡或呈现明显的腐烂迹象(皱缩、透明、微弱或变黑)，则为死根。为了检查根部的死亡状态，在根管同一位置对后期记录的连续图像进行了分析，以确保死根仍然是黑色，没有发生生长。记录所有图像的根径( $\bf \Pi _ { \mathrm { m m } } ^ { \prime }$ ）、活根和死根长度（ $\mathrm { ( c m ) }$ 以及各新根的形状。

# 1.5根系特征指标计算

活根长度(LiveLengthProduction,LLP)和死根长度（Dead LengthLoss,DLL)是从 $_ { t - 1 }$ 到 $\mathbf { \Phi } _ { t }$ （其中 $\mathbf { \Phi } _ { t }$ 为根系扫描间隔)时间段活根总长或死根总长( $_ \mathrm { c m } \cdot$ $\mathbf { m } ^ { - 2 }$ )。

$$
\mathrm { L L P } _ { _ { t - 1 , t } } ( \mathrm { D L L } _ { _ { t - 1 , t } } ) = \sum _ { n = 1 } ^ { N } \frac { l _ { n , t } - l _ { n , t - 1 } } { A }
$$

式中：1为扫描时活(死)根长度 $( \mathrm { c m } ) { \mathrm { { } } } ; n$ 为根的个数（个）， $n \in [ 1 , N ]$ $N$ 为总活根数或死根数（个）； $A$ 是扫描图像观察到的土壤面积 $( \mathbf { m } ^ { 2 } ) { \mathbf { \ H } } _ { { \mathbf { \ H } } }$ 为扫描间隔(d)。

累计活根长度(Cumulative LiveLength Production，CLLP）和累计死根长度（CumulativeDeadLength Loss,CDLL)为到 $\mathbf { \Phi } _ { t }$ 时期活根或死根的总长度（24号 $\left( \mathrm { c m } \cdot \mathrm { m } ^ { - 2 } \right)$ )。

$$
\mathrm { C L L P } _ { \tau } ( \mathrm { C D L L } _ { T } ) { = } \sum _ { t = 1 } ^ { T } \mathrm { L L P } _ { t - 1 , t } ( \mathrm { D L L } _ { t - 1 , t } )
$$

细根净生产量(NetRootProduction，NRP)时间 $\mathbf { \Phi } _ { t }$ 内累计活根长度减去累计死根长度( $( \mathrm { c m } \cdot \mathrm { m } ^ { - 2 } )$ ，得到

时间 $\mathbf { \Phi } _ { t }$ 内的净根生产量。

$$
\mathrm { N R P } _ { _ T } = \mathrm { C L L P } _ { _ T } - \mathrm { C D L L } _ { _ T }
$$

累计净根生产量（CumulativeNetRootProduction,CNRP)时间 $\mathbf { \Phi } _ { t }$ 内累计活根总长度( $\mathrm { c m } \cdot \mathrm { m } ^ { - 2 } .$ 。

$$
\mathrm { C N R P } _ { _ T } = \sum _ { t = 1 } ^ { T } \mathrm { N R P }
$$

死根数 $( D N )$ 在给定的时间间隔内，计算 $\mathbf { \Phi } _ { t }$ 和 $_ { t - 1 }$ 之间死根的差值。

$$
\mathrm { D N } _ { t - 1 , t } = \frac { \mathrm { D N } _ { t } - D N _ { t - 1 } } { A }
$$

式中： $\mathrm { D N } _ { t - 1 , t }$ 为死根数（个· $\mathrm { m } ^ { - 2 }$ ）； $\mathrm { D N } _ { t }$ 为扫描 $\mathbf { \Phi } _ { t }$ 时的死根数(个);A为扫描图像观察到的土壤面积 $( \mathbf { m } ^ { 2 } ) ; t$ $_ { t - 1 }$ 为两侧测量的时间间隔(d)。

通过计算 $_ { t - 1 }$ 和 $\mathbf { \chi } _ { t }$ 时间的根长差来计算单个根系生长，从而确定每个根生长率（RootElongationRate,RER)即在时间 $\mathbf { \chi } _ { t }$ 和 $_ { t - 1 }$ 之间所有根系长度平均值除以相对应的周期。

$$
\mathrm { R E R } _ { { } _ { t - 1 , t } } = \frac { 1 } { N } \sum _ { n = 1 } ^ { N } \frac { l _ { n , t } - l _ { n , t - 1 } } { P _ { { } _ { t - 1 , t } } }
$$

式中：RER是根生长率 $\left( \mathrm { c m } \cdot \mathrm { d } ^ { - 1 } \right) ; n \in \left[ 1 , N \right] ; N$ 为总活根数或死根数； $P$ 为两次采样的时间间隔(d)。以上所有变量对应根直径( $( 0 { \sim } 0 . 5 ~ \mathrm { m m }$ 和 $0 . 5 { \sim } 1 . 0 ~ \mathrm { m m }$ ）和土层( $0 { \sim } 2 0 \mathrm { c m }$ 和 $2 0 { \sim } 4 0 ~ \mathrm { c m }$ ）。

根的存活率 $( S )$ 和中位寿命(ML)用非参数Kaplan-Meier方法来确定，在给定时间段内生长的每个细根都相对独立，在研究结束时被分为存活（经过查看还未死亡)或死亡。公式如下：

$$
S { = } \prod _ { t < T } { \frac { r ( t ) - d ( t ) } { t ( t ) } }
$$

式中：S为存活率 $( \% ) { : } r$ 为活根数（个）； $d$ 为死根数（个）； $T$ 为整个根系监测时间的周期 $( \mathrm { d } ) { \mathbf { \Omega } } _ { \mathbf { { i } } }$ 为测定时

间(d)。

根的寿命是根据第一次观察根到死亡的天数(d)[10]。中位寿命(ML)是估计从根出现到死亡之间经过时间的中位数(d)，周转率 $( T )$ 是中位寿命的倒数，计算公式如下：

$$
T { = } \frac { 1 5 } { \mathrm { M L } }
$$

以RER、NRP和CDLL为因变量，根径和土层深度为自变量，采用广义线性模型(GLM)，考虑各因素之间的交互作用，对各因素进行单因素方差分析（ANOVA），用LSD进行差异性检验( $( P { < } 0 . 0 5 )$ 。采用SPSS21.0软件进行数据处理、统计分析和非参数Kaplan-Meier生存分析，用 $0 \mathrm { r i g i n } 8 . 5$ 软件绘制图形。

器官之一，根长、根体积和平均根直径动态变化能够反映植物对干旱胁迫的响应快慢[22],是评价植物抗旱能力的重要指标[23-24]。受到干旱胁迫后复水会对植物产生补偿效应，其中最先表现的部位是根系[9]。本研究结果表明,干旱后复水,根长、根尖数和平均根直径呈现先增加再减小的趋势，根长增长量在复水后第7d达到最大为 $4 0 . 7 \mathrm { c m }$ ,与王晓雪等[17]对干旱后复水燕麦根系生长的研究结果一致，其原因是复水促使根系产生更多的吸收根，说明复水能够恢复植物根系生长。复水后，食叶草的平均根直径减少，这是因为根长和根尖数的增加，降低了根系的平均根直径，促进了根系的生长；这与魏清江等[24]对于干旱复水下柑橘(Citrusreticulata)根系直径减少的结论一致。郝树荣等25研究表明，在水稻分蘖期干旱胁迫后复水，水稻根长和根尖数增加，表明水稻在分蘖期干旱后复水对水稻根系生长具有一定的补偿效应，这与本研究中复水对食叶草根长和根尖数的影响结果一致。

# 3.2根系直径和土壤深度对细根增长率的影响

根增长率是衡量植物根系活跃程度的指标之一，根增长率越高说明当下生长环境适宜植物生长[10.26]。研究表明,随着土层深度的增加,细根的增长率会随之减小[10]。本研究发现， $2 0 { \sim } 4 0 ~ \mathrm { c m }$ 的根系生长速率高于 $0 { \sim } 2 0 ~ \mathrm { c m }$ 土壤生长速率,这与Wang等[18]对干旱复水下红松研究结果不同。其原因可能是本试验选用的是 $3 \mathrm { ~ a ~ }$ 生食叶草,根系生长中心已近由 $0 { \sim } 2 0 ~ \mathrm { c m }$ 土层转移到 $2 0 { \sim } 4 0 ~ \mathrm { c m }$ 土层，而且 $2 0 \sim$ $4 0 \ \mathrm { c m }$ 土层在复水第2d出现了新生细根, $0 { \sim } 2 0 ~ \mathrm { c m }$ 土层在复水第4d出现新生细根(图4)，而Wang选择红松的幼苗，其根系生长中心还位于浅土层。细根在深层土壤的分布显著影响其对水分和养分的吸收[27],大部分细根(主要是 $0 { \sim } 1 . 0 \ \mathrm { m m }$ )分布在表层土壤,但在干旱期深层根系会有较大的波动[9.28]。本研究发现， $0 . 5 { \sim } 1 . 0 \ \mathrm { m m }$ 细根在 $2 0 { \sim } 4 0 ~ \mathrm { c m }$ 土层波动变化显著大于 $0 { \sim } 2 0 ~ \mathrm { c m }$ 土层，这与Mao等28研究自然条件下木荷(Schimasuperba)细根增长的结果一致,表明多年生植物可能通过深土层细根增长变化来响应土壤中水分的变化。

根直径和土壤深度均会影响到细根的增长速率和积累量[10],本研究表明, $0 { \sim } 1 . 0 \ \mathrm { m m }$ 细根主要分布在 $0 { \sim } 2 0 ~ \mathrm { c m }$ 土层，占监测根系的2/3左右，根直径和土壤深度对细根长度影响显著，这与以往研究结果相似[29],但在根直径和土壤深度的交互作用中，细根增长率没有显著性差异，这与Germon等[9研究杂交核桃树(Juglans regiaXnigra cv.NG23)细根增长对根直径和土壤响应的研究结果一致，表明根系在土壤中先是根长的增长，再是根径的增长，说明土壤对根径的增加具有一定阻碍作用。

# 3.3土层深度对食叶草根死亡率的影响

细根死亡率随着根直径的增加而降低[30-32],植物会保留较粗（ $1 { \sim } 2 ~ \mathrm { m m }$ )的细根，因为其对资源的输送和固定起着重要的作用，并且是 $0 { \sim } 1 . 0 ~ \mathrm { m m }$ 细根的产生者[33]。不同直径细根,其死亡率也各不相同[34]。粗根与细根相比,碳水化合物含量较高，主要成分是木质素和纤维素[35]，营养元素含量低，其原因是较高的C:N，可以促进细根的生长，提高根系生存能力[36-37]。以生长成本为例,细根的生长成本低，在特殊的情况下（干旱、盐碱），植物会增加对粗根的投入,从而维持基本的生存[38-39]

研究发现，复水后 $5 { \sim } 1 5 \mathrm { d }$ ， $0 { \sim } 0 . 5 ~ \mathrm { m m }$ 细根的死亡率显著高于 $0 . 5 { \sim } 1 . 0 \ \mathrm { m m } ( P { < } 0 . 0 5 )$ （图3b），结果与周永姣等[40]研究结果一致，原因可能是前期干旱地上生物量低，光合产物供应不足，无法充足供应这些细根的能量消耗，而且长距离运输需要消耗能量[41]。 $0 { \sim } 2 0 ~ \mathrm { c m }$ 根系死亡率显著大于 $2 0 { \sim } 4 0 ~ \mathrm { c m } ( P { < }$ 0.05),可能是表层根系受到的土壤蒸发[27.42]、耕作措施43和土壤温度等外界因素，造成根系死亡，这与阳维宗等"沼泽草本植物的细根死亡动态变化的研究结果相似。

# 3.4复水对食叶草细根周转率变化的影响

食叶草的细根周转率在1.08\~1.85（15d为一个周期)之间，略微高于干旱后复水燕麦的根系周转率(0.63\~1.25,14d为一个周期)[10],造成这种现象的原因可能是燕麦是一年植物，而食叶草是多年植物，不同植物其根系生长发育方式不同，导致其根系周转率存在差异。本研究表明， $0 { \sim } 2 0 ~ \mathrm { c m }$ 土层细根周转率高于 $2 0 { \sim } 4 0 ~ \mathrm { c m }$ 土层细根,原因是 $0 { \sim } 2 0 ~ \mathrm { c m }$ 土层细根生长和死亡比 $2 0 { \sim } 4 0 ~ \mathrm { c m }$ 土层更容易受到土壤含水量等因素的影响，造成细根的提前死亡分解，从而提高了根系的周转率，这与阳维宗等[8在若尔盖高寒草本沼泽的研究结果一致。赵佳宁等[34]通过对全球森林生态系统细根周转规律研究得出森林的细根周转率每年为 $0 . 6 0 2 { \sim } 1 . 3 1 2$ ，细根周转率随着纬度的升高逐渐降低，因为周转率与根系生长死亡有关，通过10个月的定位数据观察得到食叶草的周转每年在0.89\~1.36，且夏季根系周转率显著低于春秋季。

根系动态明显反映土壤中水分变化剧烈，张坤等[44]研究表明，灌水频率对加工番茄 $0 { \sim } 4 0 ~ \mathrm { c m }$ 土层根系有显著影响，每7d灌溉一次的频率下根生物量、根系分布、水分利用效率均处于最优。对燕麦、温室黄瓜研究表明，7d的灌溉间隔促使细根向深土层分布,增加总根系生物量,同时提高产量[18.45]。研究发现食叶草在盐碱地复水10d后根系增长缓慢、细根系开始出现明显死亡，从盐碱地合理开发利用与水资源节约方面考虑，将灌溉周期设为10d，既能满足食叶草生长需要，也缓解了水资源紧缺问题。

# 4结论

通过研究盐碱地干旱后复水食叶草细根生长发育及衰亡过程，根系存活率动态变化， $0 { \sim } 4 0 \ \mathrm { c m }$ 细根在复水后4\~7d的细根存活率维持在 $7 0 . 6 \% .$ $9 3 . 9 \%$ ， $1 0 \mathrm { ~ d ~ }$ 后出现大幅度下降，到 $1 5 \mathrm { ~ d ~ }$ 后下降到$2 0 . 5 \%$ 。复水后， $0 { \sim } 4 0 ~ \mathrm { c m }$ 土层食叶草细根长和根尖数较复水前均增加 $2 4 . 2 \%$ 和 $6 2 . 1 \%$ ,表明食叶草在干旱后复水通过增加根系吸水范围从而恢复生长； $0 \sim$ $2 0 \mathrm { c m }$ 土层的细根增长量和死亡量比 $2 0 { \sim } 4 0 ~ \mathrm { c m }$ 土层高 $1 4 . 8 \%$ 和 $3 5 . 4 \%$ ,两者相互抵消,表现出 $2 0 { \sim } 4 0 ~ \mathrm { c m }$ 土层周转率、存活量、中位寿命高于 $0 { \sim } 2 0 ~ \mathrm { c m }$ 土层。$0 . 5 { \sim } 1 . 0 ~ \mathrm { m m }$ 较 $0 { \sim } 0 . 5 ~ \mathrm { m m }$ 根系的增长率和中位寿命分别高 $2 3 . 6 \%$ 和 $1 4 . 6 \%$ ，说明 $0 . 5 { \sim } 1 ~ \mathrm { m m }$ 根系对外界的适应能力强于 $0 { \sim } 0 . 5 ~ \mathrm { m m }$ 根系。食叶草根增长量、存活率、累积量均在复水后 $1 0 \mathrm { d }$ 出现明显降低，分别下降了 $1 1 . 2 \% . 3 2 . 6 \%$ 和 $1 8 . 6 \%$ ，在 $1 0 \mathrm { ~ d ~ }$ 后出现了大规模死亡，表明食叶草细根可以维持10d稳定生长。食叶草细根对复水响应的动态变化，体现了其与环境的适应策略，为干旱区盐生植物的灌溉模式制定提供理论依据。

# Response of Rumex hanus by. roots to drought after rehydration

ZHENG Xu, YANG Zhixin， HAO Dongmei, WANG Runrun, Li Luhua, ZHANG Fenghua, WANG Jiaping (Agricultural College of Shihezi University,Shihezi 832Ooo,Xinjiang, China)

Abstract: The rational formulation of irrgation cycle of halophytes is the key factor to improve the utilization efficiency of water resources in arid areas.Taking the fine root ( $\scriptstyle \ d \leqslant 1 \ \mathrm { m m }$ )of leafy grass (Rumex hanus by.) as the research object,the total root length,root tip number and average root diameter at different times after rewatering were measured by drought and re-watering method,and the changing characteristics ofroot growth rate, mortality,rot life span and turnover rate were studied,soas to provide reference basis for the establishmentof the bes irrigation cycle.The results showed that the root length and the number of root tips increased the most at 7 days after re-watering,and decreased at10 to15 days after re-watering.The fine root growth rate (RER) reached the maximum on the 4th day after re-watering,and decreased significantly on the 1Oth day $( P { < } 0 . 0 0 1 \$ ).Therewere significant differences among different soil layers（ $0 { - } 2 0 ~ \mathrm { c m }$ and $2 0 { - } 4 0 ~ \mathrm { c m }$ ）and root diameter( $\mathrm { 0 - 0 . 5 ~ m m }$ and $0 . 5 \mathrm { - } 1 . 0 ~ \mathrm { m m }$ ） $( P { < } 0 . 0 0 1 )$ .Both fine root net production (NRP) and RER increased at first and then decreased.15 days after re-watering, the fine root survival rate of fine root in the $2 0 \ \mathrm { c m }$ soil layer and $2 0 { - } 4 0 ~ \mathrm { c m }$ soil layer was $3 . 6 \%$ and $1 6 . 9 \%$ respectively. The fine root survival rate of $0 . 5 \mathrm { - } 1 . 0 \ \mathrm { m m }$ diameter fine root was higher than that of $0 { - } 0 . 5 ~ \mathrm { m m }$ fine root.The results showed that after re-watering,the life span of fine roots in the soil layer of $2 0 \ \mathrm { c m }$ was significantly higher than that in the $0 { - } 2 0 ~ \mathrm { c m }$ soil layer at 8.09\~13.83 days $( P { < } 0 . 0 5 )$ .The fine root of leafy grass can achieve the purpose of survival by increasing the fine root of $0 . 5 \mathrm { - } 1 . 0 ~ \mathrm { m m }$ in $2 0 { - } 4 0 ~ \mathrm { c m }$ soil layer to cope with and adapt to drought and re-watering. Considering the growth of leaf-eating grassand the supply of agricultural water resources,the best irrigation periodof leaf-eating grass in saline-alkali land in summer is 10 days.
