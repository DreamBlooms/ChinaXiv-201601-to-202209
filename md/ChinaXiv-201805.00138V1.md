# 鼎湖山针阔叶混交林 ${ \tt C O } _ { 2 }$ 浓度及通量变化特征

黄俊²，麦博儒³，邓雪娇，肖薇1\*，王春林²，沈子琦²（1.南京信息工程大学大气环境中心，南京 210044；2.广州市气候与农业气象中心，广州 511430；3.中国气象局广州热带海洋气象研究所，广州510080)

摘要：本文采用珠三角典型森林植被生态系统的鼎湖山南亚热带常绿阔叶林通量观测站 2012 年 $\mathrm { C O } _ { 2 }$ 通量资料和气象资料，分析了 $\mathrm { C O } _ { 2 }$ 通量、净生态系统 $\mathrm { C O } _ { 2 }$ 交换量(NEE)及 $\mathrm { C O } _ { 2 }$ 浓度的变化特征，及其与气象因子的关系，为珠三角地区的碳源汇、碳循环评价及研究提供参考依据。结果表明： $( 1 ) \mathrm { C O } _ { 2 }$ 浓度日变化呈“一峰一谷"变化形态，在夜间或日出前后出现最大值，在午后达到最小值； $\mathrm { C O } _ { 2 }$ 通量日变化呈“单谷”曲线，在白天中午前后到最小值(负值)，夜间和早晨较高(正值)。 $( 2 ) \mathrm { C O } _ { 2 }$ 通量季节平均值的高低次序依次为春季>夏季>冬季 $\mathrm { : } >$ 秋季， $\mathrm { C O } _ { 2 }$ 浓度季节平均值的高低次序则为冬季 $\mathrm { > }$ 春季 $>$ 秋季 $: >$ 夏季，即非生长季高于生长季节，这可能与植物物候的季节变化所引起的 $\mathrm { C O } _ { 2 }$ 源汇强度改变有关。(3)2012年鼎湖山森林生态系统的 $\mathrm { C O } _ { 2 }$ 年均浓度为 $6 6 4 . 7 \mathrm { m g } { \cdot } \mathrm { m } ^ { - 3 }$ ， $\mathrm { C O } _ { 2 }$ 通量年均值为 $\cdot 0 . 0 7 9 \mathrm { m g } \cdot \mathrm { m } ^ { - 2 } \cdot \mathrm { s } ^ { - 1 }$ ，NEE为 $. 6 1 1 \ \mathrm { g } \mathrm { C } \cdot \mathrm { m } ^ { - 2 } \cdot \mathrm { a } ^ { - 1 }$ ，表明鼎湖山针阔叶混交林近年来处于快速生长过程中，具有较强的碳汇功能。 $( 4 ) \mathrm { C O } _ { 2 }$ 通量及浓度与气温、饱和水气压差均成显著负相关关系，其中 $\mathrm { C O } _ { 2 }$ 浓度与气温的相关性最高，其次为饱和水汽压差，表明气温和饱和水汽压差是影响 $\mathrm { C O } _ { 2 }$ 浓度和通量的关键气象因子。受人类活动及气候变化的影响，本文研究发现近年来鼎湖山森林生态系统的碳汇功能有所增强。

关键词：针阔叶混交林， $\mathrm { C O } _ { 2 }$ 通量，涡度相关法，鼎湖山

# Characteristics of $\mathbf { C O } _ { 2 }$ flux and concentration in the mixed conifer-broadleaf forest at the Dinghu mountain

(1.Yale-NUISTCenter on Atmospheric Environment, Nanjing University of Information Science and Technology, Nanjing 210044,China; 2.Guangzhou Climate and Agrometeorology Center, Guangzhou 511430,China; 3.Guangzhou Institute of Tropical and Marine Meteorology,CMA, Guangzhou 510080,China)   
Abstract: This paper aims to analyze the variation characteristics of the $\mathrm { C O } _ { 2 }$ flux, net ecosystem $\mathbf { C O } _ { 2 }$ exchange   
(NEE), $\mathrm { C O } _ { 2 }$ concentration and the relationship with meteorological variables on the basis of the $\mathbf { C O } _ { 2 }$ flux data and   
meteorological data of the southern subtropical evergreen broad-leaved forest flux observation station of Dinghu   
Mountain,a typical forest vegetation ecosystem in the Pearl River Delta,which provides a reference for the   
carbon source and carbon cycle evaluation and research in the Pearl River Delta region.The research results   
showed that: （1） The diurnal variation of $\mathbf { C O } _ { 2 }$ concentration appears to be in the form of“peak-valley".

Specifically, the $\mathrm { C O } _ { 2 }$ concentration variation is nearly at its peak at night or around sunrise. And the $\mathrm { C O } _ { 2 }$ concentration variation is nearly at its valley in the afternoon; Diurnal variation of $\mathbf { C O } _ { 2 }$ flux generally appears to be a“valley” curve，approaching minimum value (most negative) around noon and reaching maximum (positive) in the morning and at night. (2) The seasonal average of $\mathrm { C O } _ { 2 }$ flux can be shown as the following order: spring> summer> winter> autumn. The order of the seasonal average of $\mathbf { C O } _ { 2 }$ concentration can be shown as the following order: winter> spring> autumn> summer,from which we can find that the non-growth season is higher than the growth season in that it is related to the seasonal variation of $\mathbf { C O } _ { 2 }$ source/sink intensity caused by the seasonal variation of plant phenology. (3） In 2O12, the average annual concentration of $\mathbf { C O } _ { 2 }$ in Dinghushan forest ecosystem is $6 6 4 . 7 \ 6 6 5 . 7 \ \mathrm { m g } { \cdot } \mathrm { m } ^ { - 3 }$ . The annual mean value of $\mathrm { C O } _ { 2 }$ flux is $- 0 . 0 7 9 \ \mathrm { m g \cdot m ^ { - 2 } \cdot s ^ { - 1 } }$ ，and net ecosystem $\mathrm { C O } _ { 2 }$ exchange capacity (NEE） is $- 6 1 1 \ \mathrm { g C } \cdot \mathrm { m } ^ { - 2 } \cdot \mathrm { a } ^ { - 1 }$ ，which indicates that mixed confer-broadleaf forest at the Dinghu Mountain has strong carbon sink function in rapid growth during recent years.(4) $\mathrm { C O } _ { 2 }$ flux and $\mathrm { C O } _ { 2 }$ concentration have a significant negative correlation with the temperature and the pressure diffrence of saturated water,of which the correlation between $\mathrm { C O } _ { 2 }$ concentration and the temperature is the highest,and the second is the saturated vapor pressre difference，which indicates that the temperature and saturated vapor presure difference are the key meteorological factors that affect $\mathrm { C O } _ { 2 }$ flux and $\mathrm { C O } _ { 2 }$ concentration.This paper finds that carbon sink function of Dinghu Mountain forest ecosystem has been enhanced in recent years,which may be influenced by human activities and climate change.

Key words: mixed conifer-broadleaf forest， $\mathrm { C O } _ { 2 }$ flux，eddy covariance method，Dinghu mountain.

二氧化碳 $\left( \mathbf { C O } _ { 2 } \right)$ 作为全球大气中最重要的温室气体，其浓度正以平均每年2ppm的速度增加，它对全球气候变暖的贡献率高达 $60 \%$ （Nisbet&Myers et al，2007；Prentice et al，2011；刁一伟等，2015)。森林约占整个陆地表面的 $4 0 \%$ ，是陆地上面积最多且复杂的生态系统，具有独特的生态结构特征及功能，它与大气中的 $\mathrm { C O } _ { 2 }$ 交换频繁，其碳交换量占了陆地生态系统的 $90 \%$ 以上，能贮存的碳量约是大气中的3倍(Rollinger etal，1998)。植物通过光合作用吸收大气中的 $\mathrm { C O } _ { 2 }$ 并长期地储存于生态系统中，同时自身呼吸、土壤微生物呼吸及枯枝落叶分解等过程又会释放 $\mathrm { C O } _ { 2 }$ (Nisbet&Myers et al，2007;Wofsy et al，1993；吴家兵等，2003),掌握森林生态系统的 $\mathrm { C O } _ { 2 }$ 通量及浓度变化特征，对评估森林生态系统碳循环、碳源汇特征及气候变化具有重要意义。

近年来，国内已对 $\mathrm { C O } _ { 2 }$ 浓度及通量开展了较多研究，基于涡动相关系统开展 $\mathrm { C O } _ { 2 }$ 浓度及通量的变化规律方面得到了广泛应用(张弥等，2010)。张法伟等(2012）研究草原净生态系统的 $\mathrm { C O } _ { 2 }$ 通量特征及影响因子，其生长季NEE 主要受气温的影响。陈晓峰等(2016)采用7层 $\mathrm { C O } _ { 2 }$ 观测和涡度相关系统对安吉亚热带毛竹林生长季林的 $\mathrm { C O } _ { 2 }$ 浓度和通量进行了长时间的观测和研究。徐志伟等（2016）利用静态箱-气相色谱法对广州市海珠湖湿地公园 $\mathrm { C O } _ { 2 }$ 通量季节性变化特征以及相关环境因子进行了研究。彭凤姣等（2017）、徐勇峰等(2018)采用涡度相关法分别对大九湖泥炭湿地和洪泽湖湿地典型杨树林生长季的 $\mathrm { C O } _ { 2 }$ 通量进行了观测，对比分析了生长季与非生长季节 $\mathrm { C O } _ { 2 }$ 通量的变化特征及其影响因素，发现洪泽湖水位变化通过影响土壤水分进而可能导致湿地碳汇功能的改变。此外，随着遥感技术的发展，利用遥感手段也可以用来研究森林生态系统 $\mathrm { C O } _ { 2 }$ 浓度及通量的时空变化特征，再结合地面观测数据，可以进一步提高 $\mathrm { C O } _ { 2 }$ 源汇评估的准确性(Saeki et al，2009；布然等，2015；Bu et al，2015)。

珠江三角洲地区(简称珠三角)的生态系统是一个多元化综合且复杂的生态系统(麦博儒等，2014)，其包括南亚热带森林、绿地、草地及农田等生态系统。珠三角是是中国经济贸易最为发达的地区之一，随着工业化和城市化的快速发展，温室气体的排放总量迅速增加，且具有明显的区域性特点(邓雪娇等，2006)。近年来，有关珠三角地区不同生态系统的 $\mathrm { C O } _ { 2 }$ 通量研究逐步展开，并取得一些阶段性的研究成果，发现在不同的区域、不同植被类型的 $\mathrm { C O } _ { 2 }$ 浓度及通量变化及其对气象因子的响应规律并不完全一致(张倩媚等，2002；闫俊华等，2003；周存宇等，2004；王春林等，2006，2007)。受城市建设及气候变化的影响，森林生态系统 $\mathrm { C O } _ { 2 }$ 浓度、通量及固碳效益也会发生变化，关于森林生态系统尤其是南亚热带森林生态系统的$\mathrm { C O } _ { 2 }$ 通量及其与气象因子关系在近几年的研究工作中相对较少。南亚热带森林生态系统拥有丰富多样的物种数及复杂的群落结构，对区域生态系统碳平衡具有重要调节作用(Zhou etal，2003)，南亚热带森林的碳源汇评估已成为大家研究的热点。鼎湖山针阔叶林作为珠三角地区典型的南亚热带常绿阔叶林，是南亚热带地区最具特色和研究价值的地区之一(张倩媚等，2002)。

本文用珠三角地区森林生态系统鼎湖山通量观测站 2012年的数据，研究分析鼎湖山的 $\mathrm { C O } _ { 2 }$ 通量及浓度变化特征，与前人部分研究成果进行对比，并分析影响其变化的关键气象因子，为珠三角地区的碳源汇、碳循环研究提供参考依据。

# 1 材料与方法

# 1.1研究区域概况

鼎湖山自然保护区位于广东省肇庆市 $( 1 1 2 ^ { \circ } 3 0 ^ { \prime } - 1 1 2 ^ { \circ } 3 3 ^ { \prime } \mathrm { E }$ ， $2 3 ^ { \circ } 0 9 ^ { \prime }$ 一 $2 3 ^ { \circ } 1 1 ^ { \prime } \mathrm { N } )$ ，主要为丘陵和低山，海拔在 $1 4 { \sim } 1 \ 0 0 0 \ \mathrm { m }$ 间，鼎湖山的植被是北回归线上保留比较丰富的南亚热带地带性植被，为典型的南亚热带常绿阔叶林，主要为锥栗(Castanopis chinensis)、荷木(Schim asuperba)、厚壳桂(Cryptocarya chinensis)等群落(王春林等，2006)。季风常绿阔叶林样地的土壤类型为赤红壤， $\mathsf { p H }$ 值为3.86，厚度 $6 0 { \sim } 9 0 ~ \mathrm { c m }$ ，地面枯枝落叶层厚度 $1 { \sim } 3 ~ \mathrm { c m }$ ，覆盖度 $8 0 \% \sim 9 0 \%$ 。气候类型属于南亚热带季风湿润型气候，10 月至来年3月为干季，4一9月份为湿季，且光、热、水气候资源比较丰富：年太阳辐射约 $4 ~ 6 6 5 ~ \mathrm { M \cdot J \cdot m ^ { - 2 } \cdot a ^ { - 1 } }$ ，年平均日照时数为 $1 4 3 3 \mathrm { h }$ ，年平均气温 $2 1 . 0 ^ { \circ } \mathrm { C }$ ，1月的平均气温最低为 $1 2 . 0 ^ { \circ } \mathrm { C }$ ，7月的平均气温最高为 $2 8 . 0 ^ { \circ } \mathrm { C }$ 年平均相对湿度为 $82 \%$ ；年均降水量 $1 9 5 6 \mathrm { m m }$ ，其中 $76 \%$ 的降水集中在4—9月(王春林等，2007)。

# 1.2数据观测及处理

鼎湖山南亚热带常绿阔叶林通量观测站安装针阔叶混交林内，海拔高度 $2 4 0 \mathrm { ~ m ~ }$ ，位于鼎湖山自然保护区核心区，是中国科学院通量网(ChinaFLUX)中森林生态系统观测站点的成员之一。涡度相关观测系统安装在通量观测塔上第5层平台(高度 $2 7 \mathrm { m } )$ ，主要观测设备为三维超声风温仪(CSAT3，CampbellInc，USA)及开路 $\mathrm { C O } _ { 2 } / \mathrm { H } _ { 2 } \mathrm { O }$ 分析仪(Li-7500，Li Cor Inc，USA)，数据采集器(CR3000，Campbell Inc，USA)原始采样频率为 $1 0 \mathrm { { H z } }$ ，依据涡度相关方法在线计算并存储30 分钟平均的 $\mathrm { C O } _ { 2 }$ 通量 $( F _ { \mathrm { c } } )$ 和摩擦速度 $( \boldsymbol { u } ^ { * } )$ 等数据。在线计算时，软件自动进行了虚温订正和空气密度变化订正(Webb etal，1980；Schotanus et al，1983)。气温、风速、降水、相对湿度等常规气象观测由CR10X-TD数据采集器采集并记录30 分钟平均值。本文资料的观测时段为2012年，观测时对通量设备进行了零点及跨度的标定及校准，其中因设备故障原因3、4月常规气象数据缺测，11月通量数据缺测。

将 2012年30分钟平均的通量数据和气象数据进行时间一致性匹配，此外，受仪器设备故障、降雨、大气运动等影响会导致数据异常及缺失，需要对数据进行质量控制，剔除符合如下任意一个条件的 $3 0 \mathrm { { m i n } }$ 记录：(1)同期出现降雨;(2)大气湍流不充分，摩擦速度 $\mathbf { u ^ { * } }$ 低于 $0 . 2 \mathrm { m } { \cdot } \mathrm { s } ^ { - 1 }$ ；(3) $\mathrm { C O } _ { 2 }$ 通量有效数据范围为 $- 2 . 0 \sim$ $2 . 0 \mathrm { m g } \mathrm { \cdot m } ^ { - 2 } \mathrm { \cdot s } ^ { - 1 }$ ， $\mathrm { C O } _ { 2 }$ 浓度有效数据范围为 $5 0 0 { \sim } 8 0 0 \mathrm { m g } { \cdot } \mathrm { m } ^ { - 3 }$ ，水汽浓度有效数据范围为 $0 { \sim } 4 0 \ \mathrm { g } { \cdot } \mathrm { m } ^ { - 3 }$ ，超出以上有效范围的数据则剔除(孙春健等，2012)；(4)有明显异常突出的数据。

# 1.3研究方法

涡度相关法(Eddy covariance method)是通过脉动观测值直接计算湍流输送量(湍流通量)的一种方法，也称为湍流脉动法(Baldocchi et al，1988)。 $\mathrm { C O } _ { 2 }$ 通量的计算公式可以分别表示为：

$$
F _ { \mathbf { c } } = \overline { { \omega ^ { \prime } \rho _ { c } ^ { \prime } } }
$$

式中， $\omega ^ { \prime }$ 为垂直风速与平均值的瞬时偏差(即扰动值)， $\rho _ { c } ^ { \prime }$ 是空气中 $\mathrm { C O } _ { 2 }$ 密度的瞬时扰动值， $\omega ^ { \prime } \rho _ { c } ^ { \prime }$ 是瞬时垂直风速和空气中 $\mathrm { C O } _ { 2 }$ 密度的协方差(贾庆宇等，2010；何奇瑾等，2006)。涡度相关法是最直接的测定方法，它可以稳定地监测到 $1 0 \mathrm { H z }$ 的脉动值和在线计算其协方差量(何齐瑾，2006)，该技术已成为国际通量观测网络(FLUXNET)的主要观测技术(Baldocchi&Vogel，1996)。该公式使用时需要做一些假设(王介民等，2007):(1)大气准平稳湍流；(2)水平方向上均匀混合(平流可以忽略)；(3)在近地面存在常通量层；(4)影响通量的涡旋都已被测量到；(5)设备能够监测到的通量为观测所在的下垫面的通量。

饱和水汽压差(Vapor Pressure Deficit，简称VPD)是指某一空气温度下饱和水汽压与实际水汽压的差值(Rogers et al，1996)，是植被蒸散作用的重要驱动要素之一，同时也是生态模型中进行植物水和碳通量模拟最重要的气候因子之一(张红梅等，2014)，它影响着植物叶子气孔的闭合，从而控制植物蒸腾及光合作用等生理过程(Jarvis et al，1976)，进而影响植物的生长发育。VPD 可由大气相对湿度(RH)和气温(T)估算得出(Richard etal，1998).计算公式如下：

$$
\begin{array} { r } { \mathrm { V P D } = 0 . 6 1 1 \times e ^ { \frac { 1 7 . 2 7 \times T } { T + 2 7 8 . 3 } } \times ( 1 - \frac { R H } { 1 0 0 } ) } \end{array}
$$

净生态系统 $\mathrm { C O } _ { 2 }$ 交换量(Net Ecosystem Exchange，NEE)表示生态系统中植物进行光合作用、生物及非生物呼吸或消耗大气中的 $\mathrm { C O } _ { 2 }$ 引起的生态系统碳储量的变化(Garratt，1975)。NEE 可用下述方程描述:

$$
\mathrm { N E E } = F _ { c } + F _ { s }
$$

式中， $\boldsymbol { F _ { \mathrm { c } } }$ 为 $\mathrm { C O } _ { 2 }$ 通量的观测值， $F _ { \mathrm { s } }$ 为观测高度以下 $\mathrm { C O } _ { 2 }$ 储存量。当 $u ^ { * }$ 较低时，表明在垂直方向大气湍流比较微弱，大部分的 $\mathrm { C O } _ { 2 }$ 被储存在空气中，使得 $F _ { \mathrm { s } }$ 较大；当 $u ^ { * }$ 较高时，表明垂直方向大气湍流比较旺盛，植物与大气中的 $\mathrm { C O } _ { 2 }$ 交换能力增强，使得 $F _ { \mathrm { c } }$ 较大(孙春健等，2012)。在 NEE 计算的过程中假设大气的垂直湍流是充分的，平流通量和水平湍流通量可忽略(Wofsy etal，1993；Law et al，1999)。

通常， $F _ { \mathrm { s } }$ 可用多层 $\mathrm { C O } _ { 2 }$ 浓度变化和单层 $\mathrm { C O } _ { 2 }$ 浓度变化这两种方法来估算(Hollinger et al，1994；Griffiset al，2003)。根据前人已有的研究， $F _ { \mathrm { s } }$ 采用单层 $\mathrm { C O } _ { 2 }$ 浓度变化方法计算的结果与用多层 $\mathrm { C O } _ { 2 }$ 浓度变化测量基本一致(Aubinet et al，2001；闫俊华等，2003；孙春健等，2012)，且对全年 NEE 的影响较小。由于本文的研究数据只有一层观测数据，因此，根据 Hollinger 等(1994)的方法对单层 $\mathrm { C O } _ { 2 }$ 浓度变化进行 $F _ { \mathrm { s } }$ 的估算,计算方法如下:

$$
\begin{array} { r } { F _ { s } = \frac { \Delta \mathsf { C } ( \mathsf { z } ) } { \Delta t } \times \Delta \mathsf { z } } \end{array}
$$

式中，△z为通量设备离地面距离(取 $2 7 \mathrm { ~ m } )$ ， $\Delta C ( \mathbf { z } )$ 为 $\mathrm { C O } _ { 2 }$ 浓度两次观测的变化量 $\big | [ \mathrm { C O } _ { 2 } ] _ { \mathrm { t } ^ { - } } [ \mathrm { C O } _ { 2 } ] _ { ( \mathrm { t } ^ { - } \Delta \mathrm { t } ) } \big |$ ， $\Delta t$ 为两次观测数据时间间隔(取1800 s)。考虑到筛选的数据是在湍流充分 $( u ^ { * } { > } 0 . 2 \ \mathrm { m } { \cdot } \mathrm { s } ^ { - 1 } )$ 的情况下，因此需要进一步进行数据质控，剔除符合以下条件的数据： $F _ { \mathrm { s } } { > } 0 . 3 \ \mathrm { m g { \cdot } m ^ { - 2 } { \cdot } s ^ { - 1 } }$ ， $| [ \mathrm { C O _ { 2 } } ] _ { \mathrm { t ^ { - } } } [ \mathrm { C O _ { 2 } } ] _ { ( t ^ { - } \Delta t ) } | { > } 2 0 \mathrm { m g } { \cdot } \mathrm { m } ^ { - 3 }$ ，以减少异常突出的数据(孙春健等，2012)。

# 3结论与讨论

$( 1 ) \mathrm { C O } _ { 2 }$ 浓度和 $\boldsymbol { F _ { \mathrm { c } } }$ 有明显的日变化特征。 $F _ { \mathrm { c } }$ 、 $F _ { \mathrm { s } }$ 、NEE 日变化一般呈“单谷"曲线，在白天中午前后到最小值(负值)，夜间和早晨较高(正值)； $\mathrm { C O } _ { 2 }$ 浓度日变化一般在呈"一峰一谷"曲线，在夜间或日出前后出现最大值，随着太阳出来后浓度降低，在午后降至最小值，日落之后又开始逐渐上升。相关的研究表明，在理想条件下生态系统是没有平流或泄流作用的，夜间湍流较弱时， $F _ { \mathrm { s } }$ 会被日出后植物的光合作用平衡掉，所以在日尺度及更长时间的尺度上 $F _ { \mathrm { s } }$ 应该为0(Aubinet et al，1999)。因此，在计算中长时间尺度的 NEE 时可以忽略 $F _ { \mathrm { s } ^ { \prime } }$ (Lee，1998；吴家兵等，2005)。但在本研究中发现 $F _ { \mathrm { s } }$ 在日及年时间尺度上都不为0，说明了鼎湖山森林生态系统夜间存在平流或泄流作用，使得生态系统夜冠层中白天减小与夜间存储的 $\mathrm { C O } _ { 2 }$ 无法达到平衡(张弥等，2010)，即导致 $F _ { \mathrm { s } }$ 不为0。因此，对于森林等高大植被而言，冠层 $F _ { \mathrm { s } }$ 对生态系统的 NEE 也具有重要影响(Hollinger etal，1994；张弥等，2010)，在进行碳源汇评估时不可忽略。

(2) $\mathrm { C O } _ { 2 }$ 通量和 $\mathrm { C O } _ { 2 }$ 浓度有明显的季节变化特征。 $\mathrm { C O } _ { 2 }$ 通量年均值为 $\cdot 0 . 0 7 9 \mathrm { m g } \cdot \mathrm { m } ^ { - 2 } \cdot \mathrm { s } ^ { - 1 }$ ， $\mathrm { C O } _ { 2 }$ 浓度年均值$6 6 4 . 7 ~ \mathrm { m g } { \cdot } \mathrm { m } ^ { - 3 }$ ；非生长季 $\mathrm { C O } _ { 2 }$ 浓度高于生长季节，这主要是受由植物物候的季节变化所引起的 $\mathrm { C O } _ { 2 }$ 源汇强度的改变所控制，此外也与珠三角地区区域 $\mathrm { C O } _ { 2 }$ 源排放因素的影响有关。

(3)本文估算得到的鼎湖山森林生态系统2012年的年平均NEE 为-611 $\mathrm { g } \mathrm { C } { \cdot } \mathrm { m } ^ { - 2 } { \cdot } \mathrm { a } ^ { - 1 }$ ，高于鼎湖山 2003 年的通量观测结果和亚热带人工针叶林通量观测结果，显著高于我国北方森林和北美森林，符合 NEE 随纬度增加而降低的普适规律，说明鼎湖山针阔叶混交林近年来正在快速生长，使得其碳汇功能有所增强。这可能与鼎湖山处于珠三角西侧，受珠三角地区人口密度高、城市化速度快、及经济迅速等人类活动的显著影响，使得 2003年以来珠三角地区的 $\mathrm { C O } _ { 2 }$ 浓度逐年增加，根据卫星遥感资料监反演得到的2003年至2009年粤西地区和珠三角地区对流层 $\mathrm { C O } _ { 2 }$ 柱浓度的年均增长率分别为 $1 . 8 2 { \times } 1 0 ^ { - 6 } / \mathrm { a }$ 、 $1 . 6 5 { \times } 1 0 ^ { - 6 } / \mathrm { a }$ ，均高于同时期全球的平均水平(麦博儒等，2014)。此外鼎湖山光、温、水等资源比较充沛，利于森林植被向地带性群落演替的快速生长，有利于植物吸收 $\mathrm { C O } _ { 2 }$ ，使得其碳汇功能加强。

$( 4 ) \mathrm { C O } _ { 2 }$ 通量、NEE及 $\mathrm { C O } _ { 2 }$ 浓度与气温、饱和水气压差VPD均成显著负相关关系，与相对湿度和水平风速均成显著正相关关系，气温和饱和水汽压差VPD是影响 $\mathrm { C O } _ { 2 }$ 浓度和通量的关键气象因子。

此外，森林生态系统的碳收支还与林冠下层植被光合作用、呼吸作用以及土壤呼吸作用等有关，前人的相关报道显示，林冠下层的呼吸及土壤呼吸在混交林森林生态系统的碳源汇中具有重要作用，其强度可以占整个森林生态系统呼吸量的 $30 \% { \sim } 8 0 \%$ (Davidson etal，2006；郑泽梅等，2009)，本文由于缺乏辐射和土壤温度等观测资料，无法计算生态系统总呼吸量 $R _ { \mathrm { e c o } }$ ，导致NEE的估算误差可能较大。
