由于养殖条件和试验成本的限制，关于罗非鱼的研究多针对于 $0 . 8 { \sim } 3 5 . 0 \mathrm { ~ g }$ 罗非鱼幼鱼[14-17]，对大规格罗非鱼的研究鲜有报道，而体重在 $2 0 0 \ \mathrm { g }$ 以上的大规格罗非鱼是罗非鱼养殖生产中的主要经济来源。因此，研究大规格罗非鱼饲料中的不同植物油的应用效果具有重要的实践意义。本研究拟探讨不同植物油对大规格吉富罗非鱼生长性能、组织脂肪酸的影响，为大规格吉富罗非鱼饲料配方的优化提供理论依据。

# 1.2 试验动物与饲养管理

养殖试验在广东海大畜牧水产研究中心(以下简称海大研究院)沙北基地进行。试验在室外养殖系统的24个网箱 $( 1 \mathrm { m } { \times } 1 \mathrm { m } { \times } 2 \mathrm { m } )$ 中进行，水深 $3 \mathrm { m }$ 。整个试验过程中随天气变化不定时充气。试验期间水温保持在 $( 3 1 \pm 2 ) ^ { \circ } \mathrm { C }$ ，氨氮浓度小于 $0 . 5 \mathrm { m g / L }$ ，溶氧浓度大于 $5 \mathrm { m g / L }$ pH 约为6.9。

试验用吉富罗非鱼由广东肇庆高要县罗非鱼养殖场提供，经消毒后在大塘中暂养，达到试验规格后转入试验系统，投喂暂养饲料。挑取体质健壮、规格一致、初始体重为(229.92±0.31)g的试验鱼1200尾，随机分为6组，每组设4个重复，每个重复50尾。再随机取2组鱼，每组10尾，取背肌放入冰箱保存，待分析肌肉脂肪酸组成。

正式养殖试验期8周，采用自动投饵机每天投喂2次，08:00和14:30各1次，根据鱼体生长、摄食和死亡等状况及时调整投喂量。试验结束后，将试验鱼饥饿 $2 4 \mathrm { h }$ ，每箱鱼称重并计数。

3讨论

3.1不同植物油对大规格吉富罗非鱼生长性能的影响

已有大量研究证明植物油在水产饲料中的良好应用效果[7-13]。饲喂异育银鲫分别含 $5 . 4 \%$ 鱼油、豆油、菜籽油、亚麻油的饲料8周后，豆油组及菜籽油组的增重率显著高于鱼油组，豆油组、菜籽油组及亚麻油组的增重率无显著差异9。本试验结果显示，经过8周饲喂后，各组吉富罗非鱼的增重率和存活率分别在 $2 2 0 \%$ 和 $100 \%$ 以上，这表明吉富罗非鱼对各植物油即大豆油、棕榈油、棉籽油、菜籽油、磷脂油和混合油均表现出很好的吸收和利用能力。这可能是因为吉富罗非鱼为偏植食性的杂食性鱼类，因此对于植物油有着良好的转化和吸收能力。

各组吉富罗非鱼的摄食量无显著差异，但增重率、特定生长率、饲料系数及蛋白质效率率却出现差异。可见吉富罗非鱼对于含不同植物油的饲料的摄食量虽然相同，但在鱼体内的转化和吸收是有差异的，从而导致了生长性能的差异。

本试验中菜籽油组和大豆油组吉富罗非鱼的生长表现较好。菜籽油含有大量的C18:1n-9，通常首先被β氧化产生能量，为鱼体生长提供了充足的能量。大豆油和菜籽油中充足的C18:2n-6 和C18:3n-3为吉富罗非鱼提供了充足的必需脂肪酸。许多研究也证实了菜籽油和大豆油作为鱼类饲料的替代脂肪源的良好效果[9-10]。

值得注意的是，一般认为饲料中添加磷脂油对仔稚鱼的生长发育十分有利[12-13]。本研究中磷脂油组的增重率、特定生长率、蛋白质效率和饲料系数与菜籽油组无显著差异，表明磷脂油同样对大规格鱼类的生长有促进作用。

棉籽油组饲料的促生长效果不好可能与饲料中C18:1n-9和C18:3n-3含量不足且含有大量饱和脂肪酸有关，因罗非鱼不能很好利用饱和脂肪酸[14]。

混合油组的肝体指数显著高于棕榈油组和菜籽油组，这可能说明棕榈油组和菜籽油组的吉富罗非鱼肝脏能很好的进行脂质代谢，不会在肝脏积累过多的脂肪。此外，混合油组吉富罗非鱼的增重率、特定生长率和蛋白质效率均显著低于菜籽油组，促生长效果不佳。

3.2不同植物油对大规格吉富罗非鱼组织脂肪酸组成的影响

鱼体的最终脂肪酸组成是摄入饲料和内源性代谢共同作用的结果，伴随着消化和吸收，转化为另一种脂肪酸或脂肪酸衍生物而氧化或积累。研究表明，鱼类的肌肉脂肪酸组成与饲料脂肪酸组成有着高度相关性[14-16]。本试验中，饲料脂肪酸组成与组织脂肪酸组成相关性结果表明，大规格吉富罗非鱼肌肉脂肪酸组成明显受饲料脂肪酸组成的影响，肝脏脂肪酸组成受饲料脂肪酸组成的影响较小，与在团头鲂[18]和日本鲈鱼[19]中的研究结果一致。棕榈油组吉富罗非鱼的组织脂肪酸组成与饲料脂肪酸组成相关性最大，但棕榈油组组织中C18:1n-9、C18:2n-6 和C18:3n-3含量都很低，且含有大量饱和脂肪酸，这样的脂肪酸组成并不利于罗非鱼的生长。因此其原因可能是由于该组饲料脂肪酸组成与鱼体初始脂肪酸组成最接近。

在大规格吉富罗非鱼组织脂肪酸中，饱和脂肪酸一般以C16:0 和C18:0为主，单不饱和脂肪酸一般以C18:1n-9 为主，n-6 系脂肪酸主要是C18:2n-6，n-3 系脂肪酸主要是C18:3n-3。在肝脏脂肪酸组成中，饱和脂肪酸一般以C16:0 和C18:0 为主，单不饱和脂肪酸一般以C18:1n-9 为主，n-6 系脂肪酸主要是C18:2n-6，n-3 系脂肪酸主要是C22:6n-3。然而，不同脂肪源之间的脂肪酸组成差异却非常大[14-15.19]。这表明，鱼油替代物不可避免地改变了鱼肉的脂肪酸组成，但鱼体也仍然在控制脂肪酸组成的改变，使之稳定在一定范围内[13.20]。

大量研究表明，同样的脂肪源由于营养史及饲料基础配方不同等原因，而导致饲料的脂肪酸组成不尽相同。不同研究中菜籽油组饲料的 $\mathrm { n } { - } 3 / \mathrm { n } { - } 6$ 分别为 $0 . 2 9 ^ { [ 8 ] }$ 、 $0 . 4 3 ^ { [ 7 ] }$ 和 $0 . 9 0 ^ { [ 2 1 ] }$ ，在本研究中，菜籽油组饲料的 $\mathrm { n } { - } 3 / \mathrm { n } { - } 6$ 为0.20。此外，不同研究者的试验中棕榈油[14-15]、大豆油[7-8,19]的 $\mathrm { n } { - } 3 / \mathrm { n } { - } 6$ 也存在差异。本研究中，尽管大豆油组、磷脂油组和混合油组饲料的 $\mathrm { n } { - } 3 / \mathrm { n } { - } 6$ 相同，但吉富罗非鱼的组织脂肪酸组成却出现很大差异。因此，未来的主要研究方向可能不仅仅是单一的用其他油脂替代鱼油，应更多地关注于在采用替代脂肪源的同时，控制脂肪酸的比例来进行更精确的研究。

现今人类膳食中n-6多不饱和脂肪酸(PUFA)大多都是过量的,但n-3PUFA却严重不足，故认为高 $\mathrm { n } { - } 3 / \mathrm { n } { - } 6$ 的鱼肉对人类来说是更健康的[22]。在本研究中，菜籽油组吉富罗非鱼肌肉中C18:3n-3含量和 $\mathrm { n } { - } 3 / \mathrm { n } { - } 6$ 最高，而饱和脂肪酸总量最低，可以认为菜籽油组吉富罗非鱼的肌肉更符合人类营养需求。

4结论

大豆油、棕榈油、菜籽油和磷脂油可以作为大规格吉富罗非鱼（体重范围： $2 3 0 { \sim } 7 9 0 \ \mathrm { g }$ ）   
饲料中良好的脂肪源。大规格吉富罗非鱼肌肉中脂肪酸组成明显受饲料脂肪酸组成的影响，   
肝脏脂肪酸组成受饲料脂肪酸组成的影响较小。
