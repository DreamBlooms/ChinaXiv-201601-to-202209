摘要：为探究施盐磷对重要海防林树种台湾相思幼苗叶光合作用与养分特征的影响，该文设置 $0 \% ( \mathrm { B } _ { 0 } ) . 0 . 2 \% ( \mathrm { B } _ { 1 } ) . 0 . 4 \% ( \mathrm { B } _ { 2 } ) . 0 . 6 \% ( \mathrm { B } _ { 3 } )$ 和 $0 . 8 \% ( \mathrm { B } _ { 4 } )$ 的 NaCl溶液和不施 $( \mathbf { P } _ { 0 } ) { \mathrm { . 0 . 5 } } \mathbf { g } { \mathrm { . k g } } ^ { - 1 } ( \mathbf { P } _ { 1 } )$ 和 $1 . 5 \ : \mathrm { g } \cdot \mathrm { k g } ^ { - 1 } ( \mathrm { P } _ { 2 } ) \ : 3$ 个供磷水平过磷酸钙磷肥，在此基础设置盐磷6个耦合处理，测定幼苗光合作用和养分特征指标。结果表明：（1）盐胁迫显著抑制台湾相思幼苗的生长发育，盐度含量越高，影响程度越大。低盐施磷对台湾相思幼苗生长不利，中高盐施磷显著减缓盐对幼苗生长的抑制作用。（2）台湾相思幼苗光合作用受盐胁迫影响显著。中低盐施磷后气孔关闭程度上升从而加剧盐胁迫对幼苗光合作用的影响，但高盐适当施磷可显著提高台湾相思幼苗光合能力。（3）盐胁迫显著降低叶绿素含量，且对光系统Ⅱ造成危害。低盐胁迫施磷对台湾相思幼苗叶绿素合成不利，高盐适当施磷可以提高叶绿素合成量、稳定细胞膜结构以及提高叶片潜在光合能力。（4）盐胁迫对叶的Fe、Mn、Na、Cu等金属元素含量有较强干扰。盐胁迫基础上施磷有效促进幼苗对部分元素的吸收。整体上，盐胁迫条件上施磷可以提高台湾相思幼苗光合作用和吸收养分的能力。该研究结果为提高台湾相思树的耐盐性以及磷利用率提供理论依据，为沿海防护林可持续经营提供指导意义。

滨海地区土壤盐碱化和缺磷已威胁到沿海防护林生存和国家生态安全。土壤盐碱化是限制植物生产力的重要环境因素之一（Khalid，2009），当前世界土壤盐碱化面积逐年增加，仅中国土壤盐碱化面积就达3400 多万公顷,滨海盐碱地 $\mathrm { ~ 1 ~ m ~ }$ 土层内含盐量一般均在 $4 ~ \mathrm { g } { \cdot } \mathrm { k g } ^ { - 1 }$ 以上，严重危害包括海防林在内的农林业的可持续经营与发展(Fascella etal.，2018；张华新等，2008)。一方面盐胁迫会对林木产生渗透胁迫和干扰营养离子平衡，通过抑制和诱导多种酶系统影响林木的正常生长，另一方面盐胁迫通过影响PSⅡI潜在活性和PSⅡI原初光能转化效率导致林木光合效率下降（王东明等，2009）。缺磷环境造成植物营养不良极有可能加剧盐伤害作用。南方滨海地区土壤普遍缺磷(苏贻攀，2016)，大部分土壤磷的含量为$0 . 0 4 3 \% { \sim } 0 . 0 6 6 \%$ （张雨瑶和李世友，2013），限制了植物同化力的形成、卡尔文环中酶的活性、RuBP的再生以及同化物的运输进而影响植物的光合作用（潘晓华等，1997），并降低整株磷、钾、钙、氮和镁累积量（韦如萍等，2008），加剧海防林更新困难。忽视滨海地区土壤多盐和少磷共同作用对海防林树种的影响，将低估沿海土壤恶劣环境对海防林经营造成的困难，不利于海防林可持续经营和管理。

台湾相思（Acaciaconfusa）广泛分布于我国沿海地区，是一种维护成本低的重要沿海防护林树种，对我国南方水土保持等贡献巨大（张清林等，2012）。近年研究发现台湾相思具有一定的耐旱（沙桦欣等，2013）、抗低温能力（张柔等，2014）、强光胁迫(Weng etal.,2011)，也已证实高盐低磷胁迫对台湾相思种子萌发和幼苗生长均会产生不利影响，但对幼苗生长不利影响的机理还尚未完全阐明(袁锋等，2021)。现有林木抗性的研究多专注于盐或磷单一胁迫对林木或作物等植物的影响（陈健妙等，2009；尹勇刚等，2020），大面积土壤中去盐工作难度较大，施磷肥等相对较为简单易行。缺磷条件下施磷通常会促进植物生长，提高其光合作用，但补施磷是否有助于减缓或消除盐胁迫对植物幼苗的伤害尚未可知。因此，本文尝试以台湾相思幼苗为研究对象，分别测定不同NaCI浓度以及在此基础上添加磷肥下台湾相思幼苗的生物量、光合参数、叶绿素含量、叶相对电导率、叶绿素荧光以及养分含量，以期明确在盐胁迫条件下施磷多大程度上改变了台湾相思幼苗叶光合作用与养分特征，并为提高台湾相思的耐盐性和磷利用率以及沿海防护林可持续经营提供理论和实践依据。

# 3讨论

生长状况的变化是植物受环境胁迫影响最直观的响应机制（王佺珍等，2017）。盐胁迫下植株生长受影响的生理机制是多方面的，如营养离子平衡受到干扰、内源激素含量发生变化以及植物遗传物质受影响等（王东明等，2009）。本试验发现盐胁迫对台湾相思幼苗的生长起明显抑制作用，并随盐度增高而增强，该结果与多数盐胁迫研究结果一致（汪贵斌和曹福亮，2003；张建锋等，2005），主要原因是虽然高盐胁迫能够激发植物幼苗自身水分调节能力与保水能力，但这种调节须在一定范围内，无论是胞内还是胞间对盐离子的吸收、运输和转运均不能无节制破坏植物体内盐离子浓度平衡而无损植物抗性。在盐胁迫基础上补施磷的试验结果表明对台湾相思幼苗施加磷素过多会降低耐盐性提升水平，这与蒋乔峰等（蒋乔峰等，2013）研究一致，造成这种现象的原因可能是磷提高了钠在木质部中的向顶运输从而影响幼苗的耐盐性，在幼苗盐分敏感时期过量施磷很可能会加剧盐胁迫对幼苗生长的影响（廖红等，2006）。此外，通过含有盐和磷的离子进入细胞后改变植物细胞水势和细胞组分也会影响细胞生长、生物量积累和光合作用。

光合作用是植物产生有机体的重要生理进程,极易受环境胁迫的影响(尹勇刚等,2020)。盐胁迫下植物光合速率下降的原因主要有影响 $\mathrm { C O } _ { 2 }$ 扩散到结合部位、改变负责光反应的细胞器的结构和功能、改变暗反应的化学过程以及抑制同化产物转移（王东明等，2009）。本试验中低盐浓度下幼苗净光合速率、蒸腾速率、气孔导度和胞间 $\mathrm { C O } _ { 2 }$ 浓度的波动说明此时非气孔因素为光合作用的主要限制因素。中高盐处理上述指标均随盐浓度反向变化说明此时气孔因素为光合作用的主要限制因素。植物地上部分对盐胁迫反应有一定的滞后性的主要原因是盐主要从通过根系吸收进入植物体内，由根至叶等地上部位需要一定的时间。此外，叶绿素是光合作用的主要色素，随着盐浓度增加，叶绿素水解酶活性增大，使得叶绿素a与b合成减少，分解增加，从而影响光能的吸收与转化（陈健妙等，2009）。叶相对电导率随盐浓度的增加逐渐增加，主要原因是叶片细胞原生质膜的膜透性增大，离子渗透增强，从而加剧了离子伤害对光合作用的影响（孙海菁等，2009）。因叶片远离根部，结合 $F _ { \nu } / F _ { m }$ 先增加后降低的结果，表明PSII仅在高盐胁迫下受到危害，此时叶片潜在活力降低（柯裕州等，2009）。在盐胁迫基础上补施磷的试验结果表明中低盐胁迫下施磷，植物通过调节体内的水分分配维持根部的细胞内外离子平衡，可能降低叶中水分含量，造成气孔的关闭程度上升，导致光合受阻；高盐胁迫下磷营养的改善极大增加气孔导度，提高幼苗叶绿素a与b含量，并在一定程度上减弱因盐胁迫引起的叶绿素分解代谢，利于提高叶绿体光系统II的光化学效率，发挥叶绿素潜能（张玉斌，2009）。在高盐胁迫下适当增加磷含量还有助于稳定细胞膜结构（王玉凤等，2021）。低磷处理较高磷处理 $F _ { \nu } / F _ { m }$ 增加量小，可能原因是在高盐胁迫下适当增加磷含量能够提高PSI的光化学特性、幼苗潜在活性以及原初光能转化效率（连慧达，2015）。

植物对矿质元素的吸收是生物量积累的基础（邢英英等，2014），也是实现植物良好生长发育的关键。盐胁迫下土壤中的 $\mathrm { \Delta N a C l }$ 通常会改变离子平衡，导致盐离子与营养元素比例发生变化，植物正常生理代谢受影响（陈少良等，2002）。本试验中不同程度盐胁迫下，台湾相思幼苗对磷、铁、镁、铝、铜元素的吸收受到抑制，而锰、钙、钠、钾元素吸收量增加，可能原因是盐胁迫下大量的钠离子破坏了台湾相思幼苗体内的离子平衡，造成严重的单盐毒害，干扰了幼苗对磷、铁、镁、铝、铜元素的吸收。而磷、铁、镁、铝、铜元素含量一定程度上影响植物的生长速率、光合速率、叶绿素合成与多方面抗逆性（史晓龙等，2018），说明长期盐胁迫会对植物的生长速率、光合速率、叶绿素合成与多方面抗逆性产生不利影响。

在盐胁迫基础上补施磷的试验结果表明低盐胁迫下，施加磷一定程度上能够抑制钠离子对幼苗的毒害作用，促进幼苗对其他元素的吸收。中盐浓度下，施加磷能够促进台湾相思幼苗对铁、锰、钾、铜元素的吸收。高盐浓度下，低磷处理较高磷处理对于台湾相思幼苗吸收镁与钾元素起到的促进作用更大，而台湾相思幼苗对其他元素的吸收则是在高磷处理下受到的促进作用更大，主要原因可能是植物在抗盐过程中急需补充相应的金属离子。金属离子在植物体内较为活跃，盐胁迫和施磷均造成根系周围离子环境的巨大变化，植物为保证体内养分均衡，维持正常生理生命活动，金属离子浓度必然发生波动，但因涉及到众多离子，且各离子电荷不同，含量不同，因此，每种离子的变化规律还不明显。

# 4结论

不同盐分不同程度上抑制了台湾相思幼苗的生长发育、光合能力以及获取养分的能力；在盐胁迫基础上施磷，浓度不同效果不同。台湾相思幼苗对低盐浓度有一定的耐受性，随着盐胁迫程度提升，台湾相思幼苗的光合能力与获取养分的能力受到的抑制加强，从而导致台湾相思幼苗难以正常生长发育。高盐胁迫下，对台湾相思幼苗施磷能够有效促进台湾相思幼苗的生长发育、光合能力以及获取养分的能力。本文揭示了施盐磷对台湾相思幼苗叶光合作用与养分特征的影响，但对于台湾相思幼苗盐磷在生理和分子水平交互作用的机理以及各离子波动规律和内在机制尚不清楚，需要更进一步的研究。
