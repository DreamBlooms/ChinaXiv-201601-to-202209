摘要：为了综合了解党参的施硒效应，该文基于水培试验探究不同亚硒酸钠浓度处理条件下党参幼苗硒积累、生长、生理和品质的变化规律。结果表明：（1）适宜硒浓度‘ $( 0 . 2 \mathrm { m g ^ { . } L } ^ { - 1 }$ ）可促进叶片面积、株高、生物量的增长；而高硒浓度（ $\mathrm { \cdot } 1 0 \mathrm { m g { \cdot } L } ^ { - 1 }$ ）则抑制党参生长。（2）增加亚硒酸钠浓度和硒暴露时间均可提高党参幼苗在根和叶中的硒含量；不同器官硒积累表现为根 $>$ 叶>茎。（3）适宜硒浓度（ $( 0 . 2 \ \mathrm { m g { \cdot } L ^ { - 1 } }$ ）处理可提高光合色素含量和根系活力，并减少丙二醛、脯氨酸和过氧化氢的积累；高硒浓度 $\mathrm { { ( 1 0 \ m g { \cdot } L ^ { - 1 } ) } }$ 则与之相反。（4）适宜硒浓度（0.2$\mathrm { \ m g { \cdot } L ^ { - 1 } }$ ）也有利于党参炔苷、多糖、可溶性蛋白的积累；而高硒浓度（ $\mathrm { \cdot } 1 0 \mathrm { m g { \cdot } L } ^ { - 1 }$ ）可对多糖和可溶性蛋白的积累产生不利影响。综上可知，亚硒酸钠对党参具有双重效应，施加适量浓度对党参的生长、植物生理和品质有益，并以 $0 . 2 \mathrm { m g } \mathrm { \cdot L } ^ { - 1 }$ 硒浓度处理的效果最佳。该研究有助于了解亚硒酸钠浓度对施硒效应的影响，为富硒党参的基础研究和栽培提供依据。

# Effects of different selenium concentrations on growth. physiological characteristics and quality of Codonopsis

# pilosula

LI Tingting, DUAN Gengting, ZHANG Lan, WANG Dandan, WU Faming, YAO Qiuyang (Pharmacognosy Deparment, School of Pharmacy, Zunyi Medical University, Zunyi 563000, Guizhou, China)

Abstract: To comprehensively understand the effects of selenium (Se） supplementation on Codonopsis pilosula, effects of sodium selenite levels on Se accumulation, growth, physiology and quality were explored by using hydroponic experiments. The results were as follows: (1)Treatment with $0 . 2 \mathrm { m g } { \cdot } \mathrm { L } ^ { - 1 } \mathrm { S e }$ promoted leaf size, plant height and biomass of $C$ pilosula, while treatment with $1 0 \mathrm { m g { \cdot } L ^ { - 1 } S e }$ suppressed them. (2)Both increasing Se level and exposure time to $C$ pilosula seedlings could raise its $\mathrm { S e }$ content in roots and leaves,and Se was distributed in organs with an order of root>leaf>stem. (3)Treatment with $0 . 2 \ \mathrm { m g { \cdot } L ^ { - 1 } }$ Se increased photosynthetic pigment contents and root activity, and reduced malondialdehyde, proline and hydrogen peroxide content, while treatment with $1 0 \ \mathrm { m g { \cdot } L ^ { - 1 } }$ Se had opposite effects. (4)Treatment with $0 . 2 \ \mathrm { m g { \cdot } L } ^ { - 1 }$ Se had positive effects on accumulations of lobetyolin,polysaccharides,and soluble protein,while treatment with $1 0 \ \mathrm { m g { \cdot } L ^ { - 1 } }$ Se had negative effects on the accumulations of polysaccharides and soluble protein. Collectively, sodium selenite had dual effects on $C .$ ：pilosula； Application of appropriate Se was beneficial to its growth,plant physiology and quality, and $0 . 2 ~ \mathrm { m g } \cdot \mathrm { L } ^ { - 1 } :$ Selevel was the optimum. This work may improve our understanding the efect of selenite level on the biology effect of $C .$ ：pilosula,which would be useful for the basic research and cultivation of Se-enriched C. pilosula.

Keywords: effect of Se supplementation, Se-enriched C. pilosula,lobetyolin,stress response, dual effect

硒（selenium，Se）是人体所必需的微量元素，正常成人每天需摄入 $5 0 { \sim } 4 0 0 ~ \mu \mathrm { g }$ 硒元素（中国卫生行业标准WS/T578.3-2017)。由于硒代半胱氨酸是动物抗氧化酶系统多个成员如谷胱甘肽过氧化酶的活性位点，因而硒元素在提高人体抗氧化和免疫力方面非常重要（Santesmasses&Gladvshev,2021)。随着更多关于硒对人体健康的有益作用的发现和报道，富硒中药材作为特种中药材资源，其开发利用也引发关注。党参(Codonopsis Radix)来源于桔梗科植物党参(Codonopsis pilosula)、素花党参(C.pilosula var.modesta)或川党参(C.tangshen)（国家药典委员会，2020)，是我国产量和需求量最大的中药材品种之一（杨慧珍等，2016)。有研究表明硒化修饰能进一步提高党参多糖的免疫活性（刘宽辉等，2017)，说明党参与硒对人体的补益作用有一定程度的叠加效应，这为富硒党参的栽培和开发提供依据。

党参中的有机硒与总硒的比值高达 $8 0 . 1 \% { \sim } 8 3 . 6 \%$ （周武先等，2021)，但其硒含量参差不齐，一般为 $0 . 0 5 { \sim } 1 . 6 ~ \mathrm { m g { \cdot } k g ^ { - 1 } }$ （0.1\~0.4中位数）（李增禧等，2013)。党参药用剂量根据国家药典一般为每天 $9 { \sim } 3 0 ~ \mathrm { g }$ ，按照正常成人每天需要补充 $5 0 \mu \mathrm { g }$ 硒的标准，党参硒含量需达到$1 . 6 { \sim } 5 . 6 \ \mathrm { m g ^ { . } k g ^ { - 1 } }$ 方可完全补充正常人所需硒元素剂量。产区土壤硒含量不足且缺乏配套富硒栽培技术是导致现有富硒党参药材资源稀少和参差不齐的主要原因。目前党参的硒营养强化研究也处于起步阶段。席旭东等（2015）报道了根际施加硒肥（主要成分为亚硒酸钠）可使党参硒含量达到 $0 . 9 8 \mathrm { \ m g ^ { . } k g ^ { - 1 } }$ ，且能增加党参的溶出物。周武先等（2021）采用大田试验外源喷施亚硒酸钠（按硒元素计 $4 0 0 \ { \mathrm { g } } { \cdot } { \mathrm { h m } } ^ { - 2 } .$ ）不仅提高硒含量达 $0 . 9 4 7 ~ \mathrm { m g { \cdot k g } ^ { - 1 } }$ ，还增加了党参的产量以及多糖和党参炔苷的积累。上述结果表明：硒元素对党参的生长和次生代谢也存在重要影响；常规人工喷施硒肥能提高党参的硒含量，但是存在一定瓶颈效应。要发现硒含量瓶颈效应的制约因素并提出针对性的富硒栽培措施需要了解党参硒积累潜能及施硒效应。

硒元素对植物生长的毒害作用和有益作用被称为双重效应（Hawrylaketal.,2015）。与动物相比，植物的双重效应发现较晚，因为硒对植物生长有益的证据近些年才陆续报道，如水稻(Olyza Sativa)（Yin et al.,2019)、油菜(Brassica napus）（Ulhassan et al.，2019）、芥菜(Brassica juncea)（Eiche et al.,2015）、黄瓜(Cucumis sativus）（Hawrylak et al.,2015）和茶(Camellia sinensis）（王丹丹等，2021）等。硒的双重作用主要表现为浓度依赖性特征，并在一定程度上改变植物生理和代谢过程，从而影响作物产量和品质（周武先等，2021)。因此，研究外源硒对作物生长、生理和品质的影响具有重要现实意义。目前，基于水培实验的党参富硒强化研究尚未见报道，其硒积累潜能及富硒与硒过量所产生的施硒效应仍知之甚少。由于亚硒酸钠及其代谢中间体在体内有强氧化性和抗氧化性，因而能反应抗氧化系统变化的抗逆性指标越来越多被和作物的生长指标一起应用于硒响应相关的基础研究中（Hartikainen etal.,2000)。基于此，本研究拟采用水培技术研究不同水平的亚硒酸钠处理下党参的硒积累特性及其对党参生长、抗逆生理特性和品质指标的影响。

# 1材料与方法

# 1.1材料和试验设计

供试党参种子由甘肃靖远西北道地药材种植经济合作社提供，由遵义医科大学药学院吴发明副教授结合亲本和幼苗植株形态鉴定为桔梗科党参属植物党参(Codonopsispilosula)的种子。党参炔苷对照品（四川省维克奇生物科技有限公司，批号：wkq17031006，纯度： $9 8 \%$ ），乙腈为色谱纯，其余试剂均为分析纯，水为超纯水。

挑选大小均一的党参种子，播种在黄壤土、腐殖土和有机肥（5:4:1）混合均匀的培育基质中，于室内培养架中发芽生长，培养 $6 0 \mathrm { d }$ 左右，待幼苗长约 $6 \mathrm { c m }$ 时，将党参幼苗移栽至装有 $6 \mathrm { ~ L ~ }$ 营养液的通气水培箱中预培养 $1 4 \mathrm { ~ d ~ }$ 。所用营养液为改良版小西茂毅营养液（Konishi etal.,1985），营养液元素组成： $\mathrm { ~ N ~ } 1 1 2 5 ~ \mathrm { \mu m o l { \cdot } L ^ { - 1 } }$ 、 $\mathrm { K } 5 0 0 \mu \mathrm { m o l { \cdot } L } ^ { - 1 }$ 、 $\mathrm { C a } 5 0 0 \mu \mathrm { m o l } { \cdot } \mathrm { L } ^ { - 1 }$ 、$\mathrm { P 5 0 0 \mu m o l { \cdot } L } ^ { - 1 }$ 、 $\operatorname { S } 4 0 0 \ { \mu } { \mathrm { m o l } } { \cdot } { \mathrm { L } } ^ { - 1 }$ ， $\mathrm { M g 4 0 0 \mu m o l { \cdot } L ^ { - 1 } }$ 、A $1 2 0 0 \mu \mathrm { m o l } { \cdot } \mathrm { L } ^ { - 1 }$ 、Fe $2 0 \mu \mathrm { m o l } { \cdot } \mathrm { L } ^ { - 1 }$ 、 $\mathbf { B } \ : 2 \ : \mu \mathrm { m o l } { \cdot } \mathrm { L } ^ { - 1 }$ 、$\mathbf { M n } 2 \mu \mathrm { m o l } { \cdot } \mathrm { L } ^ { - 1 }$ 、 $\mathrm { Z n } 2 \mu \mathrm { m o l } { \cdot } \mathrm { L } ^ { - 1 }$ 、 $\operatorname { C u } 1 . 5 ~ { \mu \mathrm { m o l } } { \cdot } \mathrm { L } ^ { - 1 }$ 、 $\mathrm { M o } \ 0 . 1 5 \ \mathrm { \textmu m o l { \cdot } L } ^ { - 1 }$ 和 ${ \mathrm { N a } } 3 7 \mu { \mathrm { m o l } } { \cdot } { \mathrm { L } } ^ { - 1 }$ ，营养液$\mathsf { p H }$ 调至6。前期营养液筛选实验显示，相同条件下，与霍格兰、通用配方（段雪婷等，2020)这两种营养液相比，党参在小西茂毅营养液培养一个月后长势最好（待发表)。预培养后，挑选生长趋势一致的幼苗进行后期试验，以亚硒酸钠为硒源，设置不同梯度亚硒酸钠浓度，按硒元素计为0（CK）、0.05、0.1、0.2、0.3、0.4、 $1 0 \ \mathrm { \ m g { \cdot } L ^ { - 1 } }$ 共7个处理，分别记为 Se0、Se0.05、Se0.1、Se0.2、Se0.3、Se0.4、Se10，每个处理3次重复，继续水培 $3 0 \mathrm { d }$ 后，分别对根、茎和叶进行取样，对其农艺性状和相关生理生化指标进行测定。

# 1.2试验方法

1.2.1硒含量测定

硒含量测定采用氢化物原子荧光光度法（谢小雪，2021)。分别采集不同处理组党参幼苗根、茎和叶，于 $6 0 \ \mathrm { ~ \textdegree ~ }$ 烘干并研磨成粉，取 $0 . 3 \ \mathrm { g }$ 后加入 $1 0 ~ \mathrm { m L }$ 硝酸-高氯酸 $( 9 : 1 )$ 于电热板上消解，将消解后溶液定容至 $2 5 ~ \mathrm { m L }$ ，测定其硒含量。

# 1.2.2植株形态特征及生长状况

培养期间每7d观察和测量植物的颜色、大小和高度等指标，在培养 $3 0 \mathrm { d }$ 时称量叶和根干鲜重。叶绿素含量采用乙醇研磨紫外吸收法测定（李合生，2000)。

# 1.2.3生理和品质指标测定

可溶性蛋白含量采用考马斯亮蓝G-250染色法测定；丙二醛（MDA）含量采用硫代巴比妥酸法测定；脯氨酸含量采用芘三酮法测定；根系活力采用氯化三苯基四氮唑法测定（李合生，2000)；多糖含量采用硫酸苯酚法测定（成铭等，2020)；过氧化氢含量采用紫外分光光度法测定（高俊凤，2006)。

# 3讨论

# 3.1外源施硒下党参的硒积累特征

亚硒酸钠是植物常从土壤中吸收得到的重要硒源，也是富硒栽培和相关基础研究广泛使用的硒源。植物可直接吸收利用亚硒酸钠，磷酸转运子PT2被发现是植物实现该吸收运输过程的载体（Lietal.,2008)。植物吸收无机硒后，主要在地上部分的叶绿体和细胞质中转化为有机硒，然后转移并积累到不同的组织中（White，2018)。植物可根据硒积累的能力分为超聚硒植物（Se hyperaccumulator）、次级聚硒植物（Se secondary accumulator）和非聚硒植物（Se non-accumulator）（Yuan etal.,2013）。聚硒植物的干重硒含量可达 $1 0 0 \mathrm { m g } \mathrm { \cdot k g } ^ { - 1 }$ 以上，有聚硒和耐硒特性，能作为硒指示植物和硒净化物种。例如，自然生长在恩施硒矿核心区域（其土壤含可溶性硒 $2 . 4 \mathrm { m g } { \cdot } \mathrm { k g } ^ { - 1 }$ ，水中含硒 $0 . 2 7 5 \mathrm { m g } { \cdot } \mathrm { L } ^ { - 1 } )$ 的恩施碎米荠(Cardamine violifolia)的硒含量可达 $1 8 0 0 { \sim } 4 4 0 0 \mathrm { m g { \cdot } k g } ^ { - 1 }$ （Both etal.,2020)。大多数植物如常见的粮食作物、蔬菜和药用植物都是非聚硒植物,硒耐受能力不如聚硒植物，干重硒含量一般不超过 $2 5 \mathrm { m g } { \cdot } \mathrm { k g } ^ { - 1 }$ 。席旭东等（2015）在大田试验中进行施硒处理，得到党参硒含量最大值为 $0 . 9 8 \mathrm { \ m g ^ { . } k g ^ { - 1 } }$ 。本研究也进一步证明党参属于非聚硒植物。在硒浓度为 $0 . 2 \mathrm { m g ^ { . } L } ^ { - 1 }$ 的水培体系中培养，党参的生长未受抑制，其叶和根的硒含量分别可达到 20 和 ${ 5 0 } \mathrm { m g } { \cdot } \mathrm { k g } ^ { - 1 }$ ；在硒浓度为 $1 0 \ \mathrm { m g { \cdot } L ^ { - 1 } }$ 的水培条件下其叶和根的硒含量分别超过100 和 $5 0 0 \ \mathrm { \ m g { \cdot } k g ^ { - 1 } }$ ，但生长受到抑制。这表明党参有较高的富硒潜能。另外，本研究与席旭东等（2015）基于土培的结果差别较大，说明培养方式和培养基质也是影响硒积累的重要因素。其中，土壤的理化因素常会对植物硒吸收产生重要影响（王锐等，2018)。由此可见，通过改变栽培方式或提高土壤硒活化度是今后开发富硒党参及其他富硒作物的潜在途径。目前，基于严格的安全性考虑，在食品中硒限量卫生标准中，富硒茶硒含量范围为 $0 . 2 5 { \sim } 4 . 0 0 ~ \mathrm { m g { \cdot } k g } ^ { - 1 }$ ，富硒稻谷为 $0 . 0 4 { \sim } 0 . 3 0 ~ \mathrm { m g ^ { - } k g ^ { - 1 } }$ ，富硒食品为$0 . 0 2 5 { \sim } 1 \ \mathrm { m g { \cdot } k g } ^ { - 1 }$ （NY/T600-2002，2003)。本实验在水培条件下通过增加外源性亚硒酸钠很容易到达或超过上述标准，说明无土栽培能在添加少量的硒源条件下即可得到高硒浓度的富硒农产品。高硒原材料可为富硒产品的开发提供更多选择空间，可以考虑更高附加值的有机硒的分离纯化产品，例如已有以恩施碎米荠为原料成功开发出“硒蛋白精华片”（刘坤媛等，

2015)。

本文还探究了最适硒浓度（ $0 . 2 ~ \mathrm { m g ^ { . } L ^ { - 1 } }$ ）的水培条件下的硒积累时空变化特征。有报道显示，亚硒酸盐更容易在根部积累有机硒，而往地上部分迁移较少（Lietal.，2020)。本研究结果也支持这一观点，相同处理浓度下党参根部比地上部分积累更多硒元素。生产富硒农产品的目标是在食用或药用部位富集硒元素，在亚硒酸钠处理下根部积累更多硒元素，这一特征可为外源施加硒肥生产富硒党参提供有利条件。此外，党参叶和根的硒积累与亚硒酸钠浓度正相关，且硒积累还随培养时间而不断增加，这也与魔芋等植物中的报道一致（蔡阳光等，2021)，说明党参地上部分和根系都可以蓄积一定量的硒元素，这些富硒资源有待进一步综合开发利用。

# 3.2不同亚硒酸钠水平对党参生长和抗逆生理特性的影响

众所周知，硒水平过高对植物有毒性作用。这主要是因为摄入过量硒元素后植物在蛋白质合成过程中会大量非特异性地引入硒代氨基酸而造成异常蛋白快速积累（Garousi et al.,2017)。随着研究的深入，越来越多的报道显示，硒对植物的生长和发育常有明显的剂量效应，表现为低硒浓度时有促进作用而高硒浓度时有抑制作用（Zhou etal.,2021)。本研究结果与之一致：合适的硒浓度（ $( 0 . 2 \ \mathrm { m g { \cdot } L ^ { - 1 } }$ ）对党参幼苗叶长、叶宽、鲜重、株高以及干重均具有显著的促进作用，而高硒浓度（如 $1 0 \mathrm { m g { \cdot } L } ^ { - 1 }$ ）则会严重抑制党参幼苗的生长。

叶绿素是植物进行光合作用的重要物质，而光合作用产生的有机物是植物生长的物质基础。有报道显示叶绿素含量及其合成途径相关基因的表达对硒浓度敏感，如茶树（胡玉荣，2016；王丹丹等，2021）、黄瓜（Hawrylak et al.,2015）、白术(Atractylodes macrocephala)（张美德等，2021）等。党参中也有相似现象，在 $0 . 2 \mathrm { m g ^ { . } L } ^ { - 1 }$ 硒浓度处理下光合色素含量有所增加，在 $1 0 \ \mathrm { m g { \cdot } L ^ { - 1 } }$ 的硒浓度处理下光合色素含量显著降低。这表明，亚硒酸钠影响光合色素的生成是其调节党参生长的重要途径。

硒对生物体的抗氧化系统的双重作用也被许多学者认为是硒对植物生长的剂量效应的重要原因。适宜浓度下，硒能增强抗氧化酶活性（Ulhassanetal.,2019)，高浓度的硒元素是促氧化剂，可引发体内活性氧（ROS）大量积累，使抗氧化酶活性受损，产生硒胁迫效应（Hartikainen etal.,2000)。MDA和过氧化氢是生物膜系统脂质过氧化和细胞损伤的重要产物，脯氨酸作为是植物体细胞内的渗透调节物质，三者是衡量植物抗逆生理特性的重要指标。在低硒浓度 ( $\mathrm { 0 . 1 { \sim } 0 . 2 \ m g { \cdot } L ^ { - 1 } }$ ）下党参的MDA、过氧化氢和脯氨酸含量均有所降低，在高硒浓度 $( 1 0 \mathrm { m g } { \cdot } \mathrm { L } ^ { - 1 } )$ 下三者均显著升高。而根系活力随硒浓度的变化呈现先增加后回落的趋势。这提示，在适宜的亚硒酸钠浓度下，党参的过氧化酶体系活性增加，ROS物质减少，使得根系活力增强能促进植物生长；而在高亚硒酸钠浓度条件下，ROS快速积累，细胞膜受损，体细胞渗透压失衡，产生硒胁迫效应，需要更多脯氨酸以提高植物抗逆性。

# 3.3不同亚硒酸钠水平对党参品质的影响

党参含有党参炔苷、多糖等多种化学成分，具有提高免疫和抗氧化等功能（Gao et al,2020)。作为一种药食两用作物，其党参炔苷、多糖和蛋白质含量常作为党参药材质量和营养价值的重要指标。本研究显示，在最适宜生长的硒浓度 $( 0 . 2 \ \mathrm { m g \cdot L } ^ { - 1 } )$ 处理下，党参炔苷、党参多糖和蛋白质含量都有所增加。值得关注的是，在高浓度的亚硒酸钠胁迫条件下党参多糖和蛋白含量均有所降低，但是炔苷显著升高。炔苷具有抗氧化作用（Yoonetal.,2014)，党参是否可能利用炔苷在亚硒酸钠胁迫条件下增强抗氧化性从而提高其适应性仍需要进一步研究。生产富硒农产品的总体目标是在食用或药用部位富集硒元素，同时尽量减少硒营养强化对植物生长、营养价值及药材品质的损害（Jaiswaletal.,2011)。总体而言，高亚硒酸钠浓度可使党参品质下降，适当的亚硒酸钠浓度处理可以提高其品质，因此控制硒浓度是植物硒营养强化趋利避害的重要途径。综上所述，药食两用的党参可以作为一种有很好开发潜力的富硒作物，有待进一步综合开发。

# 4结论

亚硒酸钠处理水平对党参的施硒效应有重要影响。在硒浓度为 $0 . 2 \mathrm { m g } { \cdot } \mathrm { L } ^ { - 1 }$ 的水培体系中可得到叶和根的硒含量分别为20 和 $5 0 \ \mathrm { \ m g { \cdot } k g ^ { - 1 } }$ 的富硒党参，该浓度对党参的生长、植物生理和品质也有益；高硒浓度（ $\mathrm { \Omega _ { \mathrm { 1 0 \ m g { \cdot } L } } { ^ { - 1 0 \ m g { \cdot } L } } ^ { { - 1 } } }$ ）可使党参的叶和根的硒含量分别超过100和 500mg:kg，但对党参幼苗有害，使党参出现胁迫反应，抑制植物生长，降低党参品质。本研究有助于了解党参的硒积累特性和施硒效应，为富硒党参的基础研究和栽培提供依据。
