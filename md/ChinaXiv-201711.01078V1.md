1935 年Underwood 和 Marston 分别发现羊的消瘦病是由缺钴引起的，从而确定了钴是动物必需的微量元素，这也是确认的第6 种动物必需的微量元素[Il。钴是维生素 $\mathbf { B } _ { 1 2 }$ （钴胺素）的重要活性组分，主要通过维生素 $\mathbf { B } _ { 1 2 }$ 参与机体造血和营养物质的消化、代谢等，进而影响体内蛋白质、脂肪和糖的代谢。钴在机体代谢过程中，能激活磷酸酶、精氨酸酶和过氧化氢酶，且能抑制细胞色素氧化酶和琥珀酸脱氢酶的活性，改善基础代谢中糖和氮的吸收，促进动物生长[2]。在家兔上，本课题组前期试验发现，饲粮中添加不同水平 $\mathrm { ( 0 . 1 { \sim } 6 . 4 \ m g / k g ) }$ 的钴对生长獭兔的日增重和饲料转化效率均没有显著影响[3]。NRC（2005）在对禽类的研究中发现含硫氨基酸（尤其是半胱氨酸）易与钴形成复合物，降低钴的吸收率。Kincaid 等[4]研究表明，奶牛干奶期（尤其是产前 $5 5 \sim 2 0 \mathrm { d }$ ）血清中维生素 $\mathbf { B } _ { 1 2 }$ 的含量下降，饲粮中补充钴可以增加瘤胃中维生素 $\mathbf { B } _ { 1 2 }$ 的合成量,还可以增加初乳和常乳中维生素 $\mathbf { B } _ { 1 2 }$ 的含量。Schwarz等[5]对处于生长期的奶牛和成熟奶牛的研究证实机体所需钴的量大于NRC 的推荐值。这些结果说明不同物种间对钴的需要量差异较大。  
钴也可促进纤维分解菌活动，从而促进纤维消化[6。动物一旦缺钴，就会出现异嗜、拒食、生长不良、消瘦和贫血等症状，以及血液中维生素 $\mathbf { B } _ { 1 2 }$ 含量下降、甲基丙二酸和同型半肤胺含量升高、红细胞减少、血红蛋白含量下降等[7-9]。近年来，钴因其具有重要的生物学功能而被营养学家所重视。饲粮中钴可提高动物对饲料尤其是劣质饲料的消化率，Krasnodebska[1]报道，给饲喂半合成饲粮的阉公羊每天补充 $1 ~ \mathrm { m g }$ 钴，可使粗纤维消化率提高 $1 3 . 4 \%$ 。Singh 等[1给特牛饲喂低质牧草并补充尿素，再另外补充钴后发现非蛋白氮和其他养分的利用率明显提高。穆秀梅等[12]研究表明，饲料中添加钴和维生素 $\mathbf { B } _ { 1 2 }$ 有利于断奶羔羊的生长发育，增加羔羊的日增重。Hertz 等[13]研究指出，饲料中添加钴使鲤鱼的成活率、受精卵的孵化率、蛋白质的合成率等均有提高或改善。但是目前钴的研究多集中在反刍动物和水产动物方面，在家兔上钴的添加是否也能影响到饲粮中营养物质的消化代谢，目前尚未知。因此，本试验在饲粮中添加不同水平的钴，通过测定饲粮营养物质表观消化率、氮代谢和钴代谢等指标，进一步确定饲粮中钴的添加水平，为确定生长獭兔钴需要量提供参考。

# 1.3样品采集与制备

饲养试验结束前6d，每组随机抽取8只试验兔，转移到经消毒处理的代谢笼内，单笼饲养，并饲喂相应的试验饲粮，自由采食和饮水。预试期 $3 \mathrm { ~ d ~ }$ ，预试期后连续3d收集每只试验兔全天的粪样和尿样，4℃密封保存，同时记录每只试验兔每天的采食量、排粪量和排尿量。鲜粪称重后，取其中一部分称重后加 $10 \%$ 的硫酸固氮，每天取样比例相同，然后在烘箱中于65℃下烘 $7 2 \mathrm { { h } }$ ，称重即得粪样的风干重，最后将3d的风干粪样混合粉碎，混匀后-20℃冷冻保存待测。尿样测量后，取一定比例盛入备好的 $2 5 0 ~ \mathrm { m L }$ 塑料瓶中，加入 $5 \mathrm { m L }$ 浓硫酸固氮，每天取样比例相同，混匀后置于一20℃冷冻保存待测。

在试验开始后的第54天清晨空腹称重，每组随机抽取8只试验兔空腹称重后立即心脏采血 $1 0 ~ \mathrm { m L }$ ，分离血清，置于-20℃冷冻保存待测；屠宰后取组织样品，液氮保存后待测。

1.4 测定指标和方法

# 1.4.1 饲粮营养物质表观消化率

能量通过美国 Parr-6200 高压氧弹仪进行测定;干物质(DM)含量依据GB/T6435—2006所述方法测定；粗纤维（CF）含量采用酸碱洗涤法测定；粗脂肪（EE）含量采用索氏抽提法测定;粗灰分(Ash)含量采用高温灼烧法测定；中性洗涤纤维(NDF)、酸性洗涤纤维(ADF)、酸性洗涤木质素（ADL）含量采用范氏纤维洗涤法测定；钙（Ca）含量采用高锰酸钾滴定法测定，磷（P）含量采用钼黄比色法测定，钴含量采用GB/T13884—2003所述方法测定。

上述指标具体测定方法参见张丽英[5]主编的《饲料分析及饲料质量检测技术》。饲粮营养物  
质表观消化率根据以下公式计算：饲粮营养物质表观消化率 $( \% ) = 1 0 0 \times$ (食入的营养物质量一粪中相应营养物质量)/食入  
的营养物质量]。

# 1.4.2 氮代谢指标

饲粮、粪和尿中氮含量利用凯氏定氮法进行测定，具体方法参见张丽英[15]主编的《饲  
料分析及饲料质量检测技术》。氮代谢指标根据以下公式计算：可消化氮（digestible nitrogen,DN,g/d) $\mid =$ 食入氮（nitrogen intake, IN)-粪氮(fecal nitrogen,  
FN)；沉积氮(retention nitrogen，RN，g/d）=IN-FN-尿氮（urinary nitrogen，UN);氮表观消化率(nitrogen apparent digestibility，NAD， $\%$ $_ { \mathrm { \Delta \ p = 1 0 0 \times } }$ DN/IN ;氮利用率(nitrogen utilization rate，NUR， $\% ) { = } 1 0 0 { \times } \mathrm { R N / I N }$ 氮生物学效价(nitrogen biological value，NBV， $\%$ $_ { \mathrm { p = 1 0 0 } } \times$ RN/DN。

# 1.4.3 胰腺消化酶活性

胰腺胰蛋白酶和糜蛋白酶活性分别采用南京建成生物工程研究所生产的胰蛋白酶测试盒和糜蛋白酶测试盒测定。具体原理为：胰蛋白酶能催化水解底物精氨酸乙酯的酯链，使其在 $2 5 3 ~ \mathrm { n m }$ 处的吸光度值升高，根据吸光度值的变化可以计算出酶的活性；而糜蛋白酶可以水解酪蛋白产生含酚的氨基酸，而酚试剂可被含酚的氨基酸还原成蓝色物质，通过比色可测定酶的活性。

# 1.4.4 血清生化指标

血清总蛋白(TP)、白蛋白(ALB)和尿素(UR)含量均采用日本和光纯药工业株式会社提供的试剂盒，按照试剂盒说明书在日立7020型全自动分析仪上进行测定。

# 1.4.5 饲粮、粪、尿及组织中钴含量

饲粮、粪、肌肉、脾脏、肾脏样品在105℃下烘干后，取适量，在马福炉中炭化至无炭粒，然后用 $\mathrm { H N O _ { 3 } } \mathrm { - H C l O _ { 4 } }$ （3:1）消化，煮沸数分钟后冷却，定容过滤，采用原子吸收分光光度法对样品中钴含量进行测定。血清和尿用 $\mathrm { H N O _ { 3 } } \mathrm { - H C l O _ { 4 } }$ （4:1）消化后，采用原子吸收分光光度法测定其中钴含量。

用SAS9.1.3统计软件中的GLM程序进行数据的方差分析，并采用Duncan氏法进行数据的多重比较。

3讨论

3.1饲粮钴添加水平对断奶至3月龄獭兔饲粮营养物质表观消化率的影响

獭兔合成维生素 $\mathbf { B } _ { 1 2 }$ 需要钴的参与，而维生素 $\mathbf { B } _ { 1 2 }$ 作为甲基丙二酰辅酶 A 变位酶和 5-甲基四氢叶酸甲基转移酶的辅酶，参与糖的异生过程和蛋氨酸合成。在羊上，Becker 等[16]发现缺钴绵羊对CF 的表观消化率显著高于不缺钴的绵羊；Pal等[7]发现饲喂 $0 . 2 5 ~ \mathrm { m g / k g }$ 体重的硫酸钴显著提高绵羊总碳水化合物、无氮浸出物和CF的消化率。Kadim 等[18]比较了饲喂低水平钴饲粮和皮下注射 $2 0 0 0 { \mu \mathrm { g } }$ 钴山羊的养分消化率，结果发现低水平钴饲粮使DM、CP、EE 的表观消化率和消化能均降低。Lopez-Guisa 等[19]指出，在低质饲粮中给青年牛提供高于 NRC 推荐量的钴（ $0 . 2 4 ~ \mathrm { m g / k g }$ ）可促进低质饲粮的消化。Tomlinson 等[20]报道，钴的营养功能除了通过瘤胃微生物合成维生素 $\mathbf { B } _ { 1 2 }$ 外，还可提高纤维分解菌的活性，从而促进纤维的消化。然而，Saxena 等[21]在特牛饲粮中添加0.25、0.30、0.40 mg/kg 钴后表明，DM、CP、EE 和CF的表观消化率均不受钴添加水平的显著影响;Hussein 等[22]在体外试验中发现，补加高剂量 $\mathrm { 5 { \sim } 3 0 ~ m g / k g }$ ）的钴对 DM 和CF 的消化率没有显著影响；Kisidayovaas 等[23]的体外试验也得出，在含钴 $0 . 2 ~ \mathrm { m g / k g }$ 的基础饲粮中补加钴（2、4、8mg/kg）并不能促进DM、NDF 和ADF 的降解。本试验结果表明，饲粮钴添加水平对獭兔饲粮中各营养物质的表观消化率均无显著影响，但DM、CP 和钴的表观消化率随着饲粮钴添加水平的升高有先升高后降低的趋势。有关钴对营养物质表观消化率影响的研究较多，但结果不尽一致，这说明钴对营养物质表观消化率的影响存在物种差异性。

# 3.2饲粮钴添加水平对断奶至3月龄獭兔氮代谢的影响

钴作为维生素 $\mathbf { B } _ { 1 2 }$ 的重要组成部分，与氮的同化、血红素和肌肉蛋白的合成密切相关[24]饲粮缺钴，则维生素 $\mathbf { B } _ { 1 2 }$ 的合成缓慢，影响饲粮中含氮物质的利用[25]。钴能促进喂以非豆科干草的牛对尿素的利用和对纤维素的消化，并能提高牛的生长速度。Mburu 等[26]给东非小山羊饲喂缺钴饲粮，与饲喂足够量钴(硫酸钴形式)饲粮相比，UN 较高，表明瘤胃中饲粮氮的发酵减少，氮的损失增加，可导致生长率降低和身体状况不良。Roginski等[27]报道，饲粮中添加钴使鲤鱼的生长速度、成活率、受精卵的孵化率、蛋白质的合成等均有提高或改善。Anadu 等[28]报道，在罗非鱼的饲粮中添加氯化钴，会使蛋白质合成加快。在本试验中，饲粮钴添加水平对獭兔FN、DN、NAD、NBV 均无显著影响，对 IN、UN、RN、NUR 有显著影响，随着饲粮钴添加水平的升高，IN、RN 和NUR 均先降低后升高再降低，UN 则逐渐降低，且 IN、RN 和 NUR 均在钴添加水平为 $1 . 6 \ \mathrm { m g / k g }$ 时达到最大值，说明在獭兔饲粮中添加适宜水平的钴能够提高蛋白质的消化率和利用率，其消化率的提高可能是由于胰腺中蛋白质消化相关酶活性增大所致，而血清中UR 含量的降低也间接证明机体蛋白质利用率的提高。血清中 TP 和 ALB 含量并未随饲粮钴添加水平的变化而发生显著变化，这与李庆云等[29]在北京鸭饲粮中添加1.0 和 $2 . 0 ~ \mathrm { m g / k g }$ 钴显著提高血浆TP及ALB 含量的结果不一致，这可能是由于所研究物种不同所致。

# 3.3饲粮钴添加水平对断奶至3月龄獭兔钴代谢的影响

利用组织钴含量作为评价钴营养状况的指标，其优点在于样品污染降低，使测定更为准确。饲粮钴吸收的主要部位是小肠后段，与铁部分共享肠黏膜转运通道。动物体内的钴含量较低，为 $0 . 0 3 { \sim } 0 . 0 6 \ \mathrm { m g / k g }$ ，其中 $43 \%$ 分布于肌肉组织， $14 \%$ 分布于骨组织，其余的 $43 \%$ 则分布于其他各种软组织内。钴在肝脏、肾脏中的含量较高、其次是肾上腺、脾脏、胰腺和骨骼组织，其余组织和器官中的含量则较少[30]。血液中钴含量很低而且变化幅度大，全血钴含量为 $3 8 ~  { \mu \mathrm { g / d L } }$ ，而血浆钴含量 $0 . 5 { \sim } 0 . 7 \ \mathrm { m g / d L }$ ，表明大量的钴存在于红细胞中。血浆中的钴大部分与 $\mathfrak { a }$ 球蛋白和 $\beta \cdot$ -球蛋白形成不稳定化合物，少部分以离子形式存在，血液中钴含量可反映饲粮钴营养的状况[31]。在本研究中，血清钴含量未随饲粮钴添加水平的变化而发生显著变化。已有表明动物血清中的元素含量相对稳定，一般不受元素添加量的影响，而肝脏是元素主要的代谢器官，受饲粮中元素含量的影响较大，一般与饲粮元素添加量呈正相关[32-34]。在本研究中，獭兔肌肉、肝脏和肾脏中钴含量均随着饲粮钴添加水平的升高而持续增加。钴主要以阳离子形式随尿排出，少量与胆汁一起排泄，未被利用的钴大部分随粪排出，随着饲粮钴添加水平的增加，尤其是 $6 . 4 ~ \mathrm { m g / k g }$ 钴添加组，其粪和尿中钴含量显著高于其他各组，这可能是由于饲粮中钴添加过量所致。

4结论

综合本试验测定指标，断奶至3月龄獭兔饲粮钴适宜添加水平为 $0 . 4 { \sim } 1 . 6 \mathrm { m g / k g }$ （饲粮中钴含量实测值为 $0 . 6 0 { \sim } 1 . 8 3 \mathrm { m g / k g } )$ 。

Abstract: This experiment was conducted to study the effects of dietary cobalt supplemental level on dietary nutrient apparent digestibility， nitrogen metabolism，digestive enzyme activities in pancreas,serum biochemical indices and cobalt metabolism of weaned to 3-month-old Rex rabbits. Two hundred weaned Rex rabbits with the similar body weight were randomly assgned into 5 groups with 4O replicates in each group and each replicate contained 1 rabbit. Rabbits in 5 groups were fed experimental diets which supplemented with O, O.1, O.4,1.6 and $6 . 4 ~ \mathrm { m g / k g }$ cobalt ((in the form of cobalt sulfate)) on the basis of a basal diet,respectively. The trial lasted for 7 days for adaptation and 53 days for feeding test. The results showed that dietary cobalt supplemental level had no significant influence on dietary nutrient apparent digestibility $( P { > } 0 . 0 5 )$ ，but with the increase of dietary cobalt supplemental level, the apparent digestibility of dietary dry mater (DM), crude protein (CP) and cobalt tended to be increased firstly and then decreased,and the highest DM and CP apparent digestibility appeared in $1 . 6 ~ \mathrm { m g / k g }$ cobalt supplementation group and the highest cobalt apparent digestibility appeared in $0 . 4 ~ \mathrm { m g / k g }$ cobalt supplementation group. Dietary cobalt supplemental level had no significant influences on fecal nitrogen (FN), digestible nitrogen (DN), nitrogen apparent digestibility (NAD) and nitrogen biological value (NBV） $( P { > } 0 . 0 5 )$ ,but had significant influences on nitrogen intake (NI), urinary nitrogen (UN),retention nitrogen (RN) and nitrogen utilization rate (NUR) $( P { < } 0 . 0 5 )$ . With the increase of dietary cobalt supplemental level,NI, RN and NUR tended to be decreased firstly and then increased, UN gradually tended to be decreased, and the highest NI, RN and NUR all appeared in $1 . 6 ~ \mathrm { m g / k g }$ cobalt supplementation group.Dietary cobalt supplemental level had significant influence on trypsin activity in pancreas （20 $( P { < } 0 . 0 5 )$ ， and the highest value appeared in $0 . 4 ~ \mathrm { m g / k g }$ cobalt supplementation group. Dietary cobalt supplemental level had significant influence on serum urea (UR) content ( $( P { < } 0 . 0 5 )$ ,with the increase of dietary cobalt supplemental level, the serum UR content tended to be decreased firstly and then increased, and the lowest value appeared in $1 . 6 ~ \mathrm { m g / k g }$ cobalt supplementation group. Dietary cobalt supplemental level had no significant influences on serum total protein (TP) and albumin (ALB) contents $( P { > } 0 . 0 5 )$ . Dietary cobalt supplemental level had significant influences on the content of cobalt in fecal,urine,muscle,spleen and kidney $( P { < } 0 . 0 5 )$ ，with the increase of dietary cobalt supplemental level, the content of cobalt in fecal, urine, muscle,spleen and kidney continued to be increased.
