# 炭化温度和时间对盐角草生物炭产率及理化性质的影响

毛明月}²，赵振勇'，王守乐²，田长彦}（1．中国科学院新疆生态与地理研究所荒漠与绿洲生态国家重点实验室,新疆乌鲁木齐830011;2．中国科学院大学，北京100049)

摘要：新疆是中国最大盐土分布区,盐生植物资源极为丰富且具有较强的吸盐聚盐能力。以盐角草（Salicormiaeuropaea)为材料,研究炭化温度和时间对盐角草的生物炭产率及理化特性的影响。结果表明：炭化能有效增加盐角草 $\mathrm { p H } \sqrt { C }$ (电导率)和灰分含量，其中生物炭 $\mathrm { \Delta \ p H }$ 最高达10.37，生物炭灰分含量与盐角草灰分含量相比增加了$5 7 . 5 1 \% \sim 1 1 0 . 9 8 \%$ 。在 $3 0 0 \sim 6 0 0 \mathrm { ~ \textdegree C }$ ,随着炭化温度的升高，生物炭 $\mathrm { p H } , E C$ 、灰分含量显著增加，而产率降低;生物炭水溶性 $\mathrm { C a } ^ { 2 + } \cdot \mathrm { M g } ^ { 2 + }$ 含量随温度( $\geqslant 4 0 0 ~ \mathrm { ~ \mathcal ~ { C } ~ }$ 降低， $\mathrm { \Delta N a } ^ { + } \mathrm { \Omega } \cdot \mathrm { K } ^ { + }$ 等低价离子富集;生物炭全钾、全磷含量随温度逐渐增加，全碳、全氮含量则降低。随着炭化时间的增加， $\mathrm { p H } , E C$ 及灰分含量逐渐升高，碳含量逐渐降低，且水溶性离子含量增加。根据生物炭理化特性,炭化时间 $^ { 2 \mathrm { ~ h ~ } }$ 为最适宜的炭化时间。该研究可为生物炭改良酸性土壤原料选择以及植物生物炭制备工艺优化提供新途径。

关键词：盐角草；生物炭；炭化温度；炭化时间；理化性质

盐角草（Salicorniaeuropaea）为藜科盐角草属一年生草本植物，茎肉质化，是已知最耐盐的真盐生植物之一[],具有高抗盐能力和盐富集能力,常被作为生物措施用于盐碱地的改良[2],广泛分布在盐沼和低湿洼地高盐度区[3],在生态保护和恢复中起重要作用，是恶劣生境下的重要碳汇植物。刘割后的盐角草可用作生物盐添加剂，与饲用玉米混合青贮以制备青贮饲料，制作生物炭是其资源化利用的一个新途径。

生物炭特性与其材料来源及制备条件有着密切的联系。目前，常用的生物炭制备原料主要为林草和作物的废弃物（树枝、木屑、秸秆、糠皮等）及酒糟、甘蔗渣、畜禽粪等有机废物[4-7],以盐角草等真盐生植物作为原材料制取生物炭的研究尚未见报道。生物炭制备过程中炭化时间和温度是非常关键的因素。生物炭的炭化程度会随着炭化温度的升高而增加，而生物炭产率等则降低; $\mathrm { \ p H }$ 随炭化温度升高而升高,并与灰分含量呈正相关[8-]。另有研究表明，炭化温度 $5 0 0 \ \mathrm { ~ \textdegree ~ }$ 时，生物炭呈现稳定状态[12-14]。Lua 等[15]以开心果壳为原料,在 $5 0 0 \ \mathrm { ^ q C }$ 下制备生物炭，裂解时间从 $^ \mathrm { ~ 1 ~ h ~ }$ 增加至 $^ { 2 \mathrm { ~ h ~ } }$ 后，生物炭的产率显著降低，当裂解时间继续增加时，生物质炭的产率基本保持在 $30 \%$ 左右，而生物炭的比表面积则呈现先增加后减小的变化趋势。

我国南方地区土壤中淋溶作用强烈，土壤中钾、钙和镁等盐基养分含量低，土壤酸度高，使土壤阳离子交换量（CEC）减小，土壤的保肥能力下降[16]生物炭一般呈碱性,含有 $\mathrm { { N a } ^ { + } , K ^ { + } , C a ^ { 2 + } , M g ^ { 2 + } }$ 等盐基离子，进入土壤以后会有一定程度的释放，交换土壤中的 $\boldsymbol { \mathrm { ~ H ~ } ^ { + } }$ 和 $\mathrm { A l } ^ { 3 + }$ ,提高盐基饱和度并调节土壤 $\mathrm { \ p H }$ 值[17]。同时,所含的 $ { \mathrm { K } } ,  { \mathrm { C a } } ,  { \mathrm { M g } }$ 也是植物所必需的营养元素[18]。因此,生物炭既是很好的生物炭基肥料,又是酸性土壤的调节剂[19],盐角草植株体内盐含量达地上部干重的50%[2）

本研究以盐角草为研究对象，分别在不同的炭化温度、时间下进行炭化制备生物炭，测定包括产率 $\mathrm { \nabla \cdot \ p H } , E C$ （电导率）、灰分含量、离子含量等理化指标，研究不同炭化方式下制备的生物炭特性。为后续盐生植物制备生物炭基肥料改良酸性土壤方面提供参考。

# 3讨论

生物质原料的成分决定了生物炭组成及性质[21]。生物炭之所以呈碱性,是因灰分中含有大量的矿质元素如 $\mathrm { { N a , K , C a , M g } }$ 等，以氧化物或碳酸盐的形式存在其体内[22],且灰分含量越高,碱性越强。生物质本身的灰分含量高，生物炭灰分含量也相应较高[23]。牛文娟[24]研究得出,小麦、水稻、玉米、油菜和棉花等作物秸秆的灰分含量分别为 $9 . 1 9 \%$ 、$1 1 . 9 0 \% . 7 . 2 2 \% . 6 . 6 1 \%$ 和 $4 . 8 7 \%$ ,含量较低;林益明等[25]研究不同红树植物木材的灰分含量发现,红树植物灰分含量基本都在 $5 \%$ 以下,均明显低于盐角草灰分含量。Yuan 等[2的研究结果表明，生物炭的石灰效应与其碱度有关，并且发现土壤 $\mathrm { \ p H }$ 和生物炭的碱度有强烈线性相关关系。本研究中所用盐角草能够有效吸收并富集盐分， $\mathrm { { N a } ^ { + } }$ 含量较高；灰分含量为36. $1 5 \%$ ，生物炭灰分含量最高为$7 6 . 2 7 \%$ ,作为酸性土壤调理剂具有更强的石灰效应，碱性更强，同时也能够提供一定量的矿质营养元素。因此，盐角草生物炭高灰分、盐分含量的石灰效应和改善植物矿质营养的作用，可为生物炭改良酸性土壤原料选择提供新的参考。

目前，普遍认为炭化温度是造成生物炭理化性质差异的主要因素之一。生物炭的物质组成由炭化终温决定，生物炭的炭化程度会随着裂解温度的升高而增加，其 $\mathrm { ~ C ~ } , \mathrm { { H } , 0 , N }$ 的含量降低，微量元素及灰分含量有所增加[12,27]。而生物炭产率等则随着裂解温度的增加而减少； $\mathrm { \ p H }$ 随裂解温度升高而升高，并与灰分含量呈正相关(8-11]。随着制备温度的升高，无机元素不断富集，但高温条件下，无机元素会在生物炭中形成稳定性更强、溶解度更低的矿物晶体[28-29]。当裂解温度升高至 $> 5 0 0 ~ \mathrm { { ^ { \circ } C } }$ 时,生物炭表面的羟基含量减少，酚羟基和羧基也会随制备温度的升高逐渐减弱,甚至消失[30-31]。本研究中，随着炭化温度的升高，生物炭 $\mathrm { p H } , E C$ 、灰分含量增加，生物炭的产率降低，这与前人的研究结果一致{8-11）。高温 $> 4 0 0 ~ \mathrm { ^ { \circ } C }$ 时,生物炭水溶性 $\mathrm { C a } ^ { 2 + }$ 、$\mathrm { M g } ^ { 2 + }$ 发生固结作用。但同时伴随着灰分增加,水溶性 $\mathrm { { N a } ^ { + } }$ 、 $\mathrm { ~ K ~ } ^ { + }$ 等低价离子富集，含量增加， $\mathrm { \ p H }$ 和$E C$ 也相应提高。生物炭中含有大量营养元素，除C 含量较高外,N、K的含量也较高，C、N含量由于燃烧和挥发的原因，随温度的升高而降低，而K、P含量随温度的升高而增加。因此，若以改良酸性土壤为目的，可以根据土壤具体的酸碱度来选择合理的炭化温度和范围。

随着炭化时间的增加， $\mathrm { p H } , E C$ 及灰分含量逐渐升高，碳含量逐渐降低，主要是由于持续炭化、生物炭灰化、碳持续燃烧和挥发所致，因此增加了生物炭水溶性离子。本研究中当炭化时间为 $^ { 2 \mathrm { ~ h ~ } }$ 时，生物炭产率最高， $\mathrm { K } ^ { + } \cdot \mathrm { M g } ^ { 2 + }$ 、全氮、全钾含量同样出现极值；炭化 $^ 3 \mathrm { ~ h ~ }$ 时，产率显著降低。这与之前炭化时间从 $^ { \mathrm { ~ 1 ~ h ~ } }$ 增加到 $^ { 2 \mathrm { ~ h ~ } }$ 后，生物炭的产率显著降低，当炭化时间继续增加时生物炭产率基本保持不变的研究结果有所差异[15]。考虑制备成本及生物炭稳定性,炭化时间 $^ { 2 \mathrm { ~ h ~ } }$ 为最适宜的炭化时间。

# 4结论

（1）炭化能有效增加盐角草 $\mathrm { p H } , E C$ 和灰分含量，随着炭化温度的升高，生物炭 $\mathrm { p H } , E C$ 、灰分含量显著增加，产率、全碳、全氮含量逐渐降低。

(2）高温 $> 4 0 0 ~ \mathrm { ^ { \circ } C }$ 炭化条件下，生物炭水溶性$\mathrm { C a } ^ { 2 + } \cdot \mathrm { M g } ^ { 2 + }$ 发生固结作用。但同时伴随着灰分增加,水溶性 $\mathrm { \Delta N a } ^ { + } \cdot \mathrm { K } ^ { + }$ 等低价离子富集，含量增加， $\mathrm { \ p H }$ 和 $E C$ 也相应提高。

（3）盐角草生物炭中含有大量营养元素，除C含量较高外， $\smash {  { \mathrm { ~ N ~ } } _ { \mathrm { ~ s ~ } }  { \mathrm { P } _ { \perp } }  { \mathrm { K } } }$ 的含量也较高，C、N含量由于燃烧和挥发的原因，随温度的升高而降低，而K、P含量随温度的升高而增加。

（4）随着炭化时间的增加， $\mathrm { p H } , E C$ 及灰分含量逐渐升高，碳含量逐渐降低，主要是由于持续炭化，生物炭灰化所致，因此增加了生物炭水溶性离子。

（5）根据生物炭理化特性分析，考虑制备成本及生物炭稳定性，炭化 $^ { 2 \mathrm { ~ h ~ } }$ 为最适宜的炭化时间。
