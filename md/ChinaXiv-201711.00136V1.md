# 1 INTRODUCTION

Many researches focus on the synthesis and studies of single-molecule magnets (SMMs) in the past two decades because of their potential applications in ultra-high density information storage and molecular spintronicsl1-41. As the lanthanide ions have many electrons in the $f$ orbits and a strong spin-orbital coupling effect, they are very appealing candidates for constructing SMMs. Thus,a large number of lanthanide-based SMMs have been reported recently[5-8l. Among these lanthanide ions, Dy(II), a late lanthanide $( 4 f ^ { _ { 1 } } , \mathtt { n } > 7 )$ with a large value of angular momentum, has a stronger magnetic anisotropy than the early lanthanides $( \boldsymbol { \mathrm { n } } < 7 ) ^ { [ 9 ] }$ .In addition,the Dy(II) ion has an odd number of $4 f$ electrons ${ \mathrm { ( n } } = 9 { \mathrm { ) } }$ ）and is a Kramer ion with doubly-degenerate ground states[10l, which leads to a flood of interesting Dy-based SMMs, including mononuclear-Dy(II)[1,12], dinuclear $\mathrm { D y } _ { 2 } { \scriptstyle [ 1 3 - 1 6 ] }$ and multinuclear Dy(II) clusters7].

It has been reported that the magnetic relaxation of lanthanide ions isstrongly influenced by the coordination environment around them[18,19]. Carboxylate group can form strong coordination bonds with lanthanide ions to provide a high afinity for oxygen donors, which makes them appropriate candidates for creating robust SMMs[20l. Furthermore,lanthanide ions have large radi, high coordination numbers and variable geometries.The carboxylate-based ligand can provide diverse coordination modes due to its multi-coordination sites,which fit well with the flexible coordination geometries of lanthanide ions[21. In this study,the rigid ligand 2-quinolinecarboxylate (QLC-) with one Nand two carboxylate Odonors is selected for the construction of dysprosium(III) SMMs,and 1,10-phen is used as an ancillary capping ligand. The 1,10-phen with large skeleton serves as a chelating ligand to terminate the propagation of the Dy-2-quinolinecarboxylate linkage to the 1D polymeric chain. The centrosymmetric dinuclear compound is composed of two eight-coordinated Dy(II) ions.In compound 1，each Dy(II) ion is chelated by two coordinated $\mathrm { Q L C ^ { - } }$ ligands with one 1,1O-phen acting as the terminal ligand. Two Dy(II) ions are bridged by a pair of $\mu _ { \mathrm { 2 } } { \mathrm { - O H } }$ groups to form the dinuclear $[ \mathrm { D y } _ { 2 } ( \mu _ { 2 } \mathrm { - O H } ) _ { 2 } ( \mathrm { Q L C } ) _ { 4 } ( 1 , 1 0 \mathrm { - p h e n } ) _ { 2 } ]$ unit. To our knowledge, only the dinuclear lanthanide-QLC compound, tetranuclear $\mathrm { \small { [ D y _ { 4 } ( Q L C ) _ { 1 2 } ( H _ { 2 } O ) _ { 6 } ] cdot 4 H _ { 2 } O } }$ ，and 1D chain structures of $\left[ \mathrm { P r } ( \mathrm { Q L C } ) _ { 3 } ( \mathrm { H } _ { 2 } \mathrm { O } ) \right] _ { \mathrm { n } }$ and $\mathrm { \Delta [ D y ( Q L C ) _ { 3 } ( H _ { 2 } O ) _ { 2 } ] _ { n } }$ have been reported[2-25]. In these reported compounds， the lanthanide centers are bridged by the $\mathrm { Q L C ^ { - } }$ ligand, which are different from the $\mu _ { 2 }$ -OH groups bridging Dy(III) centers in the dinuclear structure of 1. Herein，we present its synthesis,crystal structure and magnetic properties.

# 3.1 Crystal structure description

Compound 1 features a centrosymmetric dinuclear structure with one Dy(II) ion, one hydroxyl group, two QLC- ligands,one 1,10-phen ligand and two solvent water molecules in the asymmetric unit. As shown in Fig. 1,the Dy(1) ion is eight-coordinated by three carboxylate oxygen atoms (O(2), O(4)and O(5))and one nitrogen atom N(1) from two QLC- ligands,two nitrogen atoms from one 1,10-phen ligand,and two hydroxyl oxygen atoms (O(1) and O(1A))(Fig. 2). The Dy-O and Dy-N bond distances range from 2.227(3) to 2.472(3) $\mathring \mathrm { A }$ and 2.563(4) to 2.634(3) A,respectively (Table 1). The two crystallographically independent QLC-ligands display two types ofcoordination modes. One binds a Dy(II) ion in a chelating mode via its carboxylate group; the other chelates a Dy(II) ion through its nitrogen and one carboxylate O atoms (Fig.1). As depicted in Fig.1, one 1,10-phen and two QLC-ligands simultaneously chelate a Dy(II） ion to give a mononuclear [Dy(QLC)2(1,10-phen)] unit. Two $\mathrm { \Delta [ D y ( Q L C ) _ { 2 } ( 1 , 1 0 { - } p h e n ) ] }$ subunits are bridged by a pair of $\mu _ { 2 }$ -OH groups to generate a centrosymmetric dinuclear $[ \mathrm { D y } _ { 2 } ( \mu _ { 2 } \mathrm { - O H } ) _ { 2 } ( \mathrm { Q L C } ) _ { 4 } ( 1 , 1 0 \mathrm { - p h e n } ) _ { 2 } ]$ unit with the $\mathrm { D y ( 1 ) \cdot \cdot \cdot D y ( 1 ) A }$ distance of 3.664(2) $\mathring \mathrm { A }$ and Dy(1)-O(1)-Dy(1)A angle of $1 0 9 . 9 2 ^ { \mathrm { o } }$ .The intramolecular $\pi \cdots \pi$ interactions between aromatic rings of the $\mathrm { Q L C ^ { - } }$ ligands and 1,1O-phen ligands are observed (Fig. 1） with the centroid-to-centroid distances of 3.7007(1), 3.7340(1), 3.7258(1) and 3.8942(1) $\mathring \mathbf { A }$ and dihedral angles of $3 ^ { \mathrm { o } }$ and $6 ^ { \mathrm { o } }$ ,respectively. In the crystal packing,the closest intermolecular Dy $\cdots$ Dy separation is $9 . 9 8 8 \mathring { \mathrm { \textrm { A } } }$ (Fig. 3). In addition,the intermolecular $\pi \cdot \cdot \pi$ interactions between aromatic rings of the QLC- ligands are observed having the centroid-to-centroid distances of 3.5000(1), 3.8092(1) and 3.6151(1) $\mathring \mathbf { A }$ and dihedral angles of $3 ^ { \mathrm { o } }$ ,20 and $0 ^ { \mathrm { o } }$ , respectively. The $\mathrm { O } ( 1 ) \mathrm { W }$ latice water molecule bonds to the dinuclear molecule through the hydrogen bond between the water molecule and hydroxyl oxygen (Table 2). Finally, the hydrogen bonds between the lattice water molecules are observed in compound 1 (Table 2).

# 3.4Magnetic properties

Direct-current (dc) magnetic susceptibilities of 1 were collected under an applied magnetic field of $1 \mathrm { k O e }$ in the temperature range of $3 0 0 { \sim } 2 \mathrm { K }$ . The magnetic properties of 1 in the form of $\chi _ { \mathrm { M } } T$ versus $T .$ ，where $\chi _ { \mathrm { M } }$ is the molar magnetic susceptibility per dinuclear molecular unit,are shown in Fig.5.The $\chi _ { \mathrm { M } } T$ value is 28.29 $\mathrm { c m } ^ { 3 } \ \mathrm { K \ m o l { \Sigma } ^ { - 1 } }$ at $3 0 0 ~ \mathrm { K }$ ， which is consistent with the expected value of two uncoupled Dy(III) ions in the $^ 6 H _ { 1 5 / 2 }$ ground state ( $S = 5 / 2$ ， $L = 5$ ， $g = 4 / 3$ ，and $C = 1 4 . 1 7 ~ \mathrm { c m ^ { 3 } ~ K ~ m o l ^ { - 1 } } .$ ). As the temperature lowered, the $\chi _ { \mathrm { M } } T$ product increases gradually to a maximum value of $3 1 . 0 2 ~ \mathrm { c m ^ { 3 } ~ K ~ m o l ^ { - 1 } }$ at $4 4 ~ \mathrm { K }$ .As the temperature further lowered, $\chi _ { \mathrm { M } } T$ product decreases sharply to a minimum value of $2 2 . 5 8 ~ \mathrm { c m } ^ { 3 } \mathrm { K } ~ \mathrm { m o l } ^ { - 1 }$ at $2 \mathrm { ~ K ~ }$ (Fig. 5). A protuberance around $4 4 ~ \mathrm { K }$ in the $\chi _ { \mathrm { M } } T$ versus $T$ curve indicates the presence of intramolecular ferromagnetic interaction between the Dy(II) ions within the dimeric unit. Such kind of interaction is strong enough to compensate the decrease of $\chi _ { \mathrm { M } } T$ resulted from the depopulated Stark states.However, this kind of intramolecular ferromagnetic interaction at higher temperature is rarely observed for lanthanide compounds, because it is generally masked by the thermal population of low-lying excited states as well as strong magnetic anisotropy for lanthanide ions[30, 31l The decrease of the $\chi _ { \mathrm { M } } T$ products at low temperature suggests that the intermolecular antiferromagnetic coupling starts to dominate the magnetic behavior for the ground states of 1.

Field dependence of the magnetization at $2 \mathrm { ~ K ~ }$ shows a rapid increase of the magnetization at low field followed by a slow linear increase at high field (Fig. 5 insert). The $M$ value at $5 0 ~ \mathrm { k O e }$ is $1 6 . 8 ~ N \beta$ at $2 \mathrm { ~ K ~ }$ which is much lower than the theoretic saturate value of $2 0 ~ N \beta$ , indicating a large magnetic anisotropy for Dy(III) ions.The small $M$ value can be ascribed to the small total quantum number for Dy(III) ions produced by the depopulation of the Stark levels of the $^ 6 H _ { 1 5 / 2 }$ ground state under the ligand-field perturbation[32]. In order to probe the slow magnetic relaxation behavior, the alternating-current (ac） magnetic susceptibility measurements are performed on 1. Under a zero-dc field with an ac field of $2 . 5 \mathrm { O e }$ ,no peak can be observed in the in-phase $( \chi ^ { \prime } )$ and out-of-phase $( \chi ^ { \prime \prime } )$ susceptibilities versus $T$ curves in the high frequency of $9 9 7 \mathrm { H z }$ for 1 (Fig.5b). Thus compound 1 shows no slow magnetic relaxation,which maybe results from zero-field quantum tunneling of the magnetization that is frequently associated with the lanthanide ions[33.To suppress the quantum tunneling of the magnetization, the dc field of $2 \ \mathrm { k O e }$ is added.As shown in Fig. 5b,the peak with good shape is observed in the $\chi ^ { ' }$ and $\chi ^ { \prime \prime }$ susceptibilities versus $T$ curves with the frequency of $9 9 7 ~ \mathrm { H z }$ （204号 (Fig. 5b). Thus the quantum tunneling of the magnetization of Dy(II) ions in compound 1 are suppressed. The temperature-dependences of $\chi ^ { ' }$ and $\chi ^ { \prime \prime }$ susceptibilities under $2 \mathrm { k O e }$ dc field are observed for 1 (Fig. 5b), suggesting that compound 1 exhibits a field-induced slow magnetic relaxation.
