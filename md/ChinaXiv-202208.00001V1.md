在以往的护理教育评价中，常采用以结果为导向的评价方法，如泰勒的目标模式，虽然可以判断护生接受教育的情况，但评价不够完整全面。而CIPP模式以决策为导向，CIPP评价模式是用来提升质量的最具实践性和全面性的模式特别适用于教育改革活动对教育评价的需要[1,2]，使评价活动更具有方向性和实用性，目前已有应用在护理教育评价中的例子，因此本文旨在讨论CIPP模式目前在我国护理教育中的应用，概述存在的问题，以及改进的意见和建议。

# 1CIPP 的概述

CIPP创立于1966年，是背景评价、输入评价、过程评价和结果评价四个部分的英文首字母缩写，分别评估评价对象的目标层面、设计层面、实施层面和影响层面[3]。背景评价的本质是诊断性评价，输入评价的本质是保障性评价，过程评价用于提供修正方案的有效信息，而结果评价的本质是总结性评价，用于考核评价效果。其中背景评价是一个不断循环的过程，投入、过程、成果评价则是在背景评价发现问题、解决问题时出现[4]。

# 2CIPP评价模式在我国护理教育中的应用

# 2.1 背景评价

护理教育中的背景评价包括办学定位、培养目标和生源质量[5]，不同类型的高校办学定位不同；人才培养目标是根据护理教育的层次决定的；录取分数、第一志愿数等指标可以衡量高校的生源质量，同时体现高校和该校护理专业的认可度和影响力。背景评价对后期的师资投入、经费投入、设备投入等方面有很大的影响。

# 2.2 输入评价

护理教育中的输入评价包括组织投入、经费投入、设备投入、师资选配、专业设置、教学方法、教材分配、课时分配等[5,6]，即开展教学所需的各项投入的充分性评价，是实现培养目标、办学定位和吸引高质量生源的充分保障。同时也可以帮助教师改善教学态度，提升备课水平，丰富自己的知识水平[7]。

# 2.3 过程评价

护理教育中的过程评价包括培养方案、理论教学、实践教学、科研促教、第二课堂和质量监控等方面[5]。培养方案是高校教学目标与人才培养目标的必要载体，体现高校的护理教育办学特色，也考核了落实度。多种教学方式的灵活应用可帮助学生丰富见识、适应专业需求，提升各个方面的能力；也有助于教师改进教学内容，实现反思教学。

# 2.4 结果评价

护理教育中的结果评价包括学生素质评价和社会效益评价[5]，可反映出教学效果。学生素质从学位获得率和学生满意度两个方面评价；社会效益从初次就业率、大学生自主创业率和用人单位满意度三个方面评价。用于衡量改进后的教学效果的达成度和满意度。

# 3CIPP评价模式应用于护理教育存在的问题

# 3.1 缺乏理论指导

目前CIPP评价模式在我国护理教育中的应用不多，且研究方向多为本科及本科以下护理学教育，对学术型研究生的研究较少，对专业型研究生和护理博士生的研究更是少之又少[6]。除缺乏不同层次护理教育理论外，针对护理课程评价中的应用与护理教学方法中的应用理论指导也十分缺乏[8]。与我国护理评价长期使用的方法相比缺乏理论指导，难以很快得到广泛认同。

# 3.2 缺乏实证研究

我国目前对CIPP评价模式在护理教育中的应用大多停留在理论阶段，缺乏大量有效的实证研究，缺乏实证研究后的充分反思与改进[6]。

# 3.3 评价的适用范围受到制约

CIPP 模式一般主要用于大规模的方案，需要有各类信息源的配合和充裕的经费及科学的分析技术，而且特别适用于需讲究推理和习惯于条理性方法的教育环境中[3]。如果缺乏制度化的持续性评价，由于时间太长,耗力较多,往往导致半途而废的现象。

# 3.4 缺乏价值判断

在整个评价过程中,更多的都是注重描述性的信息，而对活动的各个方面，包括将要做出的决策和将要采取的备选行动方案,而缺乏价值上的判断[9]。

# 3.5 可能存在虚假伪造

在CIPP评价模式中，评价者的职责就是收集、分析评价资料，并向决策者报告。但如果评价者为不影响自己高校的排名和影响力，或为迎合决策者关注的问题，就可能会回避真正的问题，伪造虚假资料。这样不仅会降低评价的有效性也会对教育质量的提升带来严重阻碍[10]。

# 4CIPP评价模式应用于护理教育展望

针对CIPP评价模式应用于护理教育存在的问题，提出以下几方面的建议和意见：

1.呼呼加强对CIPP应用的研究和深入探讨，从不同层次护理教育的应用、护理课程评价应用与护理教学方法应用等方面进行国内外比较，力图建立适合我国护理教育的理论框架；2.加强CIPP评价模式在护理教育中的应用实证研究，不仅局限于某一学科、某一教学方法，可研究护理学体系中的任何学科，理论教学、实践教学、科研促教、第二课堂、情景模拟等教学方法中任一种或者任几种的应用效果。实践出真知，只有从大量的实证案例中才能得到反思与改进的方法，实现不断完善；3.探索适合我国护理教育现状的方案，同时实验满足CIPP所有条件的最优方案，在尽力满足CIPP条件的前提下评价我国护理教育；4.附加价值的判断，在整个评价过程中，不仅关注描述性的信息，也要增加价值上的判断；5.建立监督机制，在CIPP评价过程中设置监督者，监督者的利益不与评价结果有任何关系，监督者参与评价过程，评价者不可以与监督者有任何交流，监督者直接反映评价结果给决策者。或由决策者设置可视化的评价指标，没有评估上的偏差就可以降低故意利用偏差伪造的发生率。决策者也要注意保留自己的看法，尽量不让评价者发现自己的关注问题，防止故意迎合现象。
