# 连山话单音节和双音节声调声学分析

虞梦夏

（北京大学 中国语言文学系 北京 100871)

摘要：利用实验语音学声学分析的方法研究广东连山县连山话的单音节词声调和双音节词声调。实验结果表明，连山话的单音节词声调调值可以用五度值标写为阴平51、阳平121、阴上44，阳上13，阴去24，阳去213，上阴入44，阳入21下阴入24。双音节词存在连读变调现象，一般后字不变，前字变调表现为调值变低，调型平直化。作为勾漏片粤语的共性，前轻后重的节律模式制约着连山话的连读变调。此前研究对连山话连读变调采用规则取向分析，本实验表明变调的规则性不强，提出前字的本调和变调可能是心理词典中并列的自由变体。

关键词：声调 连山方言声学分析 单音节 双音节 变调

# 1引言

声调是汉语的主要特征之一，担负着区别意义的功能。连山话属于粤语勾漏片的一个分支。此前关于连山话声调的记录，只有少量基于听感的田野调查的材料。实验语音学因其研究方法可为声调研究提供更为实证和客观的考察手段，因而本文将采用声学分析的方法对连山话的单音节和双音节声调进行描写、分析，使活的方言材料得以记录存档，为抢救方言遗产，推动勾漏片方言的研究，起到积极的作用。

# 2方言点介绍

粤语中的勾漏片分布于桂、粤交接地区的玉林、北流、陆川、容县、贺县、贵县、岑溪、藤县、封开、广宁、四会、郁南、罗定、德庆、怀集、连县、连山等县市。（《中国语言地图集》）当地群众称它为“土白话”、“地佬话”或“本地话”等。

连山壮族瑶族自治县位于南岭山脉西南麓，广东省西北隅。西至广西壮族自治区贺州市八步区，北接湖南省江华瑶族自治县，南交怀集县，东临连南瑶族自治区。现辖吉田镇、太保镇、永和镇、福堂镇、小三江镇、禾洞镇、上帅镇等7个镇，全县户籍人口12.2万（2015年）总面积1218平方公里（2009年）。连山县属于壮族瑶族自治县，汉族与壮族、瑶族等少数民族杂居，通行连山话和广州话，壮语及瑶语也是主要语言（欧晓敏，2010）。南片和北片的连山话在语音上有区别，但差异不大。

学界一般认为粤语没有连读变调，其实这指的主要是广府片粤语。勾漏片粤语普遍存在连读变调现象。（侯兴泉，2011)

# 4研究方法

信号采集：单音节声调按9个调类进行选词，每个调类选2个发音词。双音节声调按9个调类两两组合，得到 $9 { \star } 9 = 8 1$ 种双音节声调组合，每种组合选词5个。单音节和双音节词表见附表。发音人男女各选2人，分别为男发音人 $\mathbf { m } 1$ 、 $\mathrm { m } 2$ ，女发音人f1、f2。每个例词发音两遍。这样单音节声调每个调类有16 个样本，双音节声调每个组合有 40个样本，录音时采取单声道录制，采样频率为 $4 4 1 0 0 \mathrm { { H z } }$ ，保存为.wav格式。录音时间为2017年10月。

降噪处理：使用AdobeAudition软件获取环境噪音，对录音样本进行降噪参数提取：使用Praat软件进行基频（FO）的提取。采用的方法为自相关法，其原理为：之后用Praat软件标记声调段、用Praat脚本对每个单字发音等比例间隔提取基频，舒声调取20个点，促声调取15个点。

归一化处理：通过Excel软件，把每个声调组合的基频进行平均，最后的到声调的基频平均数值。其中每个单音节声调是16个样本的平均，每个双音节声调组合是 40 个样本的平均。

五度值的换算方法：通过以上方法获得的声调基频参数代表的是声调的声学特征，从理论上它们还不具备语言学研究的意义，因而为了方便和田野调查的五度标调结果进行比较，本文对所有单音节词声调和双音节词声调组合的基频数值进行了半音法转换，再归到五度值。计算两个基频之间半音的公式为：

# 半音 $= 1 2 { \star } 1 0 9 _ { 2 } ( \mathbf { f } 1 / \mathbf { f } 2 )$

其中，f1为声调频率上限，f2 为频率下限。在本文中，单音节部分f1、f2分别取四人平均后的9个声调基频数据点中的最大值和最小值，双音节部分则分别取平均后所有声调组合中基频数据的最大值和最小值。

# 7单、双音节声调模式讨论

单音节一共有9个声调。分别为阴平（51），阳平（121）、阴上（44）、阳上（13）、阴去（24）、阳去（213）

双音节声调模式有 $8 { \star } 9 { = } 7 2$ 个。阳平和阳去作前字时声调归并为21。其余声调虽然没有一致的变调，但通过变调而调位归并的趋势也很明显。

# 8总结与讨论

可以总结出，连山话双音节连读变调的特点如下：

1、前字变调，后字不变。

这是勾漏片粤语连读变调的共性。可能是受到前轻后重型节奏模式的制约（侯兴泉2011）。

# 2、属于简化型连调

总的来说，从具体声调的层面来看，连山话的变调趋势可以总结为调型平直化。这属于简化型连读变调（李小凡，2004）.具体体现为：升调、降调变平调，如阴平53$ 3 3$ ，阴去、下阴入 $3 5 \substack {  3 3 }$ ，阳上、阳入 $2 4 \substack {  2 2 }$ ；曲折调变平调，如阳平 $1 2 1 \mathrm {  2 2 }$ ，阳去 $2 2 4 {  } 2 2$ ；高调变低调，如阴上、上阴入 $5 5 \substack {  } 3 3$ 。从基频曲线来看，则表现为基频值变小，曲线斜率的绝对值变小。简化型连读变调的作用在于发音省力。

从声调系统的层面来看，连山话的变调反映了双音节声调系统的简化趋势。主要体现为，不同的单字调在双音节中通过变调归入同一种调。例如，阴平（53）、阴上（55）和阴去（35）在阳调字面前如果变调，则都变为33，即阴平、阴上和阴去在前字位置上失去对立，发生调位中和，从而减少了连调式的总数，使得本该比单字调繁复得多的连调系统得以简化，这对于单字调类多，连读变调丰富的方言尤其重要。这种中和型连调使得双音节系统受单字调系统的制约日渐减弱，而其自身的系统性和独立性则日益增强（李小凡，2004）。从本次录音的情况来看，连山话的双音节系统还是一个不稳定的活跃的系统，调位的变化还没有完成，但是简化的趋势已经很明显了。

# 3、讨论：与此前研究差异的地方

侯兴泉（2011）研究发现，勾漏片粤语两字组后字通常不变调，前字为中平（或中降）和低平（或低降）的声调通常也不变调，其他调值的声调在前字位置则通常会发生连读变调，其规律也有较强的一致性：通常是阴调类声调变成中平或中降调，阳调类声调会变成低平或低降调。

但是本实验发现，连山话阴调无论在清浊调前都可能发生变调。也就是说，连山连读变调分清浊的规律并不是很强。

在前人的研究中，一般认为连山话连读变调是“本调 $$ 变调”的语音规律，而这个规律是可以用规则表示的。从心理语言学的角度，其含义是，连山话连读变调在语言处理中是“规则运用”的过程。这个过程中，本调以底层形式存在心理词汇库中，说话人在说话的当时运用规则，将本调转成变调，也就是在线的语音生成过程。而本实验发现，在同一个双音节组合里，前字读作本调和变调的情况都占了相当的比例。即使是一个双音节词，由同一个发音人读两遍，也有读为本调和变调的不同情况。因此规则取向的分析是不适用的。

参考蔡素娟对闽南语连续变调的研究（2002），本文认为连山话连读变调符合“词素变体

选择假设”。蔡素娟提出语音规律的“词汇变体生成”和“词汇变体选择”两个机制，并根据这两个机制的应用与否解释了普通话、英语、Hausa语、闽南话的语音规律。并提出闽南语并没有声调转换的生成过程，而是一个词素的两个变体并列于心理词汇库中，依出现在句子中的位置，选择其中一个词素变体。应用这个假设，连山话的一个词素有并列在词汇库中的两个变体，在作为二字组的前字时，根据后字的声调选择其中一个变体。但这种环境的约束并不是很强，两个词素变体可视为自由变体。
