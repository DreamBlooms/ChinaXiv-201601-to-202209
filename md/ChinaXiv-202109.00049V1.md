关键词： $0 \mathrm { c m }$ 地温；时空变化；突变；地气温差；陕西省

地温作为地-气系统研究的重要参数，深刻影响着土壤的物理、化学及生物过程，土壤温度的变化直接影响作物种子萌发、生长发育及产量[1]。在全球气候变化背景下，研究不同尺度地表温度的时空变化规律及其与气候变化的响应关系，对于了解不同地区地气相互作用、地气能量交换以及水、热和碳等的循环过程具有重要意义[2]。从国家尺度来看， $\mathrm { H u }$ 等3依据1967—2002年美国的 $1 0 \ \mathrm { c m }$ 地温数据,发现美国 $1 0 \mathrm { c m }$ 地温呈升高趋势。Qian等[4]根据1958一2008年加拿大30个气候站的土壤温度数据，发现春季和夏季土壤温度呈显著的正趋势，而冬季和年平均土壤温度呈不显著的正趋势。陆晓波等[5]、乔丽等[2]、王佳琳等[6先后对中国近50a地表温度进行分析研究，发现中国 $0 \mathrm { c m }$ 地温呈先降低后升高的趋势，2000年突变后升温趋势更为明显，且在所有季节均呈现上升趋势，冬季平均地表温度增温趋势最明显，平均地表温度空间分布由北向南逐渐升高。从区域尺度来看，安彬等研究表明1960—2017年黄土高原年际、季地温均呈显著上升趋势，且呈东南高、西北低的空间分布。杨平等发现1961—2010年黔东南地区 $0 { \sim } 2 0 ~ \mathrm { c m }$ 浅层地温夏季时空分布差异剧大，冬季时空分布差异和缓，四季的浅层地温均呈上升趋势。王雪姣等发现新疆1961—2015年地温呈升高趋势，地温在1999年发生突变，地温与气温呈现出显著正相关关系。赵美亮等[10]分析了青海省地表温度及同期气象数据,指出地温与气温呈显著正相关，与风速、海拔呈显著负相关，地表平均温度呈现出明显的垂直递减规律且海拔高的地区变暖趋势更加明显。闫军辉等分析得出银川市1961—2012年 $0 { \sim } 2 0 ~ \mathrm { c m }$ 平均地温呈上升趋势,其中 $0 \mathrm { c m }$ 地温升温速率最大。综上，地温的变化具有明显的时空分异，针对不同区域的地温变化特征还需要深人探究，从而明确地温与环境因子之间的响应机制。

陕西省地处我国西北地区，境内气候条件复杂多变、敏感脆弱,近年来有学者针对西安市[12]、关中地区[3地温变化特征展开了研究，而系统研究陕西省地表温度的成果相对较少。基于此，本研究利用陕西省1958—2017年的逐月气温和 $0 \mathrm { c m }$ 地温数据，分析陕西省近60a的 $0 \mathrm { c m }$ 地温的时空变化特性及其对气候变化的响应，以期为陕西省有效利用气候资源，科学规划种植区、提高农作物产量等提供参考依据。

# 1.2 研究方法

采用一元线性回归分析地温变化趋势，确定气候倾向率，并对气候变化速率进行显著性检验，分为趋势显著( $\stackrel { \triangledown } { \boldsymbol { P } } < 0 . 0 1 ,$ ）、趋势较显著( $\left( P < 0 . 0 5 \right)$ 和趋势不显著( $\left( P > 0 . 0 5 \right)$ 。采用 $\mathrm { A r c G I S } 1 0 . 2$ 软件中的样条函数插值法，分析陕西省地温在年际和四季的空间分布特征。地气温差即地面与大气的相对冷热区别，表示地面不同冷源与热源的性质及其强度，用 $0 ~ \mathrm { c m }$ 地温减去气温所得到的数值表示。利用Mann-Kendall法诊断陕西省1958—2017年地温突变情况，并分析其突变年份。Mann-Kendall法是一种非参数统计检验方法，可以明确突变开始的时间，是一种常用的突变检测方法。计算方法如下[14]：

假定时间序列为 $\{ x _ { i } \} ( i { = } 1 , 2 , 3 , \cdots n ) , m _ { i }$ 表示第 $i$ 个样本 $\boldsymbol { x } _ { i } > \boldsymbol { x } _ { j } ( 1 \leqslant j \leqslant i )$ 的累计数。定义统计量 $d _ { k }$ ·

$$
d _ { \scriptscriptstyle k } = \sum _ { 1 } ^ { \scriptscriptstyle k } m _ { i }
$$

在时间序列独立的假定下，定义统计量如下：

$$
U F _ { k } = { \frac { d _ { k } - E { \big ( } d _ { k } { \big ) } } { \sqrt { \operatorname { V a r } { \big ( } d _ { k } { \big ) } } } } , \quad ( k = 1 , 2 , 3 , \cdots , n )
$$

$$
\left\{ \begin{array} { l } { { U B _ { k } = - U F _ { k } , ~ ( k = 1 , 2 , 3 , \cdots , n ) } } \\ { { k = n + 1 - k } } \end{array} \right.
$$

式中： $E { \left( { d } _ { k } \right) }$ 和 $\mathrm { V a r } \left( d _ { k } \right)$ 分别为累计数 $d _ { k }$ 的均值和方差。 $U F _ { k }$ 为标准正态分布，其判断依据为：当 $U F { > } 0$ 则表征该时间序列呈上升趋势； $U F { < } 0$ 则为下降趋势，当其超过 $9 5 \%$ 置信水平 $\alpha$ 时，说明该上升或下降趋势显著。当 $U F$ 和 $U B$ 曲线在临界线内存在交叉点，则该交点即为该时间序列的突变发生时间；若存在多个交点时，则结合滑动 $\mathbf { \Phi } _ { t }$ 检验法进一步判断突变时间，方法见文献[14]

# 2.3地温与气温的关系

2.3.1地气温差的时间变化特征由图6a分析可知，1958一2017年，陕西省年均地表温度和气温总体均有增加趋势，且地温较气温增幅更大，所以总体来看陕西省年均地气温差呈显著升高趋势，年均地气温差倾向率达 $0 . 0 7 \ \mathrm { { ‰ } }$ 。陕西省年均地气温差在20世纪70年代至90年代有下降趋势，最小值出现在1998年 $( 1 . 9 5 ~ \mathrm { ‰}$ ），2000年后升温幅度较大，在2008年达最大值 $2 . 9 3 ~ \mathrm { \textdegree C }$ 。地气温差的增大预示着近地层空气不稳定度增强，从而造成对流增强，致使地表水分蒸发量增加，这也是陕西省气候暖干化趋势的进一步体现[15-16]。
