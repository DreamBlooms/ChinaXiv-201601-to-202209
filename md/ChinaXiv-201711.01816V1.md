磷是维持动物健康生长不可缺少的矿物元素，参与体内诸多代谢，如核酸代谢、能量代谢、脂质代谢以及酶的催化作用等。磷对骨骼矿化、维持肌肉正常功能与细胞膜完整性、细胞信号传导以及体液酸碱平衡，也具有重要作用[1]。现代育成的肉鸡生长速度快，对磷的需要量更高[2]。肉鸡饲粮中磷缺乏会导致肉鸡骨骼发育不良、生长性能下降和死亡率增加等[3]。玉米、豆粕等常规饲料中磷含量较低，且多为不可消化吸收的植酸磷，这导致肉鸡饲粮中约$70 \%$ 的非植酸磷需通过无机矿物质磷源添加，其成本仅次于能量、蛋白质居第3位[4]。而且，大量未被消化的磷可随粪便排出体外，引起水体富营养化，因而导致环境磷污染日益严重[5-7]。因此，加强肉鸡磷的吸收和代谢利用规律研究，对于节约饲料资源、降低养殖成本及减少磷排放带来的环境污染等均具有重要的理论和现实指导意义。研究发现，维生素 ${ \bf D } _ { 3 }$ （ $\mathrm { V D } _ { 3 }$ ）可促进肉仔鸡肠道磷的吸收。如早期研究发现，结扎鸡不同肠段灌注 $\mathrm { V D } _ { 3 }$ 可促进肠道磷的吸收[8]；随后研究者采用鸡胚十二指肠培养法也得到了类似的结果[9-10]。目前研究多集中于 $\mathrm { V D } _ { 3 }$ 对肉仔鸡骨骼发育的影响等方面。研究表明，饲粮中添加 $\mathrm { V D } _ { 3 }$ 或 $\mathrm { V D } _ { 3 }$ 类似物可提高肉仔鸡胫骨长度、骨密度和灰分含量，降低腿病发生率及腿病严重程度[11-14]。本实验室前期采用原位结扎灌注十二指肠技术，发现适宜水平的1,25-二羟基维生素 $\mathrm { D } _ { 3 } [ 1 , 2 5 { \cdot } ( \mathrm { O H } ) _ { 2 } \mathrm { D } _ { 3 } ]$ 通过上调肉仔鸡十二指肠钠磷协同转运载体IIb型（NaP-IIb）mRNA表达水平而提高了无机磷的吸收[15]。 $\mathrm { V D } _ { 3 }$ 是否可通过促进肠道磷吸收再进一步促进磷在骨骼中的代谢利用还需进一步研究。因此，本试验在饲粮中添加不同水平的 $\mathrm { V D } _ { 3 }$ ，研究其对肉仔鸡生长性能和胫骨磷代谢利用相关指标的影响，为调控肉仔鸡骨磷代谢利用提供理论基础和试验依据。

# 1材料与方法

# 1.1 试验设计与处理

采用 $2 { \times } 4$ 两因子完全随机设计。试验设2个饲粮非植酸磷水平：1个为NRC(1994) $\left[ 1 6 \right] _ { 1 \sim 3 }$ 周龄阶段肉仔鸡饲粮 $0 . 4 5 \%$ 非植酸磷水平（非植酸磷实测值为 $0 . 4 4 \%$ )，1个上述非植酸磷水平 $0 . 4 5 \%$ 一半的 $0 . 2 3 \%$ 非植酸磷水平（非植酸磷实测值 $0 . 2 1 \%$ ；4个 $\mathrm { V D } _ { 3 }$ 添加水平为0、1000、2 000 和 $4 0 0 0 \mathrm { \ : I U / k g }$ ，共8个组。

# 1.2 动物与饲粮

从华都肉鸡公司购买480只商品代1日龄爱拔益加（AA）肉公雏，根据上述试验设计与处理按体重随机分为8个组，每组60只鸡，分别饲养于6个不锈钢镀塑笼中，每个重复笼 10 只鸡。试验期21d。参照 NRC（1994）[16配制肉鸡1\~21日龄玉米-豆粕型基础饲粮,其组成及营养水平见表1，并按上述处理设置分别在基础饲粮中添加不同水平的 $\mathrm { V D } _ { 3 }$ 来配制8 种饲粮。饲粮以磷酸氢钙形式添加无机磷至上述非植酸磷水平，并参考 Waldroup 等[17]、Yan 等[18用洗净的建筑沙（未检测到磷）调整饲粮中石粉和磷酸氢钙的用量。由于钙含量显著影响磷的吸收利用，为消除饲粮中钙含量对磷吸收和代谢利用的影响，各组饲粮中钙含量保持一致（均为 $1 \%$ )。饲粮以粉料喂给。

每日观察记录鸡舍环境温湿度以及鸡只健康状况、发病和死亡情况。分别于14 和 21日龄 22：00时，以重复笼为单元结料，禁食，但自由饮水，次日（15和22日龄）08：00时，以重复笼为单元称鸡空腹重，并统计各重复笼鸡只的耗料量，计算平均日增重（ADG)、平均日采食量（ADFI）、料重比（F/G）和死亡率。试验鸡按《AA肉仔鸡饲养管理手册》进行饲养管理和免疫接种。鸡只自由采食及饮用自来水， $2 4 \mathrm { h }$ 恒定光照。

# 1.3样品采集与制备

于饲粮配制前采集玉米、豆粕饲料原料样品，分析粗蛋白质、钙、总磷和非植酸磷含量，使配制的饲粮中上述关键营养素含量与饲粮配方值尽可能接近，并在配制各处理饲粮时现场采集样品，四分法缩小样品量后粉碎过200 目筛，置于自封袋中低温干燥保存，以备分析饲粮粗蛋白质、钙、总磷和非植酸磷含量。于15 和22日龄，从每个重复笼选取接近平均体重的3只鸡屠宰，取右侧胫骨于自封袋中， $- 2 0 \ { ^ { \circ } \mathrm { C } }$ 冻存，以备分析胫骨灰分含量、胫骨灰分磷含量和胫骨灰分磷重量以及磷的绝对沉积率。每个重复笼3只鸡的胫骨样品合并为1个分析样本。

# 1.4样品分析

# 1.4.1 饲料原料和饲粮样品分析

样品经浓硝酸和高氯酸湿法消化后，用 IRIS Intrepid II等离子体发射光谱仪（TE,美国）测定饲料原料和饲粮中钙含量[19]；饲料原料和饲粮中粗蛋白质含量按 AOAC（1990）[20]中所述方法测定；饲粮中总磷含量根据国家标准（GB/T6437—2002）[21]采用钼黄分光光度法测定；饲粮中植酸磷含量采用三氯化铁沉淀法测定[22-24]，饲粮总磷含量减去植酸磷含量即得到非植酸磷含量。

# 1.4.2 胫骨指标测定

用马弗炉灰化法[25]测定胫骨灰分含量；胫骨灰样品经浓硝酸和高氯酸湿法消化后，根据国家标准（GB/T6437—2002）[21]采用钼黄分光光度法测定胫骨灰分磷含量，根据胫骨灰分重量和胫骨灰分磷含量计算胫骨灰分磷重量，用胫骨灰分磷重量比鸡只进食磷量即得胫骨灰分磷绝对沉积率。

# 1.5 数据统计分析

采用SAS9.2系统中的一般线性模型（GLM）程序对所得数据进行 $2 { \times } 4$ 两因子方差分析，方差分析显著者，用LSD法比较各平均数间的差异显著性。肉鸡死亡率（百分数数据）经反正弦转换后进行方差分析。每个重复笼作为1个试验单元。以 $P { < } 0 . 0 5$ 作为各项数据差异显著性检验水平。

3讨论

刘松柏[25]报道，饲粮非植酸磷水平对肉仔鸡 ADG、ADFI、F/G 和死亡率均有显著影响，且随着饲粮非植酸磷水平的提高，ADG 和 ADFI增加，F/G 降低；胡义信[2也报道了类似的结果。上述研究报道与本试验结果一致，说明肉仔鸡饲粮中缺乏非植酸磷可显著影响肉仔鸡生长性能，并增加死亡率。Biehl等[27研究发现，低磷饲粮中添加 $\mathrm { V D } _ { 3 }$ 可降低肉仔鸡 F/G;

Edwards[28]研究表明，添加1100 和 $4 4 0 0 \mathrm { I U / k g } \mathrm { V D } _ { 3 }$ 可降低肉仔鸡 F/G；Frits等[13]报道，与添加 $1 2 5 \mathrm { \ I U / k g \ V D } _ { 3 }$ 组相比，饲粮中添加1000、2000和 $4 0 0 0 \mathrm { \ I U / k g \ V D } _ { 3 }$ 显著提高21日龄肉仔鸡ADG。上述研究报道与本试验结果一致，说明饲粮中添加 $\mathrm { V D } _ { 3 }$ 可提高肉仔鸡生长性能。本研究还发现在 $0 . 2 1 \%$ 非植酸磷水平下，饲粮中添加 $\mathrm { V D } _ { 3 }$ 可降低肉仔鸡死亡率，这些可能与 $\mathrm { V D } _ { 3 }$ 促进了小肠磷的吸收和骨骼磷的代谢利用有关，从而缓解鸡只由于缺磷而导致的生长性能下降和死亡。

胫骨灰分含量是评价饲粮磷利用的敏感指标[29]。动物缺磷会引起缺磷性骨营养不良，表现为骨灰分占脱脂骨总重量的百分比下降[30]。刘松柏[25]报道，饲粮非植酸磷水平显著影响肉仔鸡腔骨灰分含量， $0 . 4 3 \%$ 非植酸磷水平组肉仔鸡腔骨灰分含量显著高于 $0 . 2 3 \%$ 非植酸磷水平组，这与本试验结果一致。Biehl等[27研究发现，饲粮中添加 $\mathrm { V D } _ { 3 }$ 能提高肉仔鸡胫骨灰分含量，但 $\mathrm { V D } _ { 3 }$ 添加量加倍不能引起更大的胫骨灰分反应，这与本试验结果一致。胫骨灰分中矿物元素含量是衡量动物对矿物元素吸收利用的方法之一[31]。当畜禽从饲粮中摄入的磷不足时，磷就会从骨组织中动员出来以满足机体需要，从而导致骨灰分中磷含量降低[32]。本研究发现， $0 . 4 4 \%$ 非植酸磷水平组肉仔鸡胫骨灰分磷含量显著高于 $0 . 2 1 \%$ 非植酸磷水平组，这与刘松柏[25]的报道一致，而 $\mathrm { V D } _ { 3 }$ 水平对胫骨灰分磷含量没有影响，这可能是因为饲粮中一定的钙水平使得灰分钙磷比保持相对恒定的状态[33]。由于采食量的差异，动物进食磷量不同，胫骨灰分磷绝对沉积率消除了采食量所造成的进食磷量差异，更能反映动物对磷的利用效果。本研究发现，饲粮中添加 $\mathrm { V D } _ { 3 }$ 能显著提高肉仔鸡胫骨灰分磷的绝对沉积率。本研究和上述研究共同表明，饲粮添加 $\mathrm { V D } _ { 3 }$ 可促进磷在骨骼中的代谢利用，一方面可能是由于 $\mathrm { V D } _ { 3 }$ 提高了肉仔鸡肠道磷的吸收，本实验室前期研究表明了适宜水平的 $1 , 2 5 \mathrm { - } ( \mathrm { O H } ) _ { 2 } \mathrm { D } _ { 3 }$ 可提高原位结扎肉仔鸡十二指肠磷的吸收[1]；另一方面可能是由于 $\mathrm { V D } _ { 3 }$ 直接促进了磷在肉仔鸡骨骼中的代谢利用。

4结论

$\textcircled{1}$ 饲粮添加 $\mathrm { V D } _ { 3 }$ 可提高肉仔鸡的生长性能并降低死亡率;  
$\textcircled{2}$ 饲粮添加 $\mathrm { V D } _ { 3 }$ 可促进磷在肉仔鸡骨骼中的代谢利用。

Effects of Different Dietary Vitamin ${ \bf D } _ { 3 }$ Levels on Growth Performance and Bone Phosphorus Metabolic UtilizationofBroilers WEN Qian LYU Lin ZAHNG Liyang LIAO Xiudong\* LUO Xugang\*i (Mineral Nutrition Research Division, Institute of Animal Sciences, Chinese Academy of

Abstract: This experiment was conducted to investigate the effects of different dietary vitamin ${ \bf D } _ { 3 }$ $\left( \mathrm { V D } _ { 3 } \right)$ levels on growth performance and bone phosphorus metabolic utilization of broilers.A completely randomized factorial design involved two non-phytate phosphorus (NPP） contents （204号 $[ 0 . 4 5 \%$ and $0 . 2 3 \%$ (the measured content was $0 . 4 4 \%$ and $0 . 2 1 \%$ ,respectively)] and four $\mathrm { V D } _ { 3 }$ supplemental levels ( $( 0 , 1 0 0 0 , 2 0 0 0$ ,and $4 0 0 0 \mathrm { I U / k g }$ ). There were 8 groups with 6 replicate cages of 10 broilers per cage for each group.The experiment lasted for 21 days.The results showed that the broilers fed diets containing $0 . 4 4 \%$ NPP had significantly higher average daily gain (ADG), average daily feed intake (ADFI), tibia ash content and its phosphorus content ( $( P { < } 0 . 0 5 )$ ，while had significantly lower ratio of feed to gain $\mathrm { ( F / G ) }$ and tibia ash phosphorus absolute deposition rate $\left( P < 0 . 0 5 \right)$ ） compared with the broilers fed diets containing $0 . 2 1 \%$ NPP. When the broilers fed diets containing $0 . 2 1 \%$ NPP, dietary $\mathrm { V D } _ { 3 }$ significantly decreased the mortality of broilers( $( P { < } 0 . 0 5 )$ ，and significantly promoted tibia ash phosphorous weight of broilers at 21 days of age $\scriptstyle ( P < 0 . 0 5 )$ ，while dietary $2 \mathrm { \ 0 0 0 \ I U / k g \ V D } _ { 3 }$ significantly promoted tibia ash phosphorous weight of broilers at 14 days of age $( P { < } 0 . 0 5$ ） compared to the group without addition of $\mathrm { V D } _ { 3 }$ . When the broilers fed diets containing $0 . 4 4 \%$ NPP, dietary $\mathrm { V D } _ { 3 }$ significantly enhanced tibia ash phosphorous weight of broilers compared to the group without addition of $\mathrm { V D } _ { 3 }$ 号 $( P { < } 0 . 0 5 )$ .The results indicate that dietary $\mathrm { V D } _ { 3 }$ can enhance the growth performance and decrease the mortality and promoted phosphorus metabolic utilization in bone of broilers.
