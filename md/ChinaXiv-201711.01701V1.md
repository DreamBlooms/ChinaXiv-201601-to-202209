1935年Underwood 和Marston 分别发现羊的消瘦病是由缺钴引起的，从而确定了钴是动物必需的微量元素，这也是确认的第6种动物必需微量元素I。钴是维生素 $\mathbf { B } _ { 1 2 }$ 的组成成分，而维生素 $\mathbf { B } _ { 1 2 }$ 是造血性维生素，参与核酸、胆碱、蛋氨酸的合成以及脂肪与糖的代谢，对肝脏和神经系统维持正常功能也具有一定作用，同时维生素 $\mathbf { B } _ { 1 2 }$ 也是甲基丙二酞CoA 变位酶和 5-甲基四氢叶酸甲基转移酶等多种酶的辅酶，参与和调节体内的丙酸代谢、叶酸代谢[2。近年来，钴因其具有重要的生物学功能而被营养学家所重视。饲粮中钴可提高动物对饲粮尤其是劣质饲粮中营养物质的消化率[3]。姜成钢[4通过体外瘤胃发酵模拟试验表明，荷斯坦泌乳奶牛饲粮中添加 $0 . 3 ~ \mathrm { m g / k g }$ 钴可显著提高纤维素消化率。穆秀梅等[5]研究表明，饲粮中添加钴和维生素 $\mathbf { B } _ { 1 2 }$ 有利于断奶羔羊的生长发育，并增加断奶羔羊的日增重。郭孝等报道，以含 $5 \%$ 高钴的苜蓿青干草为粗饲料，杜泊羊产肉量显著增加。刘镜恪等指出，饲料中的钴能够提高鱼类的生长性能和血红细胞的数量。Paterson 等[8观察了缺钴和补钴对牛免疫的影响，证实了缺钴导致牛免疫力降低。Fisher等报道，缺钻母绵羊的产羔率下降、死胎率增多、羔羊成活率下降。董海军等[]报道，在绵羊饲粮中添加 $0 . 5 \sim$ $1 . 0 ~ \mathrm { { m g / k g } }$ 的氯化钴不仅能改善绵羊的生长发育，刺激羊羔生长，还能增加羊毛产量。王润莲等[1研究表明，添加钴可促进瘤胃维生素 $\mathbf { B } _ { 1 2 }$ 的合成，其合成量随饲喂时间的延长而增加，6周后趋于稳定。Hertz 等[12研究指出，饲料中添加钴使鲤鱼的成活率、受精卵的孵化率、蛋白质的合成量等均有提高或改善。目前对钴的研究多集中在反刍动物和水产动物方面，在家兔上的研究很少，具体到獭兔上的资料就更缺乏。本试验拟通过在饲粮中添加不同水平的钴，以探讨饲粮钴添加水平对断奶至3月龄獭兔生长性能、屠宰性能、肌肉品质、皮张质量和免疫器官发育的影响，寻求生长獭兔适宜的钴添加水平，为我国獭兔饲养标准的制订提供一定的理论依据。

1.2试验动物分组与饲养管理

选取平均体重为 $8 5 5 . 6 \ \mathrm { g }$ 的断奶獭兔200只（公母各占1/2），按性别和体重随机分为5组，每组40个重复，每个重复1只试验兔。试验兔单笼饲养，试验期间早、晚各饲喂1次。试验前对兔舍、兔笼、料盒和饮水设备进行彻底清洗和消毒，试验期间自然采光和通风，自由采食和饮水，3\~5d带兔消毒兔舍1次。预试期7d，正试期 $6 0 \mathrm { d }$ ，记录试验期间的耗料量和体增重。

1.3样品采集与制备

试验结束后，每组随机抽取8只试验兔（公母各4只)，空腹 $1 2 \mathrm { h }$ 后称重，屠宰剥皮，采集指标测定所需样品。

1.4指标测定和方法

1.4.1生长性能指标

称量试验开始和试验结束时试验兔的体重，并统计断奶至3月龄试验兔全期的喂料量，计算平均日采食量（ADFI)、平均日增重（ADG）、料重比（F/G)。

1.4.2屠宰性能和肌肉品质指标的测定

试验兔于屠宰前空腹 $1 2 \mathrm { ~ h ~ }$ 后称重，记录为活体重，并分别计算半净膛屠宰率和全净膛屠宰率。试验兔放血、去皮、四肢和消化道及内容物重量即为半净膛重；全净膛重为半净膛重去除头、心脏、肺脏、肝脏和肾脏后的重量。

pH：屠宰后 $4 5 ~ \mathrm { m i n }$ 立即用 MettlerMP120 型酸碱度计测背腰最长肌第5肋骨处的pH,将探头插入肌肉 $3 \mathrm { m m }$ 读数。 $2 4 \mathrm { h }$ 后在同一部位再次读数。

失水率：屠宰后 $^ \textrm { \scriptsize 4 h }$ 取眼肌，沿样品肌纤维方向切成 $3 \ \mathrm { c m } { \times } 1 \ \mathrm { c m } { \times } 1 \ \mathrm { c m }$ 长条后称重，记为 $M _ { 1 }$ ；用铁丝钩住肉条的一端，悬空挂于纸杯中，用保鲜膜将纸杯口密封（肉样不得与纸杯壁接触)，置于 $4 \mathrm { ~ } ^ { \circ } \mathrm { C }$ 冰箱中 $2 4 \mathrm { ~ h ~ }$ ，取出样品进行称重，记为 $M _ { 2 }$ 。通过下面公式计算失水率：

失水率 $\left( \% \right) = 1 0 0 \times \left( M _ { 1 } { - } M _ { 2 } \right) / M _ { 1 }$ 。

剪切力：采集的新鲜背腰最长肌放于 $4 \mathrm { ~ } ^ { \circ } \mathrm { C }$ 冰箱，经 $^ { 4 8 \mathrm { ~ h ~ } }$ 贮存熟化后取出，将温度计插入肌肉中心部位，于恒温水浴锅内 $8 0 ~ \mathrm { { ^ \circ C } }$ 加热，在肌肉中心的温度达到 $6 0 \ \mathrm { ^ \circ C }$ 时停止加热，取出肉样，将肉样沿着肌肉纤维的方向修剪成直径为 $1 . 1 2 ~ \mathrm { c m }$ 的圆柱形肌肉块，用C-LM型肌肉嫩度计测定剪切力。

肉色：屠宰后 $4 5 ~ \mathrm { m i n }$ 用日本产CR-1O 型号色差仪，利用CIE-Lab 输出模式，从背腰最长肌处切开3个切面分别记录亮度（L\*）、红度（a\*）、黄度（b\*）值。

1.4.3皮张质量指标的测定

试验兔屠宰后，将剥下的獭兔皮称重，桌面平铺，然后用软尺测量其长度和宽度，长度量取颈部至尾根处，宽度量取腹中部最窄处，并计算皮张面积；皮张折叠，用游标卡尺测量其厚度，皮张厚度为所测值的1/2。

# 1.4.4免疫器官指数

试验兔屠宰后，仔细解剖取出胸腺、脾脏和肝脏（去除胆囊）称重，并根据以下公式计算免疫器官指数：

脾脏指数 $( \mathrm { g / k g ) = }$ 脾脏重 $( \mathbf { g } ) /$ 活体重 $( \mathbf { k g } )$ 胸腺指数 $( { \mathrm { g } } / { \mathrm { k g } } ) { = }$ 胸腺重(g)/活体重 $( \mathbf { k g } )$ 肝脏指数 $( { \mathrm { g } } / { \mathrm { k g } } ) { = }$ 肝脏重(g)/活体重 $( \mathbf { k g } )$ 。

3.1饲粮钴添加水平对断奶至3月龄獭兔的生长性能的影响

138 钴作为动物体内必需的微量元素，对畜禽的生产性能会产生一定的影响，其中报道最  
139 多的是其对反刍动物以及水产动物的促生长作用。Mburu 等[14]给东非小山羊饲喂缺钴饲  
140 粮，与饲喂钴足量饲粮的对照组比较，饲喂缺钴饲粮的试验组的尿氮排出量较高，从而导  
141 致其生长率和体状况下降。杨文平等[15研究发现，在晋中绵羊饲粮中添加钴可提高绵羊的  
142 ADG 和 ADFI。赵振山等[1研究表明，在配合饲料中添加适量的钴对团头鱿的生长有促进  
143 作用，且在添加水平在 $0 { \sim } 2 3 . 4 ~ \mathrm { m g / k g }$ 内有递增趋势，当钴添加水平达到 $2 3 . 4 ~ \mathrm { m g / k g }$ 时，饲  
144 料系数最低，生长效果最佳。梁德海等[17}认为，钴可促进中国对虾的生长，中国对虾对钴  
145 的需要量为 $3 0 ~ \mathrm { m g / k g }$ ，当饲料中钴的含量大于 $7 5 ~ \mathrm { m g / k g }$ 时，增长率和增重率则开始下降。  
146 Tiffany 等[18]指出，饲粮中添加钴可显著提高肉牛的 ADFI、ADG 和饲粮转化率。本试验研  
147 究表明，在IBW无显著差异的情况下，饲粮钴添加水平对断奶至3月龄獭兔的FBW有显  
148 著影响，对 ADFI有极显著影响。随着饲粮钴添加水平的升高，FBW 和 ADFI均先升高后  
149 降低，并且均在饲粮钴添加水平为 $0 . 4 \mathrm { m g / k g }$ 时最高。这与上述研究的结果相似，说明饲粮  
150 钴添加水平对獭兔的生长性能有影响，并在添加水平为 $0 . 4 \mathrm { m g / k g }$ 时达到最佳。

3.2饲粮钴添加水平对断奶至3月龄獭兔屠宰性能和肌肉品质的影响

饲粮营养状况直接关系和影响畜禽的屠宰性能，半净膛率和全净膛率是衡量畜禽性能性能的主要指标。在本研究中，饲粮钴添加水平对獭兔的半净膛率和全净膛率无显著影响，但二者随饲粮钴添加水平水平的升高均有先升高后降低的趋势。这可能是由于试验獭兔的 ADG 未受到饲粮钴添加水平的显著影响所致。研究表明，除了肉品安全因素外，其他属性如肉色、风味、滋味和嫩度等也是消费者判断肉品营养价值和质量的指标。肌肉的pH、系水力、鲜嫩度及肉色是评定肌肉品质优劣的主要指标[19]。肉色虽然与其食用品质(嫩度、风味和多汁性等)无直接关系，但它却是肌肉内部的生理学、生物化学和微生物学变化的外部表现，也是消费者判断肉质和鲜度的主要依据。在本试验中，饲粮钴添加水平对断奶至3月龄獭兔屠宰性能、肌肉品质均无显著影响。这与Kadim等[20在肉牛、李庆云等[21在北京肉鸭上的研究结果有所不同，这可能与试验动物的种类不同有关。

# 3.3饲粮钴添加水平对断奶至3月龄獭兔皮张质量的影响

獭兔是典型的皮用型兔，其皮张具有很高的经济价值，皮张质量除了受品种、疾病、宰杀与剥皮、加工方法等因素影响外，饲粮营养也是一重要影响因素。张福勋[22研究发现，适宜的营养水平与采食量能够提高獭兔皮张质量。皮张质量可以从皮张重量、皮张面积、皮张厚度等诸多指标进行综合评定。吴金山[23]提出了鉴定獭兔皮质量的办法，并建议从兔皮绒毛密度、色泽、板质优劣及兔皮面积几方面来进行评定。本试验研究表明，饲粮钴添加水平对獭兔的皮张面积和皮张厚度无显著影响。随着饲粮钴添加水平的升高，断奶至3月龄獭兔皮张重量先升高后降低，且在饲粮钴添加水平为 $0 . 4 ~ \mathrm { { m g / k g } }$ 时最高。

# 3.4饲粮钴添加水平对断奶至3月龄獭兔免疫器官指数的影响

免疫器官是动物机体生命免疫功能发挥的重要物质基础，是动物执行免疫功能的组织机构，在机体免疫过程中发挥着重要的作用。胸腺是动物机体的中枢免疫器官，是T 淋巴细胞发育、增殖的重要场所，在淋巴细胞的形成和分化过程中起着重要作用；脾脏是动物机体的外周免疫器官，是T、B淋巴细胞定居的场所，也是对抗原刺激进行免疫应答的场所；肝脏内有吞噬力很强的肝巨噬细胞，可识别免疫球蛋白和补体的表面受体，并且可吞噬或胞饮血液中的细菌、病毒、衰老死亡的细胞，是净化体液的重要器官。免疫器官指数177 一定程度上能够反映了动物的免疫功能，一般认为健康动物的免疫器官指数越大动物的免178 疫力越强，免疫器官重量降低为免疫抑制所致，而免疫器官重量增加则为免疫增强的表现179 [23]。刘汉中等[24]研究指出，在断奶至2月龄家兔饲粮中添加不同水平的微量元素钴后，对180 胸腺指数有极显著影响，但对脾脏指数无显著影响。Fisher等报道，缺钴母绵羊的产羔率181 下降、死胎率增多、羔羊成活率下降。Paterson 等8发现，缺钴饲粮导致后备母牛中性粒细182 胞功能明显受损，抗感染力下降。本试验结果显示胸腺指数随着饲粮钴添加水平的升高呈183 现先升高后降低趋势，说明钴与獭兔的免疫力可能存在一定关系，但具体作用机理有待于184 进一步研究。185 4结论
