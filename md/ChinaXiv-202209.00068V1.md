# 中风恢复期中西医康复护理研究综述

张培，张华，高冬华（北京中医药大学东方医院西院区，北京100072）

摘要：中风是临床上的常见病和多发病,又称为脑卒中、脑中风,具有较高的发病率,且多发于中老年群体。随着人口老龄化进程的不断加快,我国脑卒中疾病发病率也有了明显的提高。中风的主要临床表现为脑补缺血、出血性损伤等,该疾病具有较高的致残率和致死率,对人类身体健康有着严重的危害,是世界上最重要的致死性疾病之一。中风的临床症状包括头痛、呕吐、眩晕、突发视觉障碍、意识障碍等,在很大程度上影响着患者的身体健康及生活质量，如果中风没有得到及时有效的治疗,很容易对患者生命健康造成威胁。因此,对于中风患者,给予及时有效的治疗十分重要。而为了有效提高中风患者的治疗效果,促进患者良好康复,还需要注重给予患者有效的护理干预。中风患者中有很大一部分患者都存在不同程度的残疾,如肢体功能障碍、意识障碍等,给患者及家属都带来了巨大的精神及经济负担。因此，这就需要加强对中风患者的康复护理。随着我国中医学的不断发展，中医在临床护理工作中的应用也越来越广泛，且中医所以发挥作用越来越显著。比如在中风恢复期,对患者实施有效的中西医康复护理对提高患者康复质量就具有重要的作用。本研究就对中风恢复期中西医康复护理的相关研究进行回顾性分析。

# 引言

中风是一种严重的脑血液循环障碍性疾病，该疾病具有较高的发病率、致残率、致死率，且多发于中老年人，严重危害人类健康。目前临床上对于中风一直缺乏有效的治疗措施，所以，预防是防治该疾病最好的措施。而一旦当患者发生中风,就具有较高的致死率，很多幸存的患者在发生中风后都会出现不同程度的残疾障碍，生活不能自理，严重影响患者的生活质量。为了更好的提高患者生命质量，就必须加强对患者的预后干预。西医护理在中风患者护理中具有一定的作用，但是整体护理效果不算十分理想[1。中医学认为中风和饮食不节、阴阳失衡、气血失调等有很大的关系，在火、虚、痰、瘀互相作用下导致中风的发生。中医认为在治疗中风方面应该以益气活血为原则。近年来，随着我国中医学的不断发展，中医学在临床上的应用越来越广泛，比如在中风恢复期中，应用中医康复护理对提高患者康复质量、治疗效果就具有重要作用2。将中医护理与西医护理相结合，可以使得中西医护理优点得到共同发挥，进而更好的提高中风患者恢复期康复效果。

# 1中风恢复期

中风又称脑卒中、脑血管意外，是一组由不同病因引起的急性脑血管循环障碍性疾病的总称。该疾病具有发病急、进展快、持续性的特点，如果没有得到及时有效的治疗，很有可能危及患者生命健康。目前临床上根据中风病程的长短主要将中风分为四期,包括中风急性期、中风痉挛期、中风恢复期、中风后遗症期。通过将中风进行分期划分，可以便于医护人员对患者进行更好的防治，同时也可以便于教学和学术交流的开展。中风急性期是指发病后2周之内这一阶段，在发病后的两周时间内，患者病情变化较大，所以治疗主要以抢救和控制进展为主[3]。中风痉挛期是指发病后2-4周这一阶段，该阶段患者主要表现为联动反应、共同运动、肌张力增高、痉挛状态。中风恢复期是指发病后5周-6个月这一阶段，该阶段，患者病情得到了一定的缓解和控制，只有给予有效的治疗和预后，大多数患者在该阶段内都可能治愈，只有少部分患者会进入到后遗症期。中风后遗症期是指发病后6个月以上仍未痊愈这一阶段，该阶段患者会出现各种功能障碍，包括感觉障碍、言语障碍、认知障碍等，患者基本上生活不能自理

中风恢复期是一个尤为关键的阶段，该阶段的治疗效果及护理效果将直接响到患者的预后效果[4]。我国多数研究表明,早期的院内康复护理对中风患者功能康复是尤为重要的。通过给予患者恢复期康复护理，可以有效提高患者运动功能及感觉运动反应，这对促进患者康复，降低患者住院费用都具有重要的作用。针对中风恢复期患者施实中医康复护理干预，从针灸、推拿按摩、肢体康复训练、饮食等方面入手，可以有效改善患者症状，提高患者康复效果，利于提高患者及家人的生活质量[5]。因此,想要提高患者治疗效果、康复效果，就必须提高对患者恢复期护理的重视。

# 2中西医康复护理的研究进展

中风恢复期患者的康复护理主要是通过训练逐渐修正错误的运动模式，进而促进患者恢复正确的运动模式[6。中西医康复护理是一种新型的护理模式，其主要是将中西康复护理与西医康复护理相结合，即将现代康复技术与中医康复手段有机结合，以此来达到更好的康复护理效果。近年来，随着我国中医学的不断发展，中医学治疗水平也有了很大的提高，比如在中风康复护理中，中医护理的应用就具有重要的意义[]。中医康复手段较为多样化，包括艾灸、穴位按摩、中药熏洗、中药药熨、穴位贴敷、中药热敷等中医护理适宜技术，能有效促进中风患者肢体偏瘫后功能的恢复及减轻患者疼痛，尤其在提高中风患者后遗症期的生活质量方面有着显著优势，这些康复护理技术在提高中风患者康复质量方面都发挥着重要的作用[8]。

熊琦对中风病气虚血瘀证患者实施中西医结合康复护理干预的效果也展开了深入的分析，其研究结论表明，对中风病气虚血痴证患者采用中西医康复护理干预效果显著，安全可靠，具有较高的护理应用价值。其研究结果显示，实施西医康复护理的对照组治疗总有效率为 $8 7 . 1 0 \%$ ，实施中西医结合康复护理干预的观察组治疗总有效率为96.88，观察组显著高于对照组。由此也可以看出，中西医康复护理在中风患者护理中的效果显著。

近年来，随着人们生活质量的不断提高，对临床护理工作也提出了更高的要求，为了更好的提高患者治疗效果，促进患者康复痊愈，中西医康复护理的应用也越来越广泛[10]。中西医康复护理不仅在中风恢复期患者护理中发挥着重要的作用，在其他领域中也发挥着重要的作用。如何百花、钟凌英"1就中西医结合康复护理在人工全关节置换术后患者中的应用进行了详细的分析。其研究结论表示，在人工全髋关节置换术后应用中西医结合康复护理的效果显著，中西医康复护理可以有效促进患者关节功能恢复，提高患者复位满意度及生活质量，值得推广。由此可见，中西医康复护理具有极高的临床价值，其在临床护理工作中发挥着重要的作用，对促进患者康复意义重大。

# 3中风恢复期中西医康复护理

# 3.1情志护理

情志护理对促进中风恢复期患者康复也具有重要的作用。很多中风患者在恢复期阶段，很容易由于疾病、治疗的影响产生负性情绪，如焦虑、抑郁、暴躁等，而这些负性情绪就会直接影响到患者的康复效果[12]。对此,通过对患者实施情志护理，就可以有效改善患者的情志，从而使得患者以正确积极的心态面对疾病。情志护理在应用过程中可以根据辩证的不同采取针对性的轻质护理。如肝气郁结者，疏肝解郁，可食柑橘、香蕉等水果或玫瑰花、陈皮代茶饮；心脾两虚者，益气补血安神，可少量应用莲子、桂圆、山药、党参等食物[13]。在实施情志护理过程中，护理人员还可以应用音乐疗法来改善患者情绪，根据患者不同的心态选择适当的音乐。

# 3.2推拿按摩

推拿按摩是中医中常用的康复护理手段，其对促进患者血液循环，改善肢体偏瘫患者的运动功能具有重要的作用4。推拿按摩主要是给予患者肢体相关穴位的推拿按摩，包括肩井穴、合谷穴、曲池穴、足三里穴等，手法包括按揉、捏、弹拨等。在推拿按摩下，可以有效促进患者血液循环，发送送痉挛肌肉，进而促进患者关节活动恢复。胡祎等人[15在脑卒中肢体功能障碍病人恢复期中西医结合康复护理技术研究进展中就对穴位按摩这一中医康复手段进行了详细的分析。其表示穴位按摩有利于脑卒中患者运动功能和日常生活能力的恢复。

# 3.3针灸疗法

针灸疗法是临床上一种常用的中医护理手段，其主要是通过刺激本体感觉神经，从而启动牵张反射，引起拮抗肌收缩，使得痉挛肌肉张力降低，进而达到抑制痉挛、疏通经络的目的。对于中风恢复期患者，实施针灸疗法可以有效改善患者临床症状，促进患者机体功能恢复。针灸实施过程中，需要选择相应的穴位，包括头穴、四肢穴位、吞咽相关穴位等，每次针灸 $3 0 \mathrm { { m i n } }$ ,每日1次，可以起到改善神经功能、吞咽功能、肢体功能的效果。吴运景等在电针疗法对脑卒中偏瘫痉挛患者肢体运动功能的影响中指出，中风患者的偏瘫侧进行电针治疗，结合康复训练可纠正前臂旋前和足内翻跖屈异常模式，促使肩外展功能和屈髋、屈膝运动的恢复，使患者运动功能按正确的运动模式转变，进而改善患者的运动功能，提高日常生活能力。由此可见，在中风患者康复护理中，应用针灸疗法的护理效果显著。

# 4小结

从大量的研究中可以看出，在中风恢复期阶段给予患者中西医康复护理，对提高患者康复效果具有重要的作用。这是因为中西医康复护理将中医护理优势与西医护理优势相结合，在中西医结合护理干预下,患者的护理质量和护理效率都能够得到有效的提高。其中中医康复护理对改善患者肢体血液循环、缓解患者临床症状具有重要的作用，在坚持中西医康复护理下，可以有效帮助中风患者在恢复期得到康复。中西医康复护理值得在中风恢复期护理中积极推广。
