# 苗期盐穗木在不同盐浓度处理下的渗调和抗氧化系统

冯肖莉，樊寿德，周莲洁，张富春，王艳(新疆大学新疆生物资源与基因工程重点实验室,新疆 乌鲁木齐830046)

摘要：拟通过对盐穗木2月龄幼苗和6月龄成苗在不同 $\mathrm { { N a C l } }$ 浓度 $( 0 \ 、 1 0 0 \ 、 3 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ 和 $5 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ )处理15d后的相对含水量、脂质过氧化物（MDA）无机( $\mathrm { { N a } ^ { + } }$ 、 $\mathrm { ~ K ~ } ^ { + }$ 和 $\mathrm { C a } ^ { 2 + }$ )和有机渗透调节物质(脯氨酸和甜菜碱)及抗氧化指标(SOD、CAT、APX、POD、AsA 和GSH)的检测,分析并比较盐穗木不同发育阶段应对不同盐浓度的渗透调节和抗氧化防御系统,为盐穗木的研究与开发利用以及新疆生态环境的治理提供依据。结果表明： $5 0 0 \ \mathrm { m m o l }$ ·${ \mathrm { ~ L ~ } } ^ { - 1 }$ 高盐胁迫下,盐穗木幼苗和成苗相对含水量与对照组相比均显著下降;结合相对含水量和MDA含量显示,100mmol $\cdot \mathrm { ~ L ~ } ^ { - 1 }$ 和 $3 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 } \mathrm { ~ N ~ }$ aCl均未对盐穗木造成胁迫;渗透调节系统中,无机离子 $\mathrm { { N a } ^ { + } }$ 和有机物质甜菜碱(GB)分别在盐穗木幼苗和成苗的渗透调节中发挥主导作用;抗氧化系统中,除盐穗木成苗在 $5 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ 处理下抗坏血酸(AsA)含量升高外，超氧化物歧化酶(SOD)和CAT活性及谷胱甘肽(GSH)含量随盐浓度的增加一直保持较高水平且变化很小;相关系数分析显示AsA 和 SOD 分别在盐穗木幼苗和成苗中发挥了重要的抗氧化作用。由此可见，新疆牧草盐穗木在苗期的不同生长阶段采用不同的渗透调节和抗氧化策略应对盐渍生境。

关键词：盐穗木； $\mathrm { { N a C l } }$ 处理；渗透调节；抗氧化系统；耐盐机制

盐分过高往往会对植物的生长发育造成多方面的影响，但同时植物也可通过复杂的生理生化调节应对盐胁迫。一方面， $\mathrm { { N a } ^ { + } }$ 和 $\mathrm { C l } ^ { - }$ 大量进入细胞造成的离子毒害，导致了细胞结构组分及功能的改变，从而引发细胞周期紊乱、代谢受损和蛋白合成受阻，甚至细胞的死亡(1-2)等。同时,过高盐分造成的渗透胁迫也会使植物根系吸水能力下降和叶片失水速率加快[3]。然而,盐胁迫下植物可通过调节体内积累大量离子和可溶性物质[4-5]调节细胞内水分含量，以维持细胞渗透压平衡，从而保护细胞结构，减轻盐胁迫下的伤害。另一方面,盐胁迫也会诱导细胞内活性氧ROS的产生，如超氧阴离子 $\bigl ( \mathbf { O } _ { 2 } ^ { \cdot \cdot - }$ ）过氧化氢（ $\left( \operatorname { H } _ { 2 } 0 _ { 2 } \right)$ )和羟基自由基( $\mathrm { \Delta O H } ^ { \cdot }$ ）,导致膜损伤、脂质过氧化和DNA修饰，造成无法修复的代谢和结构功能紊乱。植物可通过复杂的酶类和非酶类抗氧化系统保护细胞免受活性氧的伤害[。抗坏血酸(AsA)和谷胱甘肽(GSH)是细胞内重要的非酶类抗氧化剂。AsA能够清除多种有害形式的活性氧如

· $\mathrm { \Omega } ^ { \mathrm { 0 H } , 0 _ { 2 } ^ { \cdot \mathrm { ~ - ~ } } }$ 和 $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ ,并通过抗坏血酸过氧化物酶分解过氧化氢 $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ ，而还原型谷胱甘肽(GSH)通过脱氢抗坏血酸还原酶(DHAR)参与抗坏血酸再生[7]。此外，植物的耐盐性与酶类的抗氧化剂活性也密切相关，如超氧化物歧化酶（SOD）、过氧化氢酶（CAT）、抗坏血酸酶（APX）和过氧化物酶（POD）[8-10]  
。

植物在长期的进化过程中形成了一定的生理机制以适应高盐生境(1]。不同种类的植物往往表现出不同的耐盐生理机制[12],即使同种植物在不同生长阶段的耐盐策略也存在很大差异[13-15]。盐穗木[Halostachyscaspica（Bieb.）C．A．Mey.]生长于中亚盐碱荒漠区,具有肉质同化枝的多年生灌木[16]具有独特的生态学、经济和药用价值，可在盐分含量为 $1 0 0 \mathrm { ~ g ~ } \cdot \mathrm { ~ k g ~ } ^ { - 1 }$ 的干旱区正常生长[17]。前期,Zeng等[18]对45d龄盐穗木幼苗开展了盐胁迫下无机离子和有机物在渗透调节系统中的分布及检测,发现无机离子较有机物在盐穗木幼苗渗调系统中发挥更重要的作用，且无机离子主要积累在生殖器官和同化枝内。然而，目前有关不同苗期生长阶段的盐穗木耐盐生理机制还不清楚，因此，本研究分别检测了处于盐穗木苗期的2月龄幼苗和6月龄成苗在盐胁迫下的渗透和抗氧化系统，通过对生理生化指标的比较和分析，以期明确盐穗木不同生长阶段在不同盐浓度作用下其渗透系统和抗氧化系统是否存在差异，为盐穗木在荒漠、盐碱地的研究与开发利用以及对新疆生态环境的治理提供参考依据。

采用不同盐浓度对盐穗木幼苗和成苗进行处理,测定植株的相对含水量。在 $1 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ 和$3 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ 盐处理下，盐穗木幼苗含水量稳定，大约保持在 $9 6 \%$ 左右。当盐浓度增加到500$\mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ 时，幼苗含水量显著下降( $\cdot P < 0 . 0 5 )$ 。尽管盐穗木成苗在不同盐浓度处理下含水量的变化趋势与幼苗类似，但其在各个盐浓度下的含水量均显著低于幼苗阶段( $\ P < 0 . 0 5 )$ ，且 $5 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 } \mathrm { N a C l }$ 处理时，盐穗木成苗含水量与对照相比下降了 $2 7 \%$ （ $\textstyle P < 0 . 0 5 )$ （图1）。

# 3讨论

维持体内水分平衡一直被认为是植物应对高盐环境的重要手段之一[20]。实验结果表明,在不同$\mathrm { { N a C l } }$ 浓度处理下，盐穗木幼苗较成苗具有更高的含水量。与对照组相比，盐穗木幼苗和成苗在100$\mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ 和 $3 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 } \mathrm { N a C l }$ 处理下含水量都略有增加且保持较高水平（图1)；当盐浓度达到$5 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ 时，盐穗木幼苗含水量与对照组相比稍有下降，但成苗含水量下降了约 $2 7 \%$ ，表明高盐胁迫下盐穗木成苗的渗透平衡遭到了严重破坏。以上结果提示，盐穗木在低盐生境下具有一套有效的渗透调节系统,其中 $1 0 0 \sim 3 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 } \ \mathrm { N a C l }$ 可能是盐穗木生长的适宜浓度，这一结果与Zeng等[18]报道的相一致。

大多数盐生植物肉质化表型与离子含量间存在一定的相关性，使得植物能够通过降低水势而减轻盐胁迫下的离子毒害[2I]。在盐穗木幼苗和成苗生长阶段,NaCl处理均提高了植物体内 $\mathrm { { N a } ^ { + } }$ 含量，且盐穗木幼苗较成苗积累了更多 $\mathrm { { N a } ^ { + } }$ （图2a）。同时发现，无论是盐穗木幼苗还是成苗， $\mathrm { { N a } ^ { + } }$ 作为主要的无机渗调物以保证较低的水势（图1，表1），其中在幼苗中， $\mathrm { { N a } ^ { + } }$ 占渗调物贡献比率的 $8 4 . 2 1 \%$ ,在成苗中占 $3 9 . 0 7 \%$ 。以上结果提示，盐穗木和其他肉质真盐生植物如盐节木和盐爪爪(22-23]类似,它们在盐胁迫下能够积累大量无机离子 $\mathrm { { N a } ^ { + } }$ ,并将其隔离到液泡中作为主要的渗透调节物质，提高细胞的渗透压，吸取更多水分来应对高盐胁迫。因而，肉质真盐生植物在盐胁迫下积累的大量 $\mathrm { { N a } ^ { + } }$ 可能是其进行渗透调节的一个基本策略。大量的研究显示，盐渍生境下植物会积累过多的 $\mathrm { { N a } ^ { + } }$ 而减少其他矿物质营养离子的吸收,如 $\mathrm { ~ K ~ } ^ { + }$ 和 $\mathrm { C a } ^ { 2 + [ 1 9 , 2 4 - 2 5 ] }$ ,进而对植物造成离子毒害。然而，不同物种间响应离子毒害的策略不尽相同[26]。其中, $\mathrm { ~ K ~ } ^ { + }$ 是植物必需的元素之一，同时又是一种有效的渗透调节物，在胞质中的平衡对细胞代谢及正常功能的发挥均起着重要作用[27]。前人研究发现[28]，由于 $\mathrm { { N a } ^ { + } }$ 与 $\mathrm { ~ K ~ } ^ { + }$ 间存在竞争拮抗关系，因而盐胁迫下植物体内的 $\mathrm { ~ K ~ } ^ { + }$ 含量急剧下降，但盐生植物在盐渍生境下往往具有较好的保持高水平 $\mathrm { ~ K ~ } ^ { + }$ 的能力。本实验中，随着盐处理浓度的增加，尽管盐穗木幼苗和成苗中 $\mathrm { K ^ { + } / \mathrm { N a } ^ { + } }$ 比率逐渐降低，但它们仍然保持着较高的 $\mathrm { K ^ { + } }$ 含量（图2c），表明盐穗木具有较强的 $\mathrm { ~ K ~ } ^ { + }$ 吸收和储存能力，推测盐穗木在盐胁迫下对 $\mathrm { ~ K ~ } ^ { + }$ 的高亲和吸收及选择是其维持盐胁迫下存活的重要途径。从图2b可以看出，盐穗木成苗较幼苗积累了更多的 $\mathrm { ~ K ~ } ^ { + }$ （表1），表明 $\mathrm { K } ^ { + }$ 在盐穗木成苗抵御盐胁迫时发挥了重要功能。 $\mathrm { C a } ^ { 2 + }$ 同样也能够调节和减轻盐胁迫对植物生长带来的不利影响[29]。盐胁迫下植物细胞高水平 $\mathrm { C a } ^ { 2 + }$ 的积累往往作为植物耐盐的一个重要参数。本研究中，随着盐胁迫处理浓度的增加,盐穗木幼苗和成苗中 $\mathrm { C a } ^ { 2 + }$ 含量略有升高,但基本保持不变(图2d)。由于 $\mathrm { C a } ^ { 2 + }$ 是细胞内第二信使，为保证细胞代谢反应的正常进行， $\mathrm { C a } ^ { 2 + }$ 不能持续高水平存在[30],这可能是长期盐胁迫下 $\mathrm { C a } ^ { 2 + }$ 在盐穗木可溶性渗透调节物中所占较小比例的原因（表1）。

盐渍生境下，植物被迫积累大量的矿物盐，为保证胞质内的正常代谢，往往将这些矿物离子隔离在液泡中，而胞质中有机物的积累可保持细胞内渗透压的平衡。本文中，游离脯氨酸和甘氨酸甜菜碱的含量随着盐浓度的增加而升高，暗示其在盐穗木的渗透调节中发挥着重要作用。尽管游离脯氨酸含量分别在 $5 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ NaCl处理的盐穗木幼苗及$1 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ 和 $3 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ NaCl胁迫下的成苗中显著增加（图3b），然而，游离脯氨酸对渗透调节的贡献率非常小,仅占可溶性渗调物的 $0 . 0 1 \%$ \~$0 . 4 \%$ （表1）,该结果与Zeng 等[18]对盐穗木的报道一致。大量研究显示,苋科植物通常在盐胁迫下会积累甘氨酸甜菜碱作为主要的渗调物质，应对细胞内高钠环境。甜菜碱(GB)作为一种无毒的细胞渗透调节物质,能够调节渗透压[31并稳定蛋白质结构[32],同时,也可保护光合系统免受盐胁迫损伤[3]及减少活性氧(ROS)的产生[34-35] 。

本研究中，作为苋科的盐穗木在盐胁迫下也积累了大量甜菜碱，其在幼苗渗透调节系统中的贡献率为 $6 . 5 2 \%$ ,而在成苗中达 $4 1 . 6 7 \%$ （表1）。因此，甜菜碱作为有机渗调物在盐穗木成苗的渗透调节中发挥了重要作用(表1)。基于以上结果，盐穗木幼苗主要通过 $\mathrm { { N a } ^ { + } }$ 的积累，而成苗主要通过有机物甜菜碱的合成进行盐渍生境下的渗透调节。

植物在盐胁迫下积累大量活性氧，如超氧阴离子 $( \boldsymbol { 0 } _ { 2 } ^ { \cdot \cdot \cdot }$ ）、过氧化氢 $\mathrm { ( H } _ { 2 } \mathbf { O } _ { 2 }$ )和羟基自由基 $( \ { } ^ { \cdot } \ \mathrm { O H } )$ 。活性氧(ROS)的过量产生会加剧对细胞膜的脂质过氧化，而丙二醛(MDA)常被认为是反映植物氧化损伤和敏感程度的一个指标[36]。本研究中,除了盐穗木成苗在高盐 $\left( 5 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 } \ \mathrm { N a C l } \right)$ 胁迫下MDA含量显著增加（图4a）外， $1 0 0 \ \mathrm { \ m m o l } \ \cdot \ \mathrm { L } ^ { - 1 }$ 和300$\mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ $\cdot \mathrm { ~ L ~ } ^ { - 1 } \ \mathrm { N a C l }$ 处理并未增加盐穗木幼苗和成苗中的 MDA含量,甚至 $3 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 } \ \mathrm { N a C l }$ 胁迫下的幼苗MDA含量还有所下降。事实上，保持逆境环境下低水平的脂质氧化损伤是抗性植物的一个重要特征[37]，以上结果再次提示, $1 0 0 \sim 3 0 0 ~ \mathrm { \ m m o l }$ ：$\mathrm { { L } ^ { - 1 } \ N a C l }$ 可能是盐穗木生长的适宜浓度。

植物在漫长的进化过程中形成了一套有效而复杂的抗氧化系统，以减少ROS造成的损伤，包括抗氧化物还原性谷胱甘肽(GSH)和抗坏血酸(AsA）以及抗氧化酶 SOD、POD、CAT 和 $\mathrm { A P X } ^ { [ 3 8 - 3 9 ] }$ 。SOD是抗氧化系统的第一道防线，它能够降解 $\mathbf { O } _ { 2 } ^ { - }$ 阻止次级胁迫的产生[40],直接调节 ROS 的含量。然而,SOD活性的产物 $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ 仍然有毒,随后进一步由其他抗氧化剂如CAT、APX 和POD 将其转变为 $\mathrm { H } _ { 2 } \mathrm { O }$ ○抗氧化剂GSH和AsA可以作为盐胁迫条件下SOD活性的补充，能够直接消除几种类型的ROS，如超氧阴离子自由基和单线态氧[41]。笔者对抗氧化系统指标的检测结果显示，盐穗木幼苗在盐胁迫处理下，与对照组相比,其中APX和POD 酶活性显著升高，然而AsA和GSH含量以及抗氧化酶SOD和CAT活性均未发生显著变化(图5和图6)。盐穗木成苗中，GSH含量和SOD及CAT活性也未受到盐胁迫的影响，然而在 $1 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ 和 $3 0 0 \ \mathrm { m m o l }$ ：$\mathrm { ~ L ~ } ^ { - 1 } \ \mathrm { N a C l }$ 胁迫下,POD 活性和 $5 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ NaCl胁迫下AsA含量显著的高于对照组（图5和图6）。同时，SOD和CAT活性及GSH含量一直保持稳定的高水平，对盐分表现出不敏感的特性。

幼苗中，尽管APX活性随盐浓度增加而升高，并与MDA和 $\mathbf { O } _ { 2 } ^ { - }$ 显著相关(表2），但其在所检测的6个抗氧化系统指标中的相关系数排名(表3)却是最低的，暗示APX不是盐穗木幼苗清除ROS的关键酶。GSH作为一种亲水性抗氧化剂，也可以将超氧阴离子自由基转化为 $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ 。在盐穗木幼苗和成苗中，GSH含量和 SOD、POD、CAT 活性及AsA含量呈正相关(表2），表明高水平的GSH含量往往伴随着高水平的SOD和CAT活性。

盐穗木幼苗和成苗中，AsA与GSH含量及SOD与CAT活性之间呈正相关（表3），尽管它们对盐胁迫重要的生化指标排名顺序在盐穗木的两个发育时期基本一致，但两个发育时期这些指标反应的强度有所区别。相关性分析发现（表3），其中抗坏血酸(AsA)是盐穗木幼苗抗氧化系统的关键酶，而超氧化物歧化酶(SOD)则在盐穗木成苗中发挥了重要的抗氧化作用。

# 4结论

作为荒漠草场中典型的盐生植物，盐穗木具有抗干旱、耐盐碱、耐风蚀沙埋的特点，是防风固沙、绿化造林、保持水土的优良灌木，可广泛用于荒漠化防治、盐碱地改良以及沙漠公路的防护。本研究发现，盐穗木幼苗和成苗分别通过积累无机离子 $\mathrm { { N a } ^ { + } }$ 和有机渗调物甜菜碱的方式，以维持植物在高盐生境下的水势;抗氧化剂AsA和抗氧化物酶SOD在盐穗木幼苗和成苗期分别发挥了重要的抗氧化作用，以上结果反映了盐穗木在苗期的不同生长阶段，通过不同的渗透调节和抗氧化策略应对盐渍生境，该研究对盐穗木在荒漠盐碱地上开发奠定了基础资料。

# Osmotic and Antioxidant System in Halostachys caspica Seedlings under Salt Stress

FENG Xiao-li，FAN Shou-de， ZHOU Lian-jie， ZHANG Fu-chun， WANG Yan(XinjiangKeyboratoryoflogicalesourcedeticgineing,iangUesityUruqio,igin

Abstract：The objective of this research was to beter understand the mechanism of osmoregulationand antioxidantresponse of Halostachys caspica seedlings growing under salt stressThe effects of NaCl-salinity treatments (0,100,300 and $5 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ NaCl for 15 days）on water content,lipid peroxidation（MDA）,organic（proline and glycine betaine） and inorganic osmolytes( $\mathrm { \Delta N a \mathrm { \Omega ^ { + } \mathrm { \Omega , K ^ { + } \mathrm { \Omega } } } }$ and $\mathrm { C a } ^ { 2 + }$ ）,and antioxidant defenses （SOD,CAT,APX, POD,AsA and GSH) in seedlings and grown-up seedlings of $H .$ .caspica were investigated. The research on the development and utilization of $H$ . caspica and the improvement of ecological environment in Xinjiang was carried out. Water content in the seedlings and grown-up seedlings of $H$ .caspica growing under high-salt ( $5 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ NaCl) stress was significantly decreased compared with those under control.The data of water content and MDA content revealed that asalt stress on $H$ 、caspica plants was not induced under the conditions of $1 0 0 \ \mathrm { \ m m o l } \cdot \mathrm { \Delta L } ^ { - 1 }$ and 300 $\mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ NaCl.Na+ and GB played the dominant roles in osmoregulation of seedlings and grown-up seedlings. With increasing saltconcentration,theactivities of SODand CATas well asthe content of GSH were maintained at a high level all along except AsA content of grown-up seedlings under $5 0 0 \ \mathrm { m m o l } \cdot \mathrm { L } ^ { - 1 }$ NaCl treatment was increased.The correlation coeffcient analysis showed that AsA and SOD played an important role in antioxygenation of seedlings and grown-up seedlings of $H$ 、caspica. It could be seen that the osmotic adjustments and antioxidant strategies were different at different growth stages of $H .$ . caspica seedlings so as to adapt salt stress.
