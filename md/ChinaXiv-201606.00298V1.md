The discovery of superconductivity in $\mathbf { A } _ { \mathrm { x } } \mathrm { F e } _ { 2 \mathrm { - y } } \mathbf { S } \mathbf { e } _ { 2 }$ 0 $\mathbf { \rho } _ { \mathrm { A } } = \mathbf { K } \mathbf { \rho } $ ,Cs,Rb,Tl/Rb,TI/K)[1-6] triggered another surge of research on iron-based superconductors, which previously were only featured by iron pnictides [7-12] and $\beta$ FeSe [13]. The new series of superconductors can be regarded as formation from the intercalation of metals between the FeSe layers of $\beta$ -FeSe.In comparison to the iron pnictides,metal-intercalated iron selenides are much complicated in terms of structure, chemical composition and phases. The nature of the superconducting (SC） phase, for example, is still in debate though considerable progress has been made over the last a few years [14-16].

One of the most controversial issues is whether the SC phase is Fe-vacancy-free or not, i.e. the FeSe layers in $\mathbf { A } _ { \mathrm { x } } \mathrm { F e } _ { 2 \mathrm { - y } } \mathbf { S } \mathbf { e } _ { 2 }$ are stoichiometric $( \mathrm { y } \approx 0 )$ ； or off-stoichiometric due to the existence of considerable Fe vacancies. The origin of this issue is largely due to the phase separation that inevitably occurs in these systems at $5 0 0 \sim 5 7 8 \mathrm { ~ K ~ }$ [17]，leading to the coexistence of the insulating phase $\mathrm { ( A _ { 2 } F e _ { 4 } S e _ { 5 } ) }$ and the SC phase [18-23]. The thus-obtained SC phase is not standing freely,instead it intergrows with the insulating phase in the form of nanostrip and its volume fraction is quite low， $1 0 ~ \sim ~ 2 0 \%$ ，as estimated by various measurements [24-29]. This is the main obstacle that prohibits the precise determination of the structure and composition of the SC phase. Ying and coworkers [30.31] verified that the SC phases in the K-intercalated iron selenides are almost no Fe vacancy in the FeSe layers based on their study of the superconductors obtained via a liquid ammonia route. The neutron diffraction showed that the 43 K SC phase in Li/ammonia co-intercalated FeSe compound is Fe-vacancy-free in a separate study [32l. Guo et al. 33] synthesized a 37 K SC phase containing slight Fe vacancy by the similar liquid ammonia method. In the ${ \sf K } _ { \mathrm { x } } \mathrm { F e } _ { 2 . \mathrm { y } } { \sf S } \mathrm { e } _ { 2 }$ film grown on （204号 $\mathrm { S r T i O } _ { 3 }$ (001） substrate, an Fe-vacancy-free phase ${ \displaystyle { \mathrm { K F e } } _ { 2 } } \mathrm { S e } _ { 2 }$ with ${ \sqrt { 2 } } \times { \sqrt { 5 } }$ charge ordering was observed in the SCregion [34,35].Diferent opinions,however,exist.For example,the SC phases are thought to originate from superstructures due to Fe vacancy, such as $2 \times 4$ [36] or $\sqrt { 8 }$ （204号 $\times \sqrt { 1 0 }$ [37]. Meanwhile, phases with disordered Fe vacancies were also reported to be SC [38,39]. Apart from Fe vacancies, the possibility of the existence of about $20 \%$ excess Fe in an SC phase with transition temperature (Tc)of 44 K in the K-Fe-Se system was also reported [40l.

Another puzzling issue is the existence of several SC phases with different $\mathrm { { T _ { c } } }$ from $3 0 \mathrm { K }$ to $4 6 ~ \mathrm { K }$ , which are often mixed in a single sample.For instance,apart from a dominated $3 0 \mathrm { K }$ phase,a $4 4 { \mathrm { ~ K ~ } }$ phase of trace amount sometimes can be observed in a $\mathrm { K _ { x } F e _ { 2 - y } S e _ { 2 } }$ sample obtained by high-temperature route [41l.The amount of $4 4 \mathrm { ~ K ~ }$ phase can be enhanced in samples with a little less K content ( $\mathrm { \tilde { x } } \approx 0 . 6 \sim 0 . 7 \mathrm { \cdot }$ , but its real compositions are unknown. The results in ref.[31] revealed that the obtained SC phases differ only in K contents, ${ \bf K } _ { 0 . 3 } \mathrm { F e } _ { 2 } { \bf S } \mathrm { e } _ { 2 }$ with a $\mathrm { T _ { c } } = 4 4 \ : \mathrm { K }$ and ${ \sf K } _ { 0 . 6 } \mathrm { F e } _ { 2 } { \sf S } \mathrm { e } _ { 2 }$ with a $\mathrm { T _ { c } } = 3 0 \mathrm { K }$ .Superconductivity with $\mathrm { T _ { c } }$ of $3 0 \mathrm { K }$ and 43 K was also observed in $\mathrm { K _ { x } F e _ { 2 - y } S e _ { 2 } }$ samples with $\sqrt { 2 } \times \sqrt { 2 } \times 1$ superstructure due to K vacancy ordering,but the phases for different $\mathrm { T _ { c } }$ have not been specified [25,28,42,43l. In addition, Kintercalated iron selenide with excess Fe was also proposed to have $\mathrm { T _ { c } }$ of $4 4 \mathrm { K }$ [40].

So far,there has been no consensus on the answers to the above issues.Moreover, the underlying mechanism behind the latice stability and vacancy are also poorly understood, in particular, the origin for multifarious structures.To address these issues, here, we first study the energetic change and structural evolution as a function of intercalated metal content by taking $\mathrm { K _ { x } F e _ { 2 - y } S e _ { 2 } }$ as an example using first-principles calculations.Two competing factors are found to dominate the formation of the phases and structural evolution. One is due to the energy increase caused by the accumulation of negative charge in FeSe layers. The other is due to the Coulomb attraction between K ion layers and negatively charged FeSe layers. Then we show that the intercalated K content at $0 . 2 5 \le \mathbf { x } \le 0 . 6$ can stabilize the body centered tetragonal (bct) lattice and Fe is favored to fully occupy its site. At $\mathbf { \sigma } _ { \mathbf { X } } > 0 . 6$ ， the structure is stabilized by creating Fe vacancies in FeSe layers. At low intercalated K level, $\mathbf { x } < 0 . 2 5$ ,the structure will collapse due to the lattice instability. Then a schematic phase diagram is constructed accordingly whereby we speculate the SC phases are ${ \sf K } _ { \mathrm { x } } \mathrm { F e } _ { 2 \mathrm { - y } } { \sf S } \mathrm { e } _ { 2 }$ with $0 . 2 5 \leq \mathbf { x } \leq$ 0.6 and $\mathbf { y } \approx \mathbf { 0 }$ . Our finding sheds some light on understanding this distinct SC family and should be also applicable to other metal-intercalated iron selenides besides $\mathrm { K _ { x } F e _ { 2 - y } S e _ { 2 } }$

The variations of these four energy contributions as a function of $\mathrm { \bf K }$ content are calculated and the results are shown in Figure 1(b),respectively. First of all, FeSe layers are prominent in contributing more and more positive energy to $\Delta \mathrm { E } _ { \mathrm { I } }$ when they are negatively charged with more and more electrons. This is easily understood since charging a neutral FeSe sheet will incur additional energy like charging a capacitor. Inversely，with the increasing K content, more charges willcome into effect in the Coulomb attraction between K ion layers and the charged FeSe layers and thus greatly enhance $| \Delta \mathrm { E _ { C } } |$ ，which will offset sufficiently the increase of energy induced by electron doping into the FeSe layer. In contrast with these two contributions, the other two energy contributions due to the formation of $\mathrm { K }$ ion layers and to the deformation of FeSe layers are much smaller.Therefore,we conclude that the former two contributions dominate the energy change during the $\mathrm { \bf K }$ intercalation.

It is found that $\Delta \mathrm { E } _ { \mathrm { F e } }$ vacancy Only fluctuates above and below zero when $\mathbf { x } \leq 0 . 6$ as shown in Figure 2(a). When $\mathbf { x } > 0 . 6$ ， $\Delta \mathrm { E } _ { \mathrm { F e } }$ vacancy rapidly drops,revealing that Fe vacancies are favored at high levels of K intercalation. This is easily understood since K has a smaller electronegativity than Fe,when the doped electrons due to $\mathrm { K }$ is over a limit that Fe-Se bond can accommodate, the surplus electrons will prefer to transfer into $\mathrm { F e } ^ { 2 + }$ . In this way,Fe is repelled out of the lattice in the form of element. The final compound with Fe vacancy is more energetically favored state. The limit is around at ${ \bf { X } } = 0 . 6$ The $\mathrm { K _ { 2 } F e _ { 4 } S e _ { 5 } }$ phase contains $20 \%$ Fe vacancy corresponding to ${ \bf x } = 0 . 8$ is a manifestation of this argument. This tendency of appearance of Fe vacancy with K content is supported by the experimental results [30-32].

Besides the above considerations on the formation energy，the influence of lattice dynamics on the structural stability should also be accounted for. This is, in particular, true for the compounds with low levels of intercalated K, which can induce lattice instability. Figure 2(b) shows the density of phonon states (DOPS） of ${ \sf K } _ { 0 . 2 } \mathrm { F e } _ { 2 } { \sf S } { \sf e } _ { 2 }$ ， where negative frequencies appear,meaning the structure is unstable. Partial DOPS further indicates that it is mainly because of the considerable amount of highly unstable Se atoms,which are unbonded due to the absence of $\mathrm { \bf K }$ nearby. The phonon density, however, exhibits no imaginary mode for all other compounds with $\mathbf { x } \geq 0 . 2 5$ . As examples,Figure 2(c) shows the DOPS for $\mathrm { K } _ { 0 . 2 5 } \mathrm { F e } _ { 2 } \mathrm { S e } _ { 2 }$ and ${ \sf K } _ { 0 . 5 } \mathrm { F e } _ { 2 } { \sf S } \mathrm { e } _ { 2 }$ . The above results demonstrate that the phases of ${ \sf K } _ { \mathrm { x } } \mathrm { F e } _ { 2 } \mathrm { S e } _ { 2 }$ free of Fe vacancy are both energetically and dynamically favored in the K intercalation range of $0 . 2 5 \leq \mathbf { x } \leq 0 . 6$

Furthermore,we explore the temperature-dependent stability of structures in the K region of interest.To this end,we perform the molecular dynamics (MD） simulations on ${ \bf K } _ { 0 . 2 5 } \mathrm { F e } _ { 2 } { \bf S } \mathrm { e } _ { 2 }$ and ${ \bf K } _ { 0 . 5 } \mathrm { F e } _ { 2 } { \bf S } \mathrm { e } _ { 2 }$ . Both structures can survive for at least l picosecond using $2 \sqrt { 2 \times }$ $2 \sqrt { 2 \times 1 }$ supercells at temperatures up to $5 0 0 \mathrm { ~ K ~ }$ ， indicating that they are stable at this temperature [46]. It should be worthy to note that the atom displacements of ${ \bf K } _ { 0 . 2 5 } \mathrm { F e } _ { 2 } { \bf S } \mathrm { e } _ { 2 }$ are larger than those of ${ \sf K } _ { 0 . 5 } \mathrm { F e } _ { 2 } { \sf S } \mathrm { e } _ { 2 }$ ， suggesting that $\mathrm { K } _ { 0 . 2 5 } \mathrm { F e } _ { 2 } \mathrm { S e } _ { 2 }$ is less stable. The variations of free energy with temperature for the two phases are also calculated and shown in Figure 3(d). The free energy is more favorable for ${ \bf K } _ { 0 . 5 } \mathrm { F e } _ { 2 } { \bf S } \mathrm { e } _ { 2 }$ at temperatures above $2 6 8 \mathrm { ~ K ~ }$ ,consistent with the result obtained by MD. Despite their variations in stability，both of them have the electronic structures similar to ${ \displaystyle { \mathrm { K F e } } _ { 2 } { \mathrm { S e } } _ { 2 } }$ (see Figure S3), which suggests that they should have the similar properties to ${ \displaystyle { \mathrm { K F e } } _ { 2 } } \mathrm { S e } _ { 2 }$ . Considering stoichiometry, formation energy,stability and electronic structures, $\mathrm { K } _ { 0 . 2 5 } \mathrm { F e } _ { 2 } \mathrm { S e } _ { 2 }$ and ${ \sf K } _ { 0 . 5 } \mathrm { F e } _ { 2 } { \sf S } { \sf e } _ { 2 }$ (see Figure S2 for their structures） are proposed to responsible for the observed superconductivity at 44 K and at 30 K [31], respectively.The relative stability difference could also explain the difficulty to obtain the 44 K phase.

Based on the results presented above, we schematize a phase diagram for the $\mathrm { K _ { x } F e _ { 2 - y } S e _ { 2 } }$ system in Figure 4. In the K-rich portion with $\mathbf { \sigma } _ { \mathbf { X } } > 0 . 6$ ,phases with Fe vacancies tend to exist, agreeing well with the observed antiferromagnetic $\mathrm { K _ { 2 } F e _ { 4 } S e _ { 5 } }$ phase.As for the low level Kintercalated compounds $\left( \mathbf { X } < 0 . 2 5 \right)$ , there has been no report on the synthesis of free-standing ${ \sf K } _ { \mathrm { x } } \mathrm { F e } _ { 2 } \mathrm { S e } _ { 2 }$ . We note that the two identified SC phases ${ \bf K } _ { 0 . 3 } \mathrm { F e } _ { 2 } { \bf S } \mathrm { e } _ { 2 }$ and ${ \sf K } _ { 0 . 6 } \mathrm { F e } _ { 2 } { \sf S } \mathrm { e } _ { 2 }$ lie in the region of $0 . 2 5 \le \mathbf { x } \le 0 . 6$ [31]. $\mathrm { T _ { c } }$ is $4 4 ~ \mathrm { K }$ for the former phase and $3 0 ~ \mathrm { K }$ for the later one, suggesting that $\mathrm { { T _ { c } } }$ is dependent on $\mathrm { K }$ content or the doped electron concentrations. Although experimentally observed K contents in SC phases by far are discrete,recent reports about carrier concentration tuning of $\mathrm { { T _ { c } } }$ from 30 to above $4 0 \mathrm { K }$ in FeSe thin flakes [47,48] implies that their variation of $\mathrm { T _ { c } }$ with carrier concentration can be continuous, similar to the cases in other high-temperature superconductors. Further improvement of $\mathrm { T _ { c } }$ can be expected considering theexperimental progresses achieved in single-layer FeSefilms 49-54l.Therefore, weinferthat the $3 0 ~ \mathrm { K }$ phase is electron overdoped and the $4 4 ~ \mathrm { K }$ phase also might not be optimally tuned. Because of the calculation limit and accuracy, we do not consider the slight off-stoichiometric cases,say, the Fe vacancy concentration is less than 3 at. $\%$ . It should be pointed out that such slight off-stoichiometry is tolerated in ${ \sf K } _ { \mathrm { x } } \mathrm { F e } _ { 2 } \mathrm { S e } _ { 2 }$ just like many other materials,which may be the reason that superconductivity was observed in previous reports of refs. [33, 40].

In conclusion, we carefully investigate the energetic change and structural evolution of ${ \sf K } _ { \mathrm { x } } \mathrm { F e } _ { 2 . \mathrm { y } } \mathrm { S e } _ { 2 }$ as a function of intercalated K content using first-principles calculations. Two factors dominating the formation of the phases and the structural evolution are confirmed. One is due to the accumulation of negative charge in FeSe layers; the other is due to Coulomb attraction between K ion layers and negatively charged FeSe layers. The structural evolution of this series of phases is summarized: at $0 . 2 5 \leq \mathbf { x } \leq 0 . 6 \$ ，the bct lattice is stable and Fe is favored to full occupy its site; at $\mathbf { \sigma } _ { \mathbf { X } } > 0 . 6$ ,FeSe layers tend to exclude Fe atoms and create Fe vacancies; and at $\mathbf { x } < 0 . 2 5$ , the structure will collapse for the dynamic instability. A schematic phase diagram is constructed accordingly and the possible route to further improve $\mathrm { T _ { c } }$ is suggested. The phases responding to the observed superconductivity are proposed to be ${ \bf K } _ { 0 . 2 5 } \mathrm { F e } _ { 2 } { \bf S } \mathrm { e } _ { 2 }$ and ${ \bf K } _ { 0 . 5 } \mathrm { F e } _ { 2 } { \bf S } \mathrm { e } _ { 2 }$ in terms of stoichiometry，formation energy，stability and electronic structures. Though based on the study of $\mathrm { K _ { x } F e _ { 2 - y } S e _ { 2 } }$ ，our results should be meaningful to understand the SC and related phases in metal-intercalated iron selenides and other similar SC systems.

# 4.Methods

All the first-principles calculations employ generalized gradient approximation based on density functional theory in the form of Perdew-Burke-Ernzerhof function [3] for exchange-correlation potential, which are performed in the Cambridge Serial Total Energy Package [4l. The self-consistent field method is used with a tolerance of $5 . 0 \times$ （204号 ${ { 1 0 } ^ { - 7 } }$ eV/atom, which is in conjunction with plane-wave basis sets of cutof energy of 330 eV and ultrasoft pseudopotentials in reciprocal space [5]. According to Monkhorst-Pack special $\mathbf { k }$ point scheme [6],the first Brillouin Zone is sampled with grid spacing of $0 . 0 4 \mathring { \mathrm { ~ A ~ } } ^ { 1 }$ . Fully optimization of the atomic positions and lattice parameters of all compounds is done until the remanent Hellmann-Feynman forces on all components are less than $0 . 0 1 ~ \mathrm { e V } / \mathring { \mathrm { A } }$ .The finite displacement method is used to calculate density of phonon states. In order to examine the dynamic stability of the structures, we perform first-principles finite temperature molecular dynamics simulations with time steps of 1O femtosecond under pressure of O.1 MPa. Free energies are calculated as a function of temperature using the quasiharmonic approximation based on phonon dispersion [7].

The initial model of ${ \mathrm { K F e } } _ { 2 } { \mathrm { S e } } _ { 2 }$ is isostructural to ${ \tt B a F e } _ { 2 } \mathrm { A s } _ { 2 }$ [8]. Superstructures due to different arrangements of $\mathrm { \sf ~ K }$ ions along $c$ axis are not considered for the minor energy difference.For example, the difference of $\Delta \mathrm { E } _ { \mathrm { I } }$ between two kinds of $1 \times 1 \times 2$ （204号 supercells with composition of ${ \sf K } _ { 0 . 5 } \mathrm { F e } _ { 2 } { \sf S } \mathrm { e } _ { 2 }$ [see Figure S1(a) and S1(b)], is only 0.4 meV/unit cell. Superstructures containing one $\mathrm { \bf K }$ layer and one FeSe layer due to different arrangements of $\mathrm { K }$ ions perpendicular to $c$ axis are the main concerns. The $\Delta \mathrm { E } _ { \mathrm { I } }$ of $4 \times 1 \times 1$ supercell with composition of ${ \bf K } _ { 0 . 5 } \mathrm { F e } _ { 2 } { \bf S } \mathrm { e } _ { 2 }$ [see Figure S1(c)] is 26 meV/unit cell higher than that of $2 \times 2 \times 1$ supercell with the same composition [see Figure S1(d)]. Hence, supercells with different lattice constants in ab plane, like $3 \times 1$ $\times ~ 1 , 3 \times 2 \times 1$ and so on, are not discussed in this work for their higher $\Delta \mathrm { E } _ { \mathrm { I } }$ compared to those with the same lattice constants in ab plane. Supercells $\sqrt { 2 } \times \sqrt { 2 } \times 1 , 2 \times 2 \times 1$ $\sqrt { 5 } \times \sqrt { 5 } \times 1$ and $2 \sqrt { 2 } \times 2 \sqrt { 2 } \times 1$ are built accordingly with uniform K distribution as the Coulomb repulsion between $\mathrm { \bf K }$ ions dominates $\mathsf { K }$ distribution. To study the formation of Fe vacancy, a $2 \sqrt { 2 } \times 2 \sqrt { 2 } \times 1$ supercell $( a = b > 1 0 . 3 \mathring \mathrm { \textrm { A } }$ and $c > 1 3 . 5 \mathring { \mathrm { A } } ,$ ， which is large enough to accommodate a single Fe vacancy while keeping the tetragonal symmetry, is therefore chosen. In this case, the Fe content corresponds to y $= 0 . 0 6$ . The above structures are only designed to obtain different $\mathrm { \bf K }$ contents in $\mathsf { K } _ { \mathrm { x } } \mathrm { F e } _ { 2 \mathrm { - y } } \mathsf { S e } _ { 2 }$ 0 $\mathrm { y } = 0$ or O.06),which shall not represent the real ones.As K ion ordering cannot guarantee the tetragonal symmetry in the structures， the obtained lattice constants or bond lengths will be the average values. The bond angles are obtained using the average coordinates of Fe and Se atoms. Besides,when we calculate $\Delta \mathrm { E } _ { \mathrm { I } } ,$ （20 the total energy of ${ \sf K } _ { 0 } { \sf F e } _ { 2 } { \sf S } { \sf e } _ { 2 }$ is used instead of that of FeSe so as to offset the additional energy, tens of meV per unit cell induced by the change of cell volume.
