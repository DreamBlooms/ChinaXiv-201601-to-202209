# 老年人照护进程中子代隔代孝老伦理知信行问卷的编制及信效度

王钰’郝习君‘\*陈长香’朱颖”²

随着老年人口激增，老年人养老问题、健康问题急需解决。目前有 $80 \%$ 以上的老年人存在各种慢病如脑梗瘫痪高血压、焦虑症等，而精神关怀和生活照料的缺失是老年人出现健康问题的一大原因[]。且我国老年人群失能率维持在较高水平，达4000 余万人[2，意味着照护需求也较高。孝敬老人需要讲道德尊伦理，因此简称孝老伦理，主要包含后代的传统养老理念和法律养老责任，现阶段我国老年人的主要照料提供者是子代[3]，但隔代的养老认知问题也应重视。首先祖孙间存在对子代的资源竞争问题[4]，其次随着我国三胎政策的实施和人口预期寿命的增加，将有更多的祖辈参与照料孙辈，这意味着祖孙间有更多的日常接触和情感交流，当子女受经济条件、时间精力、或已故等影响无法很好的赡养老人时，孙辈可以对祖辈进行赡养和照顾[5]。目前国内外研究提出子代支持会影响老年人健康[6,[7],8,9]，但仅停留在对子代“行为”方面的测评,但后代对孝老伦理的认知和态度直接影响其照护行为，因此本研究以知信行理论为指导，拟构建子代、隔代孝老伦理知信行调查问卷，为科学评价子代、隔代养老认知水平提供测评工具，为后续研究提供科学实用的测评依据。

# 1问卷的编制过程及方法

1.1 初步拟定问卷

1.1.1确定问卷初始条目 以知信行模式、马斯洛需求层次理论为理论指导，结合国家养法律以及文献回顾。主题词包含孝道、子女照护、隔代赡养、传统养老理念、养法律法规、代际支持、代际关系等；文献语种为中英文；文献内容及结构完整；文献公开发表时间截至2021年6月26日。根据知信行理论确定问卷一级指标为知识、态度、行为三个维度；根据马斯洛需求层次理论，结合老年人这一特殊群体[10]以及后代赡养所涉及到的范围，确定二级指标为经济、物质、安全、情感支持四个维度。经课题组讨论，访谈提纲根据Kallio五步骤半结构式访谈提纲编制法拟定[11]，2021年8月至9月选取有健在父母的子代5人、有健在外祖父母的隔代5人进行面对面访谈，直至内容饱和。访谈提纲：子代访谈提纲包括“您对孝道这一传统文化了解多少？”、“您觉得自己作为子女，陪伴和照顾父母是应该的吗？”、“出于孝心，您给予了父母哪些照顾？”、“您对国家关于赡养老人的法律法规了解多少？”等。隔代访谈提纲包括“您对孝道这一传统文化了解多少？”、“您觉得自己作为孙辈，尊老爱老、照顾老人是应该的吗？”、“您对父母照顾祖父母或外祖父母持有什么看法？”、“您对国家关于孙子女或外孙子女赡养老人的法律法规了解多少？”等。确定形成初始问卷，子代孝老伦理知信行问卷3个维度42个初始条目池，隔代孝老伦理知信行问卷3个维度 41个初始条目池。

1.1.2 设计函询问卷包括：函询目的、专家基本情况、调查问卷条目及重要性评价三部分。采用 Likert scale 5级评分法对各条目重要性评分，并设有条目修改、添加和删除栏错误！未找到引用源。。邀请15名来自一本院校和三甲医院的专家，分析专家的得分数值计算知、信、行条目的重要性赋值、变异系数，进行修改完善，拟定第二轮函询问卷。

1.1.3 专家函询专家纳入标准： $\textcircled{1}$ 具有本科及以上学历； $\textcircled{2}$ 具有副高及以上职称; $\textcircled{3}$ 从事老年相关工作10年以上；$\textcircled{4}$ 自愿参与。最终纳入专家15名，男3人，女12人．年 $4 0 { \sim } 6 2 ( 4 9 . 1 3 { \pm } 6 . 3 7 9 )$ 岁；工作时间 $1 4 { \sim } 4 2 ( 2 5 . 4 7 \pm 9 . \ 3 0 3 )$ （24号年；正高级职称10人．副高级5人；本科2人，硕士9人，博士4人；研究生导师10人。经过两轮专家函询，整理专家意见、建议，同时根据项目筛选标准（项目保留标准：重要性评分 $\geqslant 4 . 0$ 分，且变异系数 $< 0 . 2 5 ^ { [ 1 2 ] }$ ）进行条目修改。两轮函询后子代孝老知信行问卷删除条目13个、修改1个、增加5个，最终保留3个维度34个条目，包括知识维度8个，态度维度9个，行为维度14个条目；隔代孝老知信行问卷删除条目10个，最终保留3个维度 31个条目，包括知识维度9个，态度维度9个，行为维度13个。问卷知识维度采用两分法；态度和行为维度采用Likert5 级评分法。

1.2 正式调查

1.2.1样本量的估计 根据肯德尔样本估算方法，按照问卷条目数乘以 5\~10倍的方法计算[13]，再加上 $1 5 \%$ 的问卷遗失率和无效问卷，最终两个问卷最小样本量为201例和179例。

1.2.2调查对象于2021年10月至2022年2月选取河北衡水市（清凉店镇周围村庄）60岁及以上老年人的家庭为研究对象，调查家庭中的1名子代和1名隔代成员。纳入标准： $\textcircled{1}$ 老年人有子女、孙子女； $\textcircled{2}$ 子代有赡养能力； $\textcircled{3}$ 隔代 $\geqslant 1 8$ 周岁； $\textcircled{4}$ 知情同意。排除标准： $\textcircled{1}$ 老年人没有后代； $\textcircled{2}$ 后代均无赡养能力的家庭。两周后从中各抽取50和38人再次进行调查，计算问卷重测信度。

1.2.3 资料收集 调查前对调查人员统一培训，采用统一指导语。对于填写有困难者，由调查者代其完成问卷填写，当场收回。共发放各270份，回收有效问卷均为264份，有效回收率 $9 7 . 7 8 \%$ 。

1.3统计学方法采用SPSS25.0软件进行统计分析。专家一般资料采用频数及百分数（ $. \%$ ）描述。根据临界值比法、相关系数分析法筛选条目；采用内容效度、探索性因子分析检验效度；采用Cronbach's $\alpha$ 系数、重测信度、分半信度检验信度。

# 2结果

2.1专家函询结果15名专家进行子代、隔代孝老伦理知信行问卷函询，回收率 $100 \%$ ，专家积极性高。子代孝老伦理知信行问卷两轮专家权威系数 $( C R )$ 为0.930 和0.953，肯德尔和谐系数(Kendall'sW)为0.15和0.191，均 $P { < } 0 . 0 1$ 。隔代孝老伦理知信行问卷两轮专家权威系数分别是0.930 和0.953，肯德尔和谐系数(Kendal's $\grave { W _ { \rho } }$ 为0.149和0.156，均 $P { < } 0 . 0 1$ 。

# 3讨论

3.1问卷编制的意义及可靠性

国内外大多研究聚焦在子代的养老行为，但了解传统的养老观念和法律法规，态度积极的认同自己应当履行的养义务，是产生养老行为的前提。通过知信行理论可全面评估子代隔代孝老伦理的知识、态度、行为现状，对提高子代隔代孝老伦理水平、保障老年人的养老权益具有重要意义。本研究以知信行理论和马斯洛需求层次理论为理论指导，将问卷分为知识、态度、行为3个维度，每个维度分为经济、物质、安全、情感4个方面，同时参考大量文献和国家赡养法律及质性访谈后形成子代隔代孝老伦理知信行问卷初始条目池，分别为42个和41个条目。两轮Delphi 专家函询后，根据专家意见修订问卷，最终各保留34个和31个条目。专家积极度高，且两轮专家权威系数均 ${ \geqslant } 0 . 9$ ，表明问卷具有权威性和可靠性。预调查对问卷初稿进行测试后，正式调查检验问卷的信效度，应用临界比值法、相关系数法对问卷条目进行分析，采用内容效度、探索性因子分析检验问卷效度，采用Cronbach's $\alpha$ 系数、重测信度、分半信度检验问卷的信度，确保问卷的有效性、稳定性，最终形成子代隔代孝老伦理知信行问卷。

# 3.2问卷的信效度

效度反应问卷的准确性和有效性，常用的效度评价指标包括内容效度和结构效度。本研究中两个问卷I-CVI均为 $0 . 8 6 7 \sim 1 . 0 0$ ，S-CVI分别为0.967和0.964，符合通常要求问卷的 $\mathrm { I - C V I } \mathscr { \geq } 0 . 7 8$ 和 $\mathrm { S - C V I } { \gtrsim } 0 . 9 0 ^ { [ 1 4 ] }$ ，说明问卷的内容效度良好。其次，本研究探索性因子分析提取的3个公因子（知识、态度、行为）与知信行理论相一致，且两个问卷提取的3个因公子累计方差贡献率分别为 $6 4 . 4 5 5 \%$ 和 $6 4 . 6 4 1 \%$ ，且条目在因子上的载荷均 ${ > } 0 . 4$ 且无多元负荷，高于条目的因子载荷 ${ \mathrm { \cdot } } > 0 . 4  \$ ，公因子累积方差贡献率 $\geqslant 4 0 . 0 \%$ 的标准[15]，说明问卷的结构效度良好。信度反映问卷的稳定性和一致性。本研究中两个问卷及各维度的Cronbach's $\boldsymbol { a }$ 系数分别为0.953、0.956、0.961、0.935和0.952、0.900、0.966、0.936；分半信度分别为0.811、0.929、0.952、0.880和0.837、0.892、0.945、0.889；重测信度分别为0.987、0.951、0.931、0.991和0.991、0.957、0.991、0.990。达到了要求总问卷内部一致性系数 $> 0 . 7$ ，维度内部一致性系数 $> 0 . 7 5$ ，分半信度 $> 0 . 8$ ，重测信度 $> 0 . 7 5$ 的标准[16]，说明问卷的信度良好，有较好的稳定性。

总之，子代、隔代孝老伦理知信行问卷是以知信行理论和马斯洛需求层次理论作为理论框架，通过文献分析、德尔菲专家函询、条目筛选以及信效度检验严谨设计而形成，具有较好的信效度，可以全面评估子代隔代对老年人孝老伦理的知识、态度、行为现状，以期丰富养老理论体系和实践内容，尤其为养老评价体系的完善和实施提供新方法和新工具。
