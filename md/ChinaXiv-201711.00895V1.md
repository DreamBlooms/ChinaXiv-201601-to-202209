高效液相色谱法测定瘤胃甲烷菌共存对厌氧真菌代谢葡萄糖产有机酸特性的影响李袁飞孙美洲 成艳芬\* 朱伟云（江苏省消化道营养与动物健康重点实验室，南京农业大学消化道微生物实验室，南京摘要：本试验旨在利用建立的高效液相色谱法研究瘤胃甲烷菌共存对厌氧真菌代谢葡萄糖产有机酸特性的影响。根据6种有机酸的化学特性确定紫外检测波长、缓冲液浓度、pH、流速、柱温和进样量等液相色谱条件；利用建立的高效液相色谱法检测厌氧真菌纯培养及厌氧真菌与甲烷菌共培养上清液中6种有机酸的浓度。结果表明，高效液相色谱采用的条件为：5 mmol/L磷酸二氢钾-磷酸缓冲液（ $\mathrm { p H } { = } 2 . 4$ ）作为流动相，流速为 $0 . 5 \mathrm { m L } / \mathrm { m i n }$ ，柱温 $2 5 ^ { \circ } \mathrm { C }$ 进样量为 $2 0 \mu \mathrm { L }$ ， $2 1 4 \mathrm { n m }$ 波长检测。6种有机酸能够在 $3 0 \mathrm { m i n }$ 内得到良好分离。各种有机酸的线性相关系数均大于0.999，检测限为 $0 . 2 0 { \sim } 1 . 0 0 \ \mathrm { \mu m o l / L }$ ，定量限为 $0 . 6 6 7 { \sim } 3 . 3 3 3 ~ \mathrm { \mu m o l / L }$ 回收率为 $9 2 . 1 7 \% \sim 1 0 1 . 6 1 \%$ 。甲烷菌共存影响了厌氧真菌的代谢产物。厌氧真菌利用葡萄糖的主要水溶性代谢产物为甲酸、乳酸、乙酸，以及微量琥珀酸、柠檬酸、 $\mathfrak { a }$ -酮戊二酸及乙醇;甲烷菌共存显著降低了上清液中甲酸和乳酸的含量（ $P { < } 0 . 0 5$ ），显著增加了乙酸含量（ $P { < } 0 . 0 5$ ）。综上，本文利用建立的高效液相色谱法可快捷、灵敏、有效地检测厌氧真菌代谢葡萄糖产生的6种有机酸的含量，并发现甲烷菌共存显著促进了厌氧真菌氢化酶体对碳水化合物的代谢。

厌氧真菌是瘤胃内最先定殖到植物纤维组织上的微生物之一[I。它们借助强大的假根系统和分泌的一系列木质纤维素降解酶对植物纤维进行高效地降解。厌氧真菌进行混合酸发酵，其代谢产物主要为甲酸、乙酸、乳酸、琥珀酸、乙醇、二氧化碳（ $\mathrm { C O } _ { 2 }$ ）和氢气（ $\mathbf { \bar { H } } _ { 2 }$ ）[2-4]。甲烷菌严格厌氧，它们不能利用复杂的有机物，只能利用 $\mathrm { H } _ { 2 }$ 、 $\mathrm { C O } _ { 2 }$ 、甲酸、乙酸和甲醇等简单物质作为能量来源。甲烷菌和厌氧真菌可以稳定共存，而且甲烷菌的存在可以显著提高厌氧真菌对底物的降解和利用[5-6]。当甲烷菌共存时，厌氧真菌的主要代谢产物由甲酸、乙酸、氢气和乳酸转变为乙酸、甲烷和乳酸[7-8]。因此，建立厌氧真菌代谢产物中有机酸的检测方法，对于了解甲烷菌共存对厌氧真菌代谢的影响具有重要意义。目前有关厌氧真菌有机酸检测方法的报道较少，测定的有机酸多集中于含量较高的代谢产物，如甲酸、乙酸和乳酸，而且往往是利用不同的方法分开测定的，耗时费力，也不经济。然而，目前还没有能够同时检测厌氧真菌代谢产生的这些有机酸的方法。近年来的研究发现，厌氧真菌发酵可能产生某些微量代谢产物，如 $\mathbf {  { a } }$ -酮戊二酸和柠檬酸[9-10]。事实上还没有一种定量检测厌氧真菌代谢产生的这2种微量代谢产物的方法。因此，本研究旨在建立同时检测厌氧真菌代谢产生的6种有机酸（甲酸、乙酸、乳酸、 $\mathfrak { a }$ -酮戊二酸、柠檬酸和琥珀酸）的方法，并使用该方法研究厌氧真菌产有机酸的特性以及甲烷菌共存对厌氧真菌产有机酸的影响。

# 1.3有机酸分析方法的建立

在 $2 0 0 { \sim } 6 0 0 ~ \mathrm { n m }$ 波长范围内扫描各个有机酸标准溶液，发现在 $2 1 4 ~ \mathrm { n m }$ 波长下，整体的吸光度较高，而葡萄糖在此波长下几乎没有吸收，因此选择吸收波长为 $2 1 4 ~ \mathrm { n m }$ 作为检测波长。试验中比较了不同浓度 ${ \mathrm { K H } } _ { 2 } { \mathrm { P O } } _ { 4 }$ -磷酸（ $\mathrm { H _ { 3 } P O _ { 4 } }$ ）（5、10、15、20mmol/L）缓冲液发现，浓度为5mmol/L时有机酸分离度最好。在 $\mathrm { K H _ { 2 } P O _ { 4 } \mathrm { - } H _ { 3 } P O _ { 4 } }$ 缓冲液浓度为5mmol/L条件下，用 $\mathrm { { H _ { 3 } P O _ { 4 } } }$ 调节缓冲液 $\mathrm { \ p H }$ 为1.6、2.0、2.4、2.7，结果发现有机酸在 $\mathsf { p H }$ 为2.4的流动相中分离度最好。因此选用5 $\mathrm { \dot { \Delta } m m o l / L \ K H _ { 2 } P O _ { 4 } \mathrm { - } H _ { 3 } P O _ { 4 } }$ 缓冲液（ $\mathrm { p H } { = } 2 . 4 \cdot$ ）作为流动相，检测波长设定为 $2 1 4 ~ \mathrm { n m }$ 。通过综合比较分析，最终确定有机酸高效液相色谱分析条件为：流动相为 5mmol/L $\mathrm { K H _ { 2 } P O _ { 4 } \mathrm { - } H _ { 3 } P O _ { 4 } }$ 缓冲液( $\mathrm { \ p H } = 2 . 4 \cdot$ ，流速为 $0 . 5 ~ \mathrm { m L / m i n }$ ，检测波长 $2 1 4 \mathrm { n m }$ ，柱温为 $2 5 ^ { \circ } \mathrm { C }$ 进样体积为 $2 0 \mu \mathrm { L }$ 。根据该分析条件，检测各有机酸保留时间、建立标准曲线并计算检出限与定量限。

# 1.4精密度和回收率的测定

精确吸取适宜浓度的混合标准溶液，按高效液相色谱分析条件连续进样5次，记录峰面积并计算精密度。将同一发酵液样品分装成2份，一份直接测定其中有机酸含量，另外一份添加混合标准品后测定其含量，记录峰面积，平行测定3次，计算回收率，该试验重复5次。

3讨论

3.1检测厌氧真菌代谢的6种有机酸方法的建立

高效液相色谱法具有柱效高、重复性良好、操作方便等特点，并且可以同时定量多种化合物[15]。然而，由于厌氧真菌培养基成分复杂，厌氧真菌代谢产物组分也比较复杂，因此对上清液中有机酸的测定有较大干扰。在条件摸索过程中，笔者发现样品在甲酸目标峰的附近出现了较多杂峰，干扰了甲酸的准确测定。笔者调节了缓冲液pH（ $1 . 6 { \sim } 2 . 7 \$ 、缓冲液浓度（ $5 { \sim } 2 0 ~ \mathrm { m o l / L }$ 、柱温（ $1 5 { \sim } 3 0 ^ { \circ } \mathrm { C }$ 、流动相中甲醇比例和流动相流速等多种方法来消除

杂峰对甲酸的干扰：缓冲液浓度为 $5 \ \mathrm { m m o l / L }$ 时效果好于其他浓度；pH为 2.0 和2.4时效果好于其他 $\mathrm { \ p H }$ ；柱温为 $3 0 ^ { \circ } \mathrm { C }$ 时效果最好；甲醇的加入与否对结果影响不大；当流速较小时，更有利于各有机酸的分离，然而流速过小会使分析时间过长。通过控制相关因素我们最终得到了最佳组合，最大程度地消除了杂峰对甲酸检测的干扰。  
在以往的研究中，厌氧真菌代谢的有机酸往往是利用不同的方法分开测定的，耗时费力，检测灵敏度较低，误差较大，而且测定多集中于甲酸、乙酸、乳酸等含量相对较高的代谢产物，对琥珀酸、柠檬酸和 $\scriptstyle { \mathfrak { a } } .$ -酮戊二酸等微量代谢产物的关注则较少。目前还没有一种关于同时检测瘤胃厌氧真菌多种有机酸方法的报道。在以往甲酸采用甲酸脱氢酶法测定[3.8]，所需试剂成本较高而且不易保存；乳酸采用气相测定时需衍生化处理[3]，因此降低了测定的准确性。本试验结果显示，6种有机酸在 $3 0 ~ \mathrm { m i n }$ 内能得到良好的分离效果；各有机酸的回收率在 $9 2 . 1 7 \% { \sim } 1 0 1 . 6 1 \%$ 之间，相对标准偏差小于 $10 \%$ 。同时，本方法中采用的流动相（ $\mathrm { K H _ { 2 } P O _ { 4 } \mathrm { - } H _ { 3 } P O _ { 4 } }$ 缓冲液）浓度低于其他报导[14-15]，因而大大降低了高盐对色谱系统的不利影响。而且本方法样品处理简单，整个测定过程中无需应用有毒有害试剂。因此本方法完全满足厌氧真菌发酵液中有机酸的测定。

# 3.2厌氧真菌发酵葡萄糖的代谢产物和代谢途径

厌氧真菌利用葡萄糖进行混合酸发酵的代谢产物主要为甲酸、乙酸、乙醇、乳酸、 $\mathrm { C O } _ { 2 }$ 和 $\mathrm { H } _ { 2 } ^ { [ 2 - 3 ] }$ 。与其他好氧真菌不同，厌氧真菌没有线粒体，但其胞内存在一种类似于线粒体的细胞器——氢化酶体（氢体）。厌氧真菌的氢化酶体是细胞内以膜隔离的大约 $1 \mu \mathrm { m }$ 左右的小区室，它们在厌氧条件下通过底物水平磷酸化作用产生 ATP[2]。厌氧真菌可以同时在细胞质和氢化酶体中进行碳水化合物代谢，细胞质中主要代谢产生乳酸、乙醇和甲酸，而氢化酶体中代谢产生 $\mathrm { H } _ { 2 }$ 、 $\mathrm { C O } _ { 2 }$ 、乙酸和甲酸[16]。本研究中厌氧真菌纯培养的主要代谢产物为甲酸、乳酸和乙酸，以乳酸的浓度为最高，表明有很大一部分的碳水化合物在细胞质中被代谢。Cheng 等[10]研究发现，厌氧真菌存在不完整的三羧酸循环（TCA)，包括还原支路和氧化支路；其中氧化支路的终产物为 $\scriptstyle { \mathfrak { a } } .$ -酮戊二酸，还原支路的终产物为琥珀酸。本试验在厌氧真菌上清液中检测到了代谢产物 $\scriptstyle { \mathfrak { a } } .$ -酮戊二酸和琥珀酸，这验证了Kwon 等[9]和Cheng 等[10]的推测。

# 3.3甲烷菌共存对厌氧真菌葡萄糖代谢的影响

在甲烷菌共存情况下，厌氧真菌的主要代谢产物为乳酸和乙酸，以乙酸含量最高。笔者前期的研究表明，在厌氧真菌和甲烷菌共培养的发酵体系中几乎没有氢气的积累，而厌氧真菌纯培养则有氢气积累。甲烷短杆菌属为氢营养型甲烷菌，可以利用氢气或甲酸还原二氧化碳生成甲烷，几乎不利用乙酸[17]。因此，本研究中厌氧真菌与甲烷菌存在着种间氢转移的关系。在甲烷短杆菌存在的情况下，厌氧真菌所产生的代谢产物 $\mathrm { H } _ { 2 }$ 和甲酸会被甲烷菌迅速利用，解除了 $\mathrm { H } _ { 2 }$ 对氢化酶的抑制作用[18]和甲酸对丙酮酸甲酸裂解酶的抑制作用[19]，因此氢化酶体中的代谢途径加强，产生的ATP也增多。氢化酶体在产生ATP的同时会伴随着乙酸的生成[20-21]。共培养上清液中乙酸含量显著高于厌氧真菌纯培养组，这表明共培养组中更多的碳水化合物被厌氧真菌氢化酶体所代谢。氢化酶体中含有的氢化酶催化烟酰胺腺嘌呤二核苷酸磷酸（NADPH）生成烟酰胺腺嘌呤二核苷酸（NADH）和氢气，乳酸生成过程中需NADH 的参与[4]。当细胞内的 NADH减少时，乳酸的生成也会受到影响，因此共培养组的乳酸含量显著低于纯培养组。本研究中，共培养和纯培养发酵液中的琥珀酸、柠檬酸和α-酮戊二酸含量都没有显著差异，这可能是因为从TCA还原支路和氧化支路代谢的碳源量很低；底物（葡萄糖）都已被两者用完。下一步工作将利用建立的方法测定纯培养和共培养动态发酵过程中发酵液有机酸的浓度，拟更深入地研究甲烷菌共存对厌氧真菌代谢的影响。

# 4结论

$\textcircled{1}$ 厌氧真菌代谢葡萄糖产生的主要有机酸为甲酸、乳酸、乙酸以及微量的琥珀酸、柠檬酸和 $\mathfrak { a }$ -酮戊二酸。

$\textcircled{2}$ 甲烷菌共存对琥珀酸、柠檬酸、 $\mathfrak { a }$ -酮戊二酸和乙醇的含量无显著影响，却显著减少了甲酸和乳酸的含量，显著增加了乙酸的含量。

Effects of Associated Methanogen on Organic Acid Profile of Glucose Metabolism by Anaerobic Fungus Revealed by High Performance Liquid Chromatography
