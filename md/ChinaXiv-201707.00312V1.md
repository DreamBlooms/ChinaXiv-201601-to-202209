# 言

当前，中国智库建设已上升为国家战略，如火如茶的智库建设正进入发展的黄金时代。“他山之石，可以攻玉”，本文通过对“全球第一智库”布鲁金斯学会的一手资料（学会学者撰写的学术论文、研究报告、学会年报，以及学会网站上的资料等）和二手资料（国内外学者研究布鲁金斯学会的相关著作、论文等）进行系统分析，梳理布鲁金斯学会的第一个百年（Brookings1.0）发展历史，诠释学会百年铸造与积淀的核心价值和运行机制，以及影响政府决策的方式；同时解读学会在面向第二个百年（Brookings2.0）的战略规划和发展构想，希冀对中国特色新型智库建设提供参考与借鉴。

# 布鲁金斯1.0（Brookings1.0）

# 2.1 布鲁金斯1.0 发展历史

布鲁金斯学会是美国历史最悠久的智库之一。1916年3月，圣·路易斯市的企业家、华盛顿大学理事会主席罗伯特·布鲁金斯（Robert S.Brookings）和其他华盛顿改革者创立了政府研究所，这是第一个对国家公共政策问题进行分析的民间组织，旨在为学者与政府决策者之间架起一座桥梁。随后，罗伯特·布鲁金斯又于1922年和1923年成立了两个姊妹机构一分别是经济研究所和罗伯特·布鲁金斯经济与政治研究院，1927年，3个机构合并为如今的布鲁金斯学会，致力于开展经济、政府管理、政治和社会科学领域的相关研究，促进和推动政府决策的科学化，提高行政效率，并改进政府的公共政策[1]。

创立之初，理事会承诺“坚持扎实的学术研究和深刻的现实思考与客观分析，向公众呈现基于事实的公共政策研究”。理事会副主席布鲁金斯规定：理事会不得干预学会的研究工作，确保学会的独立性和研究的科学性，并给予学者充分的学术自由性。需要提及的是，美国多数智库官方网站使用“.org”域名，以表明其社会机构的身份，而布鲁金斯学会的官方网站却使用“.edu”域名，这种现象也诠释了布鲁金斯学会这一智库所具备的学术性质和教育培训职能。因此，也有中国学者把学会称作“没有学生的大学”[2]。

本文以时间为线索，追忆布鲁金斯学会演变历程，记录学会百年发展历史（1916\~2015年）中所铸造的经典思想以及里程碑大事件，进而呈现学会的荣耀，以及面向未来的举措，

# 2.2 布鲁金斯1.0 核心价值

成立于1916年3月的布鲁金斯学会，于2015年迎来了100周年华诞。在百年的历史长河中，布鲁金斯学会作为非营利性智库机构，一直为政府公共政策提供解决方案和创新思想，并逐步在美国智库发展中发挥了越来越重要的作用。特别是自2010年以来，在国际权威的智库排名报告中，连续6年荣获“全球第一智库”殊荣。学会所积淀的全球领先智库的声誉，关键在于布鲁金斯学会建立之初创始人精心的“顶层设计”管理理念，以及学会百年间一直坚持和坚守的3个核心价值：质量、独立性与影响力（Quality,Independence,Impact）[4]，此三者融合在一起的价值体系也界定了学会与众不同的特点，在学会官方网站以及学会年报上处处彰显着这三个苍遒有力的词语，这也成为学会引以为傲的座右铭。

2.2.1质量（Quality）学会高质量、高品质的公共政策研究源于： $\textcircled{1}$ 一切以学者为核心的运作机制。目前，学会一共有学者300多人，分布在世界各地，包括驻会专家、非驻会专家，以及访问学者等[5]。学会的学者一般都拥有丰富的经历和背景，很多学者有在政府任职的职业经验，学术和时务紧密结合的研究背景，这使得布鲁金斯的研究报告既具有学术研究的扎实与深刻，又不失政策建议的务实与前沿。此外，高质量的研究还来自于学者间频繁深人的交流切磋与讨论，而学会的行政人员则为学者的研究提供和创造一系列便利条件。 $\textcircled{2}$ 研究视角现实性。布鲁金斯学会创立时的训诫为“向公众呈现基于事实的公共政策研究”。百年发展历史长河中，“不变”的研究信念与“变化”的研究视角一直是学会公共政策研究的信仰所在。学会充分重视研究视角的现实性与时效性，审时度势地关注热点问题，学会公共政策研究因此也具备了“未雨绸缪”和“与时俱进”的鲜明特点，这也是学会对政府决策施加重大影响的一个主要缘由。 $\textcircled{3}$ 研究方法科学化。学会将调研、研究、模型计算等科学的研究方法充分引入到公共政策中，成就高质量、高品质的产出（Product），进而对政府决策产生积极影响。2.2.2独立性（Independence）独立性是布鲁金斯智库成立的初衷，是学会信奉的最核心原则，也是整个机构的生命力所在。独立性包括： $\textcircled{1}$ 机构本身的独立性。学会的独立性不仅能够保证学术研究的质量，同时也保证了研究结论的公信力（Credibility）。 $\textcircled{2}$ 学者的独立性。学会强调每一个学者不应受到政治力量、经济力量、或社会（宗教）力量影响，应避免政治偏见，并根据自己的判断和研究得出客观的结论。需要说明的是，布鲁金斯学会作为机构不能持有政治立场，但学者可以持有政治立场，布鲁金斯学会尊重每一个学者的政治立场和政治倾向。 $\textcircled{3}$ 财政独立。财政独立是保障独立性的另一个因素。布鲁金斯学会作为非营利性机构，其运作资金来自于创始人布鲁金斯的基金、社会捐款及出版物收入等，由学会财务部门专门管理。

2015年，布鲁金斯学会的业务收人达到9588万美元，经费来源主要包括政府拨款（ $8 6 \%$ ），基金会、公司、大学及研究机构以及个人的捐赠（ $10 \%$ ),教育培训收入( $5 \%$ )，出版物收入( $2 \%$ ),其他杂项收入（ $2 \%$ ）等，见表2。

2.2.3影响力（Impact）影响力建立在高质量和独立性基础之上。布鲁金斯学会和其他智库一样，不断追求对政策和社会的影响力：

$\textcircled{1}$ 对国家政策的影响力。学会主要通过参与国会听证、设定议程、引领讨论、与政商界团体召开小范围闭门会议，以及设计政策来提高其影响力。“旋转门”机制是布鲁金斯学会发挥其影响力的一个重要途径[。旋转门机制是美国智库对政府发挥影响的一个主要方式，即，政府官员可以出了旋转门进入智库，智库的人也可以经由旋转门进入到政府当中。学会通过“旋转门机制”直接向政府输送人才，很多学者被邀请进入政府工作；学会也会吸纳更多具有政府工作经验的人士进入智库，政府官员也可以出了旋转门加盟智库，以此使学会与政府保持密切联系[8]。这种学术和时务紧密结合的合作方式成为学会影响政府决策的主要方式之一。

$\textcircled{2}$ 对公众观点的影响力。布鲁金斯学会研究当下的热点问题有利于学会吸引媒体和公众的关注，扩大其影响力。此外，学会还通过出版其研究成果（如，学会学者撰写的学术论文、研究报告、学会年报等）、召开面向公众的研讨会、接受各大主流媒体采访，以及在世界范围内发表评论员文章等来影响政府决策，不断促进布鲁金斯学会强大的全球影响力。

$\textcircled{3}$ 对美国国际关系的影响力。学会百年发展中研究领域多元化以及研究范围国际化的发展路径日益彰显，1946年建立的外交政策部的公共政策研究对美国国际关系影响深远。学会主导下的国际间的交流与合作为美国处理国际关系间的敏感、复杂问题提供讨论和交流的平台，一方面有助于学会参与人员建立个人关系网，提高学术影响力，同时也为美国政府制定外交政策提供了建设性的参考意见[9,这也成为美国与其他国家的“二轨外交”方式的重要组成部分。二轨外交是一种特殊的非官方外交，相对于政府间的“第一轨外交”，通常是透过学者、退休官员的交流，以民间形式进行，形式包括双边和多边对话、双边和多边会议、代表团访问、学术交流以及访问学者项目等。以中美国际关系为例，布鲁金斯学会通过举办研讨会、学术交流、邀请中国学者到学会做演讲以及设立访问学者项目等方式，积极地介入中美二轨外交进程，如2006年建立的“布鲁金斯-清华公共政策研究中心”也是中美二轨外交的一个强有力的体现。

# 2.3 布鲁金斯1.0 的运行机制

2.3.1管理架构布鲁金斯学会的领导机构是理事会，目前共有51名理事，由商界、学界、前政府官员以及其他领域的杰出人士组成，理事会的职责是处理学会的行政性事务、财务资金管理，支持学会广泛开展学术研究以及保障学会的独立性。此外，学会另有荣誉理事37名[10]。每届理事会任期3年，每年举行3次全体会议。理事会现任主席由凯雷集团创建人兼CEO大卫·鲁宾斯坦和前高盛公司总裁兼首席运营官、非洲巴瑞克黄金公司董事会主席约翰·桑顿（JohnL.Thornton）先生共同担任[]

布鲁金斯学会的执行机构实行“理事会”领导下的总裁负责制，总裁带领学会的管理团队开展工作，主要负责日常行政工作，包括制订政策、推荐项目、核准出版物，以及挑选研究人员等[]2002年7月斯特罗布·塔尔博特（Strobe.Talbott）被任命为学会现任总裁，塔尔博特职业生涯非常丰富，涉及新闻界、政府以及学术界，之前是耶鲁大学全球研究中心的创始人。塔尔·博特还是美国外交政策的专家，2011年被美国国务卿希拉里·克林顿（HillaryClinton）任命为国务院外交政策委员会主席，直接参与美国政府的外交决策[12]。

总裁负责制的管理架构下，布鲁金斯学会建立了成熟有效的管理团队。目前，拥有九位副总裁，一位总经理[1]，每个人负责各自部门的具体事务，主要涉及政策研究和行政管理两大块： $\textcircled{1}$ 政策研究的运营管理主要依据学会的研究领域设置了5大研究部门（外交政策、经济研究、城市政策、全球经济与发展以及政府研究部门）的副总裁，研究部门下设不同研究中心，由主任负责； $\textcircled{2}$ 行政管理的日常运行遵循企业管理的方法，设置有财务部、运营部、对外联络部、管理培训、出版社等部门的副总裁，直接对总裁负责。所有这些行政管理部门的日常工作都是为政策研究部门提供支持和服务。

2.3.2 布鲁金斯1.0 的组织结构学会组织结构的设置反应了学会的研究全貌与特色，组织结构的调整也体现了学会研究重点与研究领域的演进过程。创立之初，布鲁金斯学会由3个研究所（政府研究所、经济研究所、经济与政治研究院）合并而来，研究领域依据政治和经济两大学科进行划分，机构设置与研究领域相统一。随着世界形势的复杂化和多元化，国际国内出现了很多新的问题，布鲁金斯学会也开始根据新的变化调整组织结构与研究范围，逐渐形成新的发展模式[13]。目前，布鲁金斯学会主要形成了研究部、研究中心、研究项目（或计划)3个由高到低的矩阵式层级组织运行机制，如图2所示。

学会的组织结构取消了研究所的概念，不再局限于按照学科划分，而是根据实际问题而定。组织的第1层级是研究部，第2层级为研究中心，第3层级为研究项目（或计划）。组织的研究项目（或计划）根据研究课题而定，是研究的基本组织单位，研究中心由研究项目（或计划）发展而来。研究中心、研究项目（或计划）这两个依次下降的层级共同支撑研究部的研究。但是，研究部、研究中心以及研究项目（或计划）之间并没有严格的隶属关系，研究中心和研究项目（或计划）的启动与关闭也会根据世界时事演变情况进行动态调整与协调。学会目前拥有5大研究部（Programs）、15个政策研究中心（PolicyCenters）、3个海外研究中心（OperationalCenters），以及28个研究项目（或计划）（Projects/Initiatives），如表3所示。

# 布鲁金斯2.0（Brookings2.0）——发展构想

理事会联合主席约翰·桑顿、大卫·罗宾斯坦以及总裁斯特罗布·塔尔博特在《布鲁金斯学会年报（2015）》[18中的报告，充分体现了学会第一个百年和第二个百年交接之时的传承与变革。

学会的传承：“在学会跨过第一个百年（Brookings1.0）门槛、大步迈向第二个百年（Brookings2.0）的发展征程中，学会将坚守创始人的信念，传承百年来的价值体系，一如既往地维护学会声誉、保护学会品牌、确保学会业务模式可持续发展。”—约翰·桑顿和大卫·罗宾斯坦。

学会的变革：“学会将第一个百年（Brookings1.0）积淀的宝贵财富带入到新百年（Brookings2.0）之时，还将积极探索发展和变革的新路径来应对全球环境变迁和新挑战， $\textcircled{1}$ 深化和拓展研究领域； $\textcircled{2}$ 扩大学会的服务范围，积极吸纳新的公众、社群和合作伙伴进人到学会的发展中来。”——斯特罗布·塔尔博特。

我们可以看到学会在承载其百年荣耀的历史契机，已经开始着手布鲁金斯2.0战略规划（Brookings 2.0 Strategic Plan）的贯彻和实施，众志成城地展开新纪元的篇章，在世事变幻中持续提升学会影响力。

# 3.1启动“百年学者计划”（Centennial ScholarInitiative)

为了庆祝学会成立100周年，学会于2016年1月正式启动“百年学者计划”（CentennialScholarInitiative）研究中心。在全球城市化问题以及城市驱动的世纪挑战下，“百年学者计划”的研究领域不再局限于五大研究部门中任一部门，而是整合学会各部门的研究力量，推动学会跨部门、跨学科、全球视角的公共政策研究，致力于建立创新学术研究体系，不断提升学会本土和全球影响力[19]。该中心的创立者布鲁斯·凯兹也因此成为学会的第一位百年学者（CentennialScholar）[20]。由此，学会的政策中心也从之前的14个增加到15个。

# 3.2 出版物数字化

布鲁金斯学会出版社每年还会一如既往地出版学者的专著，但是运行模式将会向数字作品转型。其中一个工作就是学会将以数字形式重新出版一批对当前时事仍具有社会影响力的开创性的经典著作（Brookings Classics）[21]，为学会解决棘手公共政策问题提供指导与借鉴。

# 3.3运用社交媒体

智库的功能一是建言决策，二是影响舆论。只有智库研究和舆论传播密切结合才能扩大智库影响力[22]。数字时代下，通信技术发生翻天覆地的变化。使用多种传播手段、借助各类媒体平台推广和传播学会的研究成果正成为布鲁金斯学会发展与舆论传播的新形态。社交媒体为公众与学会进行实时交流提供了传播渠道，公众可以及时有效地了解布鲁金斯学者的研究和观点。截止到2015年6月30日，学会社交媒体（Facebook、Youtube、Twitter、Webcast等）上的公众人数逐渐增多，学会对外传播的能力以及全球影响力得到有效提升，这是布鲁金斯学会发挥影响力的重要手段。如图3所示。

# 3.4启动全新的官方网站

学会将于2016年夏天启动新网站（Brookings.edu），进一步使用数字工具和社交媒体平台传播学会研究工作，也积极吸纳全球公众、合作伙伴以及支持者为学会发展献计献策，这包括学会位于北京、多哈以及新德里的3个海外研究中心。同时，学会也会将研究成果迁移到官方网站上，方便公众免费开放获取。

# 3.5扩大研究视野，扩充研究队伍

目前，传统的政治、经济等研究部门已经疲于应对多变的世界环境和变革。在学会新纪元的起步之年，学会将延展学者的专业宽度和广度，运用跨学科研究方法解决复杂、多维的世界难题。学会将大力扩充社群力量，邀请广大公众参与到学会发展之中，多方讨论和辩论当前热门话题，寻求公共政策的最佳解决方案[21]。

# 结语

布鲁金斯学会创立至今，历经100年时间，我们从中看到了一位百年学者在智库发展过程中对核心价值信念的坚守、对学会运行机制的坚持，以及学会应对新纪元中复杂变幻世事的坚定方向。诚然，智库发展建设不是一蹴而就的工作，应有长远、可持续的战略规划。当我们把目光转移到当前我国正在大力开展的新型智库建设上时，布鲁金斯学会过去的成绩、今天“世界第一智库”的影响力以及面向未来的愿景更是给了我们许多启示与思考。

中国特色智库建设应从宏观到微观，从顶层到基层进行系统规划，这样才能使我们早日解决当前智库发展中出现的“服务产品质量不高、缺乏独立性与专业性、组织方式和管理手段比较滞后”的问题[24]，缩小我们与欧美发达国家智库之间存在的差距。

（1）顶层设计：智库的建设和发展离不开顶层理念的设计。好的管理思想和管理理念是智库的生命力和灵魂所在。这需要智库管理团队从战略决策高度对智库有高屋建瓴的宏观掌控。

（2）中坚力量：智库研究人员是智库建设和发展的血液所在。吸纳具备多元学术背景和丰富职业阅历的研究人员加盟智库是其产生高质量研究成果的关键和人才保障。研究人员应注重使用科学的研究方法体系开展信息的收集与分析工作，并为公共政策提供创新思想和解决方案。

（3）基础支撑：智库的运行机制以及组织形态是智库建设和发展的基石所在。智库的运作模式、工作性质以及组织架构是智库可持续发展的有效支撑。在完善自身建设的同时要加大智库产品的舆论传播，加强智库在政府与公众间的传播力度，促进智库与媒体间的合作深度，推动智库国际间的交流与合作，进而不断提升我国智库的综合影响力。
