# 保护性耕作对黄土高原半干旱区农田土壤 $\mathbf { N } _ { 2 } \mathbf { O }$ 排放的影响

杜梦寅¹，袁建钰'，李广1，闫丽娟²，刘兴宇¹，祁小平'，庞晔(1.甘肃农业大学林学院,甘肃 兰州730070；2.甘肃农业大学农学院,甘肃 兰州730070)

摘要：一氧化二氮 $\mathrm { ~ ( N } _ { 2 } O \mathrm { ~ ) ~ }$ )是大气中重要的温室气体之一,在全球气候变暖问题中备受关注,而农田土壤是其主要排放源。在影响农田 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放的诸多因子中,耕作措施作为关键因素，其对旱地农田土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 的排放规律尚不明确。为此，本试验以传统耕作(T)为对照、布设免耕(NT）、传统耕作 $+ i$ 秸秆覆盖(TS）、免耕 $^ +$ 秸秆覆盖(NTS)3种保护性耕作，采用静态箱-气相色谱法对春小麦地土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放通量进行测定，并对影响 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放的相关因子及春小麦产量进行同期测定。结果表明：(1)春小麦地不同耕作措施下土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 累计排放量大小顺序为 $\mathrm { T { > } N T { > } T S { > } N T S }$ ,其中T处理显著高于其他处理,分别较NT、TS、NTS处理增加了 $1 5 . 8 7 \% . 2 8 . 0 8 \%$ 和 $3 9 . 5 8 \%$ ,另外相比于T处理,保护性耕作有助于表层土壤 $\mathrm { N H } _ { 4 } ^ { + }$ -N含量及春小麦产量的增加，并减少 $N \mathrm { O } _ { 3 } ^ { - } - \mathrm { N }$ 含量的积累。(2）相关关系表明，土壤温度、土壤含水量及 $\mathrm { N H } _ { 4 } ^ { + }$ -N含量是影响土壤 ${ \bf N } _ { 2 } { \bf O }$ 排放的关键因素。其中，土壤含水量、 $\mathrm { N H } _ { 4 } ^ { + }$ -N含量与春小麦地土壤 $\mathrm { N } _ { 2 } \mathrm { O }$ 排放量均呈极显著负相关关系，土壤温度与 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放量呈极显著正相关关系。因此，以经济效益和环境效益为基础，综合考虑 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放量和作物产量，黄土高原半干旱区以NTS处理为最优耕作措施。

关键词：温室气体；保护性耕作；耕作措施；春小麦；相关关系;黄土高原

目前，全球气候变暖等问题日趋严峻，主要原因是温室气体浓度的持续增加。而 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 作为影响气候变暖的三大温室气体之一，其增温潜势大，且滞留在大气中的时间长，浓度以年均 $0 . 2 5 \%$ 的速率持续上升,严重影响着全球气候变化[]。农业活动是影响土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放的主要因素，农田作为其重要的排放源，每年所排放的 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 达到全球总量的 $6 0 \%$ 以上[2]。而旱作农田土地分布广泛,不仅在我国农业生产生活中占据极为重要的地位，更是大气中${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 的主要来源[3]。因此,研究旱地农田土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 的排放特征与如何减缓温室气体的排放显得极为重要。

黄王高原地处中国中部偏北部，干旱和水土流失是其独有的地域特征，春小麦作为黄土高原地区主要农作物之一，因其抗旱性强、生育期短、产量可观，已作为优良农作物成为黄土高原地区种植首选[3]。保护性耕作作为全球农业生产中的一项重要推荐措施，可显著减少黄土高原半干旱地区土壤水分蒸发，减缓地表径流量，涵养水土，改善土地质量，并对土壤温度、水分、结构及酶活性等理化性质造成影响,进而改变农田土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 的排放[4-5]。因此研究不同耕作措施下春小麦地土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放量、产量及二者间的平衡关系，对于黄土高原农田增产减排具有重要意义。张玉铭等研究指出，耕作措施是影响农田土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放的重要因素，与免耕相比，传统耕作更有利于土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 的排放，这主要是因为土壤翻动促进了郁闭于土壤内的 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 向大气中的释放。胡春胜等发现，秸秆还田可改善氮素的迁移过程与转化规律，有效降低氮素以气体形式的流失。Ball等"则认为，免耕措施下的土壤密度和土壤含水量相对较高，土壤中气体的扩散能力受到阻碍，并增强了土壤中的厌氧环境，使表层土壤中生物的反硝化作用进一步提高，从而促进土壤中 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 的排放。综上所述，不同耕作措施对土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放的影响因素在不同温度、水分、以及土壤条件下均有不同结果，大多研究已经证明保护性耕作对土壤养分及农作物产量的提升具有重大作用，但对影响旱地农田土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放的关键因素及其响应机制尚不清楚，需进一步研究。

为了更好揭示保护性耕作下土壤理化性质和环境因子对土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放的影响，本试验以黄土高原4种耕作措施下的春小麦地为研究对象，分析不同耕作措施下土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放特征和影响 ${ \bf N } _ { 2 } { \bf O }$ 排放的环境因子，探究二者之间的相关关系，揭示不同耕作措施下农田土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放的影响机制，从而提出增产减排的最优耕作方式，进而为区域粮食生产安全及生态环境建设提供理论依据。

研究区位于甘肃省定西市安定区安家坡村甘肃农业大学旱农综合试验站 $( 3 5 ^ { \circ } 6 4 ^ { \prime } \mathrm { N } , 1 0 4 ^ { \circ } 6 4 ^ { \prime } \mathrm { E } )$ ，属温带半干旱气候，平均海拔 $2 0 0 0 \mathrm { ~ m ~ }$ ，年均气温$6 . 4 ~ \mathrm { ^ { \circ } C }$ ，年均 $\mathrm { { } ^ { \geqslant 0 } \mathcal { C } }$ 年积温 $2 9 3 3 . 5 \mathrm { { ^ { \circ } C \cdot d } }$ ，年均 $\geqslant 1 0 \ \mathrm { { ‰ } }$ 年积温 $2 2 3 9 . 1 \ { ^ \circ } \mathrm { C } \cdot \mathrm { d }$ ，年平均降水量 $3 9 4 . 1 \ \mathrm { m m }$ ，主要集中在7一9月，年潜在蒸发量约为降水量的4倍，气候干燥，年平均气温 $6 . 4 ~ \mathrm { { ^ { 6 } C } }$ ,无霜期平均140d,属于典型的半干旱雨养农业区，试验地平坦无起伏，土壤多为当地典型的黄绵土，土层深厚，土质绵软且质地均匀,蓄水性能良好,土壤容重为 $1 . 1 9 \ \mathrm { g } \cdot \mathrm { c m } ^ { \textrm { - 3 } }$ ，土壤有机质 $1 2 . 0 1 \ \mathrm { g \cdot k g ^ { - 1 } }$ ,全氮 $0 . 7 6 ~ \mathrm { g } \cdot \mathrm { k g } ^ { - 1 }$ ，全磷1.77$\mathbf { g } \bullet \mathbf { k } \mathbf { g } ^ { - 1 }$ 。

# 1.4土壤样品的采集与处理

气体采集的同时，每块小区在靠近底座的周边位置随机选取3个点，采用直径 $5 \ \mathrm { c m }$ 土钻取0\~10cm土壤样品，剔除杂质，装入自封袋放置在装有冰袋的样品箱中低温运输，带回实验室后用于各项指标的测定，其中土壤含水量采用铝盒烘干法( $1 0 5 \pm$ $2 ^ { \circ } \mathrm { C }$ )测定，将空铝盒称重后放入土壤样品并置于$1 0 5 { \sim } 1 1 0 ^ { \circ } \mathrm { C }$ 烘箱中烘干至恒重，测得烘干土重量，再根据该值计算水分重量的百分比从而得到土壤含水量数据。土壤 $N \mathrm { O } _ { 3 } ^ { - } - \mathrm { N }$ 和 $\mathrm { N H } _ { 4 } ^ { + }$ -N采用氯化钾浸提比色法进行测定。将土样过 $2 \mathrm { m m }$ 筛后称取 $1 0 \mathrm { { g } }$ ，加入氯化钾溶液 $1 0 0 ~ \mathrm { { m L } }$ ,放于摇床震荡 $^ { \textrm { 1 h } }$ 后静置30$\operatorname* { m i n }$ 等待澄清，吸取 $2 0 ~ \mathrm { m L }$ 上清液,加 $0 . 5 \mathrm { g } \mathrm { M g } 0$ 于凯氏定氮仪中蒸馏 $3 ~ \mathrm { m i n }$ ，在定氮仪冷凝管出口加5$\mathrm { m L }$ 硼酸溶液用于对氨的吸收，用 $\mathrm { H } _ { 2 } \mathrm { S O } _ { 4 }$ 溶液进行滴定，测定土壤 $\mathrm { N H } _ { 4 } ^ { + }$ -N含量;蒸馏后溶液加 $1 ~ \mathrm { m L }$ 氨基磺酸，再加 $0 . 5 \mathrm { \ g }$ 代氏合金粉，放入凯氏定氮仪蒸馏$3 ~ \mathrm { m i n }$ ，在冷凝管出口加 $5 ~ \mathrm { m L }$ 硼酸溶液吸收，用$\mathrm { H } _ { 2 } \mathrm { S O } _ { 4 }$ 溶液滴定，测定土壤 $N \mathrm { O } _ { 3 } ^ { - } - \mathrm { N }$ 含量。土壤温度采用EM50进行长期监测[0]。小麦成熟后，每试验区随机选取9行小麦计算产量、株高、穗粒数及千粒重。

# 3.2土壤环境因子及 $\mathbf { N O } _ { 3 } ^ { - }$ -N、 $\mathbf { N H } _ { 4 } ^ { + }$ -N对 $\mathbf { N _ { 2 } O }$ 排放通量的影响

旱地农田土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 主要是土壤微生物参与硝化和反硝化作用产生的[22]，同时土壤温湿度、土壤结构、土壤 $\mathrm { \ p H }$ 等环境因子对土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放也具有一定影响，其中，土壤温度及含水量对旱作农田 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放的影响极为关键[23]。本研究发现，土壤温度与不同处理下全生育期春小麦地土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放量均呈现极显著正相关关系（ $\stackrel { \triangledown } { \cdot } \mathstrut < 0 . 0 1 ~ ,$ ，这是因为一方面温度对土壤微生物活性的影响极为明显，温度每升高$1 0 \ \mathrm { { ^ circ C } }$ ，反硝化细菌的活性则会提高1.5\~3.0倍[24；另一方面土壤温度升高，土壤中微生物的呼吸作用得到促进，导致土壤微域氧气缺失，为反硝化生物创造了厌氧条件，反硝化作用产生的 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 随之增加。土壤水分含量主要通过改变土壤透气性、土壤氧化还原状况及微生物活性进而影响 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放[25],土壤水分作为影响土壤微生物细胞代谢和养分运输的重要环境因子，对硝化和反硝化作用过程也起到了决定性影响[26-27]。本试验结果显示,土壤含水量与土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放量呈极显著负相关关系( $\stackrel { \triangledown } { \boldsymbol { P } } < 0 . 0 1 \$ ，这可能是因为随着土壤水分含量的增加，硝化过程受到抑制，土壤处于厌氧和强还原状态， ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 还原酶可将反硝化作用产生的 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 还原为 $\mathbf { N } _ { 2 }$ ${ \bf N } _ { 2 } { \bf O }$ 在土壤中的扩散也受到严重阻碍,因此排放量减少[28]。

${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 是厌氧条件下反硝化细菌还原 $\mathrm { N O } _ { 3 } ^ { - }$ 或 $\mathrm { N O } _ { 2 } ^ { - }$ 的产物之一，也是微生物通过硝化作用将 $\mathrm { N H } _ { 3 }$ 或$\mathrm { N H } _ { 4 } ^ { + }$ 氧化为 $\mathrm { N O } _ { \mathrm { x } } ^ { - }$ 的过程中的中间产物。试验中向土壤中施加的无机氮会为土壤提供 $\mathrm { N O } _ { 3 } ^ { - }$ 和 $\mathrm { N H } _ { 4 } ^ { + }$ ，以此对土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 的排放产生影响[29]。本试验结果显示，土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放量与土壤 $\mathrm { N H } _ { 4 } ^ { + }$ -N呈现极显著负相关关系，这是因为旱地土壤上表层土壤 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 的主要来源为土壤的硝化过程[30],当 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 大量排放时,土壤中的 $\mathrm { N H _ { 4 } ^ { + } - N }$ 水解，提供微生物硝化作用所需底物$\mathrm { N O } _ { 3 } ^ { - }$ ,这一过程之后造成土壤中 $N \mathrm { O } _ { 3 } ^ { - } - \mathrm { N }$ 的累积，$\mathrm { N H _ { 4 } ^ { + } - N }$ 的减少。许宏伟等3研究指出， ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放量与土壤 $\mathrm { N H _ { 4 } ^ { + } - N }$ 含量具有显著的相关性,而与 $\mathrm { N O } _ { 3 } ^ { - } -$ N含量无明显的相关关系。这可能是由于该地区土壤为碱性， ${ \bf N } _ { 2 } { \bf O }$ 还原酶Nos的活性较强而硝酸还原酶Nar以及亚硝酸还原酶Nir活性较弱，不利于反硝化作用生成 $\mathrm { N } _ { 2 } \mathrm { O } ^ { [ 3 2 ] }$ 。可见,因试验地区的环境因素、土壤 $\mathrm { p H }$ 耕作制度及无机氮的配施，土壤 ${ \mathrm { N } } _ { 2 } \mathrm { O }$ 的排放特征也有差异。

# 4结论

通过对黄土高原半干旱区不同耕作措施下农田土壤及影响 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放的关键因素进行分析，得出以下结论：

（1）不同耕作措施下 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放通量随生育期的推进变化大概一致，均在分蘖期达到排放峰，各处理下 $\mathrm { N } _ { 2 } \mathrm { O }$ 累积排放量大小顺序为： $\mathrm { T } > \mathrm { N T } > \mathrm { T S } >$ NTS,以经济效益和环境效益为基础，黄土高原半干旱区应以NTS（免耕 $^ +$ 秸秆覆盖)处理为最优耕作措施。

(2) ${ \bf N } _ { 2 } { \bf O }$ 排放量与环境因子及土壤无机氮的相关性分析表明，土壤温度，土壤含水量及 $\mathrm { N H } _ { 4 } ^ { + } - \mathrm { N }$ 含量对 ${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 排放通量的影响较大，并达到极显著相关水平，因此应着重考虑对上述因子的调控以达到${ \mathrm { N } } _ { 2 } { \mathrm { O } }$ 减排的目的。

# Effects of protective measures on $\mathbf { N } _ { 2 } \mathbf { O }$ emission from farmland soil in a semi-arid area of theLoess Plateau

DU Mengyin'， YUAN Jianyu'， LI Guang'， YAN Lijuan²， LIU Xingyu',QI Xiaoping'， PANG Ye'(1. College of Forestry,Gansu Agricultural University,Lanzhou 73oO70, Gansu, China;2. College of Agriculture,Gansu Agricultural University,Lanzhou 73oO7O,Gansu, China)

Abstract: Nitrous oxide $\mathrm { ( N } _ { 2 } \mathrm { O } )$ ）is an important greenhouse gas.It has atracted much attention as a factor in climate change,and farmland soil is the main source of emissions.Among many variables affecting $\mathrm { N } _ { 2 } \mathrm { O }$ emission from farmland, tillage is a key factor but exactly how it regulates $\mathrm { N } _ { 2 } \mathrm { O }$ emission from dryland farmland soil is unclear.In this study,traditional tilage(T)was used as a control,and three kindsof conservation tillge were used: no tillage (NT), traditional tillage $^ +$ straw mulch (TS), and no tillage $^ +$ straw mulch (NTS). The soil $\mathrm { N } _ { 2 } \mathrm { O }$ emission flux of a spring wheat field was measured by static box gas chromatography,and factors affecting ${ \bf N } _ { 2 } { \bf O }$ emission and spring wheat yield were measured concurrently. Soil $\mathrm { N } _ { 2 } \mathrm { O }$ cumulative emissions under different tillage measures in the spring wheat field decreased in the order $\mathrm { T } > \mathrm { N T } > \mathrm { T S } > \mathrm { N T S }$ ，Emissions under treatment T were significantly higher than those under other treatments (by $1 5 . 8 7 \%$ ， $2 8 . 0 8 \%$ ，and $3 9 . 5 8 \%$ , respectively, compared with NT,TS,and NTS).In addition,compared with treatment T,conservation tilage helpedto increase the surface soil content of $\mathrm { N H _ { 4 } ^ { + } \mathrm { ~ - N } }$ and the yield of spring wheat, and reduce the accumulation of $\mathrm { N O } _ { 3 } ^ { - } - \mathrm { N }$ Correlation analysis showed that soil temperature,soil water content, and $\mathrm { N H } _ { 4 } ^ { + } - \mathrm { N }$ content were associated with soil $\mathrm { N } _ { 2 } \mathrm { O }$ emission. Of these,soil water content and $\mathrm { N H _ { 4 } ^ { + } - N }$ content showed significant negative correlations with soil $\mathrm { N } _ { 2 } \mathrm { O }$ emission,and soil temperature showed a highly significant positive correlation with $\mathrm { N } _ { 2 } \mathrm { O }$ emission. Based on economic and environmental benefits and considering ${ \bf N } _ { 2 } { \bf O }$ emission and crop yield,we suggest that NTS is the best tillage strategy in semi-arid areas of the Loess Plateau.

Keywords: greenhouse gases； conservation tillage; farming practices;； spring wheat； correlation; Loess Plateau
