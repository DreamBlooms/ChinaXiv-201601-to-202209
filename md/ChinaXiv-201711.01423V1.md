3.1不同钙、磷水平饲粮对伊犁马产奶量及乳成分的影响

166 家畜产奶量及乳成分受品种、遗传、生理、饲养管理、饲粮和环境等因素的影响，在遗传、  
167 生理和环境因素相同的条件下，饲养管理或饲粮对产奶量、乳成分起着主导作用[10-11]。钙和磷  
168 广泛分布于骨骼、肌肉和牙齿中。钙与神经传导、细胞通透性、酶活性和血液凝固等生理过程  
169 有关；磷则在骨骼形成、生物能量和微生物消化等过程中起重要作用。家畜在分娩后通过饲粮、  
170 机体骨钙和肾小球对钙的重吸收来维持血钙和乳钙的恒定。当钙源不足时，会导致血钙降低而  
171 影响产奶量[2]。给处于泌乳高峰期的奶牛连续3d补充 $8 . 5 ~ \mathrm { g / d }$ 离子钙，产奶量可提高 $1 . 6 \mathrm { k g / d }$   
172 并明显延长奶牛产奶高峰期[12]。给产奶量高于平均水平 $10 5 \%$ 的奶牛在分娩后 $0 { \sim } 2 \mathrm { ~ h ~ }$ 和 $8 { \sim } 3 5 \mathrm { ~ h ~ }$   
173 内补饲2次钙丸，产奶量增加 $7 . 2 \% { \sim } 7 . 7 \% ^ { [ 2 ] }$ 。Lopez等[13]报道，给荷斯坦奶牛饲喂含磷为 $0 . 3 7 \%$   
174 或 $0 . 5 7 \%$ 饲粮并观察 $1 6 5 \mathrm { d }$ ，结果发现饲粮磷水平对产奶量、乳脂率及乳蛋白率等乳成分均无显  
175 著影响。赵恒聚等[14]也发现，给奶牛饲喂磷水平为 $0 . 3 2 \%$ 、 $0 . 4 4 \%$ 和 $0 . 5 6 \%$ 的饲粮对奶牛的产奶  
176 量无显著影响。这表明，额外添加钙有助于提高奶牛的产奶量，而磷的作用并不明显。饲粮钙、  
177 磷水平对马的产奶量及乳成分有何影响还未见报道。研究发现，母马在分娩前3天和分娩后 2  
178 天之间时，血液中PTH含量升高而总钙和离子钙含量下降，因此在马泌乳期间有必要提高钙的  
179 供应水平[7]。成熟体重为 $4 0 0 { \sim } 5 0 0 \mathrm { k g }$ 的母马处于第2泌乳月时，钙、磷的需要量分别为47.1\~58.9  
180 g/d 和 $3 0 . 5 { \sim } 3 8 . 1 \ \mathrm { g } / \mathrm { d } ^ { [ 7 ] }$ 。在本研究中，处于第2泌乳月的伊犁马钙、磷饲喂水平分别为45.0\~58.9  
181 g/d 和 $3 0 . 0 { \sim } 3 8 . 0 \ \mathrm { g } / \mathrm { d }$ ，结果发现饲粮钙、磷水平分别为51.96 和 $3 4 . 0 1 \ \mathrm { g } / \mathrm { d }$ 时产奶量最高，进一步  
182 增加饲粮钙、磷水平对产奶量并无益处。生理阶段是影响马产奶量的另一个重要因素。与奶牛  
183 泌乳期不同，马的泌乳期约为 $1 8 0 \mathrm { d }$ ；而新疆伊犁马马驹通常在5月龄时断奶，马驹断奶后母马  
184 的产奶量急剧下降并进入干奶期。姚新奎[15]的研究表明，随着泌乳期的延长，新吉马产奶量先  
185 高后低，产驹后 $3 0 \mathrm { d }$ 时产奶量最高，为 $1 9 . 1 3 \mathrm { k g / d }$ ，110d时为 $1 4 . 0 5 \mathrm { k g / d }$ ，其后急剧下降，150  
186 d时为 $1 0 . 5 1 \mathrm { k g / d }$ ；伊犁马在泌乳前 $1 1 0 \mathrm { d }$ 相对平稳，产驹后 $6 0 \mathrm { d }$ 时产奶量最高，为 $1 2 . 7 8 \mathrm { k g / d }$   
187 110d时为 $1 0 . 7 5 ~ \mathrm { k g / d }$ ，到140d时仅为 $7 . 5 1 ~ \mathrm { k g / d }$ ；而卢西塔诺（Lusitano）马在产后第31天达  
188 到泌乳高峰，为 $1 4 . 0 \mathrm { k g / d }$ ，之后开始逐渐下降，到180d时产奶量下降到 $7 . 5 7 ~ \mathrm { k g / d ^ { [ 1 1 ] } }$ 。在本研  
189 究中，伊犁马第3泌乳月末的估计产奶量为 $1 0 . 9 9 ~ \mathrm { k g / d }$ ，之后不断下降，到第5泌乳月末时为  
190 $8 . 5 4 ~ \mathrm { k g / d }$ 。由此可见，适宜的饲粮钙、磷水平有增加产奶量的作用，但随着泌乳期的延长，产  
191 奶量逐步下降；泌乳阶段不同，马的产奶性能也存在明显差异。

有关饲粮钙、磷水平对马乳成分影响的报道较少，崔政安[16]的研究表明不同钙、磷水平饲粮对泌乳水牛的乳成分影响不显著，这与本研究的结果相似。张巧娥等[17对产奶量与乳成分进行回归分析表明，产奶量与乳脂率、体细胞数和干物质含量呈显著的负相关。对马的研究则表明产奶量与乳脂率存在极强的正相关关系，与乳蛋白率无相关关系，与乳总固形物含量存在极强的负相关关系[15]。但在本研究中并未发现产奶量与乳成分间存在显著的相关关系，这可能与试验时间仅局限于第3至第5泌乳月有关。

# 3.2不同钙、磷水平饲粮对伊犁马乳脂脂肪酸组成的影响

饲粮中 $\mathrm { C a ^ { 2 + } }$ 对脂肪的消化和代谢具有重要影响。 $\mathrm { C a } ^ { 2 + }$ 可在消化道与脂类形成皂钙而影响脂类的吸收；同时 $\mathrm { C a } ^ { 2 + }$ 还可与胆酸结合，增加胆酸的排泄并减少胆固醇的形成[5]。在大鼠中的研究表明，饲粮中高水平的 $\mathrm { C a } ^ { 2 + }$ 可使体脂分解增加5.2倍[18]。究其原因是由于低 $\mathrm { C a } ^ { 2 + }$ 饲粮导致 $\mathrm { C a } ^ { 2 + }$ 代谢相关激素[PTH和 $1 { , } 2 5 { \mathrm { - } } ( \mathrm { O H } ) _ { 2 }$ -维生素D]合成增加，使 $\mathrm { C a } ^ { 2 + }$ 向细胞流入，导致脂肪细胞内 $\mathrm { C a } ^ { 2 + }$ 含量升高，提高脂肪酸合成酶的表达和活性，进而增加脂类贮存；而高 $\mathrm { C a } ^ { 2 + }$ 饲粮抑制细胞对 $\mathrm { C a } ^ { 2 + }$ 代谢相关激素的反应。增加膳食 $\mathrm { C a ^ { 2 + } }$ 水平会通过降低细胞内 $\mathrm { C a ^ { 2 + } }$ 含量，增加脂解并减少甘油三酯的累积[19]。有关饲粮钙水平对马乳脂率有何影响还未见报道。在本研究中，不同饲粮组间乳脂率并无显著变化，这可能与机体在短期内维持乳成分相对稳定的机制有关。从第3泌乳月到第5泌乳月，饲粮钙水平逐渐降低，而乳脂率则持续增加。这一现象可能与产奶量降低和低 $\mathrm { C a } ^ { 2 + }$ 引发脂质合成增加有关。饲粮钙、磷水平对乳脂脂肪酸组成有何影响还未见报道。在本研究中，不同钙、磷水平饲粮对乳脂中肉豆蔻脑酸、棕榈油酸、亚油酸、花生酸、顺-11-二十碳烯酸、 $a -$ 亚麻酸以及饱和脂肪酸、不饱和脂肪的含量均有显著影响。这是否是由于饲粮 $\mathrm { C a } ^ { 2 + }$ 改变机体脂质代谢的结果还不清楚，有待进一步研究。

# 3.3伊犁马泌乳期饲喂不同钙、磷水平饲粮对马驹生长发育和血浆生理生化指标的影响

体重和体尺是反映动物生长发育的重要指标，马驹一般在6月龄时断奶，从出生到6月龄是马驹一生中生长发育最快的阶段，体重和体尺占生后总生长量的 $5 0 \%$ 以上[9]。当给处于泌乳后期的伊犁母马饲喂粗蛋白质和消化能水平较高的饲粮后可明显提高马驹体长发育速度，母马产奶量增加可能是提高马驹生长发育的重要原因[20]。在本试验中，各组马驹补饲的精料相同，补饲量也相同，因此，马驹的生长发育与母马的产奶量有关。虽然各组间母马的产奶量存在显著差异，但各组马驹除胸围外，其他生长发育指标均无显著差异。这可能与样本量较小有关。研究发现，关中母驴在胎儿期和生后期生长发育都很快，其中胸围的生长强度比体高、管围大[21]。这表明，胸围的生长更易受摄入营养水平的影响。

血清中钙、磷代谢受PTH、CT 和 $1 { , } 2 5 { \mathrm { - } } ( \mathrm { O H } ) _ { 2 }$ -维生素D三者共同调节，以维持相对恒定的水平。当膳食中 $\mathrm { C a } ^ { 2 + }$ 水平降低时，机体增加PTH和 $1 { , } 2 5 { \mathrm { - } } ( \mathrm { O H } ) _ { 2 }$ -维生素D的释放而减少CT的释放[19]。在本研究中，各组马驹血浆中钙和磷含量虽然无显著差异，但饲喂饲粮II、II和IV的母马其乳中钙含量显著低于饲喂饲粮I的母马，致使马驹从母乳中获得的钙偏低，这可能是导致ⅡI、IⅢ和IV组马驹血浆中CT含量显著低于I组的原因。

4结论

增加饲粮钙、磷水平可提高伊犁马的产奶量，但使乳中钙含量降低；饲粮钙、磷水平不影响乳成分，但增加乳脂中饱和脂肪酸的含量，减少不饱和脂肪酸的含量；由于乳中钙含量降低，导致马驹胸围发育较慢，血浆CT含量下降。
