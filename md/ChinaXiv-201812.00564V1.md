# 呕吐毒素对IPEC-J2细胞凋亡的影响

廖美芳 2 孟英才\~ 詹济华 印遇龙² 廖鹏2\*李玲1\*（1．湖南中医药大学药学院，长沙 410208；2.中国科学院亚热带农业生态研究所，亚热带农业生态过程重点实验室，长沙 410125)

摘要：本研究以IPEC-J2 细胞为材料，研究不同浓度和时间处理下呕吐毒素（DON）对猪空肠上皮细胞凋亡的影响。通过CCK-8试剂盒检测细胞增殖毒性，筛选出合适的DON 作用时间 $( 6 、 1 2 、 2 4 \mathrm { h } )$ 。通过低浓度 $\mathrm { \Omega } 2 0 0 \mathrm { \ n g / m L }$ )和高浓度 $\scriptstyle \left( 2 0 0 0 \mathrm { n g / m L } \right)$ )DON 分别作用 IPEC-J2细胞6、12、 $2 4 \mathrm { h }$ ，检测 DON 对细胞有丝分裂周期和细胞早期凋亡的影响。结果表明：1)DON 对 IPEC-J2 细胞生长有明显抑制作用，200 和 $2 0 0 0 { \mathrm { n g / m L } }$ DON 浓度下，48 和 $7 2 \mathrm { { h } }$ 的细胞存活率显著低于 $2 4 \ : \mathrm { h } ( P { < } 0 . 0 5 )$ ； $2 0 0 0 { \mathrm { n g / m L } }$ DON 浓度下， $7 2 \mathrm { { h } }$ 的细胞存活率显著低于4 $1 8 \mathrm { h } ( P { < } 0 . 0 5 )$ 。2）不同DON 浓度对细胞周期影响不同，低浓度DON主要作用于细胞有丝分裂 S期，干扰DNA 的复制；而高浓度DON主要作用于细胞有丝分裂G2/M期，干扰有关酶与纺锤丝蛋白质的合成。3）DON作用 $6 \mathrm { { h } }$ ，低浓度DON 组和高浓度DON 组早期凋亡率和总凋亡率显著高于对照组 $( P { < } 0 . 0 5 )$ ；DON 作用 $1 2 \mathrm { h }$ ，高浓度DON 组早期凋亡率和总调亡率显著高于对照组和低浓度DON 组 $( P { < } 0 . 0 5 )$ 。由此可见，DON 抑制了IPEC-J2 细胞增殖，低浓度DON 使细胞有丝分裂停留在 S期，高浓度DON 使细胞有丝分裂停留在G2/M期。DON 促使细胞早期凋亡，使总凋亡率上升，且促凋亡作用随DON浓度升高而增强。

关键词：呕吐毒素；IPEC-J2细胞；周期；调亡；增殖 中图分类号：S811.2

脱氧雪腐镰刀菌烯醇（deoxynivalenol，DON)，属于单端孢霉烯族类化合物，主要由禾谷镰刀菌、尖孢镰刀菌、雪腐镰刀菌等镰刀菌产生。另外，头孢菌属、木霉属等都可产生该毒素。由于它可以引起猪的呕吐，故又名呕吐毒素[1-2]。该毒素最初是1970 年在日本香川县的霉病大麦病毒中发现，1972年，日本的学者首次从赤霉病大麦中分离得到并阐明了这种新的真菌毒素的化学结构，将其命名为DON[3]。当人食用了被DON 污染的食物后，会导致厌食、呕吐、发烧、腹泻、站立不稳、反应迟钝等急性中毒症状，严重时影响造血系统甚至死亡[4]。中国传统饮食习惯中粮谷比例远远高于西方，使得 DON 的危害更为突出。1998年，在国际癌症研究机构公布的评价报告中，DON被列为3类致癌物。欧盟要求饲料中DON浓度要小于 $0 . 7 5 \mathrm { m g / k g }$ ，中国饲料DON 浓度限量值为 $1 ~ \mathrm { m g / k g ^ { [ 5 - 6 ] } }$ 。

猪摄入DON以后，对脑神经产生麻痹作用。而且，即使是低浓度的DON也能引起猪脑中 5-羟吲哚乙酸含量的提高[7]。不同浓度 DON 对大鼠红细胞的溶血作用不同，其溶血作用有一个阈值，低于此浓度则红细胞不会发生溶血反应[8]。DON可能通过3种不同的方式对原核细胞产生毒性作用：1）通过渗透磷脂双层，作用于亚细胞水平；2）通过与细胞膜相互作用；3）通过自由基介导的脂质过氧化作用。DON可能以1种或1种以上的方式同时发挥毒性作用[7]。研究发现，DON 可作用于骨髓造血细胞而产生细胞毒性。其免疫毒性作用依赖于DON 的浓度、淋巴细胞亚型、组织来源和糖皮质激素[9]。另外，DON 可以抑制胸腺细胞的蛋白质合成。多数研究都表明DON具有胚胎毒性和致畸作用。动物试验表明，长期小剂量饲喂饲含 DON 的饲料，可以诱发不同器官的肿瘤[10]。经小鼠口服给药5和 $2 5 \mathrm { m g / k g }$ DON，结果显示细胞炎症因子白细胞介素-6（IL-6)、白细胞介素- $\cdot 1 \beta$ （IL-1β）和肿瘤坏死因子 $\mathbf { \sigma } \cdot \mathbf { a }$ （TNF- $\alpha$ ）的 mRNA的表达量升高，高浓度DON 可抑制体外培养的人外周淋巴细胞的增殖及免疫球蛋白的产生，并且该抑制作用有剂量依赖性；而低浓度则使免疫球蛋白含量升高[11-12]。

由于猪肠上皮细胞在免疫系统中起重要作用，通过不同的机制参与肠的先天免疫反应，例如屏障功能、抗菌肽合成、黏液分泌和参与细胞因子趋化。而目前关于肠细胞功能的大多数知识来自人类癌症或从经典动物模型产生的细胞培养物进行的研究。但是，物种在响应特异性环境时选择不同的细胞因子，所以在猪活体上需要谨慎的运用基于小鼠和人的免疫功能模型[13]。因此，基于分子水平的基础研究至关重要。

本文选取猪空肠上皮细胞系IPEC-J2 细胞，通过CellCountingKit-8(CCK-8试剂盒)检测DON 对细胞的增殖毒性，筛选合适的DON 浓度和作用时间，进而检测DON 对细胞周期和凋亡的影响。通过这些指标来观测DON对IPEC-J2细胞凋亡的影响。

# 1.4 试验方法

# 1.4.1 DON配制

将原装DON 粉末溶解在二甲基亚矾(DMSO)中，配成 $1 \mathrm { m g / m L }$ 母液，用 $0 . 2 2 \mu \mathrm { m }$ 滤膜过滤。再分别用含有 $10 \%$ FBS 和 $1 \%$ 青-链霉素的完全培养基配成浓度分别为200 和2000$\mathrm { n g / m L }$ 的 DON溶液，整个过程在超净台完成。

# 1.4.2 细胞培养

IPEC-J2细胞培养液为含体积分数 $10 \%$ FBS及 $1 \%$ 双抗的DMEM培养基，接种量为每 毫升 $3 . 0 { \times } 1 0 ^ { 6 }$ 个细胞。细胞接种于 $1 0 \mathrm { c m }$ 细胞培养皿，再转移至温度为 $3 7 \mathrm { ~ \textdegree ~ }$ 、体积分数为 $5 \%$ 的 $\mathrm { C O } _ { 2 }$ 细胞培养箱内培养。

# 1.4.3 CCK-8试剂盒试验接种和DON处理

取对数生长期的IPEC-J2 细胞，用 $2 . 5 \ : \mathrm { g / L }$ 胰蛋白酶消化，接种于3块96孔板，分别标明24、48、 $7 2 \mathrm { { h } }$ ，每孔约 $1 \times 1 0 ^ { 4 }$ 个细胞，每板A、B、C、D4个组，每个组2列10个重复，待细胞生长到合适密度，更换加药物处理的培养液，具体分组见表1。第1列为清零组，加$1 0 0 ~ \mu \mathrm { L }$ 培养液，其他列加入不同药物处理的等量 $1 0 0 \mu \mathrm { L }$ 培养液。

1.5 DON对IPCE-J2细胞周期影响

1.5.1 细胞培养和DON处理

IPEC-J2细胞接种于 $1 0 \mathrm { c m }$ 细胞培养皿，置于温度为 $3 7 \ \mathrm { { ^ \circ C } }$ 、体积分数 $5 \%$ 的 $\mathrm { C O } _ { 2 }$ 细胞培养箱内培养。待6、12、 $2 4 \mathrm { h }$ 各组细胞长到合适浓度，更换含有0、200、2000ng/mLDON的培养液。

# 1.5.2 细胞收集和检测

配制 $70 \%$ 乙醇，放置在 ${ \bf - } 2 0 ~ \mathrm { { ^ \circ C } }$ 冰箱预冷备用，准备一盒冰备用， $2 . 5 \ : \mathrm { g / L }$ 胰蛋白酶消化细胞，收集在 $1 5 ~ \mathrm { m L }$ 离心管中， $ { 1 } ~ 0 0 0 ~  { \mathrm { r / m i n } }$ 、 $2 0 \ \mathrm { { ^ \circ C } }$ 离心 $5 \mathrm { m i n }$ ，收集细胞，去上清培养基；15mL离心管中加入 $3 { \sim } 5 ~ \mathrm { m L }$ PBS，枪头轻轻吹打混匀细胞， $ { 1 } ~ 0 0 0 ~  { \mathrm { r / m i n } }$ 、 $2 0 \ \mathrm { ^ \circ C }$ 离心 $5 \mathrm { m i n }$ ，去上清，留 $2 0 0 ~ \mu \mathrm { L }$ 左右的PBS上清液，枪头轻轻吹打混匀。吸取 ${ 8 0 0 \mu \mathrm { L } }$ 预冷的 $70 \%$ 乙醇到$1 . 5 ~ \mathrm { m L }$ 离心管中，将 $2 0 0 ~ \mu \mathrm { L }$ 混匀的细胞悬液一滴滴地加入至 $70 \%$ 酒精中固定，细胞周期样品可在 ${ } - 2 0 \ { } ^ { \circ } { \mathrm { C } }$ 冰箱中保存 $3 { \sim } 4 \mathrm { d }$ 。

取出固定的样品， $8 0 0 ~ \mathrm { r / m i n }$ 离心 $5 \mathrm { m i n }$ ,弃上清。加入 $1 \mathrm { m L }$ 预冷PBS 重悬细胞， $8 0 0 ~ \mathrm { r / m i n }$ 离心 $5 \mathrm { m i n }$ ，收细胞。可重复1\~2次，以除去乙醇。加入 $1 5 0 ~ \mu \mathrm { L }$ PI工作液， $4 \mathrm { { ^ \circ C } }$ 避光染色$3 0 \mathrm { m i n }$ 。转至流式检测管，上机检测，PI用 $4 8 8 ~ \mathrm { n m }$ 氩离子激光器激发，由630带通滤光片接收，通过FSC/SSC 散点图收集10000 个细胞，采用设门技术排除粘连细胞和碎片，分析PI荧光直方图上细胞各周期的百分率。

3讨论

研究表明，DON 对原核细胞、真核细胞均具有明显的毒性作用，特别是对于生长较快的细胞如胃肠道黏膜细胞、淋巴细胞、胸腺细胞、脾细胞、骨髓造血细胞等均有损伤作用，并且抑制蛋白质合成。Alm 等[14]、Tiemann 等[15]研究表明，DON 能影响细胞周期的分布，抑制细胞进入S期，使细胞阻滞在 $\mathrm { G } _ { 0 } , \mathrm { G } _ { 1 }$ 期，具有显著的抗增殖作用。本试验结果表明，DON 具有明显的抑制细胞生长的作用，在 $2 4 \mathrm { h }$ 内作用显著，超过 $4 8 \mathrm { h }$ 后低浓度( $\mathrm { \Omega } 2 0 0 \mathrm { n g / m L }$ DON作用趋势平稳，高浓度（ $\phantom { + } 2 0 0 0 \mathrm { n g / m L }$ ）DON 持续抑制；IPEC-J2 细胞存活率随着DON浓度增高而极显著下降。低浓度DON主要作用于细胞有丝分裂S期，干扰DNA的复制;高浓度DON主要作用于细胞有丝分裂G2/M期，干扰有关酶与纺锤丝蛋白质的合成。此结果与前人报道[16]相似。

细胞凋亡指为维持内环境稳定，由基因控制的细胞自主的有序的死亡。凋亡分为3个步骤：凋亡起始、凋亡小体形成、凋亡小体被邻近的细胞或体内吞噬细胞所吞噬；机理为接受调亡信号 $$ 凋亡调控分子间的相互作用 $$ 蛋白水解酶的活化 $$ 进入连续反应过程。调亡细胞的残余物质被消化后重新利用。本试验的凋亡试验结果显示，IPEC-J2 细胞在DON 作用6h后,200和 $2 0 0 0 { \mathrm { n g / m L } }$ DON 组的早期凋亡率和总凋亡率显著升高; $1 2 \mathrm { h }$ 后， $2 0 0 0 { \mathrm { n g / m L } }$ DON组的早期凋亡率和总凋亡率显著升高； $2 4 \mathrm { h }$ 后作用不再显著。相同浓度DON处理不同时间，低浓度DON作用12和 $4 2 \mathrm { h }$ 后早期凋亡率和总凋亡率显著降低；高浓度DON作用 $2 4 \mathrm { h }$ 后早期凋亡率和总凋亡率显著降低。与低浓度不同的是，高浓度DON 促进凋亡作用持续到12h，这与启动了不同的凋亡机制有关[17]。

4结论

$\textcircled{1}$ DON对IPEC-J2 细胞生长有明显抑制作用，其抑制作用效果随着DON 浓度增高而  
增强，其中抑制细胞增殖主要作用在 $2 4 \mathrm { h }$ 。$\textcircled{2}$ 不同DON浓度对细胞周期影响不同，低浓度DON主要作用于细胞有丝分裂S期，  
干扰DNA的复制；高浓度DON主要作用于细胞有丝分裂G2/M期，干扰有关酶与纺锤丝  
蛋白质的合成。$\textcircled{3}$

DON 促使细胞早期凋亡，使总凋亡率上升，且促凋亡作用随DON 浓度升高而增强。

An appropriate exposure time (6,12,and 24 hour) used CCK-8 assay to test for cell proliferation. The effects of DON on cell mitosis cycle and early cell apoptosis were examined by treated the IPEC-J2 cells with low concentration ( $2 0 0 ~ \mathrm { n g / m L } )$ ）and high concentrations( $\scriptstyle 2 \ 0 0 0 \ \mathrm { n g / m L }$ DON for6,12,24 hours.The results showed as follows: 1) DON significantly inhibited the growth of IPEC-J2 cells. DON also can promote cell apoptosis and increase total apoptosis rate, and this effect is enhanced with DON concentration increasing.
