# 咸水滴灌对肉从蓉寄生体系盐离子分配及品质的影响

刘玲²，罗剑洪³，李丙文'，徐新文'，范敬龙'，马志国4(1.中国科学院新疆生态与地理研究所,新疆 乌鲁木齐830011；2.中国科学院大学,北京101408；3.新疆生产建设兵团第二师34团农业技术发展中心,新疆 尉犁841506；4.暨南大学药学院,广东 广州510632)

摘要：为探究咸水滴灌对人工种植肉从蓉寄生体系盐离子分配和肉从蓉品质的影响,在塔里木盆地塔克拉玛干沙漠边缘,选择4个灌水矿化度差异较大的样地,采集土壤、梭梭和肉从蓉样品,分析土壤、寄生体系各部分盐离子含量及肉从蓉主要药用成分含量，结果表明：(1）在高矿化度咸水滴灌下,接种肉从蓉的梭梭根部、主茎和同化枝盐离子含量为：同化枝>根>茎,肉从蓉体内盐离子含量与寄主根系含量相近；(2）肉从蓉体内总盐、Na+和CI含量均随灌水矿化度的增高而增加；肉从蓉体内 $\mathrm { K ^ { + } }$ 占总盐分的1/3，具有较强的吸收富集K的能力，但 $\mathrm { K ^ { + } }$ 含量并未随灌水盐浓度的变化而表现出显著性差异；（3）肉从蓉体内主要药用成分苯乙醇苷类物质含量随灌溉水矿化度的升高而增加,盐分胁迫有助于刺激肉从蓉体内主要药用成分的积累。因此，较高矿化度咸水滴灌可以提高肉从蓉的品质，但需要注意长期高盐胁迫对寄主梭梭的影响，因而制定合理的灌溉制度是不可或缺的。

关键词：咸水滴灌；肉从蓉；盐分离子；药用品质

肉从蓉(CistanchedeserticolaMa)为列当科（Oro-banchaceae)多年生根寄生草本植物，主要分布于我国西北干旱区，是传统名贵中药材1。伴随着肉从蓉市场需求量的不断上升，野生资源濒临枯竭，人工种植肉从蓉已经成为市场的主要来源。当前，干旱区生态用水与农业用水矛盾十分突出3，开发利用地下咸水资源发展肉从蓉产业日益普遍[4]。然而，较高矿化度的咸水灌溉会对植物产生离子毒害，抑制作物生长,致使作物减产,甚至死亡[5-8]。梭梭作为肉灰蓉的寄主，是典型的真盐生植物，其耐盐能力较强,有学者研究发现，梭梭在 $5 0 \ \mathrm { g \cdot L ^ { - 1 } }$ 矿化度咸水灌溉时还可以继续生长[0],在 $2 8 ~ \mathrm { g } \bullet \mathrm { L } ^ { - 1 }$ 咸水灌溉时不仅能够正常生长，还可以接种肉灰蓉[4]。而肉从蓉作为全寄生植物，所需的营养成分全部来源于寄主，通过吸器将两者紧密相连，肉从蓉可以被看作寄主梭梭的一个器官，两者之间通过源库关系构成统一整体，形成梭梭-肉灰蓉寄生体系，进行着物质与信息的交流[11-13]。有关梭梭耐盐性的研究很多[9-10.14],但从两者共同构成的体系出发,针对梭梭-肉从蓉寄生体系的研究较少，尤其是在高矿化度咸水灌溉条件下，高盐环境对肉从蓉体内盐分含量及药用品质的影响尚不清楚。本文在目前肉从蓉主要种植地的新疆南部地区，对不同矿化度咸水滴灌种植肉从蓉的梭梭-肉灰蓉寄生体系盐分离子和肉从蓉主要药用成分进行实验分析和研究，并针对上述问题展开探讨，其结果对开发利用咸水资源发展肉从蓉种植产业具有重要意义。

# 3讨论

# 3.1咸水滴灌对寄生体系盐离子分布的影响

咸水滴灌会引起土壤含盐量增加，随着灌溉水矿化度的升高,土壤含盐量也呈现上升趋势[17]。盐分胁迫会对植物生长带来不利影响，反之，植物在长期对环境的适应过程中，也产生了对盐分胁迫的忍耐机制。真盐生植物进化出了独特的保护机制，如稀盐盐生植物，能将盐分区隔化到细胞的液泡中或储存器官中,以避免盐离子对生长器官的危害[18-19]。笔者的研究显示，在寄主梭梭各器官中，盐离子含量在同化枝、主根、细根和主茎中依次降低，这与崔旭盛等2对梭梭不同器官中K和Na含量的分布结果是一致的。同化枝中盐分含量最高，这是由于梭梭肉质化的同化枝富含发达的液泡，因而梭梭能有效的将Na+区隔化在液泡中以保持细胞的渗透势[21-22]；根系是梭梭吸收营养元素和矿物质成分的主要器官，其中盐分含量较低，总盐含量仅为同化枝的1/4,表明梭梭对根系生理功能具有一定的保护作用；茎是输导组织，而非储存器官，这就是茎中盐分积累少、含量最低的主要原因[23]。由此可知，这可能是梭梭在高盐环境中生长的优势所在，与前人研究结果相似[24]。本研究结果进一步显示，在梭梭-肉灰蓉寄生体系中，肉从蓉体内盐分离子含量(包括总盐、 $\mathrm { N a } ^ { + }$ 和CI)与梭梭根系的含量相当，并远低于同化枝，这说明进入肉从蓉体内的盐分含量并不高，其机制还需要进一步研究。

通常使用 $\mathrm { K ^ { + } / N a ^ { + } }$ 值作为衡量植物耐盐能力的标准之一，适量的 $\mathrm { N a } ^ { + }$ 可以有效促进旱生植物梭梭的生长和抗旱性,当 $\mathrm { N a } ^ { + }$ 的施用量超过 $0 . 6 \mathrm { m g \cdot g ^ { - 1 } }$ 会对梭梭起到抑制作用[25]。而本研究中灌溉水中 $\mathrm { \Delta N a } ^ { + }$ 含量分别高达 $1 4 . 0 \ \mathrm { m g \cdot g ^ { - 1 } }$ 和 $3 . 8 ~ \mathrm { m g \cdot g ^ { - 1 } }$ ,均高于该阈值， $\mathrm { ~ I ~ } _ { 3 4 }$ 梭梭不同器官总盐和 $\mathrm { N a } ^ { + }$ 含量均高于 $\mathbb { I } _ { \ 3 4 }$ ，而同化枝中 $\mathrm { K } ^ { + }$ 含量则相反，这可能是由于梭梭通过积累大量的 $\mathrm { N a } ^ { + }$ 并将其区隔化至液泡中来适应环境胁迫，而不是 $\mathrm { K } ^ { + [ 2 6 ] }$ ,有研究已表明， $\mathrm { \Delta N a } ^ { + }$ 是梭梭适应干旱环境的重要调节物质[27]。在寄生体系内，梭梭细根中 $\mathrm { K } ^ { + }$ 含量以及 $\mathrm { K ^ { + } / N a ^ { + } }$ 值都维持在较高水平，而肉从蓉体内 $\mathrm { K } ^ { + }$ 含量和 $\mathrm { K ^ { + } / N a ^ { + } }$ 值均高于细根的对应值，这说明肉众蓉在体内富集大量 $\mathrm { K ^ { + } }$ ，而限制 $\mathrm { { N a } ^ { + } }$ 的进入，这与崔旭盛等28在肉从蓉生殖生长阶段的研究结果一致，表明肉从蓉能通过对离子的选择性吸收而形成自我保护，以抵抗高盐胁迫。

# 3.2咸水滴灌对肉众蓉体内盐分含量的影响

梭梭根系是肉从蓉与寄主的连结点，并通过库源关系构成统一整体，梭梭作为"源"不断向肉从蓉“库”中转移生长所需的物质成分，因而寄主的生理状态对肉从蓉的生长发育有直接影响，本研究结果显示，灌水矿化度较高的采样地梭梭根系中的盐分含量较高，肉从蓉体内总盐、 $\mathrm { \Delta N a ^ { + } }$ 和CI含量也随之增加，表明肉从蓉随寄主生长环境的变化而变化。

与总盐、 $\mathrm { . N a ^ { + } }$ 和CI含量的变化趋势不同，肉从蓉体内 $\mathrm { K } ^ { + }$ 含量在采样点之间并没有显著差异，即并未随土壤盐离子含量的升高而变化。有研究表明，不同产地的管花肉灰蓉、不同花色肉灰蓉种质以及不同生育阶段的肉从蓉对 $\mathrm { K } ^ { + }$ 都具有较强的吸收能力[28,30-31],肉从蓉对 $\mathrm { K } ^ { + }$ 具有显著的富集作用,本研究支持这个结论。

# 3.3咸水滴灌对肉从蓉主要药用有效成分的影响

肉从蓉作为名贵中药，含苯乙醇苷类、环烯醚萜类、木质素类和多糖等多种有效成分[32],而苯乙醇苷类含量是肉从蓉质量评价的主要标准之—[33-34]。这些苷类、萜类物质是植物的次生代谢物质，是植物长期适应环境胁迫的产物[35]。干旱胁迫对管花肉从蓉悬浮细胞中松果菊苷和毛蕊花糖苷的积累有明显促进作用[36],本研究结果与之相似，随着灌溉水矿化度和土壤盐分含量的升高，不同土壤立地类型接种的肉从蓉体内苯乙醇苷类物质显著增加。显然，盐分胁迫会促进肉从蓉次生代谢物质的产生，有利于肉从蓉有效药用成分的积累

在长期的高盐胁迫下，寄主梭梭的生长会受到抑制，因此，在开发利用咸水资源种植肉从蓉的生产实践过程中，如何既确保寄主正常生长和生物量的有效积累，又能尽可能提高接种肉灰蓉的品质，需要控制好两者之间的平衡，确定适宜的咸水灌溉阈值，制定合理的灌溉制度是关键所在。

# 4结论

选择灌水矿化度差异较大的4个样地，通过对土壤、寄生体系各部分盐离子含量及肉从蓉主要药用成分含量的分析，得出以下结论：

（1）不同高矿化度咸水滴灌下，在梭梭-肉从蓉

寄生体系中，肉从蓉体内的盐离子含量与寄主梭梭根系相似，并都远低于同化枝；且肉从蓉通过对离子的选择性吸收形成自我保护，以抵抗高盐胁迫。(2）在高矿化度咸水滴灌条件下，肉从蓉体内总盐、 $\mathrm { . N a ^ { + } }$ 和CI含量均随灌水矿化度的增高而增加，其中肉从蓉具有较强的吸收富集 $\mathrm { K ^ { + } }$ 的能力，但 $\mathrm { K ^ { + } }$ 含量并未随灌水盐浓度的变化而表现出显著性差异。（3）肉从蓉体内苯乙醇苷类为代表的次生代谢产物的含量随灌溉水矿化度的增加而升高，但考虑到对寄主梭梭的影响，选取适宜的咸水灌溉阈值和制定合理的灌水制度是实现肉从蓉产业长期发展的有效措施。

# Effect of saline water drip irrigation on salt ions distribution for the Haloxylon ammoedndron-Cistanche deserticola association (H-C association) and quality of Cistanche deserticola Ma

LIU Lingl²， LUO Jianhong²，LI Bingwen'， XU Xinwen’， FAN Jinglong'， MA Zhiguo4 (1. Xingjiang Institute ofEcology and Geography,Chinese Academy of Sciences,Urumqi 830011,Xinjiang, China; 2. University of Chinese Academyof Sciences,Beijing 101408,China;3.Agricultural Technology Development Center, the $3 4 ^ { \mathrm { t h } }$ Branch of the Second Division of the Xinjiang Production and Construction Corps,Yuli 841506, Xinjiang, China; 4. College of Pharmacy Jinan University,Guangzhou 51O632,Guangdong, China)

Abstract: The impact of saltwater drip irigation onsalt ion distribution and the quality of Cistanche cistanche Maand Haloxylon ammodendron-Cistanche deserticola Ma (H-C)association during artificial cultivation in the planting base of $C$ .deserticola on the Taklimakan Desert edge was examined.We selected four sample plots with obvious differences in irrigation mineralization to collect samples of $H .$ ammodendron and $C$ ，deserticola.We also collected and analyzed the salt ioncontent in soil,H-C association,and main medicinal components of $C$ deserticola from different locations.Results showed the following: (1) The salt content of roots,main stems,and assimilated branches of $C .$ .deserticola inoculated by drip irrigation with high salinity brackish water was in the order of assimilating branch>coarse roots>fine roots>stems; the salt content in $C$ 、deserticolais was similar to that of the root system.(2) The contents of total salt, $\mathrm { { N a } ^ { + } }$ ，and $\mathrm { C l } ^ { - }$ in $C$ 、deserticola increased with an increase of irrigation salinity and soil salinity; the proportion of $\mathrm { K } ^ { + }$ in $C .$ ，deserticola accounts for one-third of the total salt and can accumulate $\mathrm { K } ^ { + }$ ,but there is no significant difference in $\mathrm { K } ^ { + }$ content with irrigation salt concentration. (3) The synthesis of phenylethanol glycosides,which are the main medicinal ingredient in $C$ ，deserticola,increased with an increase of irigation salinity,and salt stress promotes the accumulation of a main pharmacological component in $C .$ deserticola. In consequence， high salinity saline water drip irrigation can improve the quality of C. deserticola.And it is necessary to pay attention to the efect of long-term high salinity stresson the host $H _ { \mathbf { \Gamma } }$ ammodendron, so it is important to develop a reasonable irrigation system.

Keywords: drip irrigation with saline water; Cistanche deserticola Ma; salt ions; officinal quality
