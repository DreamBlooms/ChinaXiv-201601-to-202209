# 1 INTRODUCTION

In the last couple of decades,more and more researchers have been paid considerable attention to the construction of various coordination compounds not only due to their multiple applications as functional materials in many fields such as luminescence,chemical and biological sensor, separation，magnetism, catalysis，gas adsorption, ion exchange， chirality, $e t c ^ { [ 1 - 9 ] }$ ， but also for their intriguing architectures and fascinating topologies[10l. Among them, zero-dimensional $( 0 D )$ multinuclear complexes, one-dimensional $( 1 D )$ （20 helical and zigzag chains, two-dimensional $( 2 D )$ grids， three-dimensional $( 3 D )$ porous structures，and interpenetrating networks have been presented[1l. Although a variety of metal coordination frameworks with beautiful topologies and interesting properties have been synthesized,it is a great challenge to control structures with desired properties,because many factors afect the result,such as the selection of solvent system,reaction temperature,time，pH,organic ligand，metal ion,the ratio between metal and ligand, counterion and so on[2-15]. The effective and facile approach to overcome this problem is the appropriate choiceof well-designed organic bridging ligands containing modifiable backbones and connectivity information, together with the metal centers with various coordination preferences.

Our group is interested in the ternary compounds constructed from the rigid N-donor ligands (polypyridine or pyrazine)，O-donorligands(polycarboxylate）and $d ^ { 1 0 }$ transitionalmetal ions. 2,6-Bis(pyrazin-2-yl)pyridine-4-carboxylic acid (Hbppc, Chart 1),as a rigid planar ligand, has not been reported in the realm of chemistry. It possesses several interesting structure characterizations.Firstly，it contains five nitrogen and two oxygen atoms from one pyridyl, two pyrazinyl and one carboxyl groups,thus it can function as either chelating or bridging ligand. Secondly, the carboxyl and N-heterocyclic groups can act not only as hydrogen-bond acceptors but also as hydrogen-bond donors,depending upon the deprotonation and protonation, which is beneficial for the construction of supramolecular architectures.In addition,the extended conjugated system can easily form $\pi { - } \pi$ interactions which have greatly influence the constructing supramolecular framework.Therefore，Hbppc asa good candidate has atracted our interest for the construction of metal-organic supramolecular architectures.

In addition to coordination bonding,other noncovalent interactions,such as hydrogen bonds[16l and $\pi ^ { \cdot \cdot \cdot } \pi$ interactions[17],also greatly affect the structures of coordination polymers,and they may link multinuclear discrete subunits or low-dimensional entities into high-dimensional supramolecular networks to make the whole structure stable.In this work,by using Hbppc ligand and changing the pH and/or using auxiliary ligands, aseries of coordination polymers $\mathrm { \small { [ Z n ( b p p c ) _ { 2 } ( H _ { 2 } O ) _ { 2 } ] \cdot 3 H _ { 2 } O } }$ (1)，[Zn2(bppc)2(m-bdc)(HO)4]·7HO (2), [Zn(bppc)(H2O)2]n·n(m-Hbdc). $\mathrm { n ( H } _ { 2 } \mathrm { O ) }$ (3）and $\mathrm { \small { [ Z n _ { 2 } ( b p p c ) ( b t c ) ( H _ { 2 } O ) _ { 3 } ] _ { n } } } \cdot \mathrm { { 5 n H _ { 2 } O } }$ (4)，were prepared under hydrothermal conditions. Moreover, their luminescent property and thermal stability were also investigated.

The experiments and structural research remind us to study the effects of reaction conditions on the final structures. Compounds 1 and 2 were synthesized under the same reaction conditions,except for the O-donor assstant ligands of the reactions. The $\mathbf { H } _ { 2 } ( \mathbf { m } { - } \mathbf { b } \mathrm { d c } )$ as a second ligand of 2 is completely deprotonated to coordinate with $Z \mathrm { n } ^ { \mathrm { I I } }$ which makes the structure of 2 different from that of 1. Complexes 2 and 3 were synthesized under the same reaction conditions, except for the $\mathsf { p H }$ values of the reactions ( $\mathrm { \Delta p H } = 5 . 5$ for 2 and 6.5 for 3). It was found that the influence of $\mathrm { p H }$ value on the structures is in fact based on the protonated extend of Hbppc ligand. For compound 2, when $\mathrm { p H } = 5 . 5$ , the Hbppc ligand is not protonated and the $( { \mathrm { m } } { \mathrm { - } } { \mathrm { b d c } } ) ^ { 2 - }$ coordinates with $Z \mathbf { n } ^ { \mathrm { I I } }$ as an O-donor assistant ligand; but for compound 3, when $\mathrm { p H } = 6 . 5$ the Hbppc ligand is deprotonated to coordinate with $Z \mathbf { n } ^ { \mathrm { I I } }$ and fulfill the coordination sites of $Z \mathrm { n } ^ { \mathrm { I I } }$ The $[ \mathrm { H } ( \mathrm { m } { \cdot } \mathrm { b d c } ) ] ^ { \cdot }$ exists as a guest molecule to balance the charge and stabilize the skeleton of the compound. Compounds 3 and 4 were also isolated under the same reaction conditions except using different auxiliary ligands ( $\mathbf { \mathrm { H } } _ { 2 } ( \mathbf { m } { \cdot } \mathbf { b } \mathbf { \mathrm { d } } \mathbf { c } )$ for 3 and Hbtc for 4),resulting in 3 possessing a $1 D$ ladder-chain structure and 4 showing a $2 D$ layer structure.

# 3.6Luminescent properties

Organic-inorganic coordination polymers, especially those with $d ^ { 1 0 }$ metal centers,have been investigated for their fluorescent properties and potential applications as fluorescent-emiting materials,such as light-emitting diodes (LEDs)[21l. Therefore, the photoluminescence properties of $\mathbf { 1 } { \sim } \mathbf { 4 }$ and free ligand (Hbppc) were studied in the solid state (Fig. 5). Intense emission bands are observed at 420 and $4 6 6 ~ \mathrm { n m }$ 0 $\lambda _ { \mathrm { e x } } = 2 9 0 \ \mathrm { n m }$ ）for Hbppc, which can be assigned to the ligand-centered charge transitions,that is, the $\pi ^ { * }  n$ and $\pi ^ { * }  \pi$ transitions[22]. The bimodal emissions are also observed at $5 0 0 \ \mathrm { n m }$ for 1, $4 7 7 ~ \mathrm { { n m } }$ for 2, $4 7 1 \ \mathrm { n m }$ for 3 and $5 2 6 ~ \mathrm { n m }$ for 4 respectively upon excitation at $2 8 0 ~ \mathrm { n m }$ . As previously reported[23,24],the solid aromatic carboxylate ligands 1,3-H2bdc and H3btc are nearly nonfluorescent in the range of $4 0 0 { \sim } 6 0 0 \ \mathrm { n m }$ at ambient temperature. Compared with the free Hbppc, the main peaks in $\mathbf { 1 } { \sim } \mathbf { 4 }$ are red-shifted, which may result from ligand-to-metal charge transfer (LMCT)[25l. Notably, a lower energy emisson was also detected at $3 8 8 ~ \mathrm { n m }$ for 1, $3 9 5 \mathrm { n m }$ for 2 and $3 9 0 \mathrm { n m }$ for 3 which can be atributed to the intraligand $\pi ^ { * }  \pi$ transition.

# 3.6Thermogravimetric analysis

In order to characterize the thermal stability of compounds $\mathbf { 1 } { \sim } 4$ ,their thermal decomposition behavior was investigated at $3 0 { \sim } 1 0 0 0 \ \mathrm { ~ \textdegree ~ }$ under nitrogen atmospheres using thermal gravimetric analysis (TGA) technique. Compound 1 first loses its lattice water molecule below $1 0 4 ^ { \circ } \mathrm { C }$ ，and the weight loss found of $7 . 1 \%$ is consistent with that calculated $( 7 . 7 \% )$ . The second stage weight loss can be detected from 108 to $1 5 0 \mathrm { ~ } ^ { \circ } \mathrm { C }$ attributed to the departure of coordinated water molecules (Calcd. $5 . 1 \%$ ; Found: $4 . 9 \%$ ). Above $3 4 8 \mathrm { ~ \textdegree C }$ ，the compound begins to lose their ligands and then starts to decompose (Fig. 6a).For 2,the weight loss of $1 8 . 2 8 \%$ under $1 1 6 ^ { \circ } \mathrm { ~ C ~ }$ corresponds to the removal of latice and coordinated water molecules (calcd. $1 8 . 6 2 \%$ ). Then, the weight loss of $1 4 . 8 4 \%$ between $1 6 0 ~ \mathrm { ~ \textdegree ~ }$ to $3 2 5 \ \mathrm { ~ \textdegree C }$ indicates the loss of $\mathrm { H } _ { 2 }$ (m-bdc) (calcd. $1 5 . 0 4 \%$ ，and the framework begins to collapse (Fig. 6b). Below $3 4 5 \ \mathrm { ~ \textdegree ~ }$ , compound 3 begins to lose water molecules and free $\mathrm { H } _ { 2 } ( \mathrm { m } { - } \mathrm { b } \mathrm { d } \mathrm { c } )$ molecules. Then the decomposition of residual components starts above $3 6 0 ~ \mathrm { ~ \textdegree ~ }$ (Fig. 5c). And the framework of compound 4 remains intact until heated to $3 3 8 ^ { \circ } \mathrm { C }$ and then begins to collapse (Fig. 6d). TGA results of these four coordination frameworks indicate they possess great thermal stability.

# 4 CONCLUSION

In summary,four new compounds have been synthesized by the assembly of Hbppc,different auxiliary ligands and zinc salts under hydrothermal conditions. Compounds $\mathbf { 1 } { \sim } \mathbf { 4 }$ show intriguing $0 D$ ， $1 D$ and $2 D$ structures. Their structural diversities show that the $\mathsf { p H }$ value of the reaction system and auxiliary ligand play a significant role in the structural self-assembly process.Additionally，the hydrogen bonding interactions contributed to the formation of supramolecular architectures.Moreover, compounds $\mathbf { 1 } { \sim } 4$ exhibit intense emissions, indicating they may be good candidates as luminescent materials.
