# ABSTRACT

A well-dispersed noble metal on perovskite support is expected to show good catalytic ability for NO oxidation under oxygen-rich conditions,which is an important step in reducing NO in lean burn exhaust gas. Inthis study,we investigated the catalytic oxidation of NO to $\mathsf { N O } _ { 2 }$ ona $\mathrm { P t } _ { 4 } / \mathrm { C a T i O } _ { 3 }$ (001) model to evaluate interfacial effect. Results showed that $0 _ { 2 }$ preferred Pt-Ti interface sites,and that the decomposition of $0 _ { 2 }$ required a dissociation activation barrier of $0 . 9 7 \mathrm { e V } .$ .The decomposed O-O on the interface required nearly no activation barrier in the subsequent NO oxidation. However,the most rate-limiting step involved the desorption of the second formed $\mathsf { N O } _ { 2 }$ from interfacial $\mathrm { P t } _ { 4 } / \mathrm { C a T i O } _ { 3 } \left( 0 0 1 \right)$ ,which yielded a significant desorption barrier of $2 . 6 3 \mathrm { e V } .$ These findings can help understand the oxidation process of NO to $\mathsf { N O } _ { 2 }$ on noble-metal-perovskite interface.

# 1. Introduction

$\mathsf { N O } _ { x }$ emission in lean burn and diesel exhaustis one of the main sources of air pollution. Although efficient in converting toxic gases fromgasoline engines,three-way catalysts are generally ineffective in reducing $\mathsf { N O } _ { x }$ in diesel engine under oxygen-rich conditions [1,2].Therefore,developing $\mathsf { N O } _ { x }$ storage reduction(NSR) catalysts consisting of a precious metal supported on metal oxides [3] has become an alternative technology for reducing $\mathsf { N O } _ { x }$ emission [4]. The presence of a precious metal is critical in NO oxidation under oxygen-rich condition[5],which isan important step in $\mathsf { N O } _ { x }$ storage.NO oxidation by excessive $0 _ { 2 }$ can occur in the presence of noble metals [6].However,an important drawback of traditional precious metal support involves the aggregation of supported noble metals,which directly leads to catalyst invalidation.

To maintain catalytic activity,various studies were conducted to find a more appropriate support for good dispersion of the noble metal [6,7].Several perovskites such as $\mathsf { C a T i O } _ { 3 }$ and $\mathrm { L a F e O } _ { 3 }$ are recently found to be capable of preventing the aggregation of supported Pt or Pd particles by refreshing them in and out of the perovskite lattice.Pt-doped $\mathsf { C a T i O } _ { 3 }$ or Pd-doped $\mathrm { L a F e O } _ { 3 }$ materialsare known as “intelligent catalysts”[2,8,9],and are expected to efficiently save the limited precious metal resources.However,the inherent electronic nature of the interface between the noble metal and perovskite has not been well studied.For instance,how the metal support interface affects the catalytic process of $\mathsf { N O } _ { x }$ remains unclear.

Given that the oxidation of NO to $\mathsf { N O } _ { 2 }$ is the most important step for reducing NO in lean burn exhaust gas [1O,11],we studied the catalytic oxidation of NO on a metal support interface based on a $\mathrm { P t } _ { 4 } / \mathrm { C a T i O } _ { 3 }$ model.First,the effects of cluster size of $ { \mathsf { P } }  { \mathsf { t } } _ { n }$ 1 $[ n = 4 , 5$ and 6) on the support was evaluated by analyzing interfacial electronic character and binding energies.Subsequently,the adsorption and dissociation of $0 _ { 2 }$ on $\mathrm { P t } _ { 4 } / \mathrm { C a T i O } _ { 3 }$ were investigated to determine the activation barrier and the most favorable sites for active dissociated Oatoms.After producingactive O atoms,the catalyticreaction processof $2 \mathsf { N O } + \mathsf { O } _ { 2 } \to 2 \mathsf { N O } _ { 2 }$ on $\mathsf { P t } _ { 4 }$ and the interface were compared to evaluate the $\mathrm { P t } _ { 4 } / \mathrm { C a T i O } _ { 3 }$ interfacialeffect.Finally, we mapped out the energy landscape of ${ \mathsf { N O } } _ { 2 }$ desorption and found that refreshing the $\mathrm { P t } _ { 4 } / \mathrm { C a T i O } _ { 3 }$ catalyst is an important and rate-limiting step for the entire NO-oxidation process.

# 2. Computational methods

All calculations were performed based on the density functional theory(DFT) using the Viennaab initio simulation package(VASP) [12,13]. The electron-ion interactions were treated by all-electron projectoraugmentedwave(PAW） approximation[14,15]. Electronic exchange-correlation energies were determined by Perdew， Burke, andEnzer（PBE）[16]functionalwithin spin-polarized generalized gradient approximation (GGA）[17]. Climbing image nudged elastic band(cNEB）method [18,19] was used to calculate the activation energies of $0 _ { 2 }$ dissociation and NO oxidation.

# 3.1. $P t _ { n } / C a T i O _ { 3 }$

To select a suitable and computationally affordable Pt cluster to deposit on the $\mathsf { C a T i O } _ { 3 }$ surface,we first compared the different isomersand cohesive energies $\left( E _ { c } \right)$ of three $\mathrm { P t } _ { n } \left( n = 4 , 5 , 6 \right)$ clusters. $E _ { c }$ was calculated as $E _ { c } = E _ { \mathrm { P t } n } / n$ to identify the most favorable $ { \mathrm { P t } } _ { n }$ cluster isomers.For $n = 4$ ,among the optimized linear 1D,square 2D, and tetrahedral 3D clusters,the tetrahedral $\mathsf { P t } _ { 4 }$ cluster with $E _ { c } = 3 . 2 8 \mathrm { e V }$ was found to be the most stable isomer.For the $\mathsf { P t } _ { 5 }$ cluster,the trigonal bipyramid shows the largest $E _ { \mathrm { c } }$ of $3 . 5 1 \mathrm { e V }$ in comparison with the square pyramid and 2D pentagon isomers. # 3.4.First NO oxidation and desorption

Combined with the NO oxidation and desorption process,the whole reaction energy in the interfacial $0 _ { \mathrm { I } }$ oxidation is favorable because of its exothermic energy of 1.82 eV.

# 3.5.Second NO oxidation and desorption

As plotted in Fig.7,although the initial oxidation step occurs virtually without barrier,the second NO oxidation and desorption on both bridge- ${ \bf \nabla } \cdot \mathsf { O } _ { \mathrm { I I } }$ and interfacial- ${ \bf \nabla } \cdot \mathsf { O } _ { \mathrm { I I } }$ lead to more difficult processes.The subsequent second $\mathsf { N O } _ { 2 }$ desorptionisthe rate-limiting step,with the activation barrier of 1.66 and $2 . 6 3 \mathrm { e V }$ respectively.In Fig.7(a), the second NO oxidation elongates the $\mathsf { O } _ { \mathrm { I I } ^ { - } } \mathsf { P t }$ bond from $1 . 8 1 \mathring { \mathsf { A } }$ to $2 . 0 5 \mathring { \mathrm { A } }$ and results in an $E _ { \mathrm { a d s } }$ of 0.91eV.The formed $\mathsf { N O } _ { 2 }$ is bent with $\angle 0 _ { \mathrm { I I } } - \mathrm { N } - 0 = 1 0 9 . 5 ^ { \circ }$ Torefresh the catalyst $\mathrm { P t } _ { 4 } / \mathrm { C a T i O } _ { 3 }$ ，the activation barrier of $1 . 6 6 \mathrm { e V }$ should leavethe $\mathsf { N O } _ { 2 }$ molecule free.In comparison, the second $\mathsf { N O } _ { 2 }$ desorption on interfacial Pt-Ti leads to a larger activation barrier of $2 . 6 3 \mathrm { e V }$ buta slightly lower $E _ { \mathrm { a d s } }$ (1.14 eV) for $\mathsf { N O } _ { 2 }$ formation,as shown in Fig.7(b). Overall, the second NO reactions on bridge ${ \bf \nabla } \cdot \mathsf { O } _ { \mathrm { I I } }$ andinterfacial- ${ \bf \nabla } \cdot \mathsf { O } _ { \mathrm { I I } }$ are both endothermic,with1.3 and $0 . 8 \ : \mathrm { e V }$ ,indicating that the second $\mathsf { N O } _ { 2 }$ ismore unfavorablydesorbed,hence rendering this process as the most energy-consuming step.

# 4. Conclusion

NO oxidation on a perovskite-supported $\mathsf { P t } _ { 4 }$ cluster was investigated based on first-principles DFT calculations.Bader charge analysis and EDD show that,when interacting with the $\mathsf { P t } _ { 4 }$ cluster, CaO- and $\mathrm { T i O } _ { 2 }$ -terminated $\mathsf { C a T i O } _ { 3 }$ （001）surfaces act as electron donor and acceptor,respectively.The calculated dissociation barriers of $_ { 0 - 0 }$ decomposition on bridge $\mathsf { P t \mathrm { - } P t }$ and interface Pt-Ti correspond to 0.53 and $0 . 9 7 \mathrm { e V }$ .The subsequent NO oxidation on the active O readily occurs because of their small reaction barriers of less than $0 . 4 7 \mathrm { e V }$ 、For the entire reaction,the second $\mathsf { N O } _ { 2 }$ desorption from $\mathrm { P t } _ { 4 } / \mathrm { C a T i O } _ { 3 }$ is the most energy-consuming step,because the desorption barrier on bridge and interface sites are 1.66 and $2 . 6 3 \mathrm { e V }$ ，respectively.We expect that our study is helpful for understanding the NO oxidation mechanism on well-dispersed perovskite-supported noble metal clusters.
