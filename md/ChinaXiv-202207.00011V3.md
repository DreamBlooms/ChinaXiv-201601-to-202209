# 1信息素养课程专业化是学科服务深入的结果

# 1.1 未跟上信息素养课程专业化节奏的图书馆管理

2018年有图书馆同行，哀叹信息素养教育“与专业教育脱节”，为“无法进入高校的人才培养方案及专业培养目标”自卑[1。笔者认为事实并非如此。有研究者对教育部高校图书馆事实数据库(下文按惯例简称“高基库”)中，截至2020年9月30日提交的“信息素养必修课听课人数”有效数据的1069所高校图书馆，进行了梳理分析，发现“双一流”建设高校图书馆、“双一流”建设外的“211工程”高校图书馆、普通本科院校图书馆、高职高专院校图书馆，信息素养开设了必修课的比例，依次为： $3 3 . 3 \% ( 1 0 0 \% - 6 6 . 7 \% ) .$ （20433.9%(100%-66.1%)、 $4 3 . 3 \% ( 1 0 0 \% 5 6 . 5 \% )$ 、 $3 3 . 9 \% ( 1 0 0 \% { - } 6 6 . 1 \% ) ^ { [ 2 ] }$ 。必修课的开设说明信息素养教育已深入到高校人才培养方案中。

以“法律检索”为例，2017年国务院学位委员会办公室的“学位办（2017）19号”文件，以2学分的“实践教学与训练”（实训课）将该课纳入到法律专业硕士培养方案中[3]。中南财经政法大学(以下正文简称中南大)的法律硕士教育中心(下文简称法硕中心)，从2017级的专业硕士开始，《法律检索》被作为2个学分的必修课，由图书馆研究馆员的授课。

信息素养教育的检索课程被要求进入专业教育、被要求进入专业培养方案时，除从事学科服务的具体的信息素养教师外，图书馆人可能一下子踏不上节奏。比如某校《法律检索》作为必修课，已经连续5年(5个年级)，一直由校级馆和学院分馆馆员授课，年均必修人数为302人。但在“高基库”系统中该校的“H1.1信息素质教育课必修人数”至今连年为0。出现这种情况，也许恰恰是因为，信息素养教育专业化速度快，专业检索课的出现，图书馆管理者没跟上节奏，视而未见。

# 1.2 未适应专业检索课多头管理现状的图书馆

一般图书馆信息素养教育公选检索课程，由教务部门（包括本科教务和研究生教务）与图书馆联合管理。这类公选课，以及由学院向图书馆预约的嵌入式课程在图书馆挂了号，进了日程，建了档，于是，图书馆对此类课程有统辖权。

与之相对应的，由各学院与教务部门联合管理的专业类检索课，各学院直接聘请、安排学院图书馆或校图书馆馆员授课的，教学管理与教学检查、包括考核管理，都由各学院教学办公室管理，图书馆未承担其管理责任。也就出现5年必修法律检索课共1595人，在“高基库”“H1.1”指标中的数据为0的原因。

随着信息素养专业化进程，会出现由各学院直辖，教务部门统一管理的专业检索课，图书馆只是作为提供教师的单位，由教师在图书馆备案。这些专业性检索课程，与学校其他非教师系列的授课教师同等对待，经过教务部门和图书馆确认，最终由人事部门进行统一的工作量管理。

图书馆为什么会忽略对图书馆学科服务有如此加持作用的，专业检索课的数据？一方面，可能图书馆人对信息素养教育理解过于泛化，只能接受高校图书馆进行的通用信息素养教育，还不能深入理解，信息素养教育是使学生具检索、管理、评价、合理使用文献信息的技能,促进终身学习,实现学术生涯的学术目标和职业生涯的社会目标[4,5]。另外，从事填报和审核数据的图书馆管理者，一方面也许对本馆的学科服务具体情况不够了解，另一方面可能源于本位主义思想，仅负责自己所辖工作范围。

# 1.3专业检索课是学科服务的亮点

高校图书馆学科服务的高亮区是信息素养教育。从教育部1984年发布的《关于在高等学校开设“文献检索与利用课”的意见》（简称“开课意见"）开始，各种形式的图书馆用户教育，成为高校图书馆教育职能与服务职能体现形式之一。随着信息化潮流，取代文献检索课表述为信息素养教育。信息素养教育与学科服务紧密结合。随之，信息素养课程的专业化，即专业或专题检索课程成为学科服务的亮点。

学科服务做得较好的是上海交通大学、北京大学以及武汉大学①。从这些大学图书馆的学科服务来看，其学科服务除了利用二次文献类、学术评价类数据库，如WOS(Web ofSciences)类数据库，做学科评价、类比分析外，再就是基于图书馆本身的资源优势的资源建设与揭示。另外一个重要服务就是信息素养教育的专业化。

以上海交通大学图书馆为例，无论是嵌入到化学化工[]，还是机械动力[7.8]，其学科服务的最有影响的产品仍然是信息素养教育，包括双语信息素养教育。即使其面对科研团队的嵌入式学科服务，也是通过“拜访教授”了解需要后，除了利用图书馆WOS的RSS 学术成果分析与推送外，“信息专员”仍然做的是通用信息素养和专题培训[9]。不过，北京大学医学图书馆在需要多学科交叉融合、相互协作的“转化医学”领域的学科服务，有所切入[10]。其他学科服务，也是除了利用WOS 做学术分析外[11]，无论是面向艺术学院的学科服务提供多媒体资源[12]，还是面向海洋研究院的建设、整理、清洗学科数据，建立整理海洋学专利信息等学科服务[13]，仍是以资源服务和信息素养教育为主要内容[14]。

# 2专业检索课需要高校图书馆进行知识管理

# 2.1专业检索课是高校图书馆信息素养教育专业化产物

以法律领域为例，法律检索一词最早源于，英美法系遵循先例的判例法国家。我国虽属于大陆法系的成文法国家。2010年起，司法制度改革出台了一系列法律文件。从2010年“应当参照”指导性案例，到建立法律大数据检索平台，再从检索的“指导意见”，到“全面检索”，甚至“强制检索”，这十年间的每一个文件的遣词，一个比一个坚定，一个比一个强烈，服务于法学教育的高校图书馆人强烈感受到法学教育对专业检索课的需求。

2010 年最高人民法院(下文简称最高法)印发《关于案例指导工作的规定》中明确“各级人民法院审判类似案例时应当参照”指导性案例。随后，最高法的“法（2014）23号”文件、“法发（2014）18号”文件、“法发（2014）23号”文件，分别对裁判文书公开、建设司法公开三大平台(审判流程公开平台、裁判文书公开平台、执行信息公开平台)，并实现与诉讼服务网的相互链接与资源共享，进行了规定。也就是说从2010年起，司法领域为统一法律适用标准，持续在法律检索对象与检索平台的建设方面发力。2013年7月1日开通了中国裁判文书网[15]。此后裁判文书网功能不断完善，2015年12月15日，新版中国裁判文书网开通[16]。平台的开通为“同案同判、类案同判”提供了物质基础。“法发（2015）13号”文件明确将类案参考作为统一裁判尺度的重要工具之一。2017年，最高法下发“加强类案检索”的指导意见[17]，并在“法发（2017）20号”文件中重申“类案与关联案件检索”，强调“全面检索”。“法发（2020）35号”文件则要求“完善类案和新类型案件强制检索报告工作机制”。这一系列文件，也使法律检索课在法学院在法律实务部门，纷纷开设。

我国法律检索课，有来自图书馆人的，也有来自法律人（包括律师和法学院的专任教师）的。由高校图书馆人开设的法学文献信息检索课、法律检索课，是通用信息素养与法律信息素养、通用检索技能与法律检索技能的结合[18]。它源自1984年教育部的“开课意见”，得到国务院学位委员会和教育部的“学位办（2017）19号”文的加持，是信息素养课程专业化的产物[19]。出自法律人开设的检索课程，来到了英美判例法的法律检索起点，以类案为主要检索对象，辅以法律条文检索，归属各学院管理。图书馆人开设的专业检索课，是图书馆学科服务和信息素养专业化的产物，应该由图书馆和学院共同管理。

# 2.2 《法律检索》是图书馆一门成熟的专业检索课

“信息素养教育是中国法学教育改革的内在需要”[20]。《法律检索》作为信息素养教育在法学领域专业化课程，在中南大是一个瓜熟蒂落的过程。

教育部“开课意见”后的第三年，即1987年《文献检索与利用课》就被中南政法学院，纳入到本科必修课体系中，后成为必选课。上世纪90年代末，因辅助开课的技术条件滞后，该课暂停。2000年中南政法学院与中南财经大学合并后，校图书馆一直以讲座形式进行用户教育。2008年《文献检索与利用》课作为全校本科生通识选修课，复课至今。复课的第二年，即自 2009年起每年选课人数千人以上，选课人数受制于师资力量和课堂人数上限。2014年和2015年，年选修人数两千人以上，为保证教学效果，目前课堂人数以120人为上限。

2015年，法学院馆员率先给该院学术硕士开设《法学文献检索与利用》选修课。2016年，《信息检索与论文写作》在全校研究生中作为公选课，由校图书馆安排馆员授课，并分法学类、人文经管类、理工类等三类课堂[2I]。2019年为提高对研究生的覆盖面，研究生部与校馆合作，制作MOOC课程，线上线下课堂结合。

2018 年法硕中心将《法律检索》作为必修实训课，覆盖面自2017年入校后所有法律专业硕士(见图1)。该课由研究馆员授课，深受好评。2021年《涉外法律检索》由于涉外法治人才培养需要，被教育部纳入到法硕涉外律师方向指导性培养方案中[22]。2022 年 5月，法硕中心启动法律硕士专业学位课程和教材建设[23]。《法律检索》被作为“培养方向特色课程"，成为2022年课程立项项目之一[24]。

从1987年《文献检索与利用》由图书馆，在中南大政法学院开课，到2022年《法律检索》在中南大法硕中心立项，历经三十五年。《法律检索》的主讲人即项目负责人，经历最初的文献课的开课、讲座、嵌入课程，到通识课的复课，再到 2017年的检索课专业化的教学。三十多年的不间断的教学，以及其他馆员的加入，使《法律检索》成为校图书馆一门成熟的专业检索课程。

# 2.3专业检索课亟需图书馆的知识管理

知识管理起源于行政管理、企业管理，是对组织的管理。1975 年美国亚利桑那州立大学尼古拉斯.亨利认为，公共行政管理者应该清楚本机构的独特性。公共行政管理中的知识概念与传统知识概念本质区别，它的获取应该是一种综合实践。必须将公共行政管理中的知识获取视为一种综合练习，而不是专门知识[25]。日本管理学家野中郁次郎等认为，知识管理起源于对组织内部无形资源和能力的关注，以及对企业的知识资产和知识经济结合的关注；不能把知识简单地理解为信息数据库，也不能将对知识管理软件的运用等同于知识管理[26]。美国生产力和质量中心（American Productivity & Quality Center，APQC）将知识管理（Knowledge Management,KM）定义为有系统地去促使信息和知识增长、流动和创造价值，是一门为了在正确的时间向正确的人提供正确的知识，并帮助人们共享信息并根据信息采取行动，以提高组织绩效的，创建和管理流程的学科②[27]。

也就是说，知识管理是对组织的管理，使组织外部知识内部化、隐性知识显性化（员工个人的经验技能分享出来)、显性知识系统化（员工个人知识变成组织的知识）、系统知识资产化（经过评估的有价值的知识建立长期的知识库）、资产知识情境化（与具体应用场景结合使用在具体情境员工随取随用）、情境知识创新化（员工在应用知识的过程中内化并再生出新知识）,寓知识于工作过程。成功的知识管理可以做到，使员工彼此连接，并与足够、及时及适合的知识资产连接，以及有针对性地与有经验或有技能的专业人员对接，这样员工可以适时、适地学习，减少重复性错误，并更好地胜任工作，同时使人们在职位变动、离职、退休的时候保留知识资产[28]。

前文述及，专业检索课是学科服务深入的结果。学科服务的深入，致使专业检索课被纳入各专业培养方案中。专业培养方案的课程又分属不同教学单位管理。没有专属的管理部门的非教学单位的课程，容易变成三不管地带。法硕中心教学管理者考虑到，对法律检索课知识资产的保留问题，于是，《法律检索》被作为 2022年“培养方向特色课程”项目，由图书馆教师团队来建设。

虽然，法硕中心将法律检索课纳入2022年的课程项目建设，但毕竟法律检索课是法学学科馆员，深入学科服务的产物，持续的课程的知识管理，亟需图书馆的参与并承担主要责任。

# 3专业检索课课程项目建设暨知识管理内容

# 3.1加强教学团队及教学条件，以保留教学知识资产

课程建设负责人由长期授课且率先开设《法律检索》的研究馆员担任。师资队伍以图书馆检索课教师队伍为基石，择优组建法律检索师资队伍。组建师资队伍时，课程负责人除考虑队员工作是否负责，还考察科研能力。尤其是通过科研论文中的文献综述，对其是否有脚踏实地，扎扎实实做文献研究的基本功进行考量。

教学团队组建后，除了要求教师了解法学基础知识外，还通过以老带新的方式，组织新教师听本课程项目的法律检索课，并向新教师推荐法律人(含专任教师)讲授的法律检索课，以及法律专业数据库商的培训课。

团队建设，从知识管理的角度来看，可以让教学团队彼此连接，隐性知识显性化，进行知识传承，同时保证人员变动时，可以保留项目建设课程的知识资产。

教学条件上，除了准备教学用品外，还购买各种类型的参考文献，如法学文献检索、法律文献检索、类案检索、论文写作与学术规范、法学引注手册，以及律师法律检索相关参考资料。同时，梳理汇集编制法律检索相关必读法律文件，让隐性知识显性化。

# 3.2加强教材、大纲及教案建设，使显性知识系统化

制定科学实用的体现高等教育和法律检索特色的教学大纲（含实训大纲)。对教材、教案、大纲进行立体化建设。

首先，对本校使用的参考教材《法学研究与文献检索》（高利红主编）补充新的内容，形成适合法律专业硕士的补充教学材料。第二步，就是编制法律检索教材，帮助法律专业硕士完成研习阶段的学术任务，和未来职业生涯的专业检索技能。

从教学大纲上，结合法律检索的特点进行增补。比如，公共网络资源检索中，增加尽职调查内容。学术论文的检索中，突出对法官的学术观点、庭审风格的检索。图书牛书的检索中，突出立法机关、司法机关的司法解释类图书的检索。再如，结合法律术语检索与法律条文检索，比较演绎研究法与从条文到案例的办案方式，梳理检索结果；结合统计数据检索与类案检索，比较归纳研究法与从案例到案例的办案方式，梳理检索结果。

从教案上，照顾不同的学生群体。比如在文献管理软件教学上，原教案只对本校购买的 Notefirst文件管理软件，进行教学演示，对其他的软件只给予答疑安排。通过这次项目建设，增加了为使用苹果电脑的同学，补充建设 Zotero 文献管理软件的图文版参考教案，以及Zotero 使用演示版教案。

在知识的再生（或曰检索产品的形成）上，结合学术搜索的文献使用策略、以及结合学术评价与文献筛选，对照法律检索的中法律效力顺位、类案参照顺位，让通用信息素养与专业素养结合。在文献综述写作中结合文献梳理，对照法律检索报告要求对法律条文与类案的梳理，完成学术搜索的论文综述与法律检索报告。整个从检索到知识再生的过程，体现了学术写作与完成法律检索报告过程的相似性。

# 3.3教学模式及考核指标建设，使资产知识情境化

中南大法律检索课，在长期的教学实践中，形成了一套标准的翻转式、问题式、案例式、目标式以及合作式混合教学模式。

该课每讲每节环环相扣，最终形成个人课程选题的文献综述与检索报告，以及检索小组报告展示。整个教学，使学生从个人独立操作到合作复盘，再到个人独立再生出实践作品（选题综述与检索报告)；每讲认知点与技能点，从课前教师呈现问题列表，到课后学生解决问题列表；每个专题检索从案例呈现到学生实操；从第一次课堂公布到最后阅卷依据的考核指标，将大纲内容囊括其中。

从知识管理角度看，这种将教师的显性知识系统化，通过教学模式及考核指标建设，经由教学团队使个人知识变成组织的知识。通过课程项目建设又将系统知识资产化，再与具体教学场景结合起来，使教师团队在具体教学情境中随取随用，并有系统地去促使信息和知识流动、增长并创新。

以考核指标为例，为保证教学效果，从第一课堂指标公布与阶段任务布置结合。教师和学生都须以考核指标为参照点，以个人选题为主线，以个人任务为每一阶段的抓手，使每一堂课的教与学，依据指标围绕主线。这种根据教学大纲统一考核指标，既提醒学生学习的基本要求，也让教师在考核打分时，有统一的参照物，还简化了阅卷流程。

每学期阅卷结束后，教师根据该学期每个指标中学生出现的不同问题，反馈给学生，并完善下学期的考核指标。指标的完善也是教学内容的更新，也是资产知识的情境化中的创新。

# 3.4让专业检索课的知识管理取于学生服务于学生

使学生全程参与课程项目建设，让知识管理取于学生，用于学生。根据学校课程项目建设经费中有关劳务费使用的规定，其指导思想之一是，让学生参与到科研或教学项目的实施中，也即通过科研或项目建设让教师和学生走到一起。通过科研育人、项目育人。本项目学生参与方式有两种，一种是长期式，一种是阶段式。

长期式是指每学期课前课堂调查和课后学生反馈意见。每学期课前问卷了解课堂学生的信息素养基础，以调整每学期课堂内容。课后反馈意见，设计到考查考核指标中，主要是通过学生完成课程选题综述与检索报告的结语（指标中指定含收获与遗憾两部分)，了解课程改进意见。

阶段式是指，课程项目建设申报成功后，组织学生团队帮助完成课程建设的具体项目。2022年学生阶段参与任务有两项。第一，形成“学长检索案例库”[29]。这也是根据学生检索报告“结语”中的“遗憾”的改进。有一些学生，即使教师在第一堂课就布置任务，但要拖到第二三次课后才开始着手，等到着手开始，前面的内容没有很好掌握，悔不当初。“学长检索案例库”就是在第一次课给学生的阅读材料。第二，利用非法学背景的法律硕士的计算机技术优势，对开源式免费的文件管理安装与使用，做成图文版和演示版，挂在图书馆网页上或课堂演示。这也恰恰适应了本馆可能取消集体方式购买文献管理软件的现实。

# 结语

法硕中心的《法律检索》课程项目建设，是图书馆对专业检索课进行知识管理的一个

契机。图书馆应借助这一机会，拓展图书馆对各学科的学科服务，加强专业检索课的知识管理。中南财经政法大学图书馆给研究生开设的三大类型（法学、人文经管、理工）的公选课，应该更加深入专业方向，朝着成为各学科专业培养方案实训课程的目标奋斗。
